# 口語（話し言葉）

## 日文解釈

日常会話で使われる、自然で柔軟な言語形態。書き言葉とは異なり、即座性、文脈依存性、非形式性が特徴。

### 定義
口頭でのコミュニケーションで使用される言語形態。会話の流れに応じて変化し、省略や言い換えが多く見られる。

### 主な特徴
1. **省略**
   - 助詞の省略：「私は学生だ」→「私、学生」
   - 主語の省略：「今日は忙しい」→「忙しい」
   - 語尾の省略：「分かりました」→「分かった」

2. **縮約形**
   - ～ている → ～てる
   - ～ていない → ～てない
   - ～ておく → ～とく
   - ～てしまう → ～ちゃう

3. **フィラー（間投詞）**
   - えっと、あのう、まあ、そうですね

4. **終助詞の多用**
   - ね、よ、ぞ、な、さ

## 英文解釈

Spoken language (話し言葉, hanashi kotoba) is the natural, flexible form of language used in everyday conversation.

### Characteristics
1. **Ellipsis**: Frequent omission of particles, subjects, and sentence endings
2. **Contractions**: Shortened forms like ～てる instead of ～ている
3. **Fillers**: Words like えっと (etto), あのう (anou) that help manage conversation flow
4. **Sentence-final particles**: Extensive use of particles like ね, よ to convey nuance
5. **Context-dependent**: Relies heavily on shared understanding between speakers

### Register Differences
- **Casual conversation**: Most informal, with maximum ellipsis
- **Polite conversation**: Uses です・ます forms but still allows contractions
- **Formal speech**: Closer to written language, less ellipsis

## 中文解釋

口語（話し言葉）是日常對話中使用的自然、靈活的語言形態。

### 核心特徵
1. **省略**
   - 助詞省略：「私は学生だ」→「私、学生」
   - 主語省略：基於情境的共同理解
   - 語尾省略：非正式場合常見

2. **縮約形**
   - ～ている → ～てる（進行式）
   - ～ておく → ～とく（預先準備）
   - ～てしまう → ～ちゃう（完成或遺憾）

3. **填充詞**
   - えっと（思考時）、あのう（猶豫時）
   - まあ（緩和語氣）、そうですね（同意或思考）

4. **終助詞豐富**
   - ね（確認、共鳴）
   - よ（強調、提醒）
   - な（自言自語）

### 與書面語的區別
- 口語更依賴情境和非語言線索
- 書面語需要更完整的文法結構
- 口語允許即時修正和重述

## 例文

### 基本用法

1. **助詞省略**
   - 書き言葉：「私は明日、東京に行きます」
   - 話し言葉：「私、明日東京行くよ」
   - 我明天去東京。

2. **縮約形**
   - 書き言葉：「今、何をしているのですか」
   - 話し言葉：「今、何してるの？」
   - 你現在在做什麼？

3. **フィラーの使用**
   - 「えっと、明日は…あのう…ちょっと用事があって」
   - 嗯，明天…那個…有點事。

### 場面別用法

4. **日常会話**
   - 「昨日の映画、どうだった？」
   - 「うん、面白かったよ。また見たいな」
   - 昨天的電影怎麼樣？嗯，很有趣。還想再看一次。

5. **依頼表現**
   - 書き言葉：「お手数ですが、ご確認いただけますでしょうか」
   - 話し言葉：「すみません、ちょっと見てもらえますか」
   - 不好意思，可以請你看一下嗎？

## 対比・関連概念

### vs 書き言葉（written_language.md）
| 観点 | 話し言葉 | 書き言葉 |
|------|---------|---------|
| 構造 | 不完全な文が多い | 完全な文法構造 |
| 省略 | 頻繁 | 稀少 |
| 修正 | その場で可能 | 推敲が必要 |
| 依存性 | 文脈依存度高い | 自己完結的 |
| 形式性 | 柔軟 | 固定的 |

### 関連概念
- **カジュアル会話**（casual-conversation.md）：最も非形式的な口語
- **丁寧語**（formal.md）：口語でも使用される丁寧な表現
- **文語**：書き言葉の中でも特に形式的な文体

## 文法ノート

### 縮約形のパターン
```
～ている → ～てる / ～てんの
～ておく → ～とく / ～どく
～てしまう → ～ちゃう / ～じゃう
～てはいけない → ～ちゃだめ / ～じゃだめ
～なければならない → ～なきゃ / ～なくちゃ
```

### 助詞の省略傾向
```
を → 省略可能（特に目的語が明確な場合）
は → 省略可能（主題が明確な場合）
が → 比較的省略されにくい
に・で → 場所や時間が明確な場合に省略可能
```

## 学習のヒント

1. **段階的習得**
   - まずは丁寧体（です・ます）の口語から
   - 次に普通体の基本形
   - 最後に縮約形と省略

2. **実践練習**
   - 日本語の会話番組やドラマを見る
   - ネイティブとの会話で実際に使ってみる
   - 音声入力で練習する

3. **注意点**
   - 場面に応じた使い分けが重要
   - ビジネス場面では過度な省略を避ける
   - 初対面では丁寧な口語を使用
