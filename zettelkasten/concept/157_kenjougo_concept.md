## 日文解釋

謙讓語（けんじょうご）是一種日文敬語系統，用於表達說話者對聽眾或文本主人的尊敬。通過降低或謙卑地表述自己或相關方的行動、屬性、物品等，來顯示禮貌和尊敬。

謙讓語主要用於：
- 將自己或親近者的行動表達得更謙虛
- 提及對方或第三方時，表達適當的敬意
- 商務溝通和正式場合

## 中文解釋

謙讓語是日語敬語體系的重要組成部分，通過使用特定的動詞形式、助詞和表達方式，將自己或內群體的行動表現得更加謙卑。這是一種社會禮儀表達方式，反映了日本文化中對他人的尊重和禮貌。

## English Explanation

Kenjougo (謙讓語) is a formal politeness system in Japanese used to express humility and respect. It involves using humble verbs, expressions, and forms to describe one's own or one's in-group's actions, possessions, or attributes in a more deferential manner. It is commonly used in business communication, formal settings, and when showing respect to superiors or clients.

## 常見的謙讓語動詞

謙讓語通常涉及特定的謙虛動詞形式，用以取代日常動詞：

| 通常形式 | 謙讓語 | 含義 |
|---------|-------|------|
| する | いたします | 做 |
| いる | おります | 在、存在 |
| もらう | いただく | 收到、得到 |
| くれる | くださる | 給予（對方） |
| 言う | 申し上げます | 說（謙虛表達） |

## 例句

### 例1：申し上げます（說）
```
私は山田と申します。
Watashi wa Yamada to mōshiagemasu.
我叫山田。
```
在自我介紹時使用謙讓語形式「申します」而不是「言います」。

### 例2：いたします（做）
```
来週の会議で説明いたします。
Raishuu no kaigi de setsumei itashimasu.
我將在下週的會議上做說明。
```
在業務場景中使用「いたします」來表達謙虛的承諾。

### 例3：いただく（接收）
```
ご指導いただき、ありがとうございます。
Go-shidō itadaki, arigatou gozaimasu.
感謝您的指導。
```
使用「いただく」表達對他人幫助或指導的感謝。

## 學習要點

1. **識別謙讓語動詞**：學習特定的謙虛動詞形式，如「いたす」、「申す」、「いただく」
2. **理解使用場景**：謙讓語主要用於商務溝通、正式場合和對上級的尊敬表達
3. **實踐應用**：在角色扮演和實際溝通中使用謙讓語，特別是在商務日本語中
4. **對比敬語系統**：比較謙讓語與敬語和丁寧語的異同

## 注釋

謙讓語是掌握日本商務禮儀和正式溝通的關鍵。理解何時使用謙讓語，以及如何正確地應用謙虛動詞，對於在日本職場中有效溝通至關重要。
