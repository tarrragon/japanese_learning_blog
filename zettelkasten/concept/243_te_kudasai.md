「〜てください」是日語中最基本的請求表達，用於禮貌地請求他人做某事。

<!--more-->

## 日文

〜てください

### 文法情報

| 項目 | 內容 |
|------|------|
| 文法類型 | 請求表達 |
| JLPT | N5 |
| 使用場面 | 日常會話、禮貌請求 |
| 形成 | 動詞て形 + ください |

### 形成規則

**基本形式**
```
動詞て形 + ください
```

**範例**
```
食べる → 食べてください
見る → 見てください
書く → 書いてください
読む → 読んでください
する → してください
来る → 来てください
```

## 日文解釋

「〜てください」は、相手に何かを依頼したり、お願いしたりするときに使う丁寧な表現である。動詞のて形に「ください」を付けて作る。「ください」は「くださる」（尊敬語）の命令形から派生した補助動詞で、直接的な命令よりも柔らかく、相手への敬意を含んだ依頼表現となる。

日常会話において最も基本的かつ頻繁に使われる依頼表現であり、店員への注文、道を尋ねるとき、友人への軽いお願いなど、幅広い場面で使用できる。ただし、目上の人や初対面の人に対しては「〜ていただけますか」「〜てくださいませんか」などのより丁寧な表現を使うことが望ましい場合もある。

否定の依頼（〜しないでほしい）を表す場合は「〜ないでください」を使う。

## 英文解釋

"~te kudasai" is the standard polite way to make requests in Japanese. It is formed by attaching "kudasai" to the te-form of a verb. This expression is softer and more respectful than a direct command, making it suitable for everyday polite requests.

The word "kudasai" derives from the honorific verb "kudasaru" (to give, from a superior to an inferior) in its imperative form. When used as an auxiliary after the te-form, it transforms a verb into a polite request. While appropriate for most daily situations, more formal expressions like "~te itadakemasu ka" may be preferred when speaking to superiors or in very formal contexts.

For negative requests (asking someone not to do something), use "~naide kudasai."

## 中文解釋

「〜てください」是日語中表達禮貌請求的基本句型。構成方式是將動詞變為て形後加上「ください」。這個表達比直接命令更加委婉，適合在日常生活中向他人提出請求。

「ください」源自敬語動詞「くださる」（給予，從上位者到下位者）的命令形。作為補助動詞接在て形後面時，它將動詞轉化為禮貌的請求表達。雖然在大多數日常場合都適用，但對長輩或非常正式的場合，可能需要使用更禮貌的表達，如「〜ていただけますか」。

表達否定請求（請對方不要做某事）時，使用「〜ないでください」。

## 核心用法

### 用法 1：基本請求

用於禮貌地請求他人做某事，是最常見的日常請求表達。

**例句 1**
```
ここに名前を書いてください。
Please write your name here.
請在這裡寫上您的名字。
```

**例句 2**
```
窓を開けてください。
Please open the window.
請打開窗戶。
```

### 用法 2：指示或說明

在給予指示或說明步驟時使用，常見於說明書、公告等。

**例句 3**
```
まず、このボタンを押してください。次に、パスワードを入力してください。
First, press this button. Next, enter your password.
首先，請按這個按鈕。接下來，請輸入密碼。
```

### 用法 3：詢問或請教

請求對方提供資訊或解釋時使用。

**例句 4**
```
もう一度説明してください。
Please explain it once more.
請再解釋一次。
```

**例句 5**
```
この言葉の意味を教えてください。
Please tell me the meaning of this word.
請告訴我這個詞的意思。
```

## 使用規則

### 語氣調整

「〜てください」的語氣可以通過添加其他元素來調整：
- **更禮貌**：「〜てくださいませんか」「〜ていただけますか」
- **更隨意**：「〜て」（省略ください，用於親近的人）
- **更正式**：「〜てくださいますようお願いいたします」

### 否定形

表達「請不要做某事」時：
- **動詞ない形 + でください**：食べないでください（請不要吃）
- **常見錯誤**：❌ 食べてくださいない（不存在此形式）

### 注意事項

1. 對上司或客戶，「〜てください」可能顯得不夠禮貌
2. 書面語中常用「〜してください」而非「〜して下さい」（漢字）
3. 連續多個「〜てください」會顯得命令感過強

## 常見錯誤

### 錯誤 1：對長輩使用時語氣不足

❌ 誤：部長、この書類を確認してください。
✅ 正：部長、この書類をご確認いただけますでしょうか。
說明：對上司應使用更禮貌的表達方式。

### 錯誤 2：否定形構成錯誤

❌ 誤：ここでタバコを吸ってくださいない。
✅ 正：ここでタバコを吸わないでください。
說明：否定請求應使用「〜ないでください」形式。

### 錯誤 3：與命令形混淆

❌ 誤：早く食べれ！（對不熟的人）
✅ 正：早く食べてください。
說明：命令形過於直接，對一般人應使用「〜てください」。

## 學習要點

1. **基礎必學**：「〜てください」是N5級別的必備句型，是學習其他敬語請求表達的基礎
2. **て形掌握**：正確使用本句型的前提是熟練掌握各類動詞的て形變化
3. **場合判斷**：要學會判斷何時需要更禮貌的表達（對上司、客戶等）
4. **否定形記憶**：「〜ないでください」是獨立的固定搭配，需要特別記憶
5. **語氣微調**：通過添加「ね」「よ」等語氣詞可以調整請求的語氣
