與格（よかく）

## 日文解釋

与格（よかく）とは、動作の対象や受け手、方向や到達点を示す文法格である。

日本語において、与格は主に助詞「に」によって標識される。例えば「彼に本を渡す」という文では、「彼に」が与格であり、動作の受け手を表す。与格は以下のような意味役割を担う：

1. **受益者**（beneficiary）：「友達にプレゼントをあげる」
2. **経験者**（experiencer）：「私には分かる」
3. **到達点**（goal）：「学校に行く」
4. **存在場所**（location）：「机の上に本がある」

与格は目的語を二つ取る動詞（二重目的語動詞）において特に重要で、直接目的語（を格）と間接目的語（に格）を区別する。例：「先生が学生に問題を出す」では、「学生に」が間接目的語（与格）、「問題を」が直接目的語（対格）である。

印欧語の多くでは与格は形態論的に標識されるが、日本語では助詞「に」が与格標識として機能する点が特徴的である。

## 英文解釋

The dative case is a grammatical case that indicates the recipient or beneficiary of an action, as well as direction or destination.

In Japanese, the dative case is primarily marked by the particle に (ni). For example, in the sentence 彼に本を渡す (I hand the book to him), 彼に represents the dative case, indicating the recipient of the action. The dative case carries the following semantic roles:

1. **Beneficiary**: 友達にプレゼントをあげる (give a present to a friend)
2. **Experiencer**: 私には分かる (I understand)
3. **Goal**: 学校に行く (go to school)
4. **Location**: 机の上に本がある (There is a book on the desk)

The dative case is particularly important with ditransitive verbs (verbs that take two objects), distinguishing between the direct object (を-case) and the indirect object (に-case). For example: 先生が学生に問題を出す (The teacher gives the students a question) - where 学生に is the indirect object (dative) and 問題を is the direct object (accusative).

While many Indo-European languages mark the dative morphologically, Japanese characteristically uses the particle に as a dative marker.

## 中文解釋

與格是表示動作接受者、受益者、方向或目的地的語法格。

在日語中，與格主要透過助詞「に」標記。例如在「彼に本を渡す」（把書遞給他）這個句子中，「彼に」是與格，表示動作的接受者。與格承擔以下語意角色：

1. **受益者**：「友達にプレゼントをあげる」（送禮物給朋友）
2. **經驗者**：「私には分かる」（我理解）
3. **到達點**：「学校に行く」（去學校）
4. **存在位置**：「机の上に本がある」（書在桌上）

與格在雙及物動詞（帶兩個賓語的動詞）中特別重要，用於區分直接賓語（を格）和間接賓語（に格）。例如：「先生が学生に問題を出す」（老師給學生出題目）中，「学生に」是間接賓語（與格），「問題を」是直接賓語（對格）。

印歐語系多數語言透過形態變化標記與格，而日語的特點是使用助詞「に」作為與格標記。

## 核心用法

### 用法 1：表示動作接受者

「に」格標記動作的接受者或受益者。

**例句 1**
```
母に手紙を書く
I write a letter to my mother
寫信給母親
```

**例句 2**
```
子供にお菓子をあげる
I give sweets to the child
給小孩糖果
```

### 用法 2：表示方向或目的地

「に」格標記動作的到達點或方向。

**例句 3**
```
東京に向かう
Head toward Tokyo
前往東京
```

### 用法 3：表示存在位置

與存在動詞（ある、いる）搭配使用。

**例句 4**
```
公園に子供がいる
There are children in the park
公園裡有小孩
```

### 用法 4：表示經驗者

標記感受、理解等心理狀態的主體。

**例句 5**
```
私には難しすぎる
It's too difficult for me
對我來說太難了
```

## 理論背景

### 格理論中的位置

在格理論體系中，與格是核心格之一，與主格（が格）、對格（を格）共同構成日語的基本格系統。與格的特點在於其多功能性，能表達多種語意關係。

### 與其他語言的對比

不同於德語、俄語等語言透過名詞詞尾變化標記與格，日語使用後置的格助詞「に」。這種分析型標記方式是日語格系統的顯著特徵。

### 在格框架中的角色

在[格框架](concept/069_case_frame.md)理論中，與格通常對應間接賓語位置或附加語位置，是動詞論元結構的重要組成部分。

## 學習要點

1. **核心功能**：與格標記接受者、方向、位置等語意角色
2. **主要標記**：日語中由助詞「に」標記
3. **與對格區別**：與格標記間接賓語，對格（を）標記直接賓語
4. **多功能性**：同一助詞「に」可表達多種語意關係
5. **句型應用**：雙及物動詞句型中必需的格成分
