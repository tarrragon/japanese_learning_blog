「お/ご～ください」是日語敬語系統中用於禮貌請求的重要句型。

<!--more-->

## 日文

お/ご～ください（お/ご〜ください）

### 文法情報

| 項目 | 內容 |
|------|------|
| 文法類型 | 敬語（丁寧語/尊敬語） |
| JLPT | N3 |
| 使用場面 | 正式場合、商務、服務業 |
| 形成 | お + 動詞ます形語幹 + ください / ご + 漢語名詞 + ください |

### 形成規則

**和語動詞（訓読み）：お + 動詞ます形語幹 + ください**
```
待つ → お待ちください
書く → お書きください
座る → お座りください
```

**漢語動詞/名詞（音読み）：ご + 名詞 + ください**
```
連絡 → ご連絡ください
確認 → ご確認ください
注意 → ご注意ください
```

## 日文解釋

「お/ご～ください」は、相手に何かをお願いする際に用いる丁寧な表現である。「～てください」よりも改まった印象を与え、ビジネスシーンやサービス業、公的な場面で頻繁に使用される。

この表現は、動詞の語幹に「お」または「ご」を付け、その後に「ください」を添えることで形成される。和語（訓読み）の動詞には「お」を、漢語（音読み）の語彙には「ご」を用いるのが基本的なルールである。

単なる命令ではなく、相手への敬意を込めた依頼であることから、聞き手に対する配慮が感じられる表現となっている。特に初対面の相手や目上の人、お客様に対して使用することで、適切な距離感と礼儀正しさを示すことができる。

## 英文解釋

"O/go ~ kudasai" is a polite request form in Japanese honorific language (keigo). It is more formal than the standard "~te kudasai" and is commonly used in business settings, customer service, and formal situations.

The construction follows a pattern where native Japanese verbs (wago) take the prefix "o-" before the verb stem, while Sino-Japanese words (kango) take "go-" before the noun form. This distinction reflects the historical development of Japanese vocabulary and honorific conventions.

This expression conveys respect toward the listener while making a request, striking a balance between politeness and directness. It is essential for professional communication in Japanese workplaces and service industries.

## 中文解釋

「お/ご～ください」是日語敬語系統中用於禮貌請求的重要句型，比「～てください」更為正式，廣泛使用於商務場合、服務業及正式場合。

構成規則上，和語動詞（訓讀）使用「お」加上動詞ます形語幹，漢語詞彙（音讀）則使用「ご」加上名詞形式。這一區分反映了日語詞彙的歷史發展及敬語體系的規律。

此表達方式既能傳達請求，又能展現對聽話者的尊重，是日語商務溝通和服務業不可或缺的基礎句型。對於學習者而言，掌握「お」與「ご」的使用區分是正確使用此句型的關鍵。

## 核心用法

### 用法 1：和語動詞的敬語請求

對於和語（訓讀）動詞，使用「お + 動詞ます形語幹 + ください」的形式。

**例句 1**
```
少々お待ちください。
Please wait a moment.
請稍等。
```

**例句 2**
```
こちらにお座りください。
Please have a seat here.
請在這裡坐下。
```

### 用法 2：漢語名詞的敬語請求

對於漢語（音讀）詞彙，使用「ご + 名詞 + ください」的形式。

**例句 3**
```
ご連絡ください。
Please contact us.
請聯繫我們。
```

**例句 4**
```
詳細についてはご確認ください。
Please confirm the details.
請確認詳細內容。
```

### 用法 3：服務業的標準用語

在服務業中，此句型是接待客人的標準表達方式。

**例句 5**
```
どうぞごゆっくりお過ごしください。
Please take your time and relax.
請慢慢享用。
```

## 常見錯誤

### 錯誤 1：「お」與「ご」的混用

❌ 誤：ご待ちください
✅ 正：お待ちください
說明：「待つ」是和語動詞，應使用「お」而非「ご」。

### 錯誤 2：對漢語詞彙使用「お」

❌ 誤：お連絡ください
✅ 正：ご連絡ください
說明：「連絡」是漢語詞彙，應使用「ご」而非「お」。

### 錯誤 3：對自己的動作使用此句型

❌ 誤：私がお説明ください
✅ 正：ご説明いたします
說明：「お/ご～ください」是對他人的請求，對自己的動作應使用謙讓語。

## 學習要點

1. **「お」與「ご」的區分**：和語（訓讀）用「お」，漢語（音讀）用「ご」，這是基本規則。

2. **語境適切性**：此句型適用於正式場合，日常朋友間對話中使用可能顯得過於生硬。

3. **與「～てください」的差異**：「お/ご～ください」更正式、更尊敬，「～てください」則較為直接。

4. **例外詞彙的記憶**：部分詞彙有固定搭配，如「お電話ください」（雖然「電話」是漢語）需要特別記憶。

5. **商務日語的基礎**：掌握此句型是進入日語商務溝通的重要一步。
