「質問する」是日文中表達「提問」的基本詞彙，在日常會話和商務場合中都非常常用。

<!--more-->

## 日文

質問する（しつもんする）

### 詞彙情報

| 項目 | 內容 |
|------|------|
| 漢字 | 質問 |
| 讀音 | しつもん |
| 詞性 | サ変動詞（名詞＋する） |
| JLPT | N3 |
| 敬語形 | ご質問なさる・ご質問される |

## 日文解釋

「質問する」は、分からないことや確認したいことについて、相手に尋ねる行為を表す動詞です。

日常会話からビジネスシーンまで幅広く使われ、特に授業や会議などの場面で頻繁に使用されます。「聞く」よりもやや改まった印象があり、明確な疑問点を尋ねる際に適しています。

類似表現の「尋ねる」「問う」と比べると、「質問する」は具体的な疑問点について答えを求めるニュアンスが強いです。また、「お聞きする」「伺う」などの敬語表現と使い分けることで、場面に応じた丁寧さを表現できます。

## 英文解釋

"質問する" (shitsumon suru) means "to ask a question" or "to inquire." It is a サ変動詞 (suru-verb) formed by combining the noun 質問 (question) with する (to do).

This expression is commonly used in both casual and formal contexts, particularly in educational settings, meetings, and customer service situations. It carries a slightly more formal nuance than the simple verb 聞く (kiku, to ask/hear).

## 中文解釋

「質問する」是日文中表示「提問」、「發問」的動詞，由名詞「質問」加上「する」構成的サ變動詞。

這個詞彙在日常對話和正式場合都很常用，特別是在課堂、會議等需要提出問題的場景中頻繁出現。相較於「聞く」，「質問する」給人較為正式的印象，適合用於詢問具體的疑問點。在商務場合中，也可以使用「ご質問」等敬語形式來表達禮貌。

## 核心用法

### 用法 1：一般提問

在日常對話或課堂中，針對不明白的事情向對方詢問。

**例句 1**
```
先生、質問してもいいですか。
Teacher, may I ask a question?
老師，我可以提問嗎？
```

**例句 2**
```
何か質問がありましたら、遠慮なくおっしゃってください。
If you have any questions, please don't hesitate to ask.
如果有任何問題，請不要客氣，儘管說。
```

### 用法 2：商務場合的正式提問

在會議或商務場合中，使用較正式的表達方式。

**例句 3**
```
この件について、いくつか質問させていただきたいのですが。
I would like to ask a few questions regarding this matter.
關於這件事，我想請教幾個問題。
```

**例句 4**
```
お客様からの質問にお答えします。
I will answer the customer's question.
我來回答客戶的問題。
```

### 用法 3：間接表達

用於描述他人的提問行為。

**例句 5**
```
学生たちが積極的に質問するようになった。
The students have started actively asking questions.
學生們變得積極提問了。
```

## 使用規則

### 助詞搭配

| 助詞 | 用法 | 例句 |
|------|------|------|
| を | 質問的內容 | 意味を質問する |
| に | 提問的對象 | 先生に質問する |
| について | 關於某事 | 計画について質問する |

### 敬語表現

| 場面 | 表現 | 例句 |
|------|------|------|
| 丁寧語 | 質問します | 一つ質問します |
| 謙讓語 | ご質問させていただく | ご質問させていただいてもよろしいでしょうか |
| 尊敬語 | ご質問なさる | 何かご質問なさいますか |

### 常見搭配

- 質問に答える（回答問題）
- 質問を受ける（接受提問）
- 質問を投げかける（提出問題）
- 質問攻め（連續發問）

## 常見錯誤

### 錯誤 1：混淆「質問する」和「聞く」

❌ 誤：道を質問しました。
✅ 正：道を聞きました。
說明：詢問道路這種簡單的日常詢問，使用「聞く」更自然。「質問する」適合用於需要說明或解釋的具體疑問。

### 錯誤 2：敬語使用錯誤

❌ 誤：お客様が質問しました。
✅ 正：お客様がご質問されました。/お客様がご質問なさいました。
說明：對客人使用尊敬語時，應使用「ご質問される」或「ご質問なさる」。

### 錯誤 3：助詞使用錯誤

❌ 誤：先生を質問する。
✅ 正：先生に質問する。
說明：表示提問對象時應使用助詞「に」，而非「を」。

## 學習要點

1. **詞性理解**：「質問する」是サ變動詞，由名詞「質問」加「する」構成，可以分開使用（質問をする）。
2. **語感區別**：相較於「聞く」，「質問する」更正式，適合用於有具體疑問點的場合。
3. **敬語轉換**：商務場合常用「ご質問させていただく」（謙讓語）或「ご質問なさる」（尊敬語）。
4. **助詞搭配**：對象用「に」，內容用「を」或「について」，務必區分清楚。
5. **場合使用**：課堂、會議、客服等場合最為常用，日常簡單詢問則多用「聞く」。
