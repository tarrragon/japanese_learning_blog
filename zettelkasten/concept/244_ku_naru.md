イ形容詞去掉「い」加上「く」再接「なる」，表示狀態的變化，意為「變得...」。

<!--more-->

## 日文

〜くなる

### 文法情報

| 項目 | 內容 |
|------|------|
| 文法類型 | 形容詞活用 + 狀態變化 |
| JLPT | N4 |
| 使用場面 | 日常會話、書面語 |
| 形成 | イ形容詞語幹 + く + なる |

### 形成規則

**基本形式**
```
イ形容詞（〜い）→ 語幹 + く + なる
```

**範例變化**
```
寒い → 寒くなる（變冷）
暑い → 暑くなる（變熱）
高い → 高くなる（變高）
安い → 安くなる（變便宜）
```

## 日文解釋

「〜くなる」は、イ形容詞の語幹に「く」を付け、さらに「なる」を接続させた形で、状態の変化を表す文法パターンである。この表現は、ある性質や状態が以前とは異なる方向へ移行することを示す。

例えば、「寒い」という形容詞の場合、「寒い」→「寒く」→「寒くなる」という変化を経て、「寒い状態に変わる」という意味になる。この変化は自然発生的なものであり、誰かの意志によるものではない点が特徴的である。

「なる」は自動詞であるため、変化の主体は自ら変化するのであって、外部からの働きかけによって変えられるのではない。これは「〜くする」（他動詞的変化）との重要な違いである。

## 英文解釋

The pattern "〜くなる" (ku naru) is used to express a change of state with i-adjectives. It is formed by taking the stem of an i-adjective (removing the final い), adding く, and then attaching the verb なる (to become).

This construction indicates that something naturally transitions into a new state described by the adjective. The change is typically spontaneous or gradual, without an external agent causing it. For example, "寒くなる" means "to become cold" — the weather or environment changes on its own.

Key distinction: While "〜くなる" describes natural change, "〜くする" (ku suru) indicates intentional change caused by someone.

## 中文解釋

「〜くなる」是日語中表達狀態變化的重要文法模式，專門用於イ形容詞（い形容詞）。其構成方式為：將イ形容詞詞尾的「い」去掉，加上「く」，再接續動詞「なる」。

這個句型表達的是自然發生的變化過程，強調某事物從一種狀態轉變為另一種狀態。例如：「暑い」（熱）→「暑くなる」（變熱），描述天氣或環境自然地變得炎熱。

重要區別：
- 「〜くなる」：自然變化（自動詞性質）
- 「〜くする」：人為改變（他動詞性質）

此文法模式是 JLPT N4 級別的基礎文法，在日常會話中使用頻率極高。

## 核心用法

### 用法 1：自然狀態變化

描述天氣、環境、時間等自然發生的變化。

**例句 1**
```
秋になると、日が短くなる。
When autumn comes, the days become shorter.
到了秋天，白天會變短。
```

**例句 2**
```
夜になって、だんだん涼しくなってきた。
As night fell, it gradually became cooler.
到了晚上，天氣漸漸變涼了。
```

### 用法 2：人的狀態或感受變化

描述人的身體狀態、心理感受、能力等的變化。

**例句 3**
```
日本語を勉強して、漢字が読めるようになった。上手くなった。
After studying Japanese, I became able to read kanji. I got better.
學了日語之後，變得會讀漢字了。變厲害了。
```

**例句 4**
```
彼女に会えなくて、寂しくなった。
I couldn't see her, so I became lonely.
見不到她，變得寂寞了。
```

### 用法 3：事物性質的變化

描述物品、價格、狀況等的變化。

**例句 5**
```
最近、野菜の値段が高くなった。
Recently, vegetable prices have become expensive.
最近蔬菜的價格變高了。
```

## 常見錯誤

### 錯誤 1：與ナ形容詞混淆

❌ 誤：静かくなる
✅ 正：静かになる
說明：ナ形容詞使用「〜になる」而非「〜くなる」。

### 錯誤 2：保留「い」

❌ 誤：寒いくなる
✅ 正：寒くなる
說明：必須去掉「い」後才能加「く」。

### 錯誤 3：與「〜くする」混淆

❌ 誤：部屋が暖かくなった。（想表達「我把房間弄暖」）
✅ 正：部屋を暖かくした。
說明：「〜くなる」表示自然變化，「〜くする」表示人為使之改變。

## 學習要點

1. **形成規則**：イ形容詞去「い」→ 加「く」→ 加「なる」
2. **自動詞性質**：表示自然發生的變化，主語自己改變
3. **與「〜くする」區別**：「なる」是自動詞（自然變化），「する」是他動詞（人為改變）
4. **ナ形容詞不適用**：ナ形容詞使用「〜になる」模式
5. **時態變化**：可接續各種時態（なる→なった→なっている→なります等）
