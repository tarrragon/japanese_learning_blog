「輸入」は、コンピューターやシステムにデータや情報を入力することを指す概念です。

## 日文

輸入（にゅうりょく）

## 日文解釋

「輸入」とは、コンピューターやその他の情報処理システムに対して、外部からデータや情報を取り込むことを意味します。キーボードでの文字入力、マウスでの操作、音声認識によるデータ取得など、様々な方法があります。

日本語の「入力」とほぼ同義ですが、「輸入」は中国語からの借用語として、特に台湾や中国のIT文脈で使用されることがあります。日本語のIT用語としては「入力（にゅうりょく）」が一般的です。

また、「輸入」には「外国から商品を持ち込む」という意味もあり、文脈によって区別が必要です。コンピューター用語としての「輸入」は、主に中華圏のIT環境で使用される表現です。

## 英文解釋

"Nyuuryoku" (輸入/input) refers to the process of entering data or information into a computer or other information processing system. This concept encompasses various methods including keyboard typing, mouse operations, voice recognition, and other data acquisition methods.

In Japanese IT terminology, "nyuuryoku" (入力) is the standard term for "input," while "yunyu" (輸入) typically refers to importing goods from foreign countries. However, in Chinese-influenced IT contexts, 輸入 is commonly used to mean data input.

## 中文解釋

「輸入」在IT領域中，指的是將資料或資訊送入電腦或其他資訊處理系統的過程。這個概念涵蓋了多種方法，包括鍵盤打字、滑鼠操作、語音辨識等資料獲取方式。

在日語中，「入力（にゅうりょく）」是標準的IT術語，而「輸入（ゆにゅう）」通常指的是從外國進口商品。但在中文影響的IT語境中，「輸入」被廣泛用於表示資料輸入。值得注意的是，這張卡片記錄的是中文「輸入」在日文學習語境中的對應概念。

## 核心用法

### 用法 1：電腦資料輸入

在IT情境下，描述將資料輸入電腦系統的動作。

**例句 1**
```
パスワードを入力してください。
Please enter your password.
請輸入您的密碼。
```

**例句 2**
```
キーボードで文字を入力する方法を学びます。
We will learn how to input characters using a keyboard.
我們將學習使用鍵盤輸入文字的方法。
```

### 用法 2：資訊系統的資料登錄

描述在系統中登錄或記錄資訊的過程。

**例句 3**
```
顧客情報をデータベースに入力する必要があります。
You need to input customer information into the database.
您需要將客戶資訊輸入資料庫。
```

### 用法 3：學習語境中的輸入假說

在語言學習理論中，「輸入」指的是學習者接收的語言材料。

**例句 4**
```
言語習得において、理解可能な入力が重要です。
In language acquisition, comprehensible input is important.
在語言習得中，可理解的輸入非常重要。
```

**例句 5**
```
リスニングとリーディングは言語入力の主な形式です。
Listening and reading are the main forms of language input.
聽力和閱讀是語言輸入的主要形式。
```

## 使用規則

### 日語中的「入力」vs「輸入」

在日語中：
- **入力（にゅうりょく）**：IT術語，表示資料輸入
- **輸入（ゆにゅう）**：表示從外國進口商品

注意：本卡片的「輸入」是中文IT用語，對應日語的「入力」。

### 常見搭配

| 日語 | 中文 | 說明 |
|------|------|------|
| データを入力する | 輸入資料 | 一般資料輸入 |
| パスワードを入力する | 輸入密碼 | 安全相關輸入 |
| 情報を入力する | 輸入資訊 | 資訊登錄 |
| 文字を入力する | 輸入文字 | 文字輸入 |

### 注意事項

1. 在日語IT語境中，使用「入力」而非「輸入」
2. 「輸入」在日語中主要表示「進口」
3. 學習日語時，注意區分這兩個詞的用法

## 常見錯誤

### 錯誤 1：混淆「入力」和「輸入」

❌ 誤：パスワードを輸入してください。
✅ 正：パスワードを入力してください。
說明：在日語中，電腦資料輸入應使用「入力」，「輸入」表示進口商品。

### 錯誤 2：語言學習輸入假說的誤用

❌ 誤：言語習得において、輸入が重要です。
✅ 正：言語習得において、入力（インプット）が重要です。
說明：語言學習理論中的「輸入」，日語通常說「入力」或使用外來語「インプット」。

## 學習要點

1. **日中詞義差異**：中文「輸入」= 日語「入力」（IT用語）
2. **日語「輸入」**：表示進口，讀作「ゆにゅう」
3. **IT語境**：日語使用「入力（にゅうりょく）」或「インプット」
4. **語言學習理論**：可理解輸入（comprehensible input）= 理解可能な入力/インプット
5. **場景區分**：根據語境選擇正確的日語表達
