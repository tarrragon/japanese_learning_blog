可能形是日文動詞的重要活用形式之一，用於表達「能夠做某事」或「有能力做某事」的意思。

可能形（かのうけい）

## 日文解釋

可能形は日本語動詞の活用形の一つで、「～することができる」「～する能力がある」という意味を表す。五段動詞は語幹のう段をえ段に変えて「る」を付け、一段動詞は「る」を「られる」に変える。可能形自体は一段動詞として活用する。

可能形は単に能力を示すだけでなく、状況や条件による可能性も表現できる。たとえば「今日は早く帰れる」は能力ではなく、状況的に帰ることが可能であることを意味する。

また、可能形を使う際、目的語の助詞「を」が「が」に変わることが一般的である。これは可能形の動詞が一種の状態を表すようになるためと考えられている。ただし、現代日本語では「を」のまま使われることも多い。

## 英文解釋

The potential form (可能形, kanoukei) is a Japanese verb conjugation that expresses ability or possibility, equivalent to "can do" or "be able to do" in English.

For godan (五段) verbs, the final u-vowel changes to the e-vowel and る is added. For ichidan (一段) verbs, る is replaced with られる. The potential form itself conjugates as an ichidan verb.

A notable grammatical feature is that the object particle を often changes to が when using the potential form, though を is increasingly accepted in modern Japanese, especially in casual speech.

## 中文解釋

可能形是日文動詞的一種變化形式，用於表達「能夠做某事」或「有做某事的能力」。這是 N4 程度的重要文法概念。

變化規則如下：
- **五段動詞**：將詞尾的う段假名改為え段，再加上「る」
- **一段動詞**：將「る」改為「られる」
- **不規則動詞**：「する」變為「できる」，「来る」變為「来られる」

使用可能形時，原本表示目的語的助詞「を」通常會變成「が」。這是因為可能形的動詞帶有狀態性質，主語與能力的關係類似於「擁有」某種能力。

口語中，一段動詞的可能形常省略「ら」，如「食べられる」說成「食べれる」，這被稱為「ら抜き言葉」。

## 核心用法

### 用法 1：表示能力

說明某人具備某種能力或技能。

**例句 1**
```
彼女は５か国語が話せます。
She can speak five languages.
她會說五種語言。
```

**例句 2**
```
私は漢字が読めるようになりました。
I have become able to read kanji.
我變得會讀漢字了。
```

### 用法 2：表示狀況性的可能

說明由於情況或條件而能夠做某事。

**例句 3**
```
今日は早く帰れます。
I can go home early today.
今天可以早點回家。
```

### 用法 3：表示許可或允許的可能

說明某事被允許或可以進行。

**例句 4**
```
この部屋では飲食ができます。
Eating and drinking is allowed in this room.
這個房間可以飲食。
```

**例句 5**
```
ここで写真が撮れますか。
Can I take photos here?
這裡可以拍照嗎？
```

## 變化規則

### 五段動詞變化

將詞尾う段改為え段，加上「る」：

| 辞書形 | 可能形 | 意味 |
|--------|--------|------|
| 書く（かく）| 書ける（かける）| 能寫 |
| 読む（よむ）| 読める（よめる）| 能讀 |
| 話す（はなす）| 話せる（はなせる）| 能說 |
| 行く（いく）| 行ける（いける）| 能去 |
| 買う（かう）| 買える（かえる）| 能買 |

### 一段動詞變化

將「る」改為「られる」：

| 辞書形 | 可能形 | ら抜き形 |
|--------|--------|---------|
| 食べる | 食べられる | 食べれる |
| 見る | 見られる | 見れる |
| 起きる | 起きられる | 起きれる |

### 不規則動詞變化

| 辞書形 | 可能形 |
|--------|--------|
| する | できる |
| 来る（くる）| 来られる（こられる）|

## 常見錯誤

### 錯誤 1：助詞使用混淆

❌ 誤：日本語を話せます。
✅ 正：日本語が話せます。
說明：雖然現代口語中「を」也被接受，但傳統上可能形搭配「が」更為正確。

### 錯誤 2：與受身形混淆

❌ 誤：先生に褒められる（想表達可能）
✅ 正：先生に褒められる（實際意思是被動：被老師稱讚）
說明：一段動詞的可能形和受身形相同（〜られる），需從上下文判斷意思。

### 錯誤 3：不規則動詞變化錯誤

❌ 誤：勉強しれる
✅ 正：勉強できる
說明：「する」的可能形是「できる」，不能類推為「しれる」。

## 學習要點

1. **五段動詞變化**：う段→え段＋る，這是最基本的變化規則
2. **助詞變化**：可能形常將「を」改為「が」，但現代口語較為寬鬆
3. **ら抜き言葉**：口語中一段動詞的「られる」常省略為「れる」
4. **一段活用**：可能形本身作為一段動詞活用（書ける→書けます）
5. **多義性注意**：一段動詞的可能形與受身形相同，需從語境判斷
