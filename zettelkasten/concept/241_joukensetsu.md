条件節是日文複合句中表達條件關係的從屬子句，與主節形成「如果...那麼...」的邏輯結構。

## 日文解釋

「条件節」とは、条件を表す従属節のことです。日本語の複文において、主節（帰結節）に対して条件を提示する役割を担います。[^dict]

条件節は「と」「ば」「たら」「なら」などの条件形式を用いて形成されます。これらの形式はそれぞれ異なる意味的・語用的特徴を持ち、場面や文脈に応じて使い分けられます。

条件節の特徴として、主節に先行して現れることが多く、「前件→後件」という論理的順序を形成します。ただし「なら」を用いる場合は、主節が先行することもあります。

条件節の述語形式には制約があり、命令形・意向形・依頼形などを直接用いることができません。これは条件節が未確定の事態を表すという性質に由来します。

[^dict]: 定義參考 スーパー大辞林（macOS 內建字典）

## 英文解釋

A "conditional clause" (条件節 joukensetsu) is a subordinate clause that expresses a condition. In Japanese complex sentences, the conditional clause functions to present a condition for the main clause (consequent clause).

Conditional clauses are formed using conditional forms such as "to", "ba", "tara", and "nara". Each form possesses distinct semantic and pragmatic features, requiring appropriate selection based on the situation and context.

A key characteristic of conditional clauses is that they typically precede the main clause, establishing the logical sequence of "antecedent → consequent". However, with "nara", the main clause may sometimes come first.

The predicate forms within conditional clauses have restrictions: imperative, volitional, and request forms cannot be used directly. This stems from the nature of conditional clauses expressing uncertain or hypothetical situations.

## 中文解釋

「条件節」是表達條件的從屬子句。在日文複合句中，條件節扮演著為主節（結果節）提供條件的角色。

條件節使用「と」「ば」「たら」「なら」等條件形式來形成。這些形式各自具有不同的語義和語用特徵，需要根據場合和語境適當選擇。

條件節的特徵之一是通常出現在主節之前，形成「前件→後件」的邏輯順序。但使用「なら」時，主節也可能先出現。

條件節的謂語形式有限制：不能直接使用命令形、意向形、請求形等。這源於條件節表達未確定事態的本質特性。

## 核心用法

### 用法 1：表達假設條件

條件節最基本的功能是表達假設情境，即「如果A成立，那麼B」的邏輯關係。

**例句 1**
```
雨が降れば、試合は中止になる。
If it rains, the match will be cancelled.
如果下雨，比賽就會取消。
```

**例句 2**
```
時間があったら、映画を見に行こう。
If we have time, let's go see a movie.
如果有時間，就去看電影吧。
```

### 用法 2：表達一般真理或習慣

使用「と」形成的條件節，常用於表達自然法則或習慣性動作。

**例句 3**
```
氷は温めると溶ける。
Ice melts when you heat it.
冰加熱就會融化。
```

### 用法 3：表達前提或話題

使用「なら」形成的條件節，常用於回應對方的話語或設定討論前提。

**例句 4**
```
東京に行くなら、新幹線が便利だよ。
If you're going to Tokyo, the Shinkansen is convenient.
如果你要去東京，新幹線很方便。
```

## 常見錯誤

### 錯誤 1：條件節內使用命令形

❌ 誤：早く起きろなら、バスに間に合う。
✅ 正：早く起きれば、バスに間に合う。
說明：條件節內不能使用命令形。應使用條件形（ば形）。

### 錯誤 2：混淆條件節與時間節

❌ 誤：明日来れば、プレゼントをあげる。（表達確定事項）
✅ 正：明日来たら、プレゼントをあげる。
說明：表達確定會發生的事情時，「たら」比「ば」更自然。

## 學習要點

1. **條件節的位置**：通常位於主節之前，但「なら」較為靈活
2. **四種條件形式**：「と」「ば」「たら」「なら」各有不同的使用場景
3. **謂語限制**：條件節內不能使用命令形、意向形等
4. **與主節的呼應**：條件節的形式會影響主節的表達方式
5. **時制與確定性**：條件節表達的確定程度影響形式選擇
