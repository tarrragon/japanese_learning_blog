## 日文定義

**會話含意理論**（会話含意理論、かいわ がんいりろん）は、言語学者グライス（H. Paul Grice）が提唱した、発話者が明確に表現していないが、聞き手が推論によって理解できる意味の理論である。言葉の字義的な意味を超えた、文脈に基づく暗黙的な意味を研究する。

## 中文解釋

**會話含意理論**是由語言學家格賴斯（H. Paul Grice）提出的理論，用於解釋發話者未明確表達但聽者能透過推理理解的意思。該理論研究超越字面意思、基於語境的隱含意義，是語用論的核心內容。

## 英文定義

**Conversational Implicature Theory** is a theory proposed by linguist H. Paul Grice that explains how speakers convey meanings that are not explicitly stated but are understood by listeners through inference. It examines implicit meanings based on context that go beyond the literal meaning of words, and is a core component of pragmatics.

## 核心概念

### 字義含意 vs. 會話含意

- **字義含意**（literal meaning）：表達的字面含義，即詞典中的意思
- **會話含意**（conversational implicature）：通過語境推論得出的隱含含義

### 格賴斯的四項準則（The Cooperative Principle）

發話者遵循以下準則來進行有效溝通：

1. **量的準則**（Maxim of Quantity）
   - 提供必要的訊息量
   - 不提供過多或過少的訊息

2. **品質準則**（Maxim of Quality）
   - 不說虛假的話
   - 沒有充分根據的話不說

3. **關係準則**（Maxim of Relation）
   - 發言要與話題相關
   - 保持與對話的連貫性

4. **方式準則**（Maxim of Manner）
   - 表達清晰明確
   - 避免歧義、不明確或過度冗長

## 日文例句

### 例 1：違反量的準則

**場景**：
```
A: 昨日の試験はどうでしたか？
B: 問題1と問題2は解けました。
```

**字義含意**：Bが問題1と問題2を解いた

**會話含意**：Bは問題3以降は解けなかった（提供された情報が限定的であるため）

### 例 2：違反品質準則

**場景**：
```
A: この映画はどう思う？
B: 確かに、色んな人が出てますね。
```

**字義含意**：この映画には多くの人が出演している

**會話含意**：この映画は面白くない（品質準則を侵害することで、映画への否定的な態度を暗示）

### 例 3：違反關係準則

**場景**：
```
A: この企画書は完成しましたか？
B: 天気がいいですね。
```

**字義含意**：天気がいい

**會話含意**：企画書はまだ完成していない、または完成させることができない（話題を無視することで、困難な状況を暗示）

## 中文例句

### 例 1：數量暗示

**對話**：
```
甲：那場會議怎麼樣？
乙：與會者都很認真，討論熱烈。
```

**字義含意**：與會者認真，討論熱烈

**會話含意**：沒有達成預期結果（只強調過程，未提結論暗示沒有好結果）

### 例 2：品質暗示

**對話**：
```
甲：你的新同事怎麼樣？
乙：他總是按時上班。
```

**字義含意**：新同事按時上班

**會話含意**：新同事在其他方面可能有問題（故意迴避其他評價）

### 例 3：關聯暗示

**對話**：
```
甲：你為什麼沒有參加昨天的聚會？
乙：最近天氣不太好。
```

**字義含意**：最近天氣不好

**會話含意**：說話者不想解釋真正原因，可能是有意推託（話題轉移）

## 英文例句

### Example 1: Quantity Maxim Violation

**Scenario**:
```
A: How did the project go?
B: We completed Phase 1 and Phase 2.
```

**Literal Meaning**: Phases 1 and 2 were completed

**Implicature**: Phases 3 and later were not completed (omission of information implies a problem)

### Example 2: Quality Maxim Violation

**Scenario**:
```
A: What do you think of the new manager?
B: He has a nice office.
```

**Literal Meaning**: The manager has a nice office

**Implicature**: The speaker doesn't want to directly criticize the manager, suggesting dissatisfaction (evasion of judgment)

### Example 3: Relation Maxim Violation

**Scenario**:
```
A: Can you help me with this problem?
B: I'm very busy these days.
```

**Literal Meaning**: The speaker is busy

**Implicature**: The speaker cannot or doesn't want to help (indirect refusal)

## 學習重點

1. **會話含意不是字面意思**：理解言語交流中的隱含含義至關重要
2. **準則違反的藝術**：發話者有意違反準則來傳達特定含意
3. **語境依賴性**：同樣的表達在不同語境中會產生不同的會話含意
4. **跨文化差異**：不同文化對會話含意的使用方式可能差異很大
