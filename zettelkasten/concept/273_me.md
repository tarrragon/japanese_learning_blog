「目」是日文中極為常用的多義詞，涵蓋身體部位、視覺功能、外觀判斷及接尾辭用法。

目（め）

## 日文解釋

「目」は、生物が光を感知して視覚情報を得るための器官を指す最も基本的な意味から、比喩的に「視線」「外観」「判断力」などの抽象的な意味へと広がっている多義語である。

また、接尾辞として「～番目」「～回目」のように順序や回数を表す用法も日常的に使われる。さらに、日本語には「目」を用いた慣用句が非常に多く、「目が高い」（審美眼がある）、「目を通す」（ざっと読む）など、視覚に関連した比喩表現が豊富に存在する。

日本文化において「目」は心の窓とされ、「目は口ほどに物を言う」という諺があるように、非言語コミュニケーションにおいて重要な役割を果たすと考えられている。

## 英文解釋

The word 目 (me) is a highly polysemous term in Japanese. Its primary meaning refers to the eye as a body part, but it extends metaphorically to concepts like "gaze," "appearance," "judgment," and "experience."

As a suffix, 目 indicates ordinal position (e.g., 一番目 = first, 二回目 = second time). Japanese contains numerous idiomatic expressions using 目, reflecting the cultural significance of visual perception in communication and social interaction.

## 中文解釋

「目」是日文中極具多義性的詞彙。其最基本的意義是指眼睛這一身體器官，但通過比喻延伸出「視線」「外觀」「判斷力」「經驗」等抽象含義。

作為接尾辭使用時，「目」表示順序或次數（如：一番目＝第一、二回目＝第二次）。日文中包含大量以「目」構成的慣用語，反映出視覺感知在日本文化中的重要地位。

## 核心用法

### 用法 1：身體部位（眼睛）

表示眼睛這一視覺器官，是最基本的用法。

**例句 1**
```
目が疲れたので、少し休憩しよう。
My eyes are tired, so let's take a short break.
眼睛累了，稍微休息一下吧。
```

**例句 2**
```
彼女は大きな目が印象的だ。
Her large eyes are impressive.
她的大眼睛令人印象深刻。
```

### 用法 2：視線・注目

表示視線或注意力的方向。

**例句 3**
```
みんなの目が彼に集まった。
Everyone's eyes turned to him.
所有人的目光都集中在他身上。
```

### 用法 3：接尾辭（順序・次數）

接在數詞後面表示順序或次數。

**例句 4**
```
これが三回目のチャレンジです。
This is my third attempt.
這是第三次挑戰。
```

**例句 5**
```
一番目の問題から解いてください。
Please solve from the first question.
請從第一題開始作答。
```

### 用法 4：慣用表現

與其他詞搭配形成慣用語。

**例句 6**
```
彼は骨董品に目が高い。
He has a good eye for antiques.
他對古董很有鑑賞力。
```

## 常見慣用句

### 目を通す（ざっと読む）

**例**：書類に目を通しておいてください。
請先過目一下文件。

### 目が高い（審美眼がある）

**例**：お客様は目が高いですね。
客人真有眼光。

### 目が回る（非常に忙しい）

**例**：年末は目が回るほど忙しい。
年末忙得團團轉。

### 目を引く（注目を集める）

**例**：彼女の赤いドレスが目を引いた。
她的紅裙子很引人注目。

### 目がない（大好き）

**例**：私は甘いものに目がない。
我超愛吃甜食。

## 常見錯誤

### 錯誤 1：接尾辭用法的數詞選擇

❌ 誤：一目（いちめ）の人
✅ 正：一番目（いちばんめ）の人 / 一人目（ひとりめ）の人
說明：表示順序時，通常使用「番目」或特定量詞＋目的形式，不能直接說「一目」。

### 錯誤 2：「目」與「眼」的混用

❌ 誤：眼が疲れた（一般口語）
✅ 正：目が疲れた
說明：日常會話中多用「目」，「眼」（まなこ/がん）較常見於書面語或複合詞（如：眼科、眼球）。

### 錯誤 3：慣用句的誤用

❌ 誤：彼は音楽に目が高い。
✅ 正：彼は音楽に耳が高い。/ 彼は音楽の耳がいい。
說明：「目が高い」用於視覺相關的鑑賞力，音樂應使用「耳」相關的表達。

## 學習要點

1. **多義詞的核心**：理解「目」從具體（眼睛）到抽象（視線、判斷）的意義延伸。
2. **接尾辭用法**：掌握「～番目」「～回目」「～人目」等表示順序的用法。
3. **慣用句豐富**：日文中以「目」構成的慣用句極多，需逐一學習記憶。
4. **與「眼」的區別**：口語用「目」，書面語或專業用語多用「眼」。
5. **文化意涵**：理解日本文化中「目」在非言語溝通中的重要性。
