## 日文解釋

「視力」とは、目で物を見分ける能力のことを指します。具体的には、物の形・大きさ・色などを識別する能力であり、医学的には「視力検査」によって数値化されます。

日本では視力は通常小数点で表され、「1.0」が標準的な視力とされています。「視力が落ちる」「視力が低下する」という表現で視力の悪化を表し、「視力を矯正する」で眼鏡やコンタクトレンズによる補正を表します。

日常会話では「目が悪い」「目がいい」という言い方も多く使われますが、医療や公的な場面では「視力」という語が正式に使用されます。

## 英文解釋

"Shiryoku" (視力) refers to visual acuity - the sharpness or clarity of vision. It measures the eye's ability to discern fine details and distinguish objects at various distances.

In Japan, visual acuity is typically expressed in decimal form (e.g., 1.0, 0.5), unlike the Snellen fraction system (20/20) used in English-speaking countries. The term encompasses both distance vision (遠視力) and near vision (近視力).

Common collocations include "shiryoku ga warui" (poor eyesight) and "shiryoku wo sokutei suru" (to measure visual acuity).

## 中文解釋

「視力」指的是眼睛辨識物體的能力，包括分辨物體的形狀、大小、顏色等細節的能力。在醫學上，視力透過視力檢查來測量並以數值表示。

日本的視力表示方式採用小數制（如 1.0、0.5），與台灣相同，但與歐美的分數表示法（如 20/20）不同。視力 1.0 被視為標準視力，低於此數值則需要矯正。

常見的搭配詞包括「視力が落ちる」（視力下降）、「視力検査」（視力檢查）、「視力矯正」（視力矯正）等。

## 核心用法

### 用法 1：描述視力狀態

說明視力的好壞或變化情況。

**例句 1**
```
最近、視力が落ちてきたので、眼科に行こうと思います。
Recently my eyesight has been getting worse, so I'm thinking of going to the eye doctor.
最近視力下降了，所以我打算去眼科檢查。
```

**例句 2**
```
私の視力は左右とも1.0です。
My visual acuity is 1.0 in both eyes.
我雙眼的視力都是 1.0。
```

### 用法 2：視力檢查相關

與視力測量和檢查相關的表達。

**例句 3**
```
運転免許の更新には視力検査が必要です。
A vision test is required for driver's license renewal.
更新駕照需要進行視力檢查。
```

**例句 4**
```
健康診断で視力を測定したら、0.3しかなかった。
When they measured my eyesight at the health checkup, it was only 0.3.
在健康檢查時測量視力，結果只有 0.3。
```

### 用法 3：視力矯正相關

與眼鏡、隱形眼鏡等視力矯正相關的表達。

**例句 5**
```
視力矯正のために、コンタクトレンズを使っています。
I use contact lenses to correct my vision.
我使用隱形眼鏡來矯正視力。
```

## 常見錯誤

### 錯誤 1：與「視覚」混淆

❌ 誤：視覚が1.0あります。
✅ 正：視力が1.0あります。
說明：「視覚」指的是視覺這一感官功能，而「視力」指的是可以測量的視覺能力數值。表達視力數值時應使用「視力」。

### 錯誤 2：誤用動詞搭配

❌ 誤：視力が減る。
✅ 正：視力が落ちる／視力が低下する。
說明：視力的下降用「落ちる」或「低下する」，不用「減る」。

## 學習要點

1. **數值表達**：日本視力用小數表示，1.0 為標準，0.1 以下為重度視力障礙
2. **常見搭配**：「視力が落ちる」（下降）、「視力を測る」（測量）、「視力を矯正する」（矯正）
3. **相關詞彙**：近視（きんし）、遠視（えんし）、乱視（らんし）、老眼（ろうがん）
4. **場景區分**：日常用「目がいい/悪い」，正式場合用「視力」
5. **複合詞**：視力検査、視力表、視力障害、矯正視力、裸眼視力
