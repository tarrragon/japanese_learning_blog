「～ていただく」は、相手に何かをしてもらう際に使う謙譲表現です。「～てもらう」の謙譲語として機能し、ビジネスや改まった場面で頻繁に使用されます。

## 日文

～ていただく（～ていただく）

### 文法情報

| 項目 | 内容 |
|------|------|
| 文法類型 | 補助動詞（授受表現） |
| JLPT | N3 |
| 使用場面 | ビジネス、正式場面、丁寧な依頼 |
| 形成 | 動詞て形 + いただく |

### 形成規則

**基本形式**
```
動詞て形 + いただく
```

**活用例**
```
見る → 見ていただく
説明する → 説明していただく
待つ → 待っていただく
読む → 読んでいただく
```

## 日文解釋

「～ていただく」は、相手の行為に対して感謝と敬意を示す謙譲表現です。

この表現は「～てもらう」の謙譲語であり、話し手が相手から恩恵を受ける行為について述べる際に使用します。相手の行為を自分にとっての恩恵として捉え、へりくだって表現することで、敬意を示すことができます。

特にビジネスシーンでは、上司・取引先・お客様に対して何かを依頼したり、相手がしてくれたことに感謝を述べたりする際に不可欠な表現です。「～てくださる」が相手を主語にするのに対し、「～ていただく」は話し手を主語にして相手の行為を受けることを表します。

## 英文解釋

"~te itadaku" is a humble expression used when receiving a favor or action from someone. It is the humble (kenjougo) form of "~te morau" and expresses gratitude and respect toward the person performing the action.

This grammar pattern is essential in formal and business Japanese. It positions the speaker as the humble recipient of someone's action, emphasizing gratitude for the favor received. The construction literally means "to humbly receive the doing of [verb]" and is frequently used in polite requests and expressions of thanks.

## 中文解釋

「～ていただく」是一種謙讓表達方式，用於請求他人為自己做某事，或表達對他人為自己所做之事的感謝。這是「～てもらう」的謙讓語形式。

這個文法表達了說話者謙卑地接受對方的行為，同時對對方表示敬意。在商務場合中特別重要，用於向上司、客戶或合作夥伴提出請求，或對對方的幫助表示感謝。與「～てくださる」不同的是，「～ていただく」以說話者為主語，強調自己「接受」對方行為的恩惠。

## 核心用法

### 用法 1：丁寧な依頼

相手に何かを丁寧に依頼する際に使います。

**例句 1**
```
この書類をご確認いただけますでしょうか。
Could you please check this document?
能否請您確認一下這份文件？
```

**例句 2**
```
少々お待ちいただけますか。
Could you wait a moment, please?
能否請您稍等一下？
```

### 用法 2：感謝の表現

相手がしてくれたことに対して感謝を述べる際に使います。

**例句 3**
```
本日はお越しいただきありがとうございます。
Thank you for coming today.
感謝您今天的蒞臨。
```

**例句 4**
```
貴重なご意見をいただき、大変参考になりました。
I received valuable feedback from you, which was very helpful.
承蒙您提供寶貴意見，非常有參考價值。
```

### 用法 3：許可を求める

相手に何かをさせてもらう許可を求める際に使います。

**例句 5**
```
お名前をお伺いしてもよろしいでしょうか。
May I ask your name?
請問可以請教您的大名嗎？
```

## 文法規則

### 規則 1：「～てもらう」との使い分け

- 「～てもらう」：カジュアルな場面、友人・家族間
- 「～ていただく」：フォーマルな場面、目上の人・お客様に対して

### 規則 2：「～てくださる」との違い

| 項目 | ～ていただく | ～てくださる |
|------|-------------|-------------|
| 主語 | 話し手 | 相手 |
| 視点 | 恩恵を受ける側 | 恩恵を与える側 |
| 敬語分類 | 謙譲語 | 尊敬語 |

### 規則 3：「お/ご～いただく」形式

動詞によっては「お/ご」を付けてより丁寧にできます。
- 待つ → お待ちいただく
- 確認する → ご確認いただく
- 連絡する → ご連絡いただく

## 常見錯誤

### 錯誤 1：主語の混同

❌ 誤：先生が私に教えていただきました。
✅ 正：先生に教えていただきました。
說明：「～ていただく」は話し手が主語。「に」で行為者を示す。

### 錯誤 2：カジュアルな場面での過剰使用

❌ 誤：（友達に）お茶を買っていただけますか。
✅ 正：（友達に）お茶を買ってもらえる？
說明：親しい間柄では「～てもらう」が自然。

### 錯誤 3：自分の行為に使用

❌ 誤：私が説明させていただきます。
✅ 正：説明させていただきます。
說明：「私が」は冗長。謙譲表現では主語を省略することが多い。

## 學習要點

1. **授受表現の体系**：「～てもらう」→「～ていただく」の敬語変換を理解する
2. **視点の違い**：「～ていただく」（謙譲）と「～てくださる」（尊敬）の視点の違いを把握する
3. **場面の使い分け**：ビジネス・正式場面では「～ていただく」、カジュアルな場面では「～てもらう」
4. **「お/ご」との組み合わせ**：より丁寧な表現「お/ご～いただく」を覚える
5. **依頼と感謝**：依頼（～ていただけますか）と感謝（～ていただきありがとうございます）の両方で使える
