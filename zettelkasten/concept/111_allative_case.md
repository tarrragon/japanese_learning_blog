## 日文解釋

向格（allative case）とは、言語学において、動作や移動の方向・目標を示す格である。

日本語では、この格機能は主に格助詞「に」と「へ」によって表される。「に」は到達点や目標を強調し、「へ」は移動の方向を強調する傾向がある。例えば「東京に行く」は東京という目的地への到達を示し、「東京へ行く」は東京の方向への移動を示す。ただし、現代日本語においては両者はほぼ同義的に使用されることも多い。

向格は移動動詞（行く、来る、帰る等）や送達動詞（送る、届ける等）と共に使用され、動作の到達点を明示する。また、抽象的な意味として、変化の結果状態（「医者になる」）や贈与の相手（「友達にあげる」）を示す用法にも拡張されている。

向格の概念は、形態論的に格標識を持つ言語（フィンランド語、エストニア語等のウラル語派）では独立した格として存在するが、日本語のような膠着語では助詞によってこの機能が担われる。

## 英文解釋

The allative case is a grammatical case in linguistics that indicates the direction or goal of an action or movement.

In Japanese, this grammatical function is primarily expressed through the case particles に (ni) and へ (e). The particle に tends to emphasize the destination or endpoint, while へ emphasizes the direction of movement. For example, 東京に行く focuses on arriving at Tokyo as a destination, whereas 東京へ行く focuses on moving toward Tokyo. However, in modern Japanese, these particles are often used interchangeably.

The allative function appears with motion verbs (行く, 来る, 帰る, etc.) and verbs of transfer (送る, 届ける, etc.) to specify the endpoint of an action. This function has also been extended to abstract meanings, such as indicating resultant states of change (医者になる - "become a doctor") or recipients of giving (友達にあげる - "give to a friend").

While some languages (such as Finnish and Estonian in the Uralic family) have a morphologically distinct allative case, in agglutinative languages like Japanese, this function is expressed through particles.

## 中文解釋

向格是語言學中用於表示動作或移動的方向、目標的格位。

在日文中，此語法功能主要由格助詞「に」和「へ」承擔。「に」傾向於強調到達點或目標，「へ」則強調移動的方向。例如「東京に行く」強調東京這個目的地的到達，「東京へ行く」強調往東京方向的移動。不過在現代日文中，兩者經常可以互換使用。

向格與移動動詞（行く、来る、帰る等）及傳遞動詞（送る、届ける等）搭配使用，明示動作的終點。此外，其用法也擴展到抽象意義，如表示變化的結果狀態（「医者になる」成為醫生）或贈與的對象（「友達にあげる」給朋友）。

在某些語言中（如芬蘭語、愛沙尼亞語等烏拉爾語系語言），向格是獨立的形態格；在日文這類膠著語中，則由助詞來承擔這一功能。

## 核心用法

### 用法 1：表示移動的目的地

與移動動詞搭配，表示動作到達的終點。

**例句 1**
```
学校に行きます。
I go to school.
我去學校。
```

**例句 2**
```
日本へ旅行する予定です。
I plan to travel to Japan.
我計劃去日本旅行。
```

### 用法 2：表示移動的方向

強調移動的方向而非具體到達點。

**例句 3**
```
北へ向かって歩く。
Walk heading north.
朝北方走。
```

**例句 4**
```
海の方へ走っていった。
Ran off toward the sea.
往海的方向跑去了。
```

### 用法 3：表示變化的結果狀態

抽象用法，表示狀態變化的終點。

**例句 5**
```
彼は医者になった。
He became a doctor.
他成為了醫生。
```

**例句 6**
```
信号が青に変わった。
The traffic light changed to green.
信號燈變成了綠色。
```

## 「に」與「へ」的區別

### 核心差異

| 比較項目 | に | へ |
|---------|-----|-----|
| 核心語感 | 到達點、著落點 | 方向、朝向 |
| 焦點 | 終點本身 | 移動過程 |
| 抽象用法 | 廣泛（變化、對象） | 較少 |
| 可替換性 | 大多可替換 | 移動動詞時可替換 |

### 用法對比

1. **可以互換的場合**
   - 移動動詞：「東京に/へ行く」（去東京）
   - 方向表達：「駅に/へ向かう」（往車站前進）

2. **只能用「に」的場合**
   - 存在位置：「椅子に座る」（坐在椅子上）
   - 時間點：「三時に会う」（三點見面）
   - 間接受詞：「友達にあげる」（給朋友）
   - 變化結果：「水が氷になる」（水變成冰）

3. **傾向用「へ」的場合**
   - 強調方向：「西へ西へと進む」（不斷往西前進）
   - 書信用語：「〇〇様へ」（致〇〇先生/女士）

## 與其他格的關係

### 向格 vs 處所格（Locative）

向格表示移動的目標方向，處所格表示動作發生的地點。

```
学校に行く。（向格：往學校去）
学校で勉強する。（處所格：在學校學習）
```

### 向格 vs 奪格/離格（Ablative）

向格表示移動的目標，奪格/離格表示移動的起點。

```
東京に行く。（向格：去東京）
東京から来た。（奪格：從東京來）
```

## 學習要點

1. **核心功能**：向格表示動作或移動的方向、目標
2. **日文表達**：主要由「に」「へ」兩個助詞表達
3. **語感差異**：「に」強調到達點，「へ」強調方向
4. **抽象擴展**：「に」可表示變化結果、間接受詞等
5. **格系統定位**：與處所格（で）、奪格（から）形成完整的空間格系統
