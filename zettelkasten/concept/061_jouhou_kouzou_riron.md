## 定義

**資訊結構理論**（information structure / 情報構造）是語用學的重要分支，研究語言如何在話語中組織和傳遞資訊，特別關注**舊資訊與新資訊**、**主題與評述**、**焦點與背景**等概念。

### 核心概念

資訊結構理論主要處理三個層次的問題：

1. **主題與評述**（Topic-Comment）
   - 主題：談話的出發點
   - 評述：對主題的陳述

2. **焦點與背景**（Focus-Background）
   - 焦點：話語中最重要的新資訊
   - 背景：已知或次要資訊

3. **舊資訊與新資訊**（Given-New）
   - 舊資訊：聽者已知的內容
   - 新資訊：聽者不知的內容

## 日文中的資訊結構

### 1. 助詞「は」與「が」的區分

```
象は鼻が長い。
（主題是「大象」，焦點是「鼻子長」）

誰が来たの？ - 田中さんが来た。
（焦點在「田中先生」，回答疑問詞）
```

### 2. 語序與焦點

```
私は昨日本を読んだ。
（中立陳述）

昨日私は本を読んだ。
（焦點在「昨天」）

本を私は昨日読んだ。
（焦點在「書」，可能有對比含義）
```

### 3. 「の」的強調用法

```
買ったのは本だ。
（分裂句，焦點在「書」）

本を買ったのは私だ。
（焦點在「我」）
```

## 英文解釋

**Information Structure** is a theory in pragmatics that examines how language organizes and conveys information in discourse. It analyzes the distribution of:

- **Topic-Comment**: What the sentence is about vs. what is said about it
- **Focus-Background**: Highlighted new information vs. presupposed content
- **Given-New**: Information already known to the listener vs. newly introduced information

In Japanese, information structure is primarily realized through:
- Particles (は for topic, が for exhaustive listing or focus)
- Word order variation (fronting for emphasis)
- Cleft constructions (のだ/のは constructions)
- Prosody (pitch accent patterns)

## 中文解釋

**資訊結構理論**是語用學的重要分支，研究語言如何在話語中組織和傳遞資訊。主要分析：

- **主題與評述**：句子談論的對象 vs. 對該對象的陳述
- **焦點與背景**：突出的新資訊 vs. 預設的背景資訊
- **舊資訊與新資訊**：聽者已知的內容 vs. 新引入的內容

在日語中，資訊結構主要通過以下方式實現：
- 助詞（は 標記主題，が 標記焦點或窮盡列舉）
- 語序變化（前置成分表達強調）
- 分裂句（のだ/のは 結構）
- 韻律（音高重音模式）

## 理論應用

### 1. 對比分析

```
（A）雨が降っている。
（中立描述，焦點在「下雨」這件事）

（B）雨は降っている。
（主題是「雨」，隱含對比：雨在下，但其他天氣現象可能不同）
```

### 2. 疑問詞與焦點

```
Q: 誰が本を読んだ？（焦點疑問）
A: 太郎が読んだ。（焦點在「太郎」）

Q: 太郎は何をした？（主題疑問）
A: 本を読んだ。（焦點在「讀書」）
```

### 3. 分裂句的使用

```
私が好きなのはコーヒーだ。
（強調「咖啡」是我喜歡的東西）

コーヒーを飲んだのは昨日だ。
（強調時間是「昨天」）
```
