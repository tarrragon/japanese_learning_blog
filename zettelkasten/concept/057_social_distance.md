# 社會距離

## 定義

### 日文
話し手と聞き手の間の心理的・社会的な距離感。親しさの程度や上下関係によって決まり、言葉遣いや敬語の使用に大きく影響する。

### 繁體中文
說話者與聽話者之間的心理與社會距離感。由親疏程度和上下關係決定，對用語和敬語使用有重大影響。

### English
The psychological and social distance between speaker and listener. Determined by degree of familiarity and hierarchical relationship, significantly affecting word choice and honorific usage.

---

## 核心概念

社會距離是日語溝通中的關鍵要素，決定了說話者選擇何種語言形式：

### 距離的維度

1. **親疏程度（親しさ）**
   - 初対面 vs 親友
   - 知人 vs 家族
   - 公的関係 vs 私的関係

2. **上下關係（上下関係）**
   - 目上 vs 目下
   - 先輩 vs 後輩
   - 上司 vs 部下

3. **內外關係（ウチ・ソト）**
   - 身内（家族、同僚）
   - 外部（客、他社）

---

## 語言表現的影響

### 敬語選擇

**距離遠（疎遠）**：
```
明日会議がございます。（丁寧語＋尊敬語）
お越しくださいませんか。（尊敬語＋謙譲語）
```

**距離近（親密）**：
```
明日会議があるよ。（普通体）
来ない？（カジュアル）
```

### 呼稱方式

| 距離 | 呼び方 | 例 |
|------|--------|-----|
| 遠い | 姓＋役職/さん | 田中部長、山田さん |
| 中間 | 姓＋さん/君 | 田中さん、山田君 |
| 近い | 名＋さん/呼び捨て | 太郎さん、花子 |
| 非常に近い | あだ名 | たーくん、はなちゃん |

### 文末表現

**遠距離**：
- ～ます・～です
- ～ございます
- ～でしょうか

**近距離**：
- ～だ・～である
- ～よ・～ね
- ～じゃん・～っしょ

## 実例分析

### 場面1：初対面の同僚

```
A：はじめまして。営業部の田中と申します。
B：こちらこそ。企画部の山田です。よろしくお願いします。
```

**分析**：
- 丁寧語使用（距離大）
- 姓＋役職
- 謙譲語「申します」

### 場面2：親しい友人

```
A：明日飲みに行かない？
B：いいね！何時にする？
```

**分析**：
- 普通体（距離小）
- 疑問形の省略
- カジュアルな終助詞

### 場面3：上司與部下

```
部下：部長、資料をお持ちしました。
部長：ああ、ありがとう。机に置いといて。
```

**分析**：
- 部下→上司：尊敬語＋謙譲語（距離大＋地位差）
- 上司→部下：普通体（地位優位）

---

## 距離變化的動態性

### 時間經過による変化

**初期（疎遠）**：
```
田中さん、この書類を確認していただけますか。
```

**数ヶ月後（やや親密）**：
```
田中さん、これちょっと見てもらえる？
```

**1年後（親密）**：
```
田中、これ見といて。
```

### 場面による調整

**同じ相手でも**：
- 会議中：丁寧語（公的）
- 休憩時：普通体（私的）
