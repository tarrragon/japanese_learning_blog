# 動詞活用（どうしかつよう）

**中文解釋**：
動詞活用是指動詞根據時態、語氣、肯定否定等語法功能改變形態的系統。日文動詞分為三大類（五段動詞、一段動詞、カ變サ變動詞），各有不同的活用規則。掌握活用形是學習日文文法的基礎。

**English Definition**:
Verb conjugation refers to the systematic modification of verb forms based on tense, mood, affirmation/negation, and other grammatical functions. Japanese verbs are classified into three groups (godan verbs, ichidan verbs, and irregular verbs), each following distinct conjugation patterns. Mastering conjugation forms is fundamental to learning Japanese grammar.

**Japanese Explanation**:
動詞活用とは、動詞が時制、語気、肯定・否定などの文法機能に応じて形を変える体系を指す。日本語の動詞は三つのグループ（五段活用動詞、一段活用動詞、カ行変格活用・サ行変格活用動詞）に分類され、それぞれ異なる活用規則に従う。活用形を習得することは日本語文法学習の基礎である。

---

## 活用形總覽

日文動詞的主要活用形包括：

| 活用形 | 用途 | 範例（書く） |
|--------|------|-------------|
| 辞書形 | 基本形、字典形式 | 書く |
| ます形 | 禮貌體現在/未來 | 書きます |
| て形 | 接續、進行式 | 書いて |
| た形 | 過去式 | 書いた |
| ない形 | 否定形 | 書かない |
| 可能形 | 能力表達 | 書ける |
| 受身形 | 被動語態 | 書かれる |
| 使役形 | 使役語態 | 書かせる |
| 命令形 | 命令語氣 | 書け |
| 意志形 | 意志、推測 | 書こう |

---

## 三類動詞活用對照

### 五段動詞（う段結尾）

**範例**：書く（かく）

| 活用形 | 形式 | 活用方式 |
|--------|------|---------|
| 辞書形 | 書く | 原形 |
| ます形 | 書きます | く→き |
| て形 | 書いて | く→いて |
| た形 | 書いた | く→いた |
| ない形 | 書かない | く→か |

### 一段動詞（る結尾）

**範例**：食べる（たべる）

| 活用形 | 形式 | 活用方式 |
|--------|------|---------|
| 辞書形 | 食べる | 原形 |
| ます形 | 食べます | る→ます |
| て形 | 食べて | る→て |
| た形 | 食べた | る→た |
| ない形 | 食べない | る→ない |

### 不規則動詞

**範例**：する・来る（くる）

| 活用形 | する | 来る |
|--------|------|------|
| 辞書形 | する | 来る |
| ます形 | します | 来ます |
| て形 | して | 来て |
| た形 | した | 来た |
| ない形 | しない | 来ない |

## 例句

1. **辞書形（字典形）**
   - **日文**：明日、友達と映画を**見る**。
   - **假名**：あした、ともだちとえいがを**みる**。
   - **中文**：明天要和朋友看電影。
   - **English**: I will watch a movie with friends tomorrow.

2. **ます形（禮貌形）**
   - **日文**：毎朝、新聞を**読みます**。
   - **假名**：まいあさ、しんぶんを**よみます**。
   - **中文**：每天早上讀報紙。
   - **English**: I read the newspaper every morning.

3. **て形（接續形）**
   - **日文**：宿題を**して**から、遊びます。
   - **假名**：しゅくだいを**して**から、あそびます。
   - **中文**：做完作業後玩耍。
   - **English**: I'll play after doing homework.

4. **た形（過去形）**
   - **日文**：昨日、友達に**会った**。
   - **假名**：きのう、ともだちに**あった**。
   - **中文**：昨天見了朋友。
   - **English**: I met my friend yesterday.

5. **ない形（否定形）**
   - **日文**：今日は学校に**行かない**。
   - **假名**：きょうはがっこうに**いかない**。
   - **中文**：今天不去學校。
   - **English**: I'm not going to school today.

## 學習重點

1. **動詞分類**：先判斷動詞屬於哪一類（五段、一段、不規則）
2. **語幹識別**：找出動詞的語幹（ます形去掉ます）
3. **音便規則**：五段動詞的て形、た形有音便變化
4. **特殊動詞**：する、来る、ある等不規則動詞需單獨記憶
5. **實際應用**：透過造句練習鞏固各活用形的使用
