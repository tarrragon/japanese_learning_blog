## 日文

連用形（れんようけい）

### 概念情報

| 項目 | 內容 |

### 定義

連用形とは、日本語の用言（動詞・形容詞・形容動詞）における六つの活用形の一つである。「連用」という名称は「用言に連なる」という意味で、他の用言や助動詞の前に置かれて接続する機能を持つ。

### 連用形の作り方

**五段動詞**：語尾を「い段」に変える
- 書く → 書き
- 読む → 読み
- 話す → 話し

**一段動詞**：「る」を取り除く
- 食べる → 食べ
- 見る → 見

**変格動詞**
- する → し
- 来る → き

**形容詞**：「い」を「く」に変える
- 高い → 高く
- 美しい → 美しく

**形容動詞**：語幹に「に」を付ける
- 静かだ → 静かに
- 綺麗だ → 綺麗に

## 日文解釋

連用形は日本語文法において極めて重要な活用形である。その名の通り、用言（動詞・形容詞・助動詞など）に「連なる」ために使われる形式で、様々な文法的機能を担う。

連用形の主な機能は以下の通りである：

1. **中止法**：文を一旦区切って次の節に続ける
2. **助動詞接続**：「ます」「たい」「ながら」などの助動詞に接続
3. **複合動詞形成**：二つの動詞を組み合わせて新しい動詞を作る
4. **名詞化**：連用形がそのまま名詞として機能する場合がある

特に「ます形」の基盤となる形式として、丁寧語を形成する上で欠かせない。また、「て形」や「た形」を作る際の出発点ともなり、日本語学習者にとって習得必須の活用形である。

## 英文解釋

The **renyoukei** (連用形), also known as the continuative or conjunctive form, is one of the six basic conjugation forms in Japanese grammar. The term literally means "form that connects to inflected words" (用言に連なる形).

This form serves several essential grammatical functions:

1. **Conjunctive function**: Connects clauses in compound sentences (中止法)
2. **Auxiliary attachment**: Serves as the base for attaching auxiliaries like ます (masu), たい (tai), and ながら (nagara)
3. **Compound verb formation**: Combines with other verbs to create compound verbs (e.g., 読み始める "start reading")
4. **Nominalization**: Some renyoukei forms function as nouns (e.g., 休み from 休む)

The renyoukei is particularly important as the foundation of the polite masu-form. For learners, understanding this form is crucial for constructing grammatically correct and naturally flowing sentences.

## 中文解釋

**連用形**是日語動詞六種活用形之一，其名稱來自「連接用言」的意思，即這種形式用於連接其他用言（動詞、形容詞等）或助動詞。

連用形的主要功能包括：

1. **中止法**：暫時中斷句子，連接下一個子句
2. **接續助動詞**：作為「ます」「たい」「ながら」等助動詞的接續基礎
3. **複合動詞形成**：與其他動詞結合形成複合動詞（如「読み始める」開始閱讀）
4. **名詞化**：部分連用形可直接作為名詞使用（如「休み」休息）

連用形是「ます形」的基礎，對於學習日語禮貌表達至關重要。此外，它也是形成「て形」和「た形」的出發點，是日語學習者必須掌握的核心活用形。

## 核心用法

### 用法 1：中止法（接續前後句）

連用形可以暫時中斷句子，連接後續的子句，表示動作的並列或時間順序。

**例句 1**
```
朝起き、顔を洗い、朝食を食べた。
I woke up, washed my face, and ate breakfast.
早上起床，洗臉，然後吃了早餐。
```

**例句 2**
```
彼女は笑い、私も笑った。
She laughed, and I laughed too.
她笑了，我也笑了。
```

### 用法 2：接續助動詞

連用形是接續「ます」「たい」「ながら」等助動詞的基礎形式。

**例句 3**
```
毎日日本語を勉強します。
I study Japanese every day.
我每天學習日語。
```

**例句 4**
```
日本に行きたいです。
I want to go to Japan.
我想去日本。
```

### 用法 3：複合動詞形成

連用形與其他動詞結合，形成表達複雜意義的複合動詞。

**例句 5**
```
本を読み始めた。
I started reading a book.
開始讀書了。
```

**例句 6**
```
電車に乗り遅れた。
I missed the train.
趕不上電車了。
```

## 文法規則

### 規則 1：五段動詞的連用形

五段動詞的連用形是將語尾從「う段」變為「い段」。

| 動詞 | 語尾 | 連用形 |
|------|------|--------|
| 書く | く→き | 書き |
| 読む | む→み | 読み |
| 話す | す→し | 話し |
| 待つ | つ→ち | 待ち |
| 死ぬ | ぬ→に | 死に |
| 遊ぶ | ぶ→び | 遊び |
| 買う | う→い | 買い |
| 乗る | る→り | 乗り |

### 規則 2：一段動詞的連用形

一段動詞的連用形只需去掉「る」，語幹不變。

- 食べる → 食べ
- 見る → 見
- 起きる → 起き
- 寝る → 寝

### 規則 3：變格動詞的連用形

變格動詞有特殊的連用形：

- する → し
- 来る（くる）→ 来（き）

### 規則 4：形容詞的連用形

形容詞將「い」變為「く」形成連用形。

- 高い → 高く
- 美しい → 美しく
- おいしい → おいしく

## 常見錯誤

### 錯誤 1：混淆五段動詞和一段動詞的變化

❌ 誤：食べる → 食べり（當作五段動詞變化）
✅ 正：食べる → 食べ

說明：「食べる」是一段動詞，連用形只需去掉「る」。

### 錯誤 2：忘記音便變化

❌ 誤：書きた
✅ 正：書いた

說明：五段動詞連用形接「た」「て」時需要音便變化，「書き」變為「書い」。

### 錯誤 3：形容詞連用形與副詞混淆

❌ 誤：高いに買った
✅ 正：高く買った

說明：形容詞的連用形是「高く」，不是「高いに」。

## 學習要點

1. **連用形是多功能活用形**：理解其接續功能是掌握日語複雜句型的基礎。

2. **中止法的運用**：連用形中止法使句子更簡潔優雅，是書面語和正式場合常用的表達方式。

3. **ます形的基礎**：連用形+ます=丁寧形，這是日語禮貌表達的核心。

4. **複合動詞的組成**：許多複合動詞由連用形+動詞構成，如「読み始める」「走り回る」「飛び出す」等。

5. **名詞化功能**：部分連用形可直接作為名詞使用，如「休み（休息）」「帰り（回程）」「始まり（開始）」。
