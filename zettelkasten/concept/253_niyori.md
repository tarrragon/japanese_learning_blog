「により」是表示原因、手段、依據的格助詞複合形式，為「によって」的書面語變體。

<!--more-->

## 日文

により（に因り・に依り）

### 文法情報

| 項目 | 內容 |
|------|------|
| 文法類型 | 格助詞複合形式 |
| JLPT | N3 |
| 使用場面 | 書面語、正式文書、新聞報導 |
| 形成 | 名詞 + により |
| 變體 | によって（口語）、による（連體修飾） |

### 形成規則

**基本形式**
```
名詞 + により
```

**範例**
```
地震 → 地震により（因為地震）
法律 → 法律により（依據法律）
努力 → 努力により（透過努力）
```

## 日文解釋

「により」は「によって」と同義の連語で、動作・作用の拠点を表す表現である。[^dict]

「により」は主に書き言葉で使用され、新聞記事、公式文書、法律文書などでよく見られる。口語では「によって」の方が自然である。

この表現には大きく分けて三つの用法がある。第一に、原因・理由を表す用法（「台風により電車が止まった」）。第二に、手段・方法を表す用法（「インターネットにより情報を得る」）。第三に、依拠・根拠を表す用法（「調査により明らかになった」）。

文体的には硬い印象を与えるため、日常会話では「によって」「〜で」「〜のせいで」などの表現が好まれる。

[^dict]: 定義參考 スーパー大辞林（macOS 內建字典）

## 英文解釋

"Niyori" is a compound particle expression meaning "by means of," "due to," or "according to." It is the formal, written variant of "niyotte."

This expression serves three main functions:
1. Indicating cause or reason (similar to "due to" or "because of")
2. Indicating means or method (similar to "by means of" or "through")
3. Indicating basis or grounds (similar to "according to" or "based on")

"Niyori" is predominantly used in formal written contexts such as news articles, official documents, and legal texts. In spoken Japanese, "niyotte" is more natural and commonly used.

## 中文解釋

「により」是表示「由於」、「透過」、「依據」的格助詞複合形式，為「によって」的書面語變體。

此表達主要有三種用法：
1. 表示原因、理由（相當於「由於」、「因為」）
2. 表示手段、方法（相當於「透過」、「藉由」）
3. 表示依據、根據（相當於「根據」、「依照」）

「により」主要用於書面語，常見於新聞報導、公文、法律文件等正式場合。在日常口語中，通常使用「によって」較為自然。

## 核心用法

### 用法 1：表示原因・理由

表達某事件或狀態的原因，常用於報導災害、事故等情況。

**例句 1**
```
台風により、多くの便が欠航となりました。
Due to the typhoon, many flights were cancelled.
由於颱風，許多航班停飛。
```

**例句 2**
```
新型ウイルスにより、イベントが中止されました。
The event was cancelled due to the new virus.
由於新型病毒，活動被取消。
```

### 用法 2：表示手段・方法

表達達成某目的所使用的手段或方法。

**例句 3**
```
本調査によりますと、若者の読書離れが進んでいます。
According to this survey, young people are reading less and less.
根據本調查，年輕人遠離閱讀的趨勢正在加劇。
```

**例句 4**
```
科学技術の発展により、生活は便利になった。
Through the development of science and technology, life has become more convenient.
透過科學技術的發展，生活變得更加便利。
```

### 用法 3：表示依據・根據

表達判斷或行動的依據。

**例句 5**
```
法律により、この行為は禁止されています。
This act is prohibited by law.
根據法律，這個行為是被禁止的。
```

## 文法規則

### 規則 1：「により」與「によって」的使用區分

「により」主要用於書面語和正式場合，而「によって」適用於口語和較輕鬆的書面語。

| 場合 | 適合的形式 |
|------|-----------|
| 新聞報導 | により |
| 公文・法律文書 | により |
| 學術論文 | により |
| 日常會話 | によって |
| 輕鬆的文章 | によって |

### 規則 2：「による」的連體修飾用法

當需要修飾名詞時，使用「による」而非「により」。

```
地震による被害（地震造成的損害）
法律による規制（法律的規範）
努力による成果（努力的成果）
```

### 規則 3：與被動態的結合

表示動作的執行者時，「により」常與被動態動詞搭配使用。

```
この法案は国会により可決された。
（這項法案被國會通過了。）
```

## 常見錯誤

### 錯誤 1：在口語中過度使用「により」

❌ 誤：雨により、試合は中止になったよ。
✅ 正：雨で、試合は中止になったよ。
說明：日常會話中，「により」顯得過於正式。應使用「で」或「によって」。

### 錯誤 2：混淆「により」和「による」

❌ 誤：地震により被害が大きかった。
✅ 正：地震による被害が大きかった。
說明：修飾名詞「被害」時，應使用連體形「による」。

### 錯誤 3：與不適當的詞彙搭配

❌ 誤：彼女の笑顔により、幸せになった。
✅ 正：彼女の笑顔で、幸せになった。
說明：「により」通常不用於描述個人情感的變化，這類情況用「で」更自然。

## 學習要點

1. **書面語特徵**：「により」是書面語表達，在新聞、公文、論文中常見，口語中較少使用。

2. **三種核心用法**：掌握原因（〜のせいで）、手段（〜を使って）、依據（〜に基づいて）三種基本意義。

3. **形式變化**：記住「により」（句中）、「による」（連體修飾）、「によって」（較口語）的使用區分。

4. **正式程度**：正式度排序為「により」＞「によって」＞「で」，根據場合選擇適當形式。

5. **搭配詞彙**：「により」常與災害、事件、法規等正式話題的名詞搭配，不適合用於日常瑣事。
