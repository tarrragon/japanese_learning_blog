## 日文

動詞（どうし）

動詞は日本語の文法において最も重要な品詞の一つです。動作、状態、変化などを表現し、文の述語として機能します。

## 日文解釋

動詞は日本語の文法体系において中核的な役割を果たす品詞です。動詞は主に動作（「走る」「食べる」）、状態（「ある」「いる」）、変化（「なる」「する」）を表現します。

日本語の動詞は活用形が非常に豊富で、時制（過去・現在・未来）、丁寧度（普通形・丁寧形）、否定、可能、受身、使役など、様々な文法的意味を動詞の語尾変化によって表現します。これは英語などの言語が助動詞を使うのとは対照的です。

動詞の活用パターンによって「一段動詞」（語幹が一定）、「五段動詞」（語幹が五十音図の段で変化）、「不規則動詞」（する・来る）の三つに分類されます。この分類は日本語学習において最も基礎的かつ重要な文法知識です。

また、動詞には「自動詞」（主語の動作・状態のみを表す）と「他動詞」（目的語を必要とする）という意味的な分類もあり、助詞の選択や文の構造に大きく影響します。例えば「窓が開く」（自動詞・が）と「窓を開ける」（他動詞・を）のように、同じ意味領域でも異なる動詞を使い分けます。

## 英文解釋

Verbs are the most important part of speech in Japanese grammar. They express actions, states, and changes, and serve as predicates in sentences.

Japanese verbs have a rich conjugation system that encodes various grammatical meanings through suffix changes, including tense (past, present, future), politeness level (plain form, polite form), negation, potential, passive, and causative forms. This is in contrast to languages like English, which rely more heavily on auxiliary verbs.

Verbs are classified into three types based on conjugation patterns: "ichidan verbs" (一段動詞), where the stem remains constant; "godan verbs" (五段動詞), where the stem changes across five rows of the hiragana chart; and "irregular verbs" (不規則動詞), which include only する (to do) and 来る (to come).

Another important classification distinguishes between intransitive verbs (自動詞), which describe actions or states of the subject alone, and transitive verbs (他動詞), which require a direct object. This distinction affects particle usage and sentence structure significantly.

## 中文解釋

動詞是日文文法中最重要的詞類之一。動詞表達動作、狀態和變化，並在句子中擔任述語的角色。

日文動詞具有豐富的變化形式，透過詞尾變化來表達各種文法意義，包括時態（過去、現在、未來）、禮貌程度（普通形、丁寧形）、否定、可能、被動、使役等。這與英文等語言主要依賴助動詞的方式形成對比。

根據活用模式，動詞分為三類：「一段動詞」（語幹固定不變）、「五段動詞」（語幹在五十音圖的段中變化）、「不規則動詞」（僅包含する和来る兩個）。這個分類是日語學習中最基礎且重要的文法知識。

此外，動詞還有「自動詞」（只描述主語的動作或狀態）和「他動詞」（需要賓語）的語義分類，這影響助詞的選擇和句子結構。例如「窓が開く」（自動詞，用が）和「窓を開ける」（他動詞，用を），雖然意思相關，但使用不同的動詞和助詞。

## 核心用法

### 用法 1：表達動作

動詞最基本的功能是表達主語執行的動作。

**例句 1**
```
私は毎日本を読みます。
I read books every day.
我每天讀書。
```

**例句 2**
```
田中さんは会社に行きました。
Tanaka-san went to the company.
田中先生去公司了。
```

### 用法 2：表達狀態或存在

某些動詞用來表達主語的狀態或存在。

**例句 3**
```
机の上に辞書があります。
There is a dictionary on the desk.
桌上有一本字典。
```

**例句 4**
```
公園に子供たちがいます。
There are children in the park.
公園裡有孩子們。
```

### 用法 3：表達變化或成為

動詞可以表達狀態的變化或轉變。

**例句 5**
```
桜の花がきれいになりました。
The cherry blossoms have become beautiful.
櫻花變美了。
```

## 文法規則

### 規則 1：活用形的基本結構

所有日文動詞都以「る」「う」「く」「す」「つ」「ぬ」「ぶ」「む」「ぐ」結尾（辭書形）。動詞的活用是透過改變這些詞尾來完成的。

- 一段動詞：去掉「る」，加上相應的詞尾（例：食べ**る** → 食べ**ます**）
- 五段動詞：改變最後一個假名的段（例：書**く** → 書**きます**）
- 不規則動詞：特殊變化（例：する → します、来る → 来ます）

### 規則 2：動詞在句中的位置

在日文中，動詞通常位於句末，是句子的核心成分。這與英文的 SVO（主語-動詞-賓語）語序不同，日文是 SOV（主語-賓語-動詞）語序。

例：
- 英文：I eat sushi.（SVO）
- 日文：私は寿司を食べます。（SOV）

### 規則 3：自動詞與他動詞的助詞搭配

- 自動詞通常與「が」搭配，表達主語的狀態或動作
- 他動詞通常與「を」搭配，表達對象物

這個區別是日文文法中最重要的概念之一，影響整個句子結構。

### 規則 4：動詞的時態表達

日文動詞只有兩個真正的時態形式：
- 非過去形（現在形/未來形）：食べる、食べます
- 過去形：食べた、食べました

非過去形可以表達現在或未來的動作，具體意義由上下文決定。

## 常見錯誤

### 錯誤 1：混淆一段動詞和五段動詞的活用

❌ 誤：食べります（將一段動詞按五段動詞變化）
✅ 正：食べます
說明：「食べる」是一段動詞，去掉「る」後加「ます」即可，不需要改變語幹。五段動詞才需要改變最後一個假名的段。

### 錯誤 2：自動詞他動詞的助詞混用

❌ 誤：電気を消える（他動詞的助詞配自動詞）
✅ 正：電気が消える（自動詞用が）/ 電気を消す（他動詞用を）
說明：「消える」是自動詞，表示「自然熄滅」，用「が」。如果要表達「使其熄滅」，應該用他動詞「消す」和「を」。

### 錯誤 3：將動詞放在句中而非句末

❌ 誤：私は食べます寿司を。（動詞不在句末）
✅ 正：私は寿司を食べます。
說明：日文是 SOV 語序，動詞必須放在句末（除非有接續句或補充說明）。這是日文語法的基本規則。

## 學習要點

1. **掌握三種動詞分類**：一段動詞、五段動詞、不規則動詞是日文文法的基礎，必須熟練辨識。
2. **理解活用形的規律**：每種動詞類型都有固定的活用規則，掌握規則比死記硬背更有效。
3. **區分自動詞與他動詞**：這個區別影響助詞選擇，是正確造句的關鍵。
4. **注意動詞的句末位置**：日文的 SOV 語序與中文、英文不同，需要習慣。
5. **從基本形開始學習**：先掌握辭書形（基本形）和ます形（丁寧形），再逐步學習其他活用形。
