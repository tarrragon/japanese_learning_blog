國際貿易中進口與出口的總稱，泛指商品與服務跨越國境的交易活動。

<!--more-->

## 日文

輸出入（ゆしゅつにゅう）

**羅馬拼音**：yushutsu nyuu

**詞性**：名詞（複合詞）

## 日文解釋

「輸出入」とは、「輸出」と「輸入」を合わせた言葉で、国際貿易における商品やサービスの国境を越えた取引を総称する用語です。「輸出」は自国から外国へ商品やサービスを送り出すこと、「輸入」は外国から自国へ商品やサービスを取り入れることを指します。輸出額と輸入額の差額は「貿易収支」と呼ばれ、経済政策の重要な指標となっています。自由貿易協定（FTA）や関税政策は輸出入に大きな影響を与え、国の経済成長や雇用に深く関係しています。

## 英文解釋

"Yushutsu nyuu" (import and export) is a compound term combining "export" (yushutsu) and "import" (yunyu), referring to the cross-border transactions of goods and services in international trade. Export refers to sending goods or services from one's own country to foreign countries, while import refers to bringing in goods or services from foreign countries. The difference between export value and import value is called the "trade balance," which serves as an important indicator for economic policy. Free trade agreements (FTAs) and tariff policies significantly impact imports and exports, deeply affecting national economic growth and employment.

## 中文解釋

「輸出入」是「輸出（出口）」與「輸入（進口）」的合稱，泛指國際貿易中商品和服務跨越國境的交易活動。「輸出」是指將本國的商品或服務銷售到國外，「輸入」是指從國外購買商品或服務進入本國。輸出額與輸入額的差額稱為「貿易收支」，是衡量一國經濟對外關係的重要指標。自由貿易協定（FTA）和關稅政策對進出口有重大影響，與國家經濟成長和就業密切相關。

## 例句

**例句1**
```
日本の輸出入額は年々増加傾向にあります。
Japan's import and export volume has been increasing year by year.
日本的進出口總額呈現逐年增加的趨勢。
```

**例句2**
```
輸出額が輸入額を上回ると、貿易収支は黒字になります。
When export value exceeds import value, the trade balance is in surplus.
當出口額超過進口額時，貿易收支呈現盈餘。
```

**例句3**
```
政府は輸出入の手続きを簡素化する新しい制度を導入しました。
The government introduced a new system to simplify import and export procedures.
政府引入了新制度以簡化進出口手續。
```

**例句4**
```
円安は輸出に有利ですが、輸入品の価格上昇を招きます。
A weak yen favors exports but causes import prices to rise.
日圓貶值有利於出口，但會導致進口商品價格上漲。
```

**例句5**
```
この港は日本の輸出入の重要な拠点となっています。
This port serves as an important hub for Japan's imports and exports.
這個港口是日本進出口的重要據點。
```

## 核心用法

### 常見詞彙搭配

| 詞彙 | 讀音 | 意義 |
|------|------|------|
| 輸出額 | ゆしゅつがく | 出口金額 |
| 輸入額 | ゆにゅうがく | 進口金額 |
| 貿易収支 | ぼうえきしゅうし | 貿易收支 |
| 輸出入品 | ゆしゅつにゅうひん | 進出口商品 |
| 輸出入業者 | ゆしゅつにゅうぎょうしゃ | 進出口業者 |
| 輸出入手続き | ゆしゅつにゅうてつづき | 進出口手續 |
| 輸出入規制 | ゆしゅつにゅうきせい | 進出口管制 |

### 相關指標

- **貿易黒字**（ぼうえきくろじ）：出口大於進口，貿易盈餘
- **貿易赤字**（ぼうえきあかじ）：進口大於出口，貿易赤字
- **貿易依存度**（ぼうえきいぞんど）：進出口總額占 GDP 的比例

## 學習要點

1. **輸出入 vs 入出力**：注意「輸出入」（ゆしゅつにゅう）指國際貿易，而「入出力」（にゅうしゅつりょく）指電腦的輸入輸出（I/O），兩者意義完全不同。

2. **複合詞結構**：「輸出」＋「輸入」→「輸出入」，省略重複的「輸」字，這是日文常見的複合詞縮略方式。

3. **經濟文章必備**：閱讀經濟新聞或日經新聞時，「輸出入」是高頻詞彙，掌握相關搭配詞對理解經濟資訊非常重要。

4. **與經常收支的關係**：輸出入是構成經常收支中「貿易收支」部分的核心，經常收支還包括服務收支、所得收支和經常轉移。
