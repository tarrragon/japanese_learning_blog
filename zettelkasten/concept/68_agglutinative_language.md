# 膠著語

## 日文解釋

膠著語（こうちゃくご）は、語形変化を表すために語幹に接辞を次々と付け加えることで文法的な意味を表現する言語類型である。日本語やトルコ語、フィンランド語などが代表的な膠著語に分類される。

各接辞は明確な境界を持ち、一つの文法的機能を担う。例えば、日本語の動詞「食べる」は「食べ（語幹）＋させ（使役）＋られ（受身）＋た（過去）」のように、複数の接辞を規則的に結合させることができる。

この特徴により、膠著語は以下の性質を持つ：
- 接辞の境界が明確で、切り離しやすい
- 一つの接辞が一つの文法機能を持つ（一対一対応）
- 語形変化が規則的で予測可能
- 複雑な文法関係を一つの語で表現できる

## 中文解釋

膠著語是一種透過在詞幹上依序附加詞綴來表達文法意義的語言類型。日語、土耳其語、芬蘭語等都是典型的膠著語。

每個詞綴都有明確的邊界，且承擔單一的文法功能。例如，日語動詞「食べる」可以組合成「食べ（詞幹）＋させ（使役）＋られ（被動）＋た（過去）」，將多個詞綴規則性地結合。

膠著語具有以下特徵：
- 詞綴邊界明確，容易分離
- 一個詞綴對應一個文法功能（一對一對應）
- 詞形變化規則且可預測
- 可用單一詞彙表達複雜的文法關係

## English Explanation

An agglutinative language is a language type that expresses grammatical meanings by successively attaching affixes to a stem. Japanese, Turkish, and Finnish are typical examples of agglutinative languages.

Each affix has clear boundaries and serves a single grammatical function. For example, the Japanese verb "taberu" (to eat) can be combined as "tabe (stem) + sase (causative) + rare (passive) + ta (past)," regularly joining multiple affixes.

Agglutinative languages have the following characteristics:
- Affixes have clear boundaries and are easily separable
- One affix corresponds to one grammatical function (one-to-one correspondence)
- Word formation is regular and predictable
- Complex grammatical relationships can be expressed in a single word

## 例文

1. **食べさせられた** (tabesaserareta)
   - 食べ（語幹）＋させ（使役）＋られ（受身）＋た（過去）
   - 中文：被迫吃了
   - English: was made to eat

2. **行かなければならない** (ikanakereba naranai)
   - 行く（語幹）＋ない（否定）＋ば（條件）＋なる（変化）＋ない（否定）
   - 中文：必須去
   - English: must go

3. **書かされている** (kakasareteiru)
   - 書く（語幹）＋される（使役受身）＋て（接續）＋いる（進行）
   - 中文：正被迫寫
   - English: being made to write

## 用法

### 1. 動詞變化的膠著性

日語動詞透過在語幹後依序附加接辭來表達文法意義：

```
食べる（基本形）
→ 食べた（過去）
→ 食べない（否定）
→ 食べられる（可能/受身）
→ 食べさせる（使役）
→ 食べさせられる（使役受身）
```

### 2. 助詞的附加

日語助詞像是後置詞一樣附加在名詞後：

```
学校（學校）
→ 学校で（在學校）
→ 学校では（在學校的話）
→ 学校でさえ（甚至在學校）
```

### 3. 敬語的層疊

敬語接辭可以層層附加：

```
言う（說）
→ 言います（丁寧形）
→ 言われます（尊敬語）
→ 言われています（尊敬語＋進行）
```
