## 中文

**一段動詞**（日文：一段動詞、いちだんどうし）是日文動詞分類中的一個主要類別，因其活用時只涉及一個段（い段或え段）而得名。也稱為「る動詞」，因為原形都以「る」結尾。

**核心特徵**：
- 動詞原形必定以「-る」結尾
- る前面的假名必須是い段（i）或え段（e）音
- 活用時去掉「る」，直接接續詞尾
- 活用規則簡單統一，無例外變化

**分類**：
1. **上一段動詞**（かみいちだんどうし）：る前是い段音
   - 例：見る（mi-ru）、起きる（o-ki-ru）、着る（ki-ru）
2. **下一段動詞**（しもいちだんどうし）：る前是え段音
   - 例：食べる（ta-be-ru）、寝る（ne-ru）、受ける（u-ke-ru）

**活用方式**：
所有一段動詞的活用方式完全相同，以「食べる」為例：

| 活用形 | 變化 | 例子 |
|--------|------|------|
| 未然形 | る → ない | 食べない |
| 連用形 | る → ます | 食べます |
| 終止形 | 原形 | 食べる |
| 連體形 | 原形 | 食べる時 |
| 假定形 | る → れば | 食べれば |
| 命令形 | る → ろ/よ | 食べろ/食べよ |

**常見搭配與用法**：
- **ます形**：食べる → 食べます（禮貌形）
- **て形**：食べる → 食べて（連接形）
- **た形**：食べる → 食べた（過去形）
- **ない形**：食べる → 食べない（否定形）
- **可能形**：食べる → 食べられる（能力表達）
- **受身形**：食べる → 食べられる（被動語態）
- **使役形**：食べる → 食べさせる（使役語態）

**與五段動詞的區別**：
- 一段動詞：活用簡單，只在一個段上變化
- [五段動詞](020_godan_verbs.md)：活用複雜，在五個段（あ段到お段）上變化
- 判斷方法：看る前的假名是否為い段或え段

**學習要點**：
1. 記憶常見的一段動詞（約 200-300 個）
2. 掌握統一的活用規則
3. 注意與五段動詞「る」結尾的區別（如「走る」是五段動詞）
4. 可能形和受身形同形（食べられる）需依語境判斷

## English

**Ichidan Verbs** (Japanese: 一段動詞, ichidan dōshi) are one of the main verb categories in Japanese, named because their conjugation involves only one vowel row (either the "i" or "e" row). Also called "ru-verbs" because they all end in る (ru) in their dictionary form.

**Core Characteristics**:
- Dictionary form always ends with る (ru)
- The syllable before る must be from the i-row or e-row
- Conjugation removes る and adds endings directly
- Unified conjugation pattern with no irregular changes

**Classification**:
1. **Kami-ichidan** (上一段): i-row before る
   - Examples: 見る (mi-ru, to see), 起きる (o-ki-ru, to wake up), 着る (ki-ru, to wear)
2. **Shimo-ichidan** (下一段): e-row before る
   - Examples: 食べる (ta-be-ru, to eat), 寝る (ne-ru, to sleep), 受ける (u-ke-ru, to receive)

**Conjugation Pattern**:
All ichidan verbs conjugate identically. Using 食べる (taberu) as example:

| Form | Change | Example |
|------|--------|---------|
| Negative stem | る → ない | 食べない (tabenai) |
| Masu-stem | る → ます | 食べます (tabemasu) |
| Dictionary | Original | 食べる (taberu) |
| Attributive | Original | 食べる時 (taberu toki) |
| Conditional | る → れば | 食べれば (tabereba) |
| Imperative | る → ろ/よ | 食べろ/食べよ (tabero/tabeyo) |

**Common Conjugations**:
- **Polite form**: 食べる → 食べます (tabemasu)
- **Te-form**: 食べる → 食べて (tabete)
- **Past form**: 食べる → 食べた (tabeta)
- **Negative form**: 食べる → 食べない (tabenai)
- **Potential**: 食べる → 食べられる (taberareru)
- **Passive**: 食べる → 食べられる (taberareru)
- **Causative**: 食べる → 食べさせる (tabesaseru)

**Distinction from Godan Verbs**:
- Ichidan verbs: Simple conjugation on one vowel row
- [Godan verbs](020_godan_verbs.md): Complex conjugation across five vowel rows
- Identification: Check if syllable before る is i-row or e-row

**Learning Points**:
1. Memorize common ichidan verbs (about 200-300)
2. Master the unified conjugation rules
3. Distinguish from godan verbs ending in る (e.g., 走る is godan)
4. Potential and passive forms are identical; context determines meaning

## 日本語

**一段動詞**（いちだんどうし）は、日本語の動詞分類における主要なカテゴリーの一つで、活用の際に一つの段（い段またはえ段）のみを使用することから名付けられた。辞書形がすべて「る」で終わるため、「る動詞」とも呼ばれる。

**基本的特徴**：
- 辞書形は必ず「る」で終わる
- 「る」の前の仮名は必ずい段（i）またはえ段（e）の音
- 活用時は「る」を取り除き、語尾を直接付ける
- 活用規則が統一的で、例外的な変化がない

**分類**：
1. **上一段動詞**（かみいちだんどうし）：「る」の前がい段音
   - 例：見る（mi-ru）、起きる（o-ki-ru）、着る（ki-ru）
2. **下一段動詞**（しもいちだんどうし）：「る」の前がえ段音
   - 例：食べる（ta-be-ru）、寝る（ne-ru）、受ける（u-ke-ru）

**活用パターン**：
すべての一段動詞は同じように活用する。「食べる」を例にすると：

| 活用形 | 変化 | 例 |
|--------|------|-----|
| 未然形 | る → ない | 食べない |
| 連用形 | る → ます | 食べます |
| 終止形 | 原形 | 食べる |
| 連体形 | 原形 | 食べる時 |
| 仮定形 | る → れば | 食べれば |
| 命令形 | る → ろ/よ | 食べろ/食べよ |

**よく使われる活用形**：
- **ます形**：食べる → 食べます（丁寧形）
- **て形**：食べる → 食べて（接続形）
- **た形**：食べる → 食べた（過去形）
- **ない形**：食べる → 食べない（否定形）
- **可能形**：食べる → 食べられる（能力表現）
- **受身形**：食べる → 食べられる（受動態）
- **使役形**：食べる → 食べさせる（使役態）

**五段動詞との違い**：
- 一段動詞：活用が簡単で、一つの段で変化する
- [五段動詞](020_godan_verbs.md)：活用が複雜で、五つの段（あ段からお段）で変化する
- 見分け方：「る」の前の仮名がい段またはえ段かどうか

**学習のポイント**：
1. よく使われる一段動詞（約200〜300語）を覚える
2. 統一的な活用規則を習得する
3. 「る」で終わる五段動詞との違いに注意（例：「走る」は五段動詞）
4. 可能形と受身形が同形（食べられる）なので、文脈で判断する

## 例文

### 例文 1: 上一段動詞の基本活用

**日本語**：毎朝六時に起きます。

**読み方**：まいあさろくじにおきます

**中文翻譯**：每天早上六點起床。

**English Translation**: I wake up at six o'clock every morning.

**文法説明**：
- 「起きる」（上一段動詞）的ます形活用
- 表示習慣性動作
- 基本句型：時間 + に + 動詞

---

### 例文 2: 下一段動詞的基本活用

**日本語**：昼ご飯を食べています。

**読み方**：ひるごはんをたべています

**中文翻譯**：正在吃午餐。

**English Translation**: I am eating lunch.

**文法説明**：
- 「食べる」（下一段動詞）的て形 + いる
- 表示進行中的動作
- 基本句型：を + 動詞ている

---

### 例文 3: 否定形的使用

**日本語**：今日はテレビを見ません。

**読み方**：きょうはテレビをみません

**中文翻譯**：今天不看電視。

**English Translation**: I won't watch TV today.

**文法説明**：
- 「見る」（上一段動詞）的否定形：見ない → 見ません
- 一段動詞否定形：去掉る + ない
- 禮貌形：ない → ません

---

### 例文 4: 過去形的使用

**日本語**：昨夜は早く寝ました。

**読み方**：さくやははやくねました

**中文翻譯**：昨晚早早就睡了。

**English Translation**: I went to bed early last night.

**文法説明**：
- 「寝る」（下一段動詞）的過去形：寝た → 寝ました
- 一段動詞過去形：去掉る + た
- 副詞「早く」修飾動詞

---

### 例文 5: 可能形與受身形

**日本語**：この料理は誰にでも食べられます。

**読み方**：このりょうりはだれにでもたべられます

**中文翻譯**：這道菜任何人都能吃。

**English Translation**: Anyone can eat this dish.

**文法説明**：
- 「食べる」的可能形：食べられる
- 此處表示「能夠吃」（可能）而非「被吃」（受身）
- 一段動詞可能形：去掉る + られる
- に + でも：表示「任何...都」

---

### 例文 6: 命令形的使用

**日本語**：早く起きなさい！

**読み方**：はやくおきなさい

**中文翻譯**：快點起床！

**English Translation**: Wake up quickly!

**文法説明**：
- 「起きる」的命令形柔和版：起きなさい
- 一段動詞命令形：去掉る + ろ（強硬）或 なさい（溫和）
- 「早く」副詞表示「快點」

---

### 例文 7: 使役形的使用

**日本語**：子供に野菜を食べさせます。

**読み方**：こどもにやさいをたべさせます

**中文翻譯**：讓孩子吃蔬菜。

**English Translation**: I make my child eat vegetables.

**文法説明**：
- 「食べる」的使役形：食べさせる
- 一段動詞使役形：去掉る + させる
- 基本句型：に + を + 使役動詞

---

### 例文 8: 條件形的使用

**日本語**：時間があれば、映画を見ます。

**読み方**：じかんがあれば、えいがをみます

**中文翻譯**：如果有時間的話，會去看電影。

**English Translation**: If I have time, I will watch a movie.

**文法説明**：
- 「見る」的條件形：見れば
- 一段動詞條件形：去掉る + れば
- 表示假定條件「如果...的話」

---

### 例文 9: 受身形的使用

**日本語**：先生に褒められました。

**読み方**：せんせいにほめられました

**中文翻譯**：被老師稱讚了。

**English Translation**: I was praised by my teacher.

**文法説明**：
- 「褒める」的受身形：褒められる
- 一段動詞受身形：去掉る + られる（與可能形同形）
- 此處依語境判斷為受身（被動）
- に：表示動作的執行者

---

### 例文 10: 意志形的使用

**日本語**：一緒に晩ご飯を食べよう。

**読み方**：いっしょにばんごはんをたべよう

**中文翻譯**：一起吃晚餐吧。

**English Translation**: Let's eat dinner together.

**文法説明**：
- 「食べる」的意志形：食べよう
- 一段動詞意志形：去掉る + よう
- 表示邀請或建議「一起...吧」
- 「一緒に」表示「一起」
