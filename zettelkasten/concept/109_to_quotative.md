## 日文
引用助詞と（いんよう じょし と）

**羅馬拼音**：inyou joshi to

**詞性**：助詞（格助詞）

**相關詞彙**：
- って - 口語引用助詞（と的口語形式）
- という - 稱為、叫做
- と思う - 認為、覺得
- と言う - 說

## 日文解釋
「と」は引用助詞として、他者の発言や思考、名称などを引用する際に使用されます。「言う」「思う」「聞く」「書く」などの動詞と共に用いられ、直接引用（そのままの言葉）と間接引用（内容を要約したもの）の両方に使えます。直接引用の場合は鉤括弧「」で囲み、間接引用の場合は鉤括弧を使わないことが多いです。話し言葉では「って」に短縮されることが一般的です。

## 英文解釋
The particle "to" functions as a quotative particle in Japanese, used to quote someone's speech, thoughts, names, or titles. It commonly pairs with verbs like "iu" (to say), "omou" (to think), "kiku" (to hear), and "kaku" (to write). It can be used for both direct quotes (exact words, often marked with quotation marks「」) and indirect quotes (paraphrased content). In casual spoken Japanese, "to" is often shortened to "tte."

## 中文解釋
「と」作為引用助詞，用於引用他人的話語、思想、名稱等內容。常與「言う」（說）、「思う」（想）、「聞く」（聽）、「書く」（寫）等動詞搭配使用。可用於直接引用（原話，通常用「」標記）和間接引用（概述內容）。在口語中，「と」經常被縮短為「って」。這個助詞是日語中表達傳聞、轉述和思想的基本語法結構。

## 例句

**例句1：直接引用**
```
先生は「明日テストがあります」と言いました。
The teacher said, "There will be a test tomorrow."
老師說：「明天有考試。」
```

**例句2：間接引用（思考）**
```
彼は来ないと思います。
I think he won't come.
我覺得他不會來。
```

**例句3：名稱引用**
```
この花は「桜」と呼ばれています。
This flower is called "sakura."
這種花被稱為「櫻花」。
```

**例句4：傳聞**
```
ニュースによると、明日は雨だと言っていました。
According to the news, they said it will rain tomorrow.
根據新聞報導，據說明天會下雨。
```

## 補充說明

### 常用句型
1. **〜と言う** - 說〜
2. **〜と思う** - 認為〜、覺得〜
3. **〜と聞く** - 聽說〜
4. **〜という** - 叫做〜、稱為〜
5. **〜とのことだ** - 據說〜（較正式）

### 「と」vs「って」的使用場合
| 場合 | と | って |
|------|----|----|
| 正式場合 | ◎ | △ |
| 日常會話 | ○ | ◎ |
| 書面語 | ◎ | × |
| 友人之間 | ○ | ◎ |

### 注意事項
- 直接引用時，引號內的內容保持原樣，不改變敬語等級
- 間接引用時，需要將引用內容轉換為適當的形式
- 「と」後面直接接動詞，不需要其他助詞

### 相關語法
- **ように言う** - 告訴某人做某事（指示、命令的傳達）
- **かどうか** - 是否（用於疑問句的間接引用）
- **なんて** - 表示驚訝或輕視的引用
