## 日文

工具格（どうぐかく）

### 概念情報

| 項目 | 內容 |
|------|------|
| 日文名稱 | 工具格 |
| 讀音 | どうぐかく |
| 英文 | Instrumental Case |
| 領域 | 語言學、格語法 |
| 日語對應 | で格（手段用法） |

## 日文解釋

工具格とは、動作を行う際に用いる道具、手段、方法を表す文法格である。

日本語では主に格助詞「で」がこの機能を担う。例えば「ペンで書く」「電車で行く」「日本語で話す」などの「で」は工具格の用法であり、動作の手段・方法を示している。

工具格は言語によって表現方法が異なり、ロシア語やラテン語などでは専用の格語尾を持つが、日本語や英語では前置詞・後置詞（助詞）を用いて表現する。日本語の「で」は工具格以外にも場所格（動作の場所）や原因格（理由）など多機能を持つ点が特徴的である。

格文法理論において、工具格は「動作主が目的を達成するために使用する実体」という意味役割を担い、動作の実現に必要な媒介を示す重要な格である。

## 英文解釋

The instrumental case is a grammatical case that marks the means, tool, or method used to perform an action.

In Japanese, this function is primarily fulfilled by the case particle で (de). For example, in expressions like ペンで書く (write with a pen), 電車で行く (go by train), and 日本語で話す (speak in Japanese), the particle で indicates the instrumental case, showing the means or method of action.

The instrumental case is expressed differently across languages. While Russian, Latin, and other inflected languages have dedicated case endings for the instrumental, Japanese and English use postpositions (particles) or prepositions. A notable feature of Japanese で is its multifunctionality, serving not only as the instrumental marker but also for locative (place of action) and causal (reason) meanings.

In case grammar theory, the instrumental case carries the semantic role of "an entity used by the agent to achieve a purpose," representing an important case that indicates the medium necessary for action realization.

## 中文解釋

工具格是語言學中標記動作執行時所使用的手段、工具或方法的語法格。

在日語中，這項功能主要由格助詞「で」承擔。例如「ペンで書く」（用筆寫）、「電車で行く」（搭電車去）、「日本語で話す」（用日語說）中的「で」即為工具格用法，表示動作的手段或方法。

工具格在不同語言中有不同的表達方式。俄語、拉丁語等屈折語有專門的工具格詞尾，而日語和英語則使用後置詞（助詞）或前置詞來表達。日語「で」的特點在於其多功能性，除工具格外，還可表示場所格（動作地點）和原因格（理由）等。

在格語法理論中，工具格承擔「施事者為達成目的所使用的實體」的語意角色，是表示動作實現所需媒介的重要格。

## 核心用法

### 用法 1：物理工具

使用具體的器具進行動作。

**例句 1**
```
箸で食べる
Eat with chopsticks
用筷子吃
```

**例句 2**
```
包丁で野菜を切る
Cut vegetables with a knife
用菜刀切蔬菜
```

### 用法 2：交通手段

表示移動時使用的交通工具。

**例句 3**
```
新幹線で東京に行く
Go to Tokyo by Shinkansen
搭新幹線去東京
```

**例句 4**
```
自転車で通勤する
Commute by bicycle
騎腳踏車通勤
```

### 用法 3：語言・媒介

表示溝通或表達時使用的語言或媒介。

**例句 5**
```
日本語で論文を書く
Write a thesis in Japanese
用日語寫論文
```

**例句 6**
```
メールで連絡する
Contact by email
用電子郵件聯絡
```

## 理論背景

### 格語法中的定位

在查爾斯·菲爾莫爾（Charles Fillmore）的格語法理論中，工具格是基本格之一，與施事格（Agent）、受事格（Patient）等並列。工具格的語意定義為「動作主用來執行動作的無生命力量或物體」。

### 日語中的多義性

日語的「で」助詞具有顯著的多義性：

| 用法 | 例句 | 語意功能 |
|------|------|----------|
| 工具格 | ペンで書く | 手段・工具 |
| 場所格 | 図書館で勉強する | 動作場所 |
| 原因格 | 病気で休む | 原因・理由 |
| 範圍格 | 日本で一番 | 限定範圍 |

### 類型學視角

從語言類型學來看，工具格的表達方式可分為：
- **形態標記型**：俄語、拉丁語等使用專門的格詞尾
- **後置詞型**：日語使用助詞「で」
- **前置詞型**：英語使用 "with", "by" 等前置詞

## 學習要點

1. **核心功能**：工具格標記動作的手段、工具或方法
2. **日語對應**：主要對應「で」格的手段用法
3. **語意角色**：在格語法中表示「被使用的實體」
4. **多義區分**：需區分「で」的工具格與其他用法（場所、原因等）
