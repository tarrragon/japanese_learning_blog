助詞「を」是日文格助詞系統中標記動作對象的核心助詞，是初學者必須掌握的基礎文法要素。

<!--more-->

## 日文

助詞を（じょしを）

### 概念定義

「を」是格助詞，主要功能是標記他動詞的直接受詞，表示動作的對象。此外，「を」也可以標記移動動詞經過或離開的場所。發音上，雖然寫作「を」（wo），但現代日語中通常發音為「o」。

## 日文解釋

「を」は格助詞の一つで、主に他動詞の直接目的語（動作の対象）を示す。また、移動動詞と共に使われる場合、移動の経路や出発点を表すこともできる。

例えば「本を読む」では「本」が動作「読む」の対象であり、「公園を歩く」では「公園」が移動の場所を示している。現代日本語では「を」は「お」と同じ発音になることが多いが、助詞として使う場合は「を」と書く。

日本語学習において、「を」の正しい使い方を習得することは文法の基礎として極めて重要である。特に、自動詞と他動詞の区別、そして「を」と「が」の使い分けを理解することが求められる。

## 英文解釋

The particle を (wo/o) is a case particle primarily used to mark the direct object of transitive verbs, indicating what receives the action. Additionally, when used with verbs of motion, it can indicate the route through which movement occurs or the point of departure.

For example, in "本を読む" (hon wo yomu - read a book), "本" is the object of the reading action. In "公園を歩く" (kouen wo aruku - walk through the park), "公園" indicates the path of movement. Although written as を, in modern Japanese it is typically pronounced as "o" rather than "wo."

Understanding the proper use of を is fundamental to Japanese grammar, particularly distinguishing between transitive and intransitive verbs, and knowing when to use を versus other particles like が or に.

## 中文解釋

「を」是格助詞之一，主要用於標記他動詞的直接受詞，表示動作的對象。此外，與移動動詞搭配時，也可以表示移動經過的路徑或離開的地點。

例如「本を読む」（讀書）中，「本」是「読む」動作的對象；「公園を歩く」（穿過公園走）中，「公園」則表示移動的場所。雖然書寫時用「を」，但現代日語發音通常與「お」相同。

學習日語時，正確掌握「を」的用法是文法基礎的關鍵，特別需要理解自動詞與他動詞的區別，以及「を」與「が」的使用差異。

## 核心用法

### 用法 1：標記直接受詞（他動詞の目的語）

最基本的用法，標記他動詞動作所作用的對象。

**例句 1**
```
りんごを食べる。
To eat an apple.
吃蘋果。
```

**例句 2**
```
日本語を勉強しています。
I am studying Japanese.
正在學日語。
```

### 用法 2：標記移動經過的場所（移動の経路）

與移動動詞搭配，表示動作經過的路徑或空間。

**例句 3**
```
橋を渡る。
To cross the bridge.
過橋。
```

**例句 4**
```
空を飛ぶ。
To fly through the sky.
在天空飛翔。
```

### 用法 3：標記離開的場所（出発点・離脱点）

與表示離開、出發的動詞搭配，表示離開的地點。

**例句 5**
```
家を出る。
To leave home.
出門／離開家。
```

**例句 6**
```
電車を降りる。
To get off the train.
下電車。
```

## 常見錯誤

### 錯誤 1：自動詞誤用「を」

❌ 窓を開いている（錯誤）
✅ 窓が開いている（正確）

解說：「開いている」是自動詞，表示狀態，應使用「が」而非「を」。

### 錯誤 2：與「が」混淆

❌ 日本語が勉強する（錯誤）
✅ 日本語を勉強する（正確）

解說：「勉強する」是他動詞，需要「を」標記受詞。

### 錯誤 3：移動動詞的場所標記

注意區分：
- 公園**を**歩く（穿過公園走）- 強調經過的路徑
- 公園**で**歩く（在公園裡走）- 強調動作發生的場所

## 學習要點

1. **他動詞判斷**：能接「を」的動詞通常是他動詞，需要有明確的動作對象。

2. **口語省略**：在日常口語中，「を」常被省略，如「ご飯（を）食べる」。

3. **書寫與發音**：雖然發音為「o」，但書寫時必須用「を」，這是日語正字法的規定。

4. **與其他助詞對比**：理解「を」與「が」「に」「で」的區別是掌握日語助詞系統的關鍵。
