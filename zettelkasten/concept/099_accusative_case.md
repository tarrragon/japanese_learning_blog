## 日文
賓格（ひんかく）/ 対格（たいかく）

**羅馬拼音**：hinkaku / taikaku

**詞性**：言語学用語、文法概念

**相關詞彙**：
- 目的語（もくてきご）- 受詞
- 格助詞（かくじょし）- 格助詞
- 他動詞（たどうし）- 他動詞

## 日文解釋
賓格（対格）は、文法用語の一つで、動詞が直接作用する対象（目的語）を示す格のことです。日本語では主に格助詞「を」がこの機能を担い、動作の対象を明示します。例えば「りんごを食べる」という文では、「りんご」が賓格（を格）で標記されており、「食べる」という動作の直接的な対象であることを示しています。英語やドイツ語などの言語では、名詞や代名詞の形態変化（格変化）によって賓格が表されますが、日本語は膠着語として助詞を用いて格を表すため、語順の自由度が高くなっています。この概念は言語学や文法研究において、文の構造を分析する際の基本的な枠組みとなっています。

## 英文解釋
The accusative case is a grammatical case that marks the direct object of a transitive verb - the entity that directly receives the action. In Japanese, this function is primarily served by the particle "wo" (を), which indicates the target of an action. For example, in "ringo wo taberu" (eat an apple), "ringo" is marked with the accusative particle を, showing it is the direct object of "taberu." Unlike languages such as German or Latin where nouns change form to show case, Japanese uses particles as an agglutinative language, allowing greater flexibility in word order. The accusative case is a fundamental concept in linguistics for analyzing sentence structure and understanding how verbs relate to their arguments.

## 中文解釋
賓格（英語：accusative case）是語法學術語，指標記動詞直接受詞的格位，即動作直接作用的對象。在日文中，這個功能主要由格助詞「を」來承擔，用於明確標示動作的對象。例如在「りんごを食べる」（吃蘋果）這個句子中，「りんご」被標記為賓格（を格），表示它是「食べる」（吃）這個動作的直接對象。與德語、拉丁語等通過名詞變形來表示格位的語言不同，日文作為黏著語使用助詞來標記格位，因此語序較為自由。賓格是語言學中分析句子結構的基本概念，有助於理解動詞與其論元之間的關係。

## 例句

**例句1**
```
本を読む。
I read a book.
讀書。
```

**例句2**
```
日本語を勉強しています。
I am studying Japanese.
我正在學習日文。
```

**例句3**
```
彼女は毎朝コーヒーを飲みます。
She drinks coffee every morning.
她每天早上喝咖啡。
```

## 補充說明

### 日文中的賓格標記
在日文中，賓格主要通過以下方式表現：
1. **を格** - 最典型的賓格標記，用於他動詞的直接受詞
2. **移動動詞的經過點** - 如「公園を歩く」（走過公園），表示移動經過的場所

### 與其他格的比較
| 格 | 助詞 | 功能 |
|---|---|---|
| 主格 | が | 標記主語 |
| 賓格 | を | 標記直接受詞 |
| 與格 | に | 標記間接受詞 |
| 屬格 | の | 標記所有關係 |

### 跨語言比較
- **德語**：Akkusativ，名詞和冠詞會變形
- **拉丁語**：名詞詞尾變化表示賓格
- **英語**：主要在代名詞體現（I→me, he→him）
- **日語**：使用助詞「を」標記，名詞本身不變形
