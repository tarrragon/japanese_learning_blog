# 社會語境

## 中文解釋

社會語境是指影響語言使用的社會因素，包括說話者與聽者的關係、場合、社會地位差異等。在日語中，社會語境直接影響敬語的選擇，是理解日本語言文化不可或缺的概念。

主要考量因素：
- **人際關係**：上下關係、內外關係、親疏關係
- **場合性質**：正式場合、非正式場合
- **社會距離**：心理距離、地位差距

## English Explanation

Social context refers to social factors that influence language use, including the relationship between speaker and listener, situation, social status differences, etc. In Japanese, social context directly affects keigo (honorific) selection and is an essential concept for understanding Japanese language and culture.

Main considerations:
- **Interpersonal relationships**: Hierarchical, in-group/out-group, intimacy levels
- **Situational formality**: Formal vs. informal settings
- **Social distance**: Psychological distance, status gaps

## 日本語の解釈

社会的文脈とは、話し手と聞き手の関係、場面、社会的地位の違いなど、言語使用に影響を与える社会的要因を指す。日本語では、社会的文脈が敬語の選択に直接影響し、日本の言語文化を理解する上で欠かせない概念である。

主な考慮要因：
- **人間関係**：上下関係、内外関係、親疎関係
- **場面性**：フォーマルな場面、インフォーマルな場面
- **社会的距離**：心理的距離、地位の格差

## 核心概念

日語敬語系統的複雜性源於對社會語境的精細感知。說話者需要同時考慮多個社會維度：

1. **垂直關係**（上下）：年齡、職位、經驗差距
2. **水平關係**（內外）：同組織成員 vs. 外部人員
3. **心理距離**（親疏）：熟悉程度、情感親密度

這三個維度交互作用，決定適當的敬語形式。

## 使用情境

### 商務場合
```
（對上司）
お疲れ様です。明日の会議の資料を確認していただけますでしょうか。
```

### 日常對話
```
（對朋友）
疲れたね。明日の会議の資料、見てくれる？
```

### 跨組織溝通
```
（對客戶）
いつもお世話になっております。資料をご確認いただけますと幸いです。
```

## 關鍵要素

### 判斷流程

1. **識別關係類型**
   - 是否有上下關係？
   - 對方是內部還是外部人員？
   - 心理距離如何？

2. **評估場合正式度**
   - 商務會議 → 高度正式
   - 部門聚餐 → 中度正式
   - 私人聚會 → 低度正式

3. **選擇適當敬語層級**
   - 尊敬語：提升對方地位
   - 謙讓語：降低自己地位
   - 丁寧語：基本禮貌表達

## 學習要點

1. **動態調整**：社會語境不是靜態的，需要隨對話進展調整
2. **文化敏感性**：理解日本社會的階層觀念和集團主義
3. **過度禮貌風險**：過度使用敬語可能造成疏離感
4. **不足禮貌風險**：敬語不足可能被視為失禮
