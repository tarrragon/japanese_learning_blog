## 日文解釋

「指数」は、ある基準となる数値を100などに設定し、それとの比較で現在の状況や変化の程度を示す数値のことである。

経済学や統計学において広く使用され、物価の変動、株式市場の動向、経済成長率など、様々な社会経済的な変化を数量的に把握するために用いられる。日本では「消費者物価指数」「日経平均株価指数」「景気動向指数」などが一般的に知られている。

また、数学においては累乗を表す数（べき指数）としても使われ、例えば2³における「3」が指数となる。このように「指数」という語は文脈によって意味が異なることに注意が必要である。

## 英文解釋

"Shisuu" (指数) refers to an index or indicator - a numerical measure that represents the relative change or ratio compared to a baseline value.

In economics and statistics, indices are used to track changes in prices, market performance, economic conditions, and other measurable phenomena. Common examples include the Consumer Price Index (CPI), stock market indices, and economic growth indicators.

In mathematics, "shisuu" also refers to an exponent - the number indicating how many times a base number is multiplied by itself (e.g., in 2³, the "3" is the exponent).

## 中文解釋

「指数」是一個表示相對變化或比率的數值概念，通常以某個基準值（如100）作為參照點。

在經濟學和統計學領域，指數被廣泛用於追蹤物價變動、市場表現、經濟狀況等可量化現象的變化。常見的例子包括消費者物價指數（CPI）、股票市場指數、經濟成長指標等。

在數學中，「指数」也指冪次方中的指數部分，表示底數自乘的次數（如 2³ 中的「3」就是指數）。

## 核心用法

### 用法 1：經濟・統計指標

表示經濟或統計數據的相對變化。

**例句 1**
```
今月の消費者物価指数は前年同月比で2.5%上昇した。
The Consumer Price Index rose 2.5% year-on-year this month.
本月的消費者物價指數較去年同期上漲了2.5%。
```

**例句 2**
```
日経平均株価指数が3万円を突破したのは歴史的な出来事だ。
It is a historic event that the Nikkei Stock Average Index broke through 30,000 yen.
日經平均股價指數突破3萬日圓是歷史性的事件。
```

### 用法 2：數學中的指數（冪次方）

表示數學運算中的指數概念。

**例句 3**
```
指数関数は、底が定数で、指数が変数となる関数のことです。
An exponential function is a function where the base is constant and the exponent is a variable.
指數函數是指底數為常數、指數為變數的函數。
```

### 用法 3：抽象的指標・水準

表示某種抽象的水準或程度。

**例句 4**
```
この地域の幸福度指数は全国平均を上回っている。
The happiness index of this region exceeds the national average.
這個地區的幸福度指數高於全國平均水準。
```

**例句 5**
```
紫外線指数が高い日は日焼け止めを必ず塗ってください。
Please make sure to apply sunscreen on days when the UV index is high.
紫外線指數高的日子請務必塗抹防曬乳。
```

## 常見錯誤

### 錯誤 1：與「指標」混淆

❌ 誤：この数値は景気の**指数**として使われています。（語意模糊）
✅ 正：この数値は景気の**指標**として使われています。
說明：「指標」是較廣義的概念，表示用來判斷某事物的標準或基準；「指数」則特指以數值形式表示的相對變化量。

### 錯誤 2：讀音錯誤

❌ 誤：「しじすう」或「さしすう」
✅ 正：「しすう」
說明：正確讀音為「しすう」，「指」讀作「し」，不讀作「さし」或「しじ」。

## 學習要點

1. **雙重含義**：記住「指数」在經濟/統計和數學中有不同的含義
2. **常見搭配**：熟記「消費者物価指数」「株価指数」等常用複合詞
3. **與「指標」的區別**：「指数」是具體的數值，「指標」是較抽象的標準
4. **讀音注意**：確認正確讀音為「しすう」
5. **使用場景**：多用於正式、專業的經濟或學術語境中
