## 日文解釋

「なる」は日本語において最も基本的かつ重要な変化を表す動詞の一つである。漢字では「成る」（完成・達成の意）または「為る」（変化・転換の意）と書き、状態や性質の変化、ある結果への到達を表現する。[^dict]

この動詞の本質的な特徴は、主語が自発的な変化を遂げるという点にある。英語の "become" や中国語の「變成」とは異なり、「なる」は変化の過程そのものよりも、**変化後の結果状態**に焦点を当てる傾向がある。

「なる」は自動詞であり、変化の主体（主語）が自然に、あるいは何らかの要因によって新たな状態に移行することを示す。他動詞「する」との対比で理解すると、「なる」は「外部からの働きかけなしに変化が起こる」というニュアンスを持つ。

使用場面は極めて広範で、日常会話から書き言葉まで、あらゆる文脈で用いられる。特に「〜になる」「〜くなる」「〜ようになる」などの文型は、日本語学習において最も早い段階で習得すべき基本表現である。

[^dict]: 定義參考 スーパー大辞林（macOS 內建字典）

## 英文解釋

"Naru" (成る/為る) is one of the most fundamental verbs in Japanese, expressing the concept of "becoming" or "turning into." Unlike the English "become," which often emphasizes the process of change, "naru" tends to focus on the **resultant state** after the transformation has occurred.

As an intransitive verb, "naru" indicates that the subject undergoes a change without direct external action. This contrasts with its transitive counterpart "suru" (to do/make), which implies deliberate action. The particle "ni" typically marks the result or new state that the subject becomes.

Key grammatical patterns include:
- **Noun + ni naru**: Becoming a noun (e.g., 医者になる - to become a doctor)
- **i-adjective stem + ku naru**: Becoming an adjective state (e.g., 大きくなる - to become big)
- **na-adjective + ni naru**: Similar pattern for na-adjectives (e.g., 静かになる - to become quiet)
- **Verb dictionary form + you ni naru**: To come to be able to / to start doing (e.g., 話せるようになる - to come to be able to speak)

## 中文解釋

「なる」是日文中表達「變成、成為」概念的核心動詞，屬於五段動詞，在日常會話和書面語中都極為常用。

與中文的「變成」不同，日文的「なる」更強調**變化後的結果狀態**，而非變化的過程本身。作為自動詞，「なる」表示主語自然地或因某種因素而轉變為新的狀態，不需要外力的直接作用。

基本句型結構：
- **名詞 + になる**：成為某種身分或狀態（如：先生になる - 成為老師）
- **い形容詞詞幹 + くなる**：變得如何（如：暖かくなる - 變暖）
- **な形容詞 + になる**：變得如何（如：元気になる - 變精神）
- **動詞辭書形 + ようになる**：開始能夠...（如：日本語が話せるようになる - 變得會說日語）

理解「なる」與「する」的對立關係是掌握日文動詞體系的關鍵。「なる」表示自發性變化（自動詞），而「する」表示人為造成的變化（他動詞）。這種自他動詞的對立是日文語法的重要特徵。

## 核心用法

### 用法 1：名詞 + になる（成為某種身分或狀態）

表示主語轉變為某種職業、身分、狀態或性質。這是「なる」最基本的用法，助詞「に」標示變化的結果。

**例句 1**
```
彼女は来年、弁護士になります。
She will become a lawyer next year.
她明年會成為律師。
```

**例句 2**
```
春になると、桜が咲きます。
When it becomes spring, the cherry blossoms bloom.
一到春天，櫻花就會開。
```

### 用法 2：い形容詞詞幹 + くなる（變得...）

い形容詞去掉「い」，加上「く」後接「なる」，表示性質或狀態的變化。

**例句 3**
```
最近、日が短くなってきましたね。
The days have been getting shorter recently, haven't they?
最近白天變短了呢。
```

### 用法 3：な形容詞 + になる（變得...）

な形容詞直接加上「に」後接「なる」，表示性質或狀態的變化。

**例句 4**
```
息子がやっと野菜を食べられるようになりました。
My son has finally become able to eat vegetables.
兒子終於變得能吃蔬菜了。
```

### 用法 4：〜ようになる（開始能夠、變得會...）

動詞可能形或辭書形加上「ようになる」，表示能力的獲得或習慣的形成。

**例句 5**
```
毎日練習したら、漢字が読めるようになりました。
After practicing every day, I became able to read kanji.
每天練習之後，我變得會讀漢字了。
```

## 常見錯誤

### 錯誤 1：與「する」混淆

❌ 誤：部屋を静かにしてください。（請讓房間安靜）→ 這是「する」的用法
✅ 正：部屋が静かになりました。（房間變安靜了）
說明：「なる」是自動詞，表示自然發生的變化；「する」是他動詞，表示人為造成的變化。選擇哪個動詞取決於是否強調外力作用。

### 錯誤 2：い形容詞接續錯誤

❌ 誤：寒いになりました。
✅ 正：寒くなりました。
說明：い形容詞必須變成「く形」（去掉い，加く）才能接「なる」。

### 錯誤 3：助詞使用錯誤

❌ 誤：医者をなりたいです。
✅ 正：医者になりたいです。
說明：「なる」前面使用助詞「に」表示變化的結果，不能用「を」。

## 學習要點

1. **自他動詞對立**：「なる」（自動詞）與「する」（他動詞）形成對立，理解這個對比是掌握日文動詞體系的基礎。

2. **助詞「に」的重要性**：「に」標示變化的結果狀態，是「なる」最重要的搭配助詞。

3. **形容詞接續規則**：い形容詞用「〜くなる」，な形容詞用「〜になる」，這是常考的文法重點。

4. **「〜ようになる」的習慣用法**：表示能力獲得或習慣形成，是 N4-N3 的重要句型。

5. **與時間表達的搭配**：「なる」常與季節、時間等名詞搭配，如「春になる」「夜になる」。
