## 日文

お〜/ご〜（お〜/ご〜）

**羅馬拼音**: o~/go~

**詞性**: 接頭詞（敬語接頭詞）

日本語で最も基本的で広く使われる敬語接頭詞です。名詞や動詞に付けることで、敬意や丁寧さを表現します。「お」は和語に、「ご」は漢語に付けるのが一般的な原則で、[敬語システム](../grammar/005_keigo_system.md)の基礎をなす重要な要素です。

### 接頭詞情報

| 項目 | 内容 |
|------|------|
| 種類 | 敬語接頭詞 |
| JLPT | N5 |
| 使用場面 | 正式、丁寧、敬語表現 |
| 付加対象 | 名詞、動詞語幹 |

### 基本規則

**和語（お）**
```
茶 → お茶
金 → お金
名前 → お名前
```

**漢語（ご）**
```
飯 → ご飯
意見 → ご意見
住所 → ご住所
```

**動詞語幹（敬語形成）**
```
読む → お読みになる（尊敬語）
送る → お送りする（謙譲語）
連絡する → ご連絡する（謙譲語）
```

## 日文解釋

「お」と「ご」は、日本語の敬語システムにおいて最も基本的かつ重要な接頭詞です。これらは二つの主要な機能を持っています。

**第一の機能：美化語としての使用**

名詞に付けることで、言葉を美しく丁寧にします。これは「美化語」と呼ばれ、相手への直接的な敬意というよりも、話し手の品格や教養を示す役割を果たします。日常会話でも頻繁に使用され、特に女性の言葉遣いでは重要な要素となっています。

例えば「お茶」「ご飯」「お水」などは、敬語というよりも日本語の自然な表現として定着しており、これらを使わないと逆に粗野な印象を与えることがあります。

**第二の機能：敬語形成の一部**

動詞の語幹に付けることで、尊敬語や謙譲語を形成します。「お〜になる」は[尊敬語の基本パターン](../grammar/025_o_go_ni_naru.md)として、「お〜する」は[謙譲語の基本パターン](../grammar/026_o_go_suru.md)として、ビジネスや正式な場面で不可欠な表現となっています。

**選択規則**

基本的に、和語由来の語には「お」を、漢語由来の語には「ご」を付けます。ただし、この規則には多くの例外があり、実際の使用は慣習に大きく依存します。

長年の言語使用の中で、「お」と「ご」の使い分けは日本語話者の言語感覚として内在化されており、適切な使い分けは日本語の洗練度を示す重要な指標となっています。

## 英文解釋

"O" and "go" are the most fundamental and important honorific prefixes in the Japanese keigo (honorific language) system. They have two primary functions.

**First Function: Beautification (Bikago)**

When attached to nouns, they make the language more refined and polite. This is called "bikago" (beautification language), which serves to demonstrate the speaker's refinement and education rather than direct respect to the listener.

Words like "ocha" (tea), "gohan" (rice/meal), and "omizu" (water) have become so naturalized that not using these prefixes can sound crude or rough. This usage is especially important in women's speech patterns.

**Second Function: Honorific Formation**

When attached to verb stems, they form respectful (sonkeigo) or humble (kenjougo) expressions. "O~ni naru" is the basic pattern for [respectful language](../grammar/025_o_go_ni_naru.md), while "o~suru" is the basic pattern for [humble language](../grammar/026_o_go_suru.md), both essential in business and formal situations.

**Selection Rule**

Generally, "o" is attached to native Japanese words (wago), while "go" is attached to Sino-Japanese words (kango). However, there are many exceptions to this rule, and actual usage depends heavily on convention.

Through years of language use, the distinction between "o" and "go" has become internalized in Japanese speakers' linguistic intuition, and proper usage is an important indicator of Japanese language proficiency.

## 中文解釋

「お」和「ご」是日語敬語系統中最基本且最重要的敬語接頭詞。它們有兩個主要功能。

**第一個功能：美化語**

附加在名詞前，使語言更加優雅和禮貌。這被稱為「美化語」，其作用不是直接表達對聽者的敬意，而是展現說話者的修養和教育程度。

像「お茶」（茶）、「ご飯」（飯）、「お水」（水）等詞彙已經自然化到不使用這些接頭詞會顯得粗俗。這種用法在女性用語中尤為重要。

**第二個功能：敬語形成**

附加在動詞語幹上，形成尊敬語或謙讓語。「お〜になる」是[尊敬語的基本模式](../grammar/025_o_go_ni_naru.md)，而「お〜する」是[謙讓語的基本模式](../grammar/026_o_go_suru.md)，兩者在商務和正式場合都不可或缺。

**選擇規則**

一般來說，「お」附加在和語（日本原有詞彙）上，而「ご」附加在漢語（漢字詞彙）上。但是，這個規則有許多例外，實際使用很大程度上取決於習慣。

經過多年的語言使用，「お」和「ご」的區分已經內化為日語使用者的語言直覺，正確使用是日語精通程度的重要指標。

## 核心用法

### 用法 1：美化語（名詞修飾）

名詞に付けて言葉を丁寧にする最も基本的な用法です。

**例句 1**
```
お茶をどうぞ。
Please have some tea.
請喝茶。
```

**例句 2**
```
お名前を教えていただけますか。
Could you tell me your name?
可以告訴我您的名字嗎？
```

**例句 3**
```
ご意見をお聞かせください。
Please let me know your opinion.
請告訴我您的意見。
```

### 用法 2：尊敬語形成（お/ご〜になる）

相手や目上の人の動作を尊敬して表現します。

**例句 4**
```
社長はもうお帰りになりました。
The president has already gone home. (respectful)
社長已經回去了。（尊敬語）
```

**例句 5**
```
先生はこの本をお読みになりますか。
Will the teacher read this book?
老師會讀這本書嗎？
```

**例句 6**
```
お客様がご注文になった商品です。
This is the product the customer ordered.
這是客人訂購的商品。
```

### 用法 3：謙譲語形成（お/ご〜する）

自分の動作を控えめに表現して相手を立てます。

**例句 7**
```
資料をお送りします。
I will send you the materials. (humble)
我會把資料寄給您。（謙讓語）
```

**例句 8**
```
後ほどご連絡します。
I will contact you later. (humble)
我稍後會聯絡您。（謙讓語）
```

**例句 9**
```
詳しくご説明します。
I will explain in detail. (humble)
我會詳細說明。（謙讓語）
```

### 用法 4：固定表現

慣用的に「お」「ご」が付いた表現。

**例句 10**
```
おはようございます。
Good morning.
早安。
```

**例句 11**
```
お疲れ様です。
Thank you for your hard work.
辛苦了。
```

**例句 12**
```
ご馳走様でした。
Thank you for the meal.
謝謝款待。
```

## 使用規則

### 規則 1：和語と漢語の区別

**和語（日本原有の言葉）→ お**
- お茶、お金、お水、お名前、お酒、お菓子
- 動詞：お読みになる、お書きになる、お待ちになる

**漢語（中国由来の言葉）→ ご**
- ご飯、ご意見、ご住所、ご家族、ご注文
- 動詞：ご説明になる、ご連絡になる、ご確認になる

**例外**
- ご飯（和語だが「ご」を使う）
- お電話（漢語だが「お」を使う）
- お勉強（漢語だが「お」を使う）

### 規則 2：付けない場合

すべての名詞に「お」「ご」を付けられるわけではありません。

**付けない例**
- 外来語：×お/ごコーヒー（通常は付けない）
- 既に敬語的な語：先生、社長、お客様
- カジュアルな場面：友達との会話

**自然な使用**
- 食べ物・飲み物：お茶、お水、お酒、ご飯
- 日常用品：お金、お箸、お皿
- 相手に関する物：お名前、ご住所、ご家族

### 規則 3：過度な使用を避ける

丁寧にしようとして過度に使うと、かえって不自然になります。

**不自然な例**
```
❌ お私のお家のお犬
（過度に「お」を付けすぎ）
✅ 私の家の犬
（普通の表現）
```

### 規則 4：相手のものと自分のもの

相手や目上の人のものには付けますが、自分のものには基本的に付けません。

**相手のもの**
```
✅ お名前は何ですか。
What is your name?
您的名字是什麼？
```

**自分のもの（ビジネス場面）**
```
✅ 名前は田中です。（×お名前）
My name is Tanaka.
我的名字是田中。
```

**例外：美化語として定着した語**
```
✅ お茶が好きです。（自分のことでも使う）
I like tea.
我喜歡茶。
```

## 常見錯誤

### 錯誤 1：お/ごの使い分け間違い

❌ 誤：ご茶
✅ 正：お茶
說明：「茶」は和語なので「お」を使う。

❌ 誤：お意見
✅ 正：ご意見
說明：「意見」は漢語なので「ご」を使う。

### 錯誤 2：外来語への誤用

❌ 誤：おコーヒー、ごコーヒー
✅ 正：コーヒー（通常は付けない）
說明：外来語には通常「お」「ご」を付けない。

**例外**
```
✅ おビール（習慣的に使われる）
Beer (polite)
啤酒（禮貌表達）
```

### 錯誤 3：過度な使用

❌ 誤：私のお家のお犬はお元気です。
✅ 正：私の家の犬は元気です。
說明：自分のものに過度に「お」を付けると不自然。

### 錯誤 4：敬語動詞との重複

❌ 誤：お召し上がりになる
✅ 正：召し上がる / お食べになる
說明：「召し上がる」は既に敬語なので「お」は不要。

## 學習要點

1. **基本区別**：和語には「お」、漢語には「ご」
2. **二つの機能**：美化語（名詞）と敬語形成（動詞）
3. **自然な使用**：慣習的な表現を覚える
4. **適度な使用**：過度に使うと不自然
5. **例外に注意**：「ご飯」「お電話」など
6. **相手のもの**：相手のものには付ける
7. **固定表現**：「おはよう」「お疲れ様」など
8. **外来語**：通常は付けない（おビールなど例外あり）
