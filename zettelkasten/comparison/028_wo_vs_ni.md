## 概述

「を」和「に」是日語中最基本且重要的助詞，它們在句子中扮演著不同的文法角色。「を」主要標記動作的直接賓語（對象），而「に」則有多種功能，包括標記方向、時間點、接受者等。理解這兩個助詞的區別是掌握日語基本句型的關鍵。

## 核心區別表

| 特徵 | を | に |
|-----|----|----|
| **基本功能** | 直接賓語標記（Direct Object Marker） | 間接賓語/方向/時間標記（Indirect Object/Direction/Time Marker） |
| **標記對象** | 動作的直接對象 | 動作的方向、目的地、接受者、時間點 |
| **動詞類型** | 他動詞 | 移動動詞、授受動詞、存在動詞等 |
| **語義角色** | 受事者（Patient） | 接受者（Recipient）、目標（Goal）、位置（Location） |
| **常見搭配** | 食べる、飲む、読む、見る、買う | 行く、来る、あげる、いる、ある |
| **可否省略** | 口語中常省略 | 通常不省略 |

## を（Direct Object Marker）

### 日文解釋

「を」は格助詞の一つで、主に他動詞の直接目的語を示します。動作や行為の対象となるものを明確に標示し、「何を」という質問に答える役割を果たします。また、移動の経路を表す用法もあります。日常会話では省略されることも多いですが、書面や丁寧な表現では必ず使用します。

### 英文解釋

"wo" (を) is a case particle that primarily marks the direct object of transitive verbs. It identifies what is being acted upon, answering the question "what" in relation to the verb's action. This particle is essential for indicating the target or patient of an action. In casual speech, it is sometimes omitted, but it remains crucial in formal writing and polite expressions. Additionally, "wo" can mark the route or path of movement.

### 中文解釋

「を」是格助詞，主要用於標記他動詞的直接賓語。它標示動作的對象，回答「什麼」的問題。這個助詞在日語句子結構中扮演著關鍵角色，明確指出動作作用的對象。在口語中有時會省略，但在書面語和正式表達中通常不能省略。此外，「を」也可以標記移動的路徑。

### を 的核心用法

#### 1. 標記動作的直接對象

表示動作直接作用的對象：

**例句 1**
```
朝ごはんを食べます。
I eat breakfast.
我吃早餐。
```

**例句 2**
```
本を読みます。
I read a book.
我看書。
```

**例句 3**
```
水を飲みます。
I drink water.
我喝水。
```

#### 2. 標記移動的路徑

表示經過或通過的場所：

**例句 4**
```
公園を散歩します。
I walk through the park.
我在公園散步。
```

**例句 5**
```
橋を渡ります。
I cross the bridge.
我過橋。
```

#### 3. 標記離開的地點

與表示離開的動詞搭配：

**例句 6**
```
家を出ます。
I leave home.
我離開家。
```

**例句 7**
```
部屋を出ました。
I left the room.
我離開了房間。
```

## に（Indirect Object/Direction/Time Marker）

### 日文解釋

「に」は多機能な格助詞で、主に方向、着点、時点、対象、受け手などを示します。移動動詞とともに使われて目的地を表したり、授受動詞とともに使われて受け手を示したり、存在動詞とともに使われて位置を表したりします。また、時刻や時点を示す用法も重要です。「を」と異なり、「に」は動作の間接的な対象や、動作が向かう方向を示す役割が中心です。

### 英文解釋

"ni" (に) is a versatile case particle with multiple functions, primarily indicating direction, destination, time point, target, or recipient. When used with motion verbs, it marks the destination; with giving/receiving verbs, it marks the recipient; with existence verbs, it marks the location. It also marks specific points in time. Unlike "wo," which marks direct objects, "ni" typically marks indirect objects or the direction/goal of an action, representing where something is going or to whom something is given.

### 中文解釋

「に」是一個多功能的格助詞，主要用於標記方向、目的地、時間點、對象或接受者。與移動動詞搭配時表示目的地，與授受動詞搭配時表示接受者，與存在動詞搭配時表示位置。它也用於標記特定的時間點。與標記直接賓語的「を」不同，「に」主要標記間接賓語或動作的方向目標，表示某物去往何處或給予何人。

### に 的核心用法

#### 1. 標記移動的目的地

表示去往的地方：

**例句 8**
```
学校に行きます。
I go to school.
我去學校。
```

**例句 9**
```
東京に来ました。
I came to Tokyo.
我來到東京了。
```

**例句 10**
```
部屋に入ります。
I enter the room.
我進入房間。
```

#### 2. 標記存在的位置

與存在動詞（いる、ある）搭配：

**例句 11**
```
机の上に本があります。
There is a book on the desk.
書桌上有一本書。
```

**例句 12**
```
公園に子供がいます。
There are children in the park.
公園裡有孩子。
```

#### 3. 標記授受的對象（接受者）

表示給予或接受的對象：

**例句 13**
```
友達にプレゼントをあげます。
I give a present to my friend.
我給朋友一份禮物。
```

**例句 14**
```
先生に質問します。
I ask the teacher a question.
我向老師提問。
```

#### 4. 標記時間點

表示具體的時刻或時間：

**例句 15**
```
7時に起きます。
I wake up at 7 o'clock.
我7點起床。
```

**例句 16**
```
月曜日に会いましょう。
Let's meet on Monday.
星期一見面吧。
```

## 關鍵對比情境

### 情境 1：同一動詞的不同用法

某些動詞可以同時使用「を」和「に」，但意義不同：

**例句 17 (使用 を - 經過)**
```
空を飛びます。
Fly through the sky.
在天空中飛翔。
```

**例句 18 (使用 に - 目的地)**
```
東京に飛びます。
Fly to Tokyo.
飛往東京。
```

### 情境 2：移動動詞的基本區別

**例句 19 (を - 離開起點)**
```
家を出ます。
Leave home.
離開家。
```

**例句 20 (に - 到達終點)**
```
学校に着きます。
Arrive at school.
到達學校。
```

### 情境 3：複合句型中的搭配

在表示目的的句型中，兩者可以共存：

**例句 21**
```
映画を見に行きます。
I go to see a movie.
我去看電影。
```

**說明**：「映画を見る」(看電影) + 「に行く」(去) = 「映画を見に行く」(去看電影)

**例句 22**
```
本を買いに店に行きます。
I go to the store to buy a book.
我去商店買書。
```

**說明**：「本を買う」+ 「店に行く」

### 情境 4：動作與位置

**例句 23 (を - 動作對象)**
```
日本語を勉強します。
I study Japanese.
我學習日文。
```

**例句 24 (に - 場所)**
```
図書館で勉強します。
I study at the library.
我在圖書館學習。
```

**例句 25 (結合使用)**
```
図書館で日本語を勉強します。
I study Japanese at the library.
我在圖書館學習日文。
```

## 常見陷阱與錯誤

### 陷阱 1：移動動詞的誤用

❌ 誤：学校を行きます。
✅ 正：学校に行きます。
說明：移動動詞「行く」表示目的地時必須用「に」，不能用「を」。「を」只能用於表示經過的路徑。

### 陷阱 2：存在動詞的誤用

❌ 誤：部屋を本があります。
✅ 正：部屋に本があります。
說明：存在動詞「ある」「いる」表示存在的位置時必須用「に」。

### 陷阱 3：授受動詞的混淆

❌ 誤：友達をプレゼントをあげます。
✅ 正：友達にプレゼントをあげます。
說明：「あげる」(給予) 中，接受者用「に」，給予的物品用「を」。

### 陷阱 4：時間表達的錯誤

❌ 誤：7時を起きます。
✅ 正：7時に起きます。
說明：具體時間點必須用「に」，不能用「を」。

### 陷阱 5：經路與目的地的混淆

✅ 正：公園を散歩します。(在公園散步 - 經過)
✅ 正：公園に散歩に行きます。(去公園散步 - 目的地)
說明：「を」表示移動的路徑，「に」表示移動的目的地。

## 學習要點

1. **「を」標記直接對象**：動作直接作用的事物（他動詞的賓語）
2. **「に」標記間接對象**：動作的方向、目的地、接受者、時間點
3. **移動動詞的區別**：目的地用「に」，經路用「を」
4. **授受動詞的固定搭配**：接受者用「に」，物品用「を」
5. **存在動詞必用「に」**：表示存在的位置
6. **時間點用「に」**：具體時刻或日期
7. **可以共存**：複合句型中兩者經常搭配使用
