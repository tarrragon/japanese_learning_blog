# を vs に：移動表達的區別

## 中文解釋

在日文中表示移動時，助詞「を」和「に」有明確的語義區別：

- **を**：表示移動**經過**的場所或空間（強調路徑、通過）
- **に**：表示移動的**目的地**或**到達點**（強調終點、方向）

這個區別不僅影響句意，也反映了說話者對移動行為的不同視角。

## 英文解釋

When expressing movement in Japanese, the particles を and に have distinct semantic differences:

- **を**: Indicates the place or space that movement **passes through** (emphasizes path, passage)
- **に**: Indicates the **destination** or **arrival point** of movement (emphasizes endpoint, direction)

This distinction affects not only the meaning but also reflects the speaker's different perspective on the movement.

## 日文解釋

日本語で移動を表すとき、助詞「を」と「に」には明確な意味の違いがあります：

- **を**：移動が**通過する**場所や空間を表す（経路・通過を強調）
- **に**：移動の**目的地**や**到着点**を表す（終点・方向を強調）

この違いは文の意味に影響するだけでなく、話し手の移動行為に対する視点の違いも反映しています。

## 核心區別表

| 項目 | を | に |
|------|----|----|
| **語義焦點** | 經過的路徑 | 到達的終點 |
| **動作性質** | 通過、離開 | 到達、進入 |
| **常見動詞** | 通る、渡る、出る、歩く、走る | 行く、来る、帰る、入る、着く |
| **空間概念** | 空間作為媒介 | 空間作為目標 |
| **視覺化** | 箭頭穿過 (→→→) | 箭頭指向 (→●) |

## 對比例句

### を的用法（經過）

1. **公園を散歩する**
   - 在公園裡散步（經過公園的空間）
   - Walk through the park
   - 強調在公園內移動的過程

2. **橋を渡る**
   - 過橋（通過橋這個空間）
   - Cross the bridge
   - 橋是被通過的路徑

3. **道を歩く**
   - 走在路上（沿著道路移動）
   - Walk along the road
   - 道路作為移動的媒介

### に的用法（到達）

1. **公園に行く**
   - 去公園（公園是目的地）
   - Go to the park
   - 強調公園作為終點

2. **東京に着く**
   - 抵達東京（東京是到達點）
   - Arrive in Tokyo
   - 東京是移動的終點

3. **家に帰る**
   - 回家（家是目的地）
   - Return home
   - 家是移動的目標點

## 同一動詞的對比

### 入る的差異

- **部屋に入る**：進入房間（強調進去這個動作的終點）
- **～を通って部屋に入る**：通過～進入房間（經過某路徑才進入）

### 出る的差異

- **家を出る**：離開家（強調從家這個空間出來）
- **外に出る**：出去外面（強調到外面這個目的地）

## 常見搭配模式

### を + 移動動詞（經過系）

```
場所 + を + 通る/渡る/歩く/走る/飛ぶ/出る/離れる
```

### に + 移動動詞（到達系）

```
場所 + に + 行く/来る/帰る/着く/入る/向かう/戻る
```

## 使用注意事項

1. **動詞決定助詞**
   - 有些動詞固定搭配を（如：出る、通る）
   - 有些動詞固定搭配に（如：着く、向かう）

2. **雙助詞可能**
   - 可以同時使用：**駅を出て、公園に向かう**（出站後前往公園）
   - を表示離開點，に表示目的地

3. **語境影響**
   - 同一地點可以用を或に，取決於強調重點
   - **空港を出る**（離開機場）vs **空港に着く**（抵達機場）
