# 因果接續詞比較

## 日文解釋

因果関係を表す接続詞の使い分けと文法的・語用論的な違いを比較する。

## 英文解釋

Comparison of Japanese causal conjunctions, examining their grammatical and pragmatic differences in expressing cause-and-effect relationships.

## 中文解釋

比較日文因果接續詞的用法差異，分析其在表達因果關係時的文法與語用特徵。

## 主要對比

### だから vs. ので

| 項目 | だから | ので |
|------|--------|------|
| **正式度** | 口語、較直接 | 書面、較正式 |
| **主觀性** | 強調說話者判斷 | 客觀陳述因果 |
| **連接方式** | 句子層級 | 從屬連接 |
| **接續** | 獨立句開頭 | 句中連接 |

**だから 範例**：
```
今日は雨だ。だから、家にいる。
今天下雨。所以待在家。
```

**ので 範例**：
```
雨が降っているので、家にいます。
因為在下雨，所以待在家。
```

### から vs. ので

| 項目 | から | ので |
|------|------|------|
| **語氣** | 主觀理由、強調 | 客觀原因、陳述 |
| **因果關係** | 主觀判斷 | 客觀事實 |
| **禮貌度** | 較隨意 | 較正式 |
| **使用場合** | 解釋、辯解 | 說明、報告 |

**から 範例**：
```
眠いから、もう寝ます。
因為睏了，要去睡了。（主觀感受）
```

**ので 範例**：
```
体調が悪いので、休みます。
因為身體不適，所以請假。（客觀說明）
```

### したがって vs. そのため

| 項目 | したがって | そのため |
|------|------------|----------|
| **正式度** | 非常正式 | 正式 |
| **邏輯性** | 強調邏輯必然 | 一般因果 |
| **文體** | 學術、法律 | 書面、報告 |
| **語感** | 推論結果 | 基於原因 |

**したがって 範例**：
```
証拠が不十分である。したがって、無罪とする。
證據不足。因此判定無罪。（邏輯推論）
```

**そのため 範例**：
```
台風が接近している。そのため、運動会は延期します。
颱風接近中。因此運動會延期。（一般因果）
```

## 文法特徵

### だから
- **接續**：句子 + だから + 句子
- **位置**：必須在句首
- **語氣**：較強、直接
- **限制**：不用於正式請求或命令句

### ので
- **接續**：[動詞/い形容詞] + ので / [名詞/な形容詞] + なので
- **位置**：句中連接
- **語氣**：委婉、客觀
- **優點**：適合提出請求

**請求句對比**：
```
❌ 遅刻したから、すみません。
✅ 遅刻したので、すみません。
因為遲到了，很抱歉。
```

### から
- **接續**：同 ので
- **特點**：可後接命令、請求
- **語感**：主觀理由明確

```
危ないから、触らないで。
危險，不要碰。（主觀判斷 + 命令）
```

### したがって/そのため
- **接續**：句子 + したがって/そのため + 句子
- **位置**：句首
- **文體**：書面語
- **特點**：邏輯性強

## 語用場景

### 日常對話
- **最常用**：だから、から
- **禮貌場合**：ので

### 商務書信
- **首選**：ので、そのため
- **避免**：だから、から

### 學術論文
- **首選**：したがって
- **次選**：そのため、ので
- **避免**：だから、から

### 道歉/請求
- **首選**：ので
- **避免**：から、だから

## 常見錯誤

### 錯誤 1：正式場合使用 だから
```
❌ 会議が延期になりました。だから、ご連絡いたします。
✅ 会議が延期になりましたので、ご連絡いたします。
會議延期了，所以聯絡您。
```

### 錯誤 2：命令句使用 ので
```
❌ 危ないので、逃げろ！
✅ 危ないから、逃げろ！
危險，快逃！
```

### 錯誤 3：請求句使用 から
```
❌ 急いでいるから、お願いします。
✅ 急いでいるので、お願いします。
因為趕時間，拜託了。
```
