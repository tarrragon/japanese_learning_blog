# とても vs 大変

## 基本資訊

### 日文解釋

**とても**と**大変**は両方とも程度を表す副詞ですが、使い方に違いがあります。

- **とても**：程度が高いことを表す一般的な副詞。肯定文でも否定文でも使える
- **大変**：非常に高い程度や困難な状況を強調する副詞。より強い印象を与える

### 中文解釋

**とても**和**大変**都是表示程度的副詞，但使用方式有所不同：

- **とても**：表示程度高的一般性副詞，可用於肯定句和否定句，語氣較為中性
- **大変**：強調非常高的程度或困難的狀況，給人更強烈的印象，常暗示辛苦或嚴重性

### English Explanation

Both **とても** and **大変** are adverbs expressing degree, but they differ in usage:

- **とても**: A general adverb indicating high degree; can be used in both affirmative and negative sentences; neutral tone
- **大変**: Emphasizes very high degree or difficult situations; gives a stronger impression; often implies hardship or seriousness

## 比較表格

| 項目 | とても | 大変 |
|------|--------|------|
| **語氣強度** | 中等 | 強烈 |
| **使用範圍** | 廣泛（肯定/否定） | 主要用於肯定句 |
| **感情色彩** | 中性 | 常帶有辛苦、嚴重的含義 |
| **正式程度** | 日常對話 | 較為正式或強調 |
| **否定句** | ✓ とても好きじゃない | △ 較少使用 |

## 例句

### とても 的使用

1. **日文**：この料理はとてもおいしいです。
   **中文**：這道料理非常好吃。
   **English**: This dish is very delicious.

2. **日文**：彼女はとても親切な人です。
   **中文**：她是一個非常親切的人。
   **English**: She is a very kind person.

3. **日文**：今日はとても疲れました。
   **中文**：今天非常累。
   **English**: I'm very tired today.

4. **日文**：その映画はとても面白くなかった。（否定句）
   **中文**：那部電影不太有趣。
   **English**: That movie wasn't very interesting.

### 大変 的使用

1. **日文**：今日は大変忙しかったです。
   **中文**：今天非常忙碌（很辛苦）。
   **English**: Today was very busy (tough).

2. **日文**：試験の準備は大変でした。
   **中文**：考試準備非常辛苦。
   **English**: Exam preparation was very tough.

3. **日文**：大変申し訳ございません。
   **中文**：非常抱歉。
   **English**: I sincerely apologize.

4. **日文**：この仕事は大変難しいです。
   **中文**：這項工作非常困難。
   **English**: This work is very difficult.

### 對比例句

1. **日文**：
   - とても楽しかったです。（單純表示很開心）
   - 大変楽しかったです。（強調非常盡興，可能超出預期）
   **中文**：
   - 非常開心。
   - 非常盡興（超乎預期的開心）。
   **English**:
   - It was very fun.
   - It was extremely fun (beyond expectations).

2. **日文**：
   - この問題はとても難しい。（客觀描述難度高）
   - この問題は大変難しい。（強調困難程度，暗示需要很大努力）
   **中文**：
   - 這個問題非常難。
   - 這個問題非常困難（需要很大努力）。
   **English**:
   - This problem is very difficult.
   - This problem is extremely difficult (requires great effort).

## 使用要點

### とても 的特點

1. **通用性高**：可修飾形容詞、形容動詞、副詞
2. **否定句友善**：とても〜ない 表示「不太〜」
3. **日常口語**：自然、常用於日常對話
4. **中性語氣**：不帶特殊感情色彩

### 大変 的特點

1. **強調性強**：表達程度極高或情況嚴重
2. **暗示辛苦**：常用於困難、辛苦的情境
3. **禮貌表達**：如「大変申し訳ございません」
4. **正式場合**：較常用於正式或強調的情境

## 選擇建議

### 使用 とても 的情況

- 一般性的程度描述
- 需要使用否定句時
- 日常輕鬆對話
- 描述正面、中性事物

### 使用 大変 的情況

- 強調困難或辛苦
- 表達深刻的歉意或感謝
- 描述超出預期的程度
- 正式或需要強調的場合

## 常見錯誤

1. ❌ **大変好きじゃない**
   ✓ **とても好きじゃない**
   說明：否定句通常使用 とても，不用 大変

2. ❌ **とても申し訳ございません**
   ✓ **大変申し訳ございません**
   說明：正式道歉習慣使用 大変

3. ❌ **この花はとてもきれいです**（想強調驚艷程度）
   ✓ **この花は大変きれいです**
   說明：若要強調超乎尋常的美麗，用 大変 更合適
