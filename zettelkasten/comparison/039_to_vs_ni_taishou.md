# と vs に 對象用法比較

## 核心區別

| 項目 | と | に |
|------|-------|-------|
| **語義** | 相互動作、雙向關係 | 單向動作、動作對象 |
| **動作性質** | 平等互動 | 不對等作用 |
| **語感** | 「一起」「共同」 | 「對著」「朝向」 |

## 詳細說明

### と：相互動作

表示雙方處於**平等互動**的關係。

**中文**：和、跟、與
**英文**：with (mutual action)

**例句**：
```
友達と話す
(ともだちとはなす)
中文：跟朋友聊天
英文：Talk with friends
說明：雙方都在說話，相互交流
```

```
先生と相談する
(せんせいとそうだんする)
中文：和老師商量
英文：Consult with teacher
說明：雙方共同討論
```

### に：單向對象

表示動作的**單向接受者或目標**。

**中文**：對、向、給
**英文**：to (one-way action)

**例句**：
```
先生に質問する
(せんせいにしつもんする)
中文：向老師提問
英文：Ask a question to teacher
說明：只有學生問，老師是接受對象
```

```
友達に手紙を書く
(ともだちにてがみをかく)
中文：給朋友寫信
英文：Write a letter to friend
說明：單向傳遞動作
```

## 對比分析

### 同一動詞的不同語義

**話す（說話）**：
- 友達**と**話す → 互相交談（聊天）
- 友達**に**話す → 告訴朋友（單向告知）

**会う（見面）**：
- 友達**と**会う → 和朋友見面（約見）
- 友達**に**会う → 去見朋友（拜訪）

**戦う（戰鬥）**：
- 敵**と**戦う → 與敵人交戰（雙方對抗）
- ❌ 敵**に**戦う → 不自然（戰鬥需要相互性）

## 選擇判斷法

### 使用 と 的情況

✅ 動作需要雙方參與：
- 結婚する、離婚する、喧嘩する、相談する

✅ 共同活動：
- 一緒に～、協力する、競争する

### 使用 に 的情況

✅ 單向傳遞動作：
- 教える、伝える、渡す、送る

✅ 感情或意見的對象：
- 感謝する、謝る、怒る、言う

## 常見錯誤

❌ 友達**に**結婚する
✅ 友達**と**結婚する
（結婚需要雙方意願）

❌ 先生**と**質問する
✅ 先生**に**質問する
（提問是單向行為）
