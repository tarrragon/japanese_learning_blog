## 核心區別

| 項目 | いる | ある |
|------|------|------|
| **對象** | 有生命的存在（人、動物） | 無生命的物體、抽象概念 |
| **動詞分類** | 一段動詞 | 五段動詞 |
| **否定形** | いない | ない |
| **丁寧形** | います | あります |

## 詳細說明

### Japanese
「いる」と「ある」は両方とも「存在」を表す動詞だが、使い分けには明確な規則がある。

**いる**：
- 人間や動物など、自分で動けるものの存在を表す
- 一段活用：いる → いない → います
- 「～ている」の形で継続状態も表す

**ある**：
- 物や植物、抽象的な概念の存在を表す
- 五段活用：ある → ない → あります
- 所有や予定の意味でも使用される

### English
Both "いる" and "ある" express existence, but they have distinct usage rules.

**いる**:
- Used for living things that can move by themselves (people, animals)
- Ichidan verb conjugation
- Also used in "～ている" form for continuous states

**ある**:
- Used for inanimate objects, plants, and abstract concepts
- Godan verb conjugation
- Also used to express possession and scheduled events

### 繁體中文
「いる」和「ある」都表示「存在」，但有明確的使用區別。

**いる**：
- 用於有生命且能自主移動的存在（人、動物）
- 一段動詞變化
- 也用於「～ている」形式表示持續狀態

**ある**：
- 用於無生命物體、植物、抽象概念
- 五段動詞變化
- 也用於表達所有和預定事項

## 對比例句

### 有生命 vs 無生命

**人・動物（いる）**：
```japanese
1. 部屋に猫がいます。
   房間裡有貓。
   There is a cat in the room.

2. 公園に子供たちがいる。
   公園裡有孩子們。
   There are children in the park.
```

**物・概念（ある）**：
```japanese
3. 机の上に本があります。
   桌子上有書。
   There is a book on the desk.

4. 明日、試験がある。
   明天有考試。
   There is an exam tomorrow.
```

### 特殊情況

**植物**：
```japanese
5. 庭に木があります。（一般用法）
   庭院裡有樹。
   There are trees in the garden.
```

**持續狀態**：
```japanese
6. 彼は立っている。（×立ってある）
   他站著。
   He is standing.

7. 窓が開いている。（自動詞 + ている）
   窗戶開著。
   The window is open.

8. 窓が開けてある。（他動詞 + てある）
   窗戶被打開著（有人刻意打開）。
   The window has been opened (intentionally).
```

## 常見錯誤

❌ 本があります → ⭕ 本があります
❌ 犬があります → ⭕ 犬がいます
❌ 友達があります → ⭕ 友達がいます
