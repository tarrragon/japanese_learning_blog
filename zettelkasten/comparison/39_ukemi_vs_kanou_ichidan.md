## 核心區別

| 比較項目 | 受身形（Passive） | 可能形（Potential） |
|---------|-----------------|-------------------|
| **形態** | 動詞ます形 + られる | 動詞ます形 + られる |
| **意義** | 被動承受動作 | 能力或可能性 |
| **主語** | 動作的接受者 | 動作的執行者 |
| **助詞** | に（動作者） | が（對象） |
| **焦點** | 誰做了什麼給我 | 我能不能做什麼 |

## られる的歧義性

一段動詞的受身形和可能形完全相同，必須透過**語境**和**助詞**判斷：

### 受身形語境
```
先生に褒められる。
（被老師稱讚）
```
- 主語是「我」（動作接受者）
- 「先生に」表示動作來源
- 焦點：老師稱讚了我

### 可能形語境
```
この本が読める。
（這本書能讀懂）
```
- 主語是「我」（能力擁有者）
- 「本が」表示對象
- 焦點：我有能力讀這本書

## 判斷方法

### 方法 1：檢查助詞
- **に + られる** → 高機率是受身形
  - `先生に叱られた`（被老師責罵）
- **が + られる** → 高機率是可能形
  - `漢字が書ける`（能寫漢字）

### 方法 2：檢查語意
- 如果主語是**被動接受者** → 受身形
  - `彼に見られた`（被他看到）
- 如果主語是**能力擁有者** → 可能形
  - `彼が見える`（能看到他）

### 方法 3：轉換測試
- **受身形**可以轉換成主動句
  - `母に起こされた` → `母が私を起こした`
- **可能形**無法轉換成主動句
  - `朝早く起きられる` → ❌ 無對應主動句

## 例句對比

### 例句 1：食べられる
**受身形**：
```
犬にケーキを食べられた。
【中】蛋糕被狗吃掉了。
【英】My cake was eaten by the dog.
```

**可能形**：
```
辛い物が食べられる。
【中】能吃辣的東西。
【英】I can eat spicy food.
```

### 例句 2：見られる
**受身形**：
```
人に日記を見られた。
【中】日記被別人看到了。
【英】My diary was seen by someone.
```

**可能形**：
```
遠くの山が見られる。
【中】能看到遠處的山。
【英】I can see the distant mountains.
```

### 例句 3：褒められる
**受身形**：
```
先生に褒められて嬉しい。
【中】被老師稱讚很高興。
【英】I'm happy to be praised by the teacher.
```

**可能形**（罕見，通常不用）：
```
この作品は褒められる。
【中】這個作品值得稱讚。
【英】This work can be praised.
```

## 特殊情況

### 自發用法
られる還有**自發**（spontaneous）用法，表示自然產生的感情：
```
故郷が思い出される。
【中】自然想起故鄉。
【英】I naturally recall my hometown.
```
- 不是受身，也不是可能
- 表示「不由自主地」產生的感覺

### 尊敬用法
られる也可表示**尊敬**：
```
先生が来られる。
【中】老師會來。（敬語）
【英】The teacher will come (honorific).
```
- 提升動作者的地位
- 常見於商業或正式場合

## 學習建議

1. **初學者**：優先記憶助詞搭配（に=受身，が=可能）
2. **中級者**：練習語境判斷，閱讀時主動分析
3. **高級者**：掌握自發和尊敬用法的細微差異

## 註釋

[^1]: 一段動詞的受身形和可能形形態完全相同，這是日語學習的常見困惑點。
[^2]: 五段動詞沒有這個問題，因為可能形是「動詞ます形 + える」，與受身形「動詞ます形 + られる」明顯不同。
[^3]: 在口語中，可能形有時會省略「ら」，變成「見れる」「食べれる」（ら抜き言葉），但這是非標準用法。
