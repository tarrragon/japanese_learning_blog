# 可能形 vs できる

## 核心區別

| 項目 | 可能形 | できる |
|------|--------|--------|
| **文法形式** | 動詞本身變化（五段動詞：う段→え段+る、一段動詞：る→られる） | 「動詞辞書形 + ことができる」或「名詞 + ができる」 |
| **助詞** | 通常用「が」標記對象 | 通常用「を」或「が」標記對象 |
| **語感** | 較為直接、口語化、自然 | 較為正式、強調能力或可能性 |
| **使用場景** | 日常會話、一般能力表達 | 書面語、強調特殊能力、禮貌表達 |
| **否定形** | 動詞可能形 + ない（例：食べられない） | できない（例：食べることができない） |

## 對比例句

### 例句 1：基本能力

**可能形**：
- 日文：私は日本語が話せます。
- 中文：我會說日文。
- 英文：I can speak Japanese.

**できる**：
- 日文：私は日本語を話すことができます。
- 中文：我能夠說日文。
- 英文：I am able to speak Japanese.

### 例句 2：具體動作

**可能形**：
- 日文：明日、来られますか。
- 中文：明天能來嗎？
- 英文：Can you come tomorrow?

**できる**：
- 日文：明日、来ることができますか。
- 中文：明天能夠過來嗎？
- 英文：Are you able to come tomorrow?

### 例句 3：否定表達

**可能形**：
- 日文：今日は行けません。
- 中文：今天不能去。
- 英文：I can't go today.

**できる**：
- 日文：今日は行くことができません。
- 中文：今天無法前往。
- 英文：I am not able to go today.

### 例句 4：技能表達

**可能形**：
- 日文：彼は泳げます。
- 中文：他會游泳。
- 英文：He can swim.

**できる**：
- 日文：彼は泳ぐことができます。
- 中文：他能夠游泳。
- 英文：He is able to swim.

## 使用建議

### 優先使用可能形的場景
- 日常會話
- 簡單能力表達
- 需要簡潔表達時

### 優先使用できる的場景
- 正式場合、商務溝通
- 強調特殊能力或成就
- 書面文章
- 需要禮貌表達時

## 特殊用法

### 可能形的特殊性
1. **見える/聞こえる**：視覺和聽覺的可能形有特殊形式
   - 見る → 見える（能看見、看得到）
   - 聞く → 聞こえる（能聽見、聽得到）

2. **食べられる的歧義**：
   - 可能形：能吃
   - 被動形：被吃
   - 通常通過上下文判斷

### できる的特殊性
1. **名詞 + ができる**：
   - 日文：私はピアノができます。
   - 中文：我會彈鋼琴。
   - 英文：I can play the piano.

2. **完成、出現的意思**：
   - 日文：新しい建物ができました。
   - 中文：新建築蓋好了。
   - 英文：A new building has been completed.

## 常見錯誤

❌ **錯誤**：私は日本語を話せます。
⭐ **正確**：私は日本語が話せます。
- 可能形通常用「が」而非「を」

❌ **錯誤**：明日来れることができますか。
⭐ **正確**：明日来られますか。/ 明日来ることができますか。
- 不要混用兩種形式
