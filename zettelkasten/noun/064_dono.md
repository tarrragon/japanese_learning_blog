## 日文解釋

「どの」は、疑問詞系の連体詞であり、複数の選択肢の中から特定のものを尋ねる際に使用される。日本語の「こそあど」体系の中で、疑問の「ど」系列に属し、常に名詞を直接修飾する必要がある点が「どれ」との大きな違いである。

「どれ」は代名詞として単独で使用できるのに対し、「どの」は連体詞であるため、「どの本」「どの日」「どの人」というように常に後ろに名詞を伴わなければならない。これは「この」「その」「あの」と同じ文法機能を持つ。

使用場面は日常生活のあらゆる場面に及ぶ。買い物の時、複数の商品の中から特定のものを指す場合、旅行計画において複数の日程や場所の中から選択する場合、また学校や職場で複数のオプションの中から選ぶ場合など、極めて頻繁に使用される基本的な疑問詞である。

「どの」は形式度が中立的で、カジュアルな会話から丁寧な場面まで、あらゆる場面で使用できる。疑問形として機能するため、それを用いた文は常に相手からの応答や選択を期待する表現となる。これは学習初期段階で習得が必須の基本的な疑問表現である。

## 英文解釋

"どの" (dono) is an interrogative determiner in Japanese, used when asking to identify a specific item or person from a set of multiple options. It belongs to the "do-" series of the "ko-so-a-do" demonstrative system, and unlike the pronoun "どれ" (dore), it always directly modifies a noun.

The key distinction between "どの" and "どれ" is grammatical: "どの" is a determiner and must always be followed by a noun (e.g., "どの本" - which book, "どの人" - which person), while "どれ" is a pronoun and can stand alone. This parallel structure mirrors the other determiners in the system: "この" (this), "その" (that), and "あの" (that over there).

"どの" is stylistically neutral and can be used in both casual and formal contexts, with politeness adjusted through the verb forms that follow. It is one of the most basic and frequently used interrogative forms in Japanese, essential for everyday communication such as shopping, dining, planning, and making choices.

## 中文解釋

「どの」是日文的疑問詞連體詞，用於詢問多個選項中的特定項目或人物。它屬於日語「こそあど」指示詞體系中的疑問系列（ど系列）。

「どの」與「どれ」的核心區別在於詞性：「どの」是連體詞，必須修飾名詞使用（例如「どの本」哪本書、「どの人」哪個人），而「どれ」是代名詞，可以單獨使用。這一結構平行於系統中的其他連體詞：「この」（這個）、「その」（那個）、「あの」（那個）。

「どの」的使用場景非常廣泛，涵蓋日常生活的各個方面——購物時選擇商品、點餐時選菜、計劃旅行時選擇日期或地點、在工作或學校環境中做出選擇等。語域上屬於中性，既可用於隨意會話，也可用於正式場合，只需透過後續動詞的敬體或常體形式調整禮貌度。

這是學習日語初期階段必須掌握的基本疑問表達。正確區分「どの」（連體詞）和「どれ」（代名詞）對於準確使用日語疑問表達至關重要。

## 核心用法

### 用法 1：詢問具體物品

直接修飾名詞，詢問多個具體物品中的特定項目。

**例句 1**
```
A: このお店はお菓子がいっぱいありますね。どの菓子が好きですか。
A: There are so many sweets in this shop. Which sweet do you like?
A: 這家店有很多糖果。你喜歡哪種糖果？

B: これが一番好きです。
B: I like this one the best.
B: 我最喜歡這個。
```

**例句 2**
```
図書館で本を探しています。どの本を読みたいですか。
I'm looking for books at the library. Which book would you like to read?
我在圖書館找書。你想看哪本書？
```

### 用法 2：詢問時間、日期

修飾時間詞彙，詢問多個時間選項中的特定時間。

**例句 3**
```
A: 映画はいつ見に行きますか。
A: When will we go to see the movie?
A: 我們什麼時候去看電影？

B: どの日曜日がいいですか。
B: Which Sunday is good for you?
B: 哪個星期日比較方便？
```

### 用法 3：詢問人物

修飾人物名詞，詢問多人中的特定人選。

**例句 4**
```
A: 会社に新しい同僚が三人来ました。どの人が田中さんですか。
A: Three new colleagues have come to the company. Which one is Tanaka-san?
A: 公司來了三個新同事。哪一個是田中先生？

B: あの背の高い人です。
B: It's that tall person.
B: 是那個個子高的。
```

**例句 5**
```
色々なオプションがあります。どの選択肢が一番いいと思いますか。
There are various options. Which option do you think is the best?
有很多選項。你認為哪個選項最好？
```

## 使用規則

### 文法結構

「どの」は必ず名詞を直接修飾します：

| 構造 | 例 | 説明 |
|------|------|------|
| どの + 名詞 | どの本 | 哪本書（正確） |
| どれ（単独） | どれ？ | 哪個？（代名詞） |
| ❌ どの（单独） | ❌ どの？ | 文法エラー |

### 常見搭配

1. **どの＋名詞＋が/を/に**
   - 「どの本がいいですか」（哪本書好？）
   - 「どの人を選びますか」（選擇哪個人？）
   - 「どの駅に行きますか」（去哪個車站？）

2. **どの＋名詞＋か**
   - 「どの映画か知りません」（不知道是哪部電影）
   - 「どの日か忘れました」（忘記了是哪一天）

3. **複合表達**
   - 「どのような」（什麼樣的）
   - 「どの程度」（哪種程度）

### 與「どれ」的使い分け

| 表現 | 詞性 | 例 | 說明 |
|------|------|------|------|
| どの + 名詞 | 連體詞 | どの本 | 必須修飾名詞 |
| どれ（単独） | 代名詞 | 本はどれ？ | 可單獨使用 |

## 常見錯誤

### 錯誤 1：「どの」的單獨使用

「どの」是連體詞，不能像「どれ」一樣單獨使用。

❌ 誤：これとあれ、どの？
✅ 正：これとあれ、どれ？

說明：多個選項中的選擇應使用代名詞「どれ」。如果後面有名詞，則使用連體詞「どの」。

### 錯誤 2：省略名詞的錯誤用法

「どの」後面不能省略名詞，否則文法不成立。

❌ 誤：どの欲しいですか。
✅ 正：どの本が欲しいですか。

說明：「どの」必須與名詞搭配使用。即使對話中已經知道說的是什麼，也要明確說出名詞。

### 錯誤 3：與「何」的混淆

「どの」用於已知選項中的選擇，「何」用於未知事物的詢問。

❌ 誤：これはどのですか。（想問「這是什麼」時）
✅ 正：これは何ですか。

說明：「どの」總是用於限定在已知的多個選項之中，「何」則用於詢問未知的事物。

## 學習要點

1. **連體詞特性**：「どの」必須修飾名詞，不能單獨使用，這是與「どれ」的根本區別
2. **こそあど體系**：理解「この・その・あの・どの」的連體詞系統對學習指示詞至關重要
3. **日常高頻**：「どの」是日常對話中最常用的疑問詞之一，在購物、選擇、詢問等場景中廣泛應用
4. **結構規律**：記住「どの＋名詞」的結構，掌握基本搭配（が、を、に等助詞）
5. **禮貌調整**：「どの」本身中立，透過後續動詞、句尾助詞調整禮貌度和語調
