## 日文解釋

相対時間詞とは、発話時点を基準として時間を表す語彙のことである。「今日」「明日」「昨日」「来週」「先月」などが代表的な例であり、これらの語は話している時点によって指し示す具体的な日時が変わる。例えば、「明日」という語は、1月1日に発話すれば1月2日を指すが、1月10日に発話すれば1月11日を指す。

相対時間詞は絶対時間詞（「2025年1月1日」のような固定された日付表現）と対比される概念である。日常会話では相対時間詞の使用頻度が高く、文脈に依存した柔軟な時間表現を可能にする。日本語学習において、これらの語彙を正確に使いこなすことは、自然な会話能力の基礎となる。

## 英文解釋

Relative time expressions (相対時間詞) are temporal words whose reference point shifts according to the moment of speech. Unlike absolute time expressions such as "January 1, 2025," relative time words like "today" (今日), "tomorrow" (明日), "yesterday" (昨日), "next week" (来週), and "last month" (先月) derive their meaning from when they are uttered.

This concept is fundamental in Japanese, as the language has a rich system of relative temporal expressions that extend beyond simple day references to include weeks, months, and years. Understanding these expressions is essential for natural conversation and comprehension.

## 中文解釋

相對時間詞是指以說話時間點為基準來表示時間的詞彙。與「2025年1月1日」這類固定日期的絕對時間表達不同，「今天」「明天」「昨天」「下週」「上個月」等相對時間詞的具體所指會隨著說話時間而改變。

例如，「明日（あした）」這個詞在1月1日說出時指的是1月2日，但在1月10日說出時則指1月11日。日語中的相對時間詞系統非常豐富，涵蓋了日、週、月、年等不同時間單位，是日常對話中最常用的時間表達方式。

## 核心用法

### 用法 1：日單位的相對時間詞

以「今日」為中心，向前後延伸的日期表達。

**例句 1**
```
昨日は雨だったが、今日は晴れている。
Yesterday it was rainy, but today it's sunny.
昨天下雨，但今天是晴天。
```

**例句 2**
```
明日の会議は午前10時からです。
Tomorrow's meeting starts at 10 AM.
明天的會議從上午10點開始。
```

### 用法 2：週單位的相對時間詞

以「今週」為基準的週表達。

**例句 3**
```
先週の金曜日に映画を見に行きました。
I went to see a movie last Friday.
上週五我去看了電影。
```

**例句 4**
```
来週の月曜日までにレポートを提出してください。
Please submit your report by next Monday.
請在下週一之前提交報告。
```

### 用法 3：月・年單位的相對時間詞

較長時間跨度的相對表達。

**例句 5**
```
去年の今頃は海外に住んでいました。
Around this time last year, I was living abroad.
去年這個時候我住在國外。
```

## 常見錯誤

### 錯誤 1：「おととい」與「あさって」的混淆

❌ 誤：今日は月曜日だから、おとといは水曜日ですね。
✅ 正：今日は月曜日だから、あさっては水曜日ですね。
說明：「おととい」是「前天」，「あさって」是「後天」。容易混淆方向。

### 錯誤 2：「先〜」與「前の〜」的使用場合

❌ 誤：前の週に東京へ行きました。
✅ 正：先週、東京へ行きました。
說明：「先週」「先月」「先年」是固定表達，一般不用「前の週」的說法。

## 學習要點

1. **核心概念**：相對時間詞的所指會隨說話時間改變，這與絕對時間詞形成對比。
2. **日單位系統**：おととい → 昨日 → 今日 → 明日 → あさって（前天 → 昨天 → 今天 → 明天 → 後天）
3. **週單位系統**：先々週 → 先週 → 今週 → 来週 → 再来週
4. **月年單位**：先月/先年（去年）→ 今月/今年 → 来月/来年
5. **文化特點**：日語中還有「一昨年（おととし）」「明後年（あさって の とし→みょうごねん）」等更遠的相對表達。
