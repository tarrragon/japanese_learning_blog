## 日文解釋

尊敬語とは、日本語の敬語体系における三つの主要分類（尊敬語・謙譲語・丁寧語）の一つであり、話し手が相手や第三者の行為・状態・所有物などを高めることで敬意を表す表現方法です。

尊敬語の形成には主に以下の方法があります：

1. **お/ご〜になる形**：動詞の連用形に「お」を付け、「になる」を添える（例：お読みになる）
2. **れる・られる形**：動詞に受身形と同じ「れる・られる」を付ける（例：読まれる）
3. **特別尊敬語**：特定の動詞に対応する専用の尊敬語（例：いらっしゃる、召し上がる）

尊敬語は相手を「上げる」表現であり、謙譲語が自分を「下げる」表現であることと対照的です。ビジネスや公式な場面では、尊敬語の適切な使用が社会人としての基本的なマナーとされています。

## 英文解釋

The Sonkeigo System (Honorific Language System) is one of the three main categories of Japanese keigo (polite language), alongside kenjougo (humble language) and teineigo (polite language). Sonkeigo is used to elevate the actions, states, or possessions of the listener or a third party to express respect.

The system operates through several grammatical mechanisms:
- **O/Go-ni naru form**: Adding "o" prefix and "ni naru" suffix to verbs
- **Reru/Rareru form**: Using passive-like conjugations for respect
- **Special honorific verbs**: Dedicated vocabulary like "irassharu" (to be/go/come) and "meshiagaru" (to eat/drink)

Understanding the sonkeigo system is essential for proper communication in Japanese business and formal contexts.

## 中文解釋

尊敬語系統是日語敬語體系中的三大分類之一（尊敬語、謙讓語、丁寧語），用於抬高對方或第三者的行為、狀態或所有物，以表達敬意。

尊敬語的形成方式主要有三種：

1. **お/ご〜になる形**：在動詞連用形前加「お」，後接「になる」（如：お読みになる）
2. **れる・られる形**：使用與被動形相同的「れる・られる」（如：読まれる）
3. **特別尊敬語**：特定動詞對應的專用敬語（如：いらっしゃる、召し上がる）

尊敬語是「抬高對方」的表達方式，與謙讓語「降低自己」的表達形成對比。在商務和正式場合中，正確使用尊敬語是社會人的基本禮儀。

## 核心用法

### 用法 1：お/ご〜になる形

最常見的尊敬語形成方式，適用於大多數動詞。

**例句 1**
```
社長はもうお帰りになりました。
The president has already left.
社長已經回去了。
```

**例句 2**
```
先生がこの本をお書きになりました。
The teacher wrote this book.
老師寫了這本書。
```

### 用法 2：れる・られる形

使用被動形態表示尊敬，較為簡便但尊敬程度較低。

**例句 3**
```
部長は何時に来られますか。
What time will the department manager come?
部長幾點來呢？
```

### 用法 3：特別尊敬語

使用專用的尊敬語動詞，尊敬程度最高。

**例句 4**
```
お客様がいらっしゃいました。
The customer has arrived.
客人到了。
```

**例句 5**
```
どうぞ、お茶を召し上がってください。
Please have some tea.
請用茶。
```

## 使用規則

### 尊敬語の三つの形式

| 形式 | 敬度 | 適用場面 |
|------|------|---------|
| 特別尊敬語 | 最高 | 顧客対応、公式場面 |
| お/ご〜になる | 高 | ビジネス一般 |
| れる・られる | 中 | 社内、準フォーマル |

### 主な特別尊敬語一覧

| 基本形 | 尊敬語 |
|--------|--------|
| いる・行く・来る | いらっしゃる |
| 食べる・飲む | 召し上がる |
| 見る | ご覧になる |
| 言う | おっしゃる |
| 知っている | ご存じだ |
| する | なさる |

### 注意事項

1. 尊敬語は相手や第三者に使い、自分の行為には使わない
2. 二重敬語（例：お読みになられる）は避ける
3. 場面に応じた敬度の選択が重要

## 常見錯誤

### 錯誤 1：自分に尊敬語を使う

❌ 誤：私がお書きになりました。
✅ 正：私が書きました。／私がお書きしました。
說明：尊敬語は相手を高める表現なので、自分の行為には使えません。自分の行為には謙譲語を使います。

### 錯誤 2：二重敬語

❌ 誤：先生がお話しになられました。
✅ 正：先生がお話しになりました。
說明：「お〜になる」と「られる」を重ねると二重敬語になり、不自然です。

### 錯誤 3：敬度の不適切な選択

❌ 誤：（顧客に対して）部長は来られます。
✅ 正：（顧客に対して）部長がまいります。
說明：社外の人に対して社内の人を高める尊敬語は使いません。この場合は謙譲語を使います。

## 學習要點

1. **三種形成方式の理解**：お/ご〜になる、れる・られる、特別尊敬語の使い分けを覚える
2. **特別尊敬語の暗記**：頻出の特別尊敬語（いらっしゃる、召し上がる、ご覧になる等）を優先的に習得する
3. **場面による使い分け**：ビジネス場面では敬度の高い形式を選ぶ
4. **二重敬語の回避**：過剰な敬語は逆効果になることを認識する
5. **謙譲語との対比**：尊敬語（相手を上げる）と謙譲語（自分を下げる）の違いを理解する
