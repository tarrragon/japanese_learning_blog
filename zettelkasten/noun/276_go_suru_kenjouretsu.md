謙讓語構成「ご〜する」是日語敬語系統中表達謙遜的重要句型。

<!--more-->

## 日文

ご〜する（ご〜いたす）

### 文法情報

| 項目 | 內容 |
|------|------|
| 類型 | 謙讓語構成 |
| 構成 | ご + 漢語動詞詞幹 + する/いたす |
| JLPT | N3 |
| 使用場面 | 商務、正式場合 |

### 形成規則

**基本形式**
```
ご + 漢語動詞詞幹 + する
ご + 漢語動詞詞幹 + いたす（更謙遜）
```

**常見組合**
```
ご連絡する → ご連絡いたす
ご説明する → ご説明いたす
ご案内する → ご案内いたす
ご報告する → ご報告いたす
```

## 日文解釋

「ご〜する」は、自分の行為を低めて表現する謙譲語の構成パターンである。漢語動詞の語幹に「ご」を付け、「する」または「いたす」を後ろに付けることで形成される。

この形式は、相手に対する敬意を示しながら、自分の行動を謙遜して表現する機能を持つ。「いたす」を使用すると、「する」よりもさらに丁寧で謙遜した印象を与える。

ビジネスシーンや正式な場面で頻繁に使用され、特に顧客やお客様、目上の人に対して自分の行動を伝える際に適している。和語動詞には「お〜する」の形式を使用するのに対し、漢語動詞には「ご〜する」の形式を使用するという使い分けがある。

## 英文解釋

"Go-[verb stem]-suru" is a humble expression pattern (kenjougo) in Japanese honorific language. It is formed by adding the prefix "go-" to a Sino-Japanese (kango) verb stem, followed by "suru" or the more humble variant "itasu."

This construction lowers the speaker's own actions to show respect to the listener or a third party being discussed. It is commonly used in business contexts and formal situations when describing one's actions to customers, clients, or superiors.

The distinction between "go-" (for Sino-Japanese verbs) and "o-" (for native Japanese verbs) is an essential aspect of mastering humble language.

## 中文解釋

「ご〜する」是日語謙讓語系統中的重要構成模式。透過在漢語動詞詞幹前加上「ご」，並在後面接上「する」或更謙遜的「いたす」來形成。

這種表達方式藉由貶低說話者自身的行為來表示對聽者或話題中第三者的敬意。在商務場合和正式場合中頻繁使用，特別是在向顧客、客戶或上級說明自己的行為時。

需要注意的是，漢語動詞使用「ご〜する」，而和語動詞則使用「お〜する」，這種區分是掌握謙讓語的關鍵。

## 核心用法

### 用法 1：向對方報告自己的行為

表達自己將要做某事或已做某事時，使用謙讓語表示對聽者的敬意。

**例句 1**
```
後ほどご連絡いたします。
I will contact you later.
稍後我會與您聯繫。
```

**例句 2**
```
詳細についてご説明いたします。
I will explain the details.
我來為您說明詳細內容。
```

### 用法 2：商務書信和郵件

書面語中更常使用「いたす」形式，顯得更加正式和謙遜。

**例句 3**
```
ご質問があれば、ご回答いたします。
If you have any questions, I will answer them.
如果您有任何疑問，我會為您解答。
```

**例句 4**
```
資料をご送付いたしますので、ご確認ください。
I will send the documents, so please confirm.
我會將資料寄給您，請您確認。
```

### 用法 3：服務業的接客用語

在接待客戶時使用，展現專業和禮貌。

**例句 5**
```
ただいまご案内いたします。
I will guide you right now.
我現在為您帶路。
```

## 使用規則

### 「ご」與「お」的區分

- **ご + 漢語動詞**：ご連絡、ご説明、ご案内、ご報告
- **お + 和語動詞**：お待ちする、お届けする、お伝えする

### する vs いたす

| 形式 | 丁寧度 | 使用場面 |
|------|--------|----------|
| ご〜する | 標準謙讓 | 一般商務場合 |
| ご〜いたす | 更謙遜 | 正式書信、VIP 對應 |

### 常見搭配動詞

- ご連絡する（聯繫）
- ご説明する（說明）
- ご案内する（引導）
- ご報告する（報告）
- ご確認する（確認）
- ご相談する（商量）

## 常見錯誤

### 錯誤 1：和語動詞誤用「ご」

❌ 誤：ご待ちする
✅ 正：お待ちする
說明：「待つ」是和語動詞，應使用「お」而非「ご」。

### 錯誤 2：對自己的行為使用尊敬語

❌ 誤：私がご説明になります。
✅ 正：私がご説明いたします。
說明：「ご〜になる」是尊敬語，用於對方的行為。自己的行為應使用謙讓語「ご〜する/いたす」。

### 錯誤 3：重複敬語

❌ 誤：ご説明させていただきます。
✅ 正：ご説明いたします。
說明：「ご〜する」本身已經是謙讓表現，不需要再加「させていただく」。

## 學習要點

1. **漢語與和語的區分**：「ご」用於漢語動詞，「お」用於和語動詞，這是最基本的使用規則。

2. **いたす的使用時機**：書面語和非常正式的場合使用「いたす」，口語中「する」已經足夠禮貌。

3. **謙讓語的本質**：透過貶低自己的行為來抬高對方，這與尊敬語抬高對方行為的方式不同。

4. **常用動詞的記憶**：熟記常見的「ご〜する」組合，如ご連絡、ご説明、ご案内等。

5. **避免過度敬語**：不要將多種敬語表現重疊使用，保持表達的自然流暢。
