## 日文解釋

「小さい」は「容積・面積・身長などの占める場所・空間が少ない」という意味です。[^dict]

サイズや容量が標準より小さいこと。物理的な大きさだけでなく、数量や規模、音量などが小さい場合にも使われる。反対語は「大きい」。日常会話で最もよく使われる基礎形容詞の一つであり、様々な文脈で幅広く活用される。

**基本的な意味**：
- 物の大きさが小さい（サイズが小さい）
- 音や声が小さい（音量が小さい）
- 数や量が少ない（規模が小さい）
- 若い、幼い（年齢が小さい）

**語用特徴**：
- い形容詞として、名詞を直接修飾できる（小さい家）
- 述語としても使用可能（この家は小さい）
- 副詞形「小さく」で動詞を修飾（小さく切る）

[^dict]: 定義參考 スーパー大辞林（macOS 內建字典）

## 英文解釋

**small; little; tiny**

Describes something that is below standard size, volume, or magnitude. Used for physical size as well as quantity, scale, or volume. The opposite is "大きい" (ookii, big).

**Common uses**:
- Physical size (a small house, little box)
- Volume (quiet voice, soft sound)
- Quantity or scale (small number, little company)
- Young age (small child)

## 中文解釋

**小的；細小的；年幼的**

形容尺寸、容量或程度低於標準。除了物理大小外，也可用於描述數量、規模、音量等。反義詞為「大きい」（大的）。

**主要用法**：
- 物理尺寸小（小房子、小箱子）
- 音量小（小聲音）
- 數量或規模小（小數字、小公司）
- 年齡小（小孩子）

**注意事項**：
- 作為い形容詞，可以直接修飾名詞（小さい家）
- 也可以作為述語使用（この家は小さい）
- 「小さい」也可表示「年幼的」（小さい子供）

## 核心用法

### 用法 1：物理尺寸（サイズ）

描述物體的物理大小，是最基本的用法。

**例句 1**
```
この箱は小さいです。
This box is small.
這個箱子很小。
```

**例句 2**
```
彼の部屋は小さいですが、きれいです。
His room is small, but clean.
他的房間雖小但很乾淨。
```

### 用法 2：音量・聲音（音・声）

描述聲音或音量小。

**例句 3**
```
小さい声で話してください。
Please speak in a quiet voice.
請小聲說話。
```

### 用法 3：年齡・成長（年齢）

表示年幼、年紀小。

**例句 4**
```
小さい子供が公園で遊んでいます。
Small children are playing in the park.
小孩子們在公園玩耍。
```

### 用法 4：規模・程度（規模）

描述規模或程度較小。

**例句 5**
```
私は小さい町で育ちました。
I grew up in a small town.
我在小鎮長大。
```

## 常見錯誤

### 錯誤 1：「小さい」與「小さな」混淆

❌ 誤：この部屋は小さなです。
✅ 正：この部屋は小さいです。
說明：「小さな」是連體詞，只能修飾名詞（小さな部屋），不能作述語。「小さい」可以作述語。

### 錯誤 2：副詞形使用錯誤

❌ 誤：野菜を小さい切ってください。
✅ 正：野菜を小さく切ってください。
說明：修飾動詞時需使用副詞形「小さく」，不是原形「小さい」。

### 錯誤 3：否定形活用錯誤

❌ 誤：この箱は小さいない。
✅ 正：この箱は小さくない。
說明：い形容詞否定時，需去掉「い」加「くない」，不是直接加「ない」。

## 學習要點

1. **基礎活用**：掌握い形容詞的基本活用規則（去「い」加詞尾）
2. **「小さい」vs「小さな」**：「小さい」可作述語和連體修飾，「小さな」只能修飾名詞
3. **多重語義**：除物理尺寸外，可表示音量、年齡、規模
4. **副詞形應用**：「小さく」可修飾動詞（小さく切る、小さく話す）
5. **反義詞配對**：與「大きい」形成基礎反義詞對
