# 悪い（わるい）

## 基本資訊

| 項目 | 內容 |
|------|------|
| **假名** | わるい |
| **漢字** | 悪い |
| **詞性** | い形容詞 |
| **JLPT** | N5 |

## 定義

### 日文解釋

1. 品質・状態・性質などが劣っている。好ましくない。
2. 道徳的に正しくない。不正である。
3. 健康状態がよくない。
4. 都合がよくない。不利である。

### 中文解釋

1. 壞的、不好的、品質差的
2. 邪惡的、不道德的
3. （身體）不舒服的、有病的
4. 不方便的、不利的

### 英文解釋

1. bad, poor, inferior (in quality, condition, or nature)
2. evil, wrong, wicked (morally)
3. ill, sick, unwell (in health)
4. inconvenient, unfavorable, disadvantageous

## い形容詞活用

| 活用形 | 形式 | 用法 |
|--------|------|------|
| **基本形** | 悪い | 天気が悪い |
| **否定形** | 悪くない | 天気が悪くない |
| **過去形** | 悪かった | 天気が悪かった |
| **過去否定形** | 悪くなかった | 天気が悪くなかった |
| **て形** | 悪くて | 天気が悪くて、出かけなかった |
| **假定形** | 悪ければ | 天気が悪ければ、中止する |
| **副詞形** | 悪く | 悪く言わないで |

## 例句

### 例句 1
**日文**：この牛乳は悪くなっている。

**中文**：這牛奶壞掉了。

**英文**：This milk has gone bad.

**說明**：表示物品品質變差、腐敗。

### 例句 2
**日文**：今日は体の調子が悪い。

**中文**：今天身體不舒服。

**英文**：I'm not feeling well today.

**說明**：表示健康狀態不佳。

### 例句 3
**日文**：悪いことをしてはいけません。

**中文**：不可以做壞事。

**英文**：You must not do bad things.

**說明**：表示道德上不正確的行為。

## 相關用法

### 常用搭配

- **頭が悪い** - 頭腦不好、笨
- **目が悪い** - 視力不好
- **性格が悪い** - 性格差
- **運が悪い** - 運氣不好
- **印象が悪い** - 印象差
- **悪く言う** - 說壞話
- **悪く思う** - 往壞處想、介意

### 慣用表現

- **悪いけど** - 不好意思（用於請求或拒絕時）
- **悪気はない** - 沒有惡意
- **悪くない** - 不錯、還可以

## 學習要點

### 重點提示

1. 「悪い」是最基本的負面評價形容詞，使用範圍極廣
2. 在道歉或請求時，「悪いけど」是常用的客套語
3. 「悪くない」（不壞）常用來表示「還不錯」的肯定意思
4. 可接在名詞後形成複合詞，如「運が悪い」

### 容易混淆

- **悪い vs ひどい**：悪い指一般的「壞」；ひどい強調「過分、嚴重」
- **悪い vs まずい**：悪い泛指壞；まずい多指味道不好或情況棘手

## 文法說明

作為い形容詞，「悪い」可以：
1. 直接修飾名詞：悪い人（壞人）
2. 作為述語：天気が悪い（天氣不好）
3. 變化為副詞：悪く（不好地）
4. 接續て形：悪くて（因為不好）

## 補充資訊

### 文化註記

- 日本人在請求他人幫忙或打擾別人時，常說「悪いけど」表示歉意
- 「悪くない」在年輕人之間常用來表達肯定評價（類似中文的「不錯啊」）

### 相關表現

- **申し訳ない** - 抱歉（更正式）
- **すみません** - 對不起（一般道歉）
- **ごめん** - 抱歉（口語）
