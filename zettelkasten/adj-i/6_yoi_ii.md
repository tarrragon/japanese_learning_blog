# 良い／いい（よい／いい）

## 日本語の説明

「良い」は肯定的な評価を表す基本的なイ形容詞。二つの読み方があり、日常会話では「いい」が多く使われる。「よい」は改まった場面や書き言葉でよく見られる。

**読み方の違い**：
- **よい**：書き言葉・改まった表現に多い（例：「これは良い考えだ」）
- **いい**：話し言葉・日常会話に多い（例：「いい天気だね」）

**活用の特徴**：
- 否定形：「よくない」（「いくない」は誤り）
- 連用形：「よく」（「いく」は誤り）
- 仮定形：「よければ」「よかったら」（「いければ」は誤り）
- 辞書形・現在形では「いい」も「よい」も使えるが、活用形は「よ」を使う

## English Explanation

"良い/いい (yoi/ii)" is a basic i-adjective expressing positive evaluation or quality. It has two readings:
- **yoi**: More formal, common in written language
- **ii**: Casual, common in spoken language

**Important conjugation rule**: Despite both forms existing for the dictionary form, conjugations always use the "yo-" stem:
- Negative: よくない (yokunai) - NOT いくない
- Adverbial: よく (yoku) - NOT いく
- Conditional: よければ (yokereba) - NOT いければ

## 中文解釋

「良い／いい」是表示肯定評價的基本い形容詞，意為「好的、良好的」。有兩種讀音：
- **よい**：較正式，多用於書面語
- **いい**：較隨意，多用於口語

**活用注意事項**：
- 雖然終止形兩種讀音都可以，但活用形一律使用「よ」開頭
- 否定形：よくない（不可說「いくない」）
- 連用形：よく（不可說「いく」）
- 假定形：よければ（不可說「いければ」）

## 例文

### 例文1

いい天気ですね。
今日は公園へ行きましょう。

**English**: Nice weather, isn't it? Let's go to the park today.

**中文**：天氣真好呢。今天去公園吧。

---

### 例文2

この案は良いと思います。
来週の会議で提案しましょう。

**English**: I think this proposal is good. Let's present it at next week's meeting.

**中文**：我認為這個方案很好。在下週的會議上提案吧。

---

### 例文3

もっとよく考えてから決めたほうがいい。

**English**: You should think about it more carefully before deciding.

**中文**：最好再仔細考慮後再決定。

---

### 例文4

時間があれば、一緒に映画を見に行きませんか。
いいですよ。

**English**: If you have time, would you like to go watch a movie together? Sure, sounds good.

**中文**：如果有時間的話，要不要一起去看電影？好啊。

## 使い方・文法ノート

### 1. 活用形の使い分け

| 活用形 | 形 | 例 |
|--------|-----|-----|
| 終止形 | よい／いい | これはいい／よい |
| 否定形 | よくない | あまりよくない |
| 連用形 | よく | よく考える |
| 仮定形 | よければ | よければどうぞ |
| 過去形 | よかった | よかったですね |

### 2. 「いい」と「よい」の使い分け基準

**「いい」を使う場面**：
- 日常会話
- カジュアルな文章
- 若者言葉

**「よい」を使う場面**：
- フォーマルな文書
- 改まった会議
- 書き言葉
- ニュース原稿

### 3. 活用形での注意

❌ 誤り：
- このケーキはいくないです（✗）
- いく考えてください（✗）
- いければ教えてください（✗）

✅ 正しい：
- このケーキはよくないです
- よく考えてください
- よければ教えてください

### 4. 慣用表現

- **いいですよ**：同意・許可を表す
- **よくできました**：褒める表現
- **よろしくお願いします**：挨拶表現（「よろしく」は「よい」の連用形から派生）

---

## 語源・豆知識

「良い」の語源は古語「よし」に由来する。「いい」という読み方は江戸時代に一般化した口語形。現代では両方が標準語として認められているが、活用形は常に「よ」を使うという規則が確立している。

「よろしく」という表現も「よい」の連用形「よく」が変化したもので、「適切に・良く」という意味が挨拶表現に発展した。
