## 日文
もっと

**羅馬拼音**: motto

**詞性**: 副詞（程度副詞[^degree-adverb]）

## 日文解釋

「もっと」は、程度や量の増加を表す副詞[^degree-adverb]で、現在の状態よりもさらに多く、さらに高く、さらに進んでいることを示します。「もっと」は「もっと〜する」「もっと〜い」という形で動詞[^verb]や形容詞[^adjective]と一緒に使われ、追加や強化を表現します。日常会話[^casual]で非常によく使われる基本的な副詞で、願望や要求を表現する際にも頻繁に用いられます。比較表現[^comparison]と共に使うことも多く、「AよりもっとB」のような形で比較の強調を行います。また、「もう少し」という意味で婉曲的な依頼や提案にも使われ、丁寧な表現[^polite]としても機能します。

## 英文解釋

"Motto" is a degree adverb[^degree-adverb] that indicates an increase in degree or quantity, expressing "more," "further," or "even more" than the current state. It is used with verbs[^verb] and adjectives[^adjective] in patterns like "motto + verb" or "motto + adjective" to express addition or intensification. This is an extremely common adverb in everyday conversation[^casual], frequently used to express wishes or requests. It is often used with comparative expressions[^comparison] in forms like "A yori motto B" (even more B than A) to emphasize comparisons. It can also function as a polite expression[^polite], used in indirect requests or suggestions with the meaning of "a little more."

## 中文解釋

「もっと」是程度副詞[^degree-adverb]，表示程度或數量的增加，意為「更加」「再多一點」「更進一步」。它與動詞[^verb]或形容詞[^adjective]搭配使用，如「もっと〜する」「もっと〜い」，表達追加或強化的意思。這是日常會話[^casual]中非常常用的基本副詞，經常用來表達願望或要求。它也常與比較表現[^comparison]一起使用，如「A比もっとB」的形式來強調比較。此外，它帶有「再多一點」的婉轉含義，可用於委婉的請求或建議，也能作為禮貌表達[^polite]使用。

## 核心用法

### 用法 1：表示程度增加

用於表達希望增加某種程度、數量或強度。

**例句 1**
```
もっと速く走ってください。
Please run faster.
請跑快一點。
```

**例句 2**
```
日本語をもっと勉強したいです。
I want to study Japanese more.
我想更多地學習日文。
```

### 用法 2：表示願望或要求

表達希望獲得更多的東西或更高的程度。

**例句 3**
```
もっと食べたいですか。
Do you want to eat more?
你還想再吃一點嗎？
```

**例句 4**
```
もっと詳しく説明してください。
Please explain in more detail.
請解釋得更詳細一點。
```

### 用法 3：比較強調

與比較表現一起使用，強調某一方的程度更高。

**例句 5**
```
この本はあの本よりもっと面白いです。
This book is even more interesting than that book.
這本書比那本書更有趣。
```

## 使用規則

### 位置規則

- 通常置於被修飾的動詞或形容詞之前
- 可以修飾動詞、形容詞、副詞
- 在比較句中，「もっと」置於形容詞或副詞之前

### 常見搭配

- **もっと〜たい**: 表示「想要更加〜」（願望）
- **もっと〜する**: 表示「更加〜地做」（動作增強）
- **もっと〜い**: 表示「更加〜」（性質增強）
- **〜よりもっと**: 表示「比〜更加」（比較強調）

### 注意事項

1. **與「もう」的區別**: 「もっと」強調「更多」，而「もう」[^mou]表示「已經」或「再一次」
2. **禮貌表達**: 在請求時使用「もっと」比直接命令更婉轉禮貌
3. **重複使用**: 避免在同一句中重複使用「もっと」，會顯得不自然

## 常見錯誤

### 錯誤 1：與「もう」混淆

❌ 誤: もう速く走ってください。（意思變成「已經快點跑」或「再一次快點跑」）
✅ 正: もっと速く走ってください。（更快一點地跑）
說明: 「もう」[^mou]表示「已經」或「再一次」，而「もっと」表示程度的增加（更加）。

### 錯誤 2：位置錯誤

❌ 誤: 勉強したいもっとです。
✅ 正: もっと勉強したいです。
說明: 「もっと」必須放在被修飾的動詞「勉強」之前，不能放在句尾。

### 錯誤 3：重複使用

❌ 誤: もっともっと早く来てください。（口語中可接受，但書面語不正式）
✅ 正: もっと早く来てください。（更早一點來）
說明: 在正式場合避免重複使用「もっと」，雖然口語中「もっともっと」可表達「越來越」的強調，但一次「もっと」已經足夠。

## 學習要點

1. **基本意義**: 「もっと」表示程度、數量的增加，意為「更加」「再多一點」
2. **常用搭配**: 「もっと〜たい」（想要更多）、「〜よりもっと」（比〜更加）
3. **位置規則**: 通常置於被修飾的動詞或形容詞之前
4. **區別「もう」**: 「もっと」= 更加（程度增加），「もう」[^mou] = 已經/再一次（時間或次數）
5. **語用功能**: 可用於表達願望、要求、比較強調，也可作為禮貌的婉轉表達

## 註解

[^degree-adverb]: **程度副詞** - 表示程度、數量、強度等的副詞，如「もっと」「とても」「少し」等。用於修飾動詞、形容詞或其他副詞，表達程度的高低。

[^verb]: **動詞** - 表示動作、狀態或變化的詞類。日文動詞有多種活用形式，可與副詞搭配表達不同的意義和語氣。

[^adjective]: **形容詞** - 表示性質、狀態的詞類。日文形容詞分為い形容詞和な形容詞，可與副詞搭配表達程度的變化。

[^casual]: **日常會話語境** - 朋友、家人之間的非正式對話。「もっと」是口語中極為常用的詞彙，適用於各種日常交流場景。

[^comparison]: **比較表現** - 用於比較兩個或多個事物的程度、性質等的文法結構，如「AよりB」（B比A更〜）。「もっと」常與比較表現結合使用。

[^polite]: **禮貌表達** - 使用禮貌的語言形式和措辭。「もっと」在請求或建議時，比直接命令更婉轉，能夠軟化語氣。

[^mou]: **「もう」副詞** - 表示「已經」或「再一次」的副詞。與「もっと」容易混淆，但意義不同：「もう」關注時間或次數，「もっと」關注程度或數量的增加。
