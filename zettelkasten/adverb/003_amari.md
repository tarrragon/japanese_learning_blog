## 日文

あまり

**讀音**：あまり (amari)

**詞性**：副詞（程度副詞[^degree-adverb]）

## 日文解釋

「あまり」は、主に否定表現[^negation]と一緒に使われる程度副詞です。「それほど〜ない」という意味で、動作や状態の程度が低いことを表します。日常会話[^daily-conversation]で頻繁に使用される基本的な表現で、控えめな[^modest]否定の気持ちを伝える際に便利です。

肯定文で使う場合は「あまりに」や「あまりにも」の形になり、「過度に」という意味になりますが、N5レベルでは否定形での使用が中心です。話し言葉でも書き言葉でも自然に使えます。

「あまり〜ない」の形で、完全な否定ではなく、「そこまで〜というわけではない」という柔らかい否定のニュアンス[^nuance]を持ちます。

## 英文解釋

"Amari" is a degree adverb[^degree-adverb] primarily used with negative expressions[^negation]. It means "not very" or "not much," indicating a low degree of an action or state. This is a fundamental expression frequently used in daily conversation[^daily-conversation], useful for conveying modest[^modest] negative feelings.

When used in affirmative sentences, it takes the form "amari ni" or "amari ni mo" and means "excessively," but at the N5 level, its use with negatives is central. It can be naturally used in both spoken and written language.

In the form "amari~nai," it carries a soft negative nuance[^nuance], meaning "not so much~" rather than a complete negation.

## 中文解釋

「あまり」是主要與否定表達[^negation]一起使用的程度副詞[^degree-adverb]。意思是「不太」、「不怎麼」，表示動作或狀態的程度較低。這是日常會話[^daily-conversation]中頻繁使用的基本表達，適合用來傳達謙虛[^modest]的否定感受。

在肯定句中使用時，會變成「あまりに」或「あまりにも」的形式，意思是「過度地」，但在 N5 程度主要是與否定形式一起使用。無論在口語或書面語都可以自然使用。

以「あまり〜ない」的形式，帶有柔和的否定語氣[^nuance]，意思是「沒有那麼〜」而不是完全否定。

## 核心用法

### 用法 1：あまり + 否定形（基本用法）

表示程度不高的否定，相當於「不太」、「不怎麼」。

**例句 1**
```
このラーメンはあまりおいしくない。
This ramen is not very delicious.
這碗拉麵不太好吃。
```

**例句 2**
```
私はあまり日本語を話せません。
I can't speak Japanese very well.
我不太會說日語。
```

### 用法 2：與形容詞否定形搭配

用於描述狀態或性質的程度較低。

**例句 3**
```
今日はあまり暑くないです。
It's not very hot today.
今天不太熱。
```

**例句 4**
```
この本はあまり面白くありませんでした。
This book was not very interesting.
這本書不太有趣。
```

### 用法 3：與動詞否定形搭配

用於表示動作或行為的頻率或程度不高。

**例句 5**
```
週末はあまり外出しません。
I don't go out much on weekends.
週末我不太外出。
```

## 使用規則

### 語法搭配

1. **基本模式**：あまり + 否定形
   - あまり + 動詞ない形[^nai-form]
   - あまり + い形容詞くない
   - あまり + な形容詞ではない/じゃない

2. **位置**：通常放在被修飾的動詞或形容詞之前

3. **書面語與口語**：兩者都可自然使用

### 常見搭配詞彙

- あまり好きではない（不太喜歡）
- あまりわからない（不太明白）
- あまり使わない（不太使用）
- あまり多くない（不太多）
- あまり上手ではない（不太擅長）

### 注意事項

1. **主要與否定形搭配**：在 N5 階段，幾乎總是與否定形一起使用
2. **程度表達**：表示「沒有那麼〜」，而非「完全不〜」
3. **禮貌性**：比直接否定更委婉、更有禮貌
4. **發音注意**：第二個「り」不要漏掉

## 常見錯誤

### 錯誤 1：在肯定句中單獨使用「あまり」

❌ 誤：このケーキはあまりおいしいです。
✅ 正：このケーキはとてもおいしいです。（或：あまりにもおいしいです。）
說明：N5 階段的「あまり」主要用於否定句。肯定句要用「とても」或「あまりにも」。

### 錯誤 2：忘記否定形

❌ 誤：私はあまり日本語を話します。
✅ 正：私はあまり日本語を話しません。
說明：「あまり」必須與否定形[^negation]搭配使用。

### 錯誤 3：與完全否定混淆

❌ 誤理解：あまり食べない = 完全不吃
✅ 正確理解：あまり食べない = 吃得不多
說明：「あまり〜ない」表示程度低，而非完全否定。完全否定要用「全然〜ない」。

## 學習要點

1. **核心意思**：「あまり」+ 否定形 = 「不太」、「不怎麼」
2. **使用頻率**：日常會話中非常常用的表達方式
3. **委婉表達**：比直接說「ない」更有禮貌和謙虛
4. **搭配限制**：N5 階段主要記住與否定形搭配
5. **實用性**：適用於各種情境，從日常對話到正式場合

## 註解

[^degree-adverb]: **程度副詞** - 表示動作或狀態程度的副詞，如「とても」（很）、「少し」（稍微）、「あまり」（不太）等。詳見 [degree-adverbs](../concept/degree-adverbs.md)

[^negation]: **否定表現** - 日語的否定形式，包括動詞ない形、形容詞くない形等。詳見 [negation-forms](../grammar/negation-forms.md)

[^daily-conversation]: **日常會話** - 朋友、家人、同事之間的非正式對話情境。詳見 [daily-conversation](../context/daily-conversation.md)

[^modest]: **控えめな表現（謙虛表達）** - 日本文化中重視的委婉、謙虛的表達方式，避免過於直接的否定。詳見 [modest-expression](../concept/modest-expression.md)

[^nuance]: **ニュアンス（語氣）** - 詞語或表達所帶有的細微感情色彩和暗示意義。詳見 [nuance](../concept/nuance.md)

[^nai-form]: **ない形（否定形）** - 日語動詞和形容詞的否定變化形式。詳見 [nai-form](../grammar/nai-form.md)
