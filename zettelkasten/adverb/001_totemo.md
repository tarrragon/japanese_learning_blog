## 日文
とても

**羅馬拼音**：totemo

**詞性**：副詞（程度副詞[^degree-adverb]）

## 日文解釋

「とても」は程度副詞[^degree-adverb]で、形容詞や形容動詞を修飾し、その程度が高いことを表します。日常会話[^casual]で最も頻繁に使われる基本的な副詞の一つです。肯定文[^affirmative]で使用されることが一般的で、「非常に」「大変」よりもカジュアルな印象を与えます。話し言葉では「すごく」と置き換えることもできますが、「とても」の方がやや丁寧な響きがあります。否定文では通常使われず、代わりに「あまり」や「全然」が用いられます。

## 英文解釋

"Totemo" is a degree adverb[^degree-adverb] that modifies adjectives and adjectival nouns to indicate a high degree of something. It is one of the most frequently used basic adverbs in everyday conversation[^casual], commonly translated as "very," "really," or "extremely." While "totemo" is typically used in affirmative sentences[^affirmative], it carries a more casual tone than formal alternatives like "非常に" (hijou ni) or "大変" (taihen). In spoken language, it can be replaced with "すごく" (sugoku), though "totemo" sounds slightly more polite. Note that "totemo" is not normally used in negative sentences, where "あまり" (amari) or "全然" (zenzen) are preferred instead.

## 中文解釋

「とても」是程度副詞[^degree-adverb]，用來修飾形容詞和形容動詞，表示程度很高。這是日常會話[^casual]中最常用的基本副詞之一，通常翻譯為「非常」、「很」或「十分」。「とても」一般用於肯定句[^affirmative]中，比正式的「非常に」（hijou ni）或「大變」（taihen）更口語化。在口語中可以用「すごく」（sugoku）替換，但「とても」聽起來稍微更有禮貌一些。需要注意的是，「とても」通常不用於否定句，否定句中會使用「あまり」（amari）或「全然」（zenzen）。

## 核心用法

### 用法 1：修飾い形容詞

修飾い形容詞時，「とても」直接放在形容詞前面，表示程度很高。

**例句 1**
```
今日はとても暑いです。
It's very hot today.
今天非常熱。
```

**例句 2**
```
この映画はとても面白かったです。
This movie was very interesting.
這部電影非常有趣。
```

### 用法 2：修飾な形容詞

修飾な形容詞時，「とても」同樣放在形容詞前面。

**例句 3**
```
先生の説明はとても丁寧[^teinei]でした。
The teacher's explanation was very polite/thorough.
老師的解釋非常詳細。
```

**例句 4**
```
この町はとても静か[^shizuka]です。
This town is very quiet.
這個城鎮非常安靜。
```

### 用法 3：修飾動詞

雖然較少見，「とても」也可以修飾某些動詞，表示程度很高。

**例句 5**
```
日本語の勉強がとても好きです。
I really like studying Japanese.
我非常喜歡學日文。
```

## 註解

[^degree-adverb]: **程度副詞** - 表示動作、狀態或性質程度的副詞。日語中常見的程度副詞包括：とても（非常）、すごく（很）、ちょっと（一點）、かなり（相當）、非常に（極其）。程度副詞通常修飾形容詞、形容動詞或某些動詞。

[^casual]: **日常會話語境** - 朋友、家人之間的非正式對話。「とても」在日常對話中非常常用，適合各種非正式場合，比「非常に」更自然。詳見 [日常會話](_meta/tags/context/002_casual.md)

[^affirmative]: **肯定文** - 肯定句，表達肯定意義的句子。「とても」主要用於肯定句中。在否定句中，應使用「あまり」（不太）或「全然」（完全不）等副詞。例：× とても高くない（錯誤）→ ○ あまり高くない（正確）

[^teinei]: **丁寧（ていねい）** - 形容動詞，意為「有禮貌的」、「細緻的」、「周到的」。在商務和正式場合中是重要的品質。

[^shizuka]: **静か（しずか）** - 形容動詞，意為「安靜的」、「寧靜的」。注意形容動詞後接「です」時不需要「な」。

## 常見錯誤

### 錯誤 1：在否定句中使用「とても」

❌ 誤：このレストランはとても美味しくないです。
✅ 正：このレストランはあまり美味しくないです。
說明：「とても」不用於否定句。在否定句中應使用「あまり」（不太）或「全然」（完全不）。

### 錯誤 2：與動詞一起使用時的位置錯誤

❌ 誤：私は好きとても日本語を勉強します。
✅ 正：私は日本語の勉強がとても好きです。
說明：「とても」應放在修飾的詞語前面。修飾動詞時，要注意語序和文法結構。

### 錯誤 3：過度使用導致表達單調

❌ 誤：とても美味しくて、とても綺麗で、とても安いです。
✅ 正：とても美味しくて、綺麗で、しかも安いです。
說明：避免在一個句子中重複使用「とても」。可以只在最重要的形容詞前使用，或者混合使用其他程度副詞如「かなり」「すごく」等。

## 學習要點

1. **基本用法**：「とても」是 N5 級別的基礎副詞，主要用於修飾形容詞和形容動詞，表示「非常」的意思
2. **肯定句專用**：「とても」只用於肯定句，否定句中要用「あまり」或「全然」
3. **語感差異**：「とても」比「非常に」「大変」更口語化，比「すごく」稍微正式一點
4. **位置固定**：「とても」必須放在所修飾的詞語前面，不能放在後面或中間
5. **自然表達**：在日常對話中，「とても」是表達程度的最自然選擇，使用頻率極高
