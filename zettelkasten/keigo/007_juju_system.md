日文授受表現的完整系統，包含普通形式（あげる/もらう/くれる）與敬語形式（差し上げる/いただく/くださる）的視點轉換與使用規範。

## 日文

授受表現系統（じゅじゅひょうげんシステム）

## 日文解釋

授受表現とは、「あげる・もらう・くれる」という三つの動詞を中心に、物や恩恵のやり取りを表す日本語の体系である。それぞれの動詞は視点と方向性が異なり、敬語形式では「差し上げる・いただく・くださる」という形になる。この体系は日本語の人称意識と恩を重んじる文化を反映している。

## 英文解釋

The giving and receiving expression system (juju hyōgen) is a Japanese linguistic framework centered on three verbs: "ageru," "morau," and "kureru." Each verb represents different perspectives and directional relationships in the exchange of objects or favors. Their honorific forms are "sashiageru," "itadaku," and "kudasaru." This system reflects Japanese cultural values of perspective-taking and recognizing indebtedness.

## 中文解釋

授受表現系統是日文中表達「給予」與「接受」行為的完整體系，核心包含三組動詞：あげる（給予）、もらう（得到）、くれる（被給予）。每個動詞代表不同的視點和方向性，在敬語形式中分別對應差し上げる、いただく、くださる。此系統反映日本文化中的人稱意識和恩惠觀念，是理解日語敬語的關鍵。

## 授受動詞三組對應

### 普通形式 vs 敬語形式

| 普通形式 | 敬語形式 | 敬語類型 | 視點 | 方向性 |
|---------|---------|---------|------|-------|
| あげる | 差し上げる | 謙讓語 | 給予者（我方） | 我 → 對方 |
| もらう | いただく | 謙讓語 | 接受者（我方） | 對方 → 我 |
| くれる | くださる | 尊敬語 | 給予者（對方） | 對方 → 我 |

### 視點轉換原理

```
【同一行為的三種表達方式】

情境：先生が私に本をくれた（老師給了我一本書）

視點 1（對方給予）：先生が本をくださった。
        → 尊敬語，強調老師的恩惠

視點 2（我方接受）：私が先生に本をいただいた。
        → 謙讓語，強調自己謙卑地接受

視點 3（我方給予）：私が先生に本を差し上げた。
        → 謙讓語，強調自己謙卑地給予
        （此例較少用，因通常不會給老師東西）
```

## 核心用法

### 用法 1：差し上げる（謙讓語：給予）

**基本句型**：
```
（私は）〜に　[名詞]を　差し上げる
```

**使用時機**：
- 我方給予對方物品或恩惠
- 降低自己的位置表示尊敬
- 對上司、長輩、客戶使用

**例句**：
- 社長にお土産を差し上げました。（我給社長獻上了伴手禮）
- お客様に資料を差し上げます。（我將資料提供給客戶）

### 用法 2：いただく（謙讓語：接受）

**基本句型**：
```
（私は）〜から／に　[名詞]を　いただく
（私は）〜に　[動詞て形]＋いただく
```

**使用時機**：
- 我方從對方接受物品或恩惠
- 請求對方為我做某事（〜ていただく）
- 對上司、長輩、客戶使用

**例句**：
- 部長からアドバイスをいただきました。（從部長那裡獲得建議）
- 確認していただけますか。（能請您確認嗎）

### 用法 3：くださる（尊敬語：給予）

**基本句型**：
```
〜が　（私に）[名詞]を　くださる
〜が　（私に）[動詞て形]＋くださる
```

**使用時機**：
- 對方給予我方物品或恩惠
- 提升給予者的位置
- 強調對方的恩惠和好意

**例句**：
- 先生が参考書をくださいました。（老師給了我參考書）
- ご説明くださりありがとうございます。（感謝您的說明）

## 例句

### 例句 1：差し上げる（謙讓語：我給對方）

**日文**：お客様にカタログを差し上げました。
**假名**：おきゃくさまにカタログをさしあげました。
**羅馬字**：Okyakusama ni katarogu o sashiagemashita.
**中文**：我將目錄提供給客戶。
**英文**：I offered the catalog to the customer.

**語境**：商務場合，向客戶提供資料
**敬語要點**：使用「差し上げる」表示對客戶的尊敬

### 例句 2：いただく（謙讓語：我從對方得到）

**日文**：部長から貴重なご意見をいただき、感謝しております。
**假名**：ぶちょうからきちょうなごいけんをいただき、かんしゃしております。
**羅馬字**：Buchō kara kichō na goiken o itadaki, kansha shite orimasu.
**中文**：非常感謝部長給予寶貴的意見。
**英文**：I am grateful for the valuable opinions received from the department manager.

**語境**：正式場合，感謝上司的指導
**敬語要點**：「いただく」表示謙卑地接受對方的恩惠

### 例句 3：くださる（尊敬語：對方給我）

**日文**：先生が推薦状を書いてくださいました。
**假名**：せんせいがすいせんじょうをかいてくださいました。
**羅馬字**：Sensei ga suisenjō o kaite kudasaimashita.
**中文**：老師為我寫了推薦信。
**英文**：The teacher kindly wrote a recommendation letter for me.

**語境**：感謝老師的幫助
**敬語要點**：「くださる」強調老師的恩惠和好意

### 例句 4：三組對比（同一情境）

**情境**：上司給了我建議

**視點 A（對方給予）**：
**日文**：部長がアドバイスをくださいました。
**中文**：部長給了我建議。（強調部長的恩惠）

**視點 B（我方接受）**：
**日文**：部長からアドバイスをいただきました。
**中文**：我從部長那裡得到建議。（強調自己謙卑地接受）

**視點 C（我方給予）**：
**日文**：（較少用於此情境）
**中文**：私が部長にアドバイスを差し上げる（不符合上下關係）

### 例句 5：補助動詞用法對比

**情境**：請求對方確認文件

**謙讓語（我方視點）**：
**日文**：確認していただけますか。
**中文**：能請您確認嗎？（我請求您為我做）

**尊敬語（對方視點）**：
**日文**：確認してくださいますか。
**中文**：能請您確認嗎？（請您為我做）

**差異**：
- 「〜ていただく」：強調我接受對方的恩惠
- 「〜てくださる」：強調對方的好意和行為

## 學習要點

### 1. 視點轉換的靈活性

同一行為可用不同動詞表達，取決於說話者的視點：
- **對方視點**：くださる（尊敬語）
- **我方視點**：いただく（謙讓語）

### 2. 方向性的一致性

```
【方向性規則】
あげる/差し上げる：我 → 對方（向外）
もらう/いただく：對方 → 我（向內）
くれる/くださる：對方 → 我（向內，但強調對方）
```

### 3. 敬語選擇的文化考量

- **謙讓語（差し上げる/いただく）**：降低自己，適合主動表達感謝
- **尊敬語（くださる）**：提升對方，適合強調對方的恩惠

### 4. 補助動詞的實用性

```
【補助動詞用法】
〜てあげる（普通）→ 〜て差し上げる（謙讓）
〜てもらう（普通）→ 〜ていただく（謙讓）
〜てくれる（普通）→ 〜てくださる（尊敬）
```

商務日語中「〜ていただく」和「〜てくださる」是最常用的請求表達。

### 5. 避免過度使用

對平輩或下屬使用敬語形式會顯得生疏或諷刺，應根據關係選擇普通形式或敬語形式。
