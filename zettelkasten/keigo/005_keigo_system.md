## 基本資訊

### 中文解釋
「敬語系統」是日文中用來表達敬意和禮貌的語言體系，由三大類組成：**尊敬語**（提升對方地位）、**謙讓語**（降低自己地位）、**丁寧語**（基本禮貌表達）。這三類敬語各有不同的功能和使用場合，共同構成了日本社會溝通的基礎框架。掌握敬語系統是學習商務日語和正式場合對話的核心能力。

### 英文解釋
The "keigo system" is a Japanese linguistic framework for expressing respect and politeness, composed of three main categories: **sonkeigo** (respectful language - elevating others), **kenjougo** (humble language - lowering oneself), and **teineigo** (polite language - basic courtesy). Each category has distinct functions and usage contexts, collectively forming the foundational framework for communication in Japanese society. Mastering the keigo system is essential for business Japanese and formal conversation.

### 日文解釋
「敬語システム」とは、日本語における敬意や丁寧さを表現するための言語体系であり、三つの主要カテゴリーで構成される：**尊敬語**（相手の地位を高める）、**謙譲語**（自分の地位を低める）、**丁寧語**（基本的な礼儀表現）。これら三つの敬語はそれぞれ異なる機能と使用場面を持ち、日本社会のコミュニケーションの基盤を成している。敬語システムの習得は、ビジネス日本語や改まった場面での会話において核心的な能力である。

## 敬語三分類

日本語の敬語体系は、以下の三つの基本類型に分けられる：

### 1. 尊敬語（そんけいご）Sonkeigo

**定義**：相手や第三者の動作・状態を高めることで敬意を表す

**文法手段**：
- 特殊尊敬動詞：いらっしゃる、おっしゃる、召し上がる
- お〜になる：お読みになる、お書きになる
- れる・られる：読まれる、書かれる

**代表例**：
```
基本形：先生が来る
尊敬語：先生がいらっしゃる
```

### 2. 謙讓語（けんじょうご）Kenjougo

**定義**：自分や内集団の動作・状態を低めることで、相対的に相手を高め敬意を表す

**文法手段**：
- 特殊謙譲動詞：伺う、申す、いただく、参る
- お〜する：お送りする、お待ちする
- ご〜する：ご説明する、ご案内する

**代表例**：
```
基本形：私が行く
謙譲語：私が伺う
```

**細分類**：
- **謙譲語 I**：向かう先（相手）を高めることを主目的とする
  - 例：先生に伺う
- **謙譲語 II（丁重語）**：聞き手への丁寧さを表す
  - 例：私が参ります

### 3. 丁寧語（ていねいご）Teineigo

**定義**：聞き手に対して丁寧な態度を示す基本的な敬語形式

**文法手段**：
- ます・です：行きます、学生です
- ございます：ありがとうございます

**代表例**：
```
普通形：行く
丁寧語：行きます
```

## 敬語三分類の比較表

| 分類 | 日文 | 目的 | 高める対象 | 代表動詞 |
|------|------|------|-----------|---------|
| 尊敬語 | そんけいご | 相手を高める | 動作主（相手） | いらっしゃる、おっしゃる |
| 謙譲語 | けんじょうご | 自分を低める | 相手（相対的） | 伺う、申す、いただく |
| 丁寧語 | ていねいご | 丁寧に話す | 聞き手 | 〜ます、〜です |

## 例句

### 例句 1：尊敬語的使用

**日文**: 先生がいらっしゃいました。
**假名**: せんせいがいらっしゃいました。
**羅馬字**: Sensei ga irasshaimashita.
**中文**: 老師來了。（尊敬表達）
**英文**: The teacher has arrived. (respectful)

**語境**: 正式場合，向他人說明老師到來
**敬語要點**: 使用尊敬語「いらっしゃる」提升老師的地位

### 例句 2：謙讓語的使用

**日文**: 明日、御社に伺います。
**假名**: あした、おんしゃにうかがいます。
**羅馬字**: Ashita, onsha ni ukagaimasu.
**中文**: 明天我將前往貴公司拜訪。（謙讓表達）
**英文**: I will visit your company tomorrow. (humble)

**語境**: 商務場合，向客戶表達拜訪意願
**敬語要點**: 使用謙讓語「伺う」降低自己的地位，相對提高對方

### 例句 3：丁寧語的使用

**日文**: 私は学生です。
**假名**: わたしは　がくせいです。
**羅馬字**: Watashi wa gakusei desu.
**中文**: 我是學生。（禮貌表達）
**英文**: I am a student. (polite)

**語境**: 一般對話，基本禮貌表達
**敬語要點**: 使用丁寧語「です」表示對聽話者的基本禮貌

### 例句 4：三種敬語並用（商務郵件）

**日文**: お忙しいところ恐れ入りますが、資料をご確認いただけますでしょうか。
**假名**: おいそがしいところおそれいりますが、しりょうをごかくにんいただけますでしょうか。
**羅馬字**: Oisogashii tokoro osoreirimasu ga, shiryou o gokakunin itadakemasu deshou ka.
**中文**: 抱歉打擾您百忙之中，能否請您確認一下資料？
**英文**: I apologize for interrupting you while you're busy, but could you please check the documents?

**語境**: 商務郵件，禮貌請求對方確認文件
**敬語要點**:
- 尊敬語：お忙しい（對方的狀態）
- 謙讓語：いただけます（請求對方為自己做事）
- 丁寧語：でしょうか（基本禮貌形式）

### 例句 5：內外區分（對外談論上司）

**日文**: 部長の田中が申しておりました。
**假名**: ぶちょうのたなかがもうしておりました。
**羅馬字**: Buchou no Tanaka ga moushite orimashita.
**中文**: 部長田中說過。（對外部人士用謙讓語）
**英文**: Department Manager Tanaka said (this). (humble when speaking to outsiders)

**語境**: 對外部客戶談論公司內部上司
**敬語要點**: 對外部人士談論內部人士時，即使是上司也使用謙讓語「申す」

## 使用要點

### 敬語選擇的基本原則

1. **誰的動作？**
   - 對方的動作 → 尊敬語
   - 自己的動作 → 謙讓語
   - 任何動作（禮貌） → 丁寧語

2. **內外關係**
   - 對外部人士談論內部人士 → 謙讓語（即使是上司）
   - 對內部人士談論上司 → 尊敬語

3. **場合與關係**
   - 商務場合 → 尊敬語 + 謙讓語 + 丁寧語
   - 一般禮貌對話 → 丁寧語為主
   - 親密朋友 → 普通形

### 敬語並用的模式

在實際使用中，三種敬語常常組合使用：

| 情境 | 組合模式 | 例句 |
|------|---------|------|
| 商務請求 | 謙讓語 + 丁寧語 | 伺います（謙讓） + ます（丁寧） |
| 報告上司動作 | 尊敬語 + 丁寧語 | おっしゃいます（尊敬） + ます（丁寧） |
| 對外談論內部 | 謙讓語 + 丁寧語 | 申しております（謙讓 + 丁寧） |

### 常見場合的敬語使用

| 場合 | 推薦敬語類型 | 避免 |
|------|------------|------|
| 對上司、客戶 | 尊敬語 + 謙讓語 + 丁寧語 | 普通形 |
| 對同事（禮貌） | 丁寧語 | 過度謙讓 |
| 對晚輩、學生 | 丁寧語 | 尊敬語、謙讓語 |
| 服務業對客戶 | 尊敬語 + 謙讓語 + 丁寧語 | 普通形 |

## 學習重點

1. **三大類的核心差異**：尊敬語提升對方、謙讓語降低自己、丁寧語表達基本禮貌
2. **視角固定性**：尊敬語用於對方的動作，謙讓語用於自己的動作
3. **內外區分**：商務場合特別注意內部與外部的界線
4. **敬語並用**：實際溝通中三種敬語常組合使用
5. **文化背景**：敬語反映日本社會的階層意識和謙遜文化
6. **避免過度**：根據場合選擇適當的敬語等級，過度使用反而不自然
