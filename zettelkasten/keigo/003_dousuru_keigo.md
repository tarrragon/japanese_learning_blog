# どうするの敬語

## 定義

### 日本語
動詞を敬語に変える方法。**尊敬語**（相手や第三者の動作を高める）、**謙譲語**（自分の動作をへりくだる）、**丁寧語**（丁寧な表現）の三種類がある。

### English
Methods for converting verbs into honorific forms. There are three types: **sonkeigo** (respectful language elevating others' actions), **kenjougo** (humble language lowering one's own actions), and **teineigo** (polite language).

### 繁體中文
將動詞轉換為敬語的方法。包含三種類型：**尊敬語**（提升對方或第三者的動作）、**謙讓語**（降低自己的動作）、**丁寧語**（禮貌表達）。

---

## 敬語の種類と形式

### 1. 尊敬語（そんけいご）
相手や第三者の動作を高める表現。

**作り方**：
- お + 動詞ます形 + になる
- 動詞られる形（受身形と同じ）
- 特別な尊敬語動詞

**例**：
- 書く → お書きになる / 書かれる
- 行く → いらっしゃる（特別）
- 言う → おっしゃる（特別）

### 2. 謙譲語（けんじょうご）
自分の動作をへりくだって表現し、相手を立てる。

**作り方**：
- お + 動詞ます形 + する
- お + 動詞ます形 + いたす
- 特別な謙譲語動詞

**例**：
- 聞く → お聞きする / 伺う（特別）
- 見る → 拝見する（特別）
- 言う → 申す / 申し上げる（特別）

### 3. 丁寧語（ていねいご）
「です・ます」で丁寧に表現する。

**作り方**：
- 動詞ます形 + ます

**例**：
- 行く → 行きます
- 食べる → 食べます
- する → します

## 例文

### 尊敬語の例

1. **先生がお書きになった論文を読みました。**
   - The teacher wrote a paper (and) I read it.
   - 我讀了老師寫的論文。

2. **部長はもう帰られました。**
   - The department head has already gone home.
   - 部長已經回家了。

3. **社長がおっしゃったことは正しいです。**
   - What the president said is correct.
   - 社長說的是對的。

### 謙譲語の例

4. **明日、お伺いいたします。**
   - I will visit (you) tomorrow.
   - 明天我會去拜訪。

5. **資料をお送りします。**
   - I will send you the materials.
   - 我會寄送資料給您。

6. **先生に申し上げたいことがあります。**
   - There is something I would like to say to the teacher.
   - 我有話想對老師說。

### 丁寧語の例

7. **毎朝、コーヒーを飲みます。**
   - I drink coffee every morning.
   - 我每天早上喝咖啡。

8. **週末は友達と会います。**
   - I meet with friends on weekends.
   - 週末我會和朋友見面。

## 使い分けのポイント

### 尊敬語を使う場面
- 目上の人（上司、先生、お客様）の動作
- 例：「課長が**お読みになります**」

### 謙譲語を使う場面
- 自分の動作を相手に対して述べる
- 例：「私が**お読みします**」

### 丁寧語を使う場面
- 誰の動作でも丁寧に表現したい時
- 例：「毎日本を**読みます**」

## 特別な敬語動詞

### 尊敬語

| 普通形 | 尊敬語 | 意味 |
|--------|--------|------|
| 言う | おっしゃる | say |
| 行く / 来る / いる | いらっしゃる | go / come / be |
| 食べる / 飲む | 召し上がる | eat / drink |
| 見る | ご覧になる | look / see |
| 知る | ご存じ | know |
| する | なさる | do |

### 謙譲語

| 普通形 | 謙譲語 | 意味 |
|--------|--------|------|
| 言う | 申す / 申し上げる | say |
| 行く | 伺う / 参る | go |
| 来る | 参る | come |
| いる | おる | be |
| 食べる / 飲む | いただく | eat / drink |
| 見る | 拝見する | look / see |
| 聞く | 伺う / 承る | hear / ask |
| する | いたす | do |
| もらう | いただく | receive |

## 注意点

### 1. 二重敬語を避ける
❌ **お召し上がりになる**（召し上がる + お〜になる）
✅ **召し上がる** または **お食べになる**

### 2. 尊敬語と謙譲語の混同
❌ 私が**伺います**（自分の動作に尊敬語）
✅ 私が**参ります**（自分の動作に謙譲語）

### 3. 相手の動作に謙譲語を使わない
❌ 先生が**拝見する**
✅ 先生が**ご覧になる**

## 学習のヒント

1. **まず丁寧語から**：「です・ます」の使い方を確実にする
2. **特別動詞を覚える**：よく使う動詞の特別な敬語形を暗記
3. **状況で判断**：誰の動作かを意識して尊敬語・謙譲語を選ぶ
4. **二重敬語に注意**：過度な敬語は不自然になる
