## 日文

〜なければならない

### 文法情報

| 項目 | 內容 |
|------|------|
| 文法類型 | 義務表現 |
| JLPT | N4 |
| 使用場面 | 正式、書面語、客觀義務 |
| 形成 | 動詞ない形 + ければならない |

### 形成規則

**基本形式**
```
動詞ない形 + ければならない
```

**範例**
```
食べる → 食べない → 食べなければならない
行く → 行かない → 行かなければならない
する → しない → しなければならない
来る → 来ない → 来なければならない
```

**縮約形式（口語）**
```
なければならない → なきゃならない → なきゃ
```

## 日文解釋

「なければならない」は「〔が形容詞の仮定形の場合〕ないというわけにはいかない」という意味です。[^dict]

この文法形式は、話し手や主語が何かをする必要性や義務があることを表します。「なければ」は「ない」の仮定形で、「ならない」は「いけない、だめだ」という意味を持ちます。直訳すると「〜しないならば、いけない」となり、つまり「〜しなければいけない」「〜する必要がある」という義務や必要性を示します。

この表現は比較的フォーマルで、書き言葉や公式な場面で頻繁に使用されます。日常会話では「なくてはいけない」や「ないといけない」といった、より口語的な形式がよく使われます。また、口語ではさらに「なきゃならない」や「なきゃ」といった縮約形も一般的です。

「なければならない」は、個人的な希望や意志ではなく、状況や規則、社会的な要求によって課される客観的な義務を表現する際に適しています。そのため、法律、規則、職務上の要求、社会的な慣習などを説明する際によく使われます。

[^dict]: 定義參考 スーパー大辞林（macOS 內建字典）

## 英文解釋

"〜なければならない" is a formal grammatical pattern expressing obligation or necessity, equivalent to "must" or "have to" in English. The structure combines the conditional form of the negative verb (〜なければ, "if not") with ならない ("won't do/is not acceptable"), literally meaning "if [one] doesn't do [something], it won't do."

This pattern emphasizes objective necessity imposed by circumstances, rules, or social expectations rather than personal desire. It's commonly used in formal contexts, written language, legal documents, and official situations. In everyday conversation, speakers often use less formal variants like "〜なくてはいけない" or "〜ないといけない," and colloquial contractions like "〜なきゃ."

The pattern can be applied to any verb by converting it to its negative form (ない form) and replacing ない with なければならない. This creates a strong sense of duty or requirement that must be fulfilled.

## 中文解釋

「〜なければならない」是表達「必須」「不得不」的正式文法形式，相當於英文的 "must" 或 "have to"。這個結構結合了動詞否定形的假定形（〜なければ，「如果不」）與ならない（「不行」「不可以」），字面意思是「如果不做某事的話就不行」。

此文法模式強調由情況、規則或社會期待所施加的客觀必要性，而非個人意願。它常用於正式場合、書面語、法律文件和官方情境。在日常對話中，說話者通常使用較不正式的變體如「〜なくてはいけない」或「〜ないといけない」，以及口語縮約形式如「〜なきゃ」。

此文法模式可以應用於任何動詞，只需將動詞轉換成否定形（ない形），然後將ない替換成なければならない。這會產生強烈的義務感或必須履行的要求。

## 核心用法

### 用法 1：法律或規則上的義務

表達由法律、規則、制度所規定的客觀義務。

**例句 1**
```
日本では20歳以上でなければお酒を飲んではならない。
In Japan, you must not drink alcohol unless you are 20 years old or older.
在日本，必須年滿20歲才能喝酒。
```

**例句 2**
```
図書館では静かにしなければならない。
You must be quiet in the library.
在圖書館裡必須保持安靜。
```

### 用法 2：職務或責任上的必要性

表達工作、職位或角色所要求的責任和義務。

**例句 3**
```
医者として、患者の秘密を守らなければならない。
As a doctor, I must keep patient confidentiality.
作為醫生，必須保守患者的秘密。
```

**例句 4**
```
明日までにこの書類を提出しなければならない。
I must submit these documents by tomorrow.
我必須在明天之前提交這些文件。
```

### 用法 3：社會常識或生活上的必要事項

表達為了達成某個目標或維持正常生活所需要做的事。

**例句 5**
```
健康を維持するためには、バランスの取れた食事をしなければならない。
To maintain health, you must eat a balanced diet.
為了維持健康，必須攝取均衡的飲食。
```

## 文法規則

### 規則 1：動詞變化

將動詞轉換為ない形，然後將ない替換為なければならない。

- 一段動詞：去掉る，加ない → なければならない
  - 食べる → 食べない → 食べなければならない
- 五段動詞：將詞尾改為あ段，加ない → なければならない
  - 書く → 書かない → 書かなければならない
- 不規則動詞：
  - する → しない → しなければならない
  - 来る（くる）→ 来ない（こない）→ 来なければならない

### 規則 2：否定形式

要表達「不必要」「不需要」時，使用「〜なくてもいい」或「〜必要はない」。

- ○ 行かなくてもいい（不用去）
- × 行かなければならなくない（錯誤）

### 規則 3：時態變化

時態變化反映在「ならない」部分。

- 現在/未來：なければならない
- 過去：なければならなかった（必須做過，但可能沒做）
- 過去完成義務：なければならなかった（表達過去的義務）

### 規則 4：口語縮約

在日常對話中，常用以下縮約形式：

- なければならない → なくちゃならない → なくちゃ
- なければならない → なきゃならない → なきゃ
- なければならない → なくてはならない → なくては

## 常見錯誤

### 錯誤 1：混淆否定形式

❌ 誤：行かないなければならない
✅ 正：行かなければならない
說明：ない形的ない要去掉，直接接ければならない。不要保留ない。

### 錯誤 2：錯誤表達「不必要」

❌ 誤：食べなければならなくない（想表達「不需要吃」）
✅ 正：食べなくてもいい
說明：「不必要做」應該用「〜なくてもいい」，而非將なければならない變成否定形。

### 錯誤 3：與べきだ混用

❌ 誤：早く寝るべきならない（想表達「應該早點睡」）
✅ 正：早く寝なければならない / 早く寝るべきだ
說明：べきだ和なければならない是兩種不同的表達，不能混合使用。べきだ表示建議或道德義務，なければならない表示客觀必要性。

## 學習要點

1. **正式程度**：なければならない是較正式的表達，書面語和正式場合使用；日常對話建議使用なくてはいけない或ないといけない
2. **客觀義務**：強調由外部因素（規則、法律、情況）所產生的義務，而非個人意願
3. **縮約形式**：口語中可縮約為なきゃならない或なきゃ，但要注意場合的正式程度
4. **否定表達**：「不必要」用なくてもいい，不是用なければならない的否定形
5. **與べきだ的區別**：なければならない強調客觀必要性和規則，べきだ強調道德建議和應該做的事
