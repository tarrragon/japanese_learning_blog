## 日文

受身形（うけみけい）

**羅馬拼音**：ukemi-kei

**文法類型**：動詞變化

### 形成規則

**基本形式**

受身形是日語動詞的一種活用形，用來表示動作的接受者，相當於英語的被動語態。

**る動詞（一段動詞）**
```
辞書形：る → 受身形：られる
食べる → 食べられる
見る → 見られる
```

**う動詞（五段動詞）**
```
辞書形：う段 → 受身形：あ段 + れる
書く → 書かれる
読む → 読まれる
話す → 話される
```

**不規則動詞**
```
する → される
来る → 来られる（こられる）
```

## 日文解釋

受身形は、動作を受ける側の視点から表現する日本語の重要な文法形式です。英語のpassive voiceに相当しますが、日本語特有の用法として「迷惑の受身」という表現があります。

この文法形式の特徴は、主語が動作を自分で行うのではなく、他者からの動作を受ける立場にあることを示す点です。日本語では、話者の視点や感情を表現する際に、この受身形が頻繁に使用されます。

受身形には大きく分けて三つの用法があります。第一に「直接受身」は、動作を直接受ける通常の被動表現です（例：「先生に褒められた」）。第二に「迷惑の受身」は、日本語特有の表現で、動作によって間接的に影響を受けることを表します（例：「雨に降られた」）。第三に「持ち主の受身」は、所有物に対する動作を表現します（例：「財布を盗まれた」）。

文化的な観点から見ると、日本語の受身形は、話者の謙遜や委婉的な表現に適しています。自分の行為を直接的に述べるよりも、受身形を使うことで、より控えめで丁寧な印象を与えることができます。

また、受身形と可能形が同じ形になる一段動詞の場合、文脈や助詞（「に」vs「が」）によって意味を判別する必要があります。

## 英文解釋

The passive form (ukemi-kei) is a crucial grammatical structure in Japanese that expresses actions from the perspective of the receiver. While it corresponds to the passive voice in English, Japanese has unique uses, particularly the "adversity passive" (meiwaku no ukemi).

The key characteristic of this form is that the subject doesn't perform the action themselves but receives it from another party. In Japanese, this passive form is frequently used to express the speaker's perspective and emotions.

There are three main types of passive constructions. First, the "direct passive" is a standard passive expression where the action is directly received (e.g., "sensei ni homerareta" - was praised by the teacher). Second, the "adversity passive" is unique to Japanese, expressing indirect effects of actions (e.g., "ame ni furareta" - got rained on/suffered from rain). Third, the "possessive passive" expresses actions toward one's possessions (e.g., "saifu wo nusumareta" - had wallet stolen).

From a cultural perspective, the passive form in Japanese is well-suited for expressing humility and indirectness. Using the passive form instead of stating one's own actions directly can create a more modest and polite impression.

Additionally, for ichidan verbs where the passive and potential forms are identical, meaning must be distinguished through context and particles ("ni" for passive vs "ga" for potential).

## 中文解釋

受身形（被動形）是日語中重要的文法結構，用於從接受者的視角表達動作。雖然對應英語的被動語態，但日語有其獨特用法，特別是「迷惑被動」（meiwaku no ukemi）。

這個文法形式的關鍵特徵是，主語不是自己執行動作，而是接受他人的動作。在日語中，經常使用被動形來表達說話者的視角和情感。

被動形主要有三種用法。第一，「直接被動」是標準的被動表達，直接接受動作（例如：「先生に褒められた」被老師稱讚）。第二，「迷惑被動」是日語特有的表達，表示間接受到動作影響（例如：「雨に降られた」遇到下雨）。第三，「所有者被動」表達對所有物的動作（例如：「財布を盗まれた」錢包被偷）。

從文化角度來看，日語的被動形適合表達謙遜和委婉。使用被動形而非直接陳述自己的行為，可以營造更謙虛、有禮的印象。

此外，對於一段動詞而言，被動形和可能形的形式相同，需要透過語境和助詞（被動用「に」，可能用「が」）來區分意思。

## 核心用法

### 用法 1：直接被動（肯定情境）

表達接受正面或中性的動作。

**例句 1**
```
先生に褒められて、とても嬉しかったです。
I was very happy to be praised by my teacher.
被老師稱讚，非常高興。
```

**例句 2**
```
友達にパーティーに誘われました。
I was invited to a party by my friend.
被朋友邀請參加派對。
```

### 用法 2：迷惑被動（日語特有）

表達因他人行為而間接受到負面影響或困擾。這是日語獨特的表達方式。

**例句 3**
```
出かける時に雨に降られて、服が濡れてしまった。
I got caught in the rain when I went out, and my clothes got wet.
出門時遇到下雨，衣服都濕了。
```

**例句 4**
```
電車の中で隣の人にタバコを吸われて、困りました。
I was troubled when the person next to me smoked on the train.
在電車上旁邊的人抽菸，讓我很困擾。
```

### 用法 3：所有者被動

表達自己的所有物或身體部位受到動作影響。

**例句 5**
```
駅で財布を盗まれてしまいました。
I had my wallet stolen at the station.
在車站錢包被偷了。
```

## 文法規則

### 規則 1：動詞變化規則

**一段動詞（る動詞）**
- 去掉「る」，加上「られる」
- 食べる → 食べられる
- 見る → 見られる
- 教える → 教えられる

**五段動詞（う動詞）**
- 將語尾從う段變為あ段，加上「れる」
- 書く → 書かれる（かく → かかれる）
- 読む → 読まれる（よむ → よまれる）
- 話す → 話される（はなす → はなされる）

**不規則動詞**
- する → される
- 来る（くる）→ 来られる（こられる）

### 規則 2：助詞的使用

**動作主用「に」標記**
- 先生**に**褒められる（被老師稱讚）
- 友達**に**笑われる（被朋友笑）

**直接受身的賓語用「を」**
- 犬**に**足**を**噛まれる（腳被狗咬）

**迷惑被動中的動作**
- 雨**に**降られる（遇到下雨）
- 赤ちゃん**に**泣かれる（被嬰兒哭鬧）

### 規則 3：與可能形的區別（一段動詞）

一段動詞的受身形和可能形形式相同，需透過助詞判斷：

**受身形：用「に」**
- この本は多くの人**に**読まれている。
- （這本書被很多人閱讀）

**可能形：用「が」**
- 漢字**が**読める。
- （能讀漢字）

## 常見錯誤

### 錯誤 1：動作主使用錯誤的助詞

❌ 誤：先生によって褒められた。
✅ 正：先生に褒められた。
說明：日語口語中，動作主通常用「に」，不用「によって」。「によって」較正式，多用於書面語或強調動作主時。

### 錯誤 2：混淆受身形和可能形（一段動詞）

❌ 誤：魚が食べられる。（想表達「被魚吃」）
✅ 正：魚に食べられる。
說明：一段動詞的受身形和可能形相同。「が」表示可能（能吃魚），「に」表示受身（被魚吃）。

### 錯誤 3：誤用迷惑被動

❌ 誤：母に料理を作られて、嬉しかった。
✅ 正：母が料理を作ってくれて、嬉しかった。
說明：迷惑被動通常表示負面影響或困擾。表達正面感激應該使用「〜てくれる」。

## 學習要點

1. **三種被動類型**：直接被動、迷惑被動、所有者被動，各有不同的使用情境和語感。

2. **迷惑被動是關鍵**：這是日語特有的表達，英語和中文沒有對應結構，常表達困擾或負面影響。

3. **助詞的重要性**：動作主用「に」，區別受身形和可能形（一段動詞）時，「に」是受身，「が」是可能。

4. **文化意涵**：受身形反映日語重視接受者視角的特性，常用於謙遜和委婉表達。

5. **變化規則必須熟記**：一段動詞、五段動詞、不規則動詞的變化規則各不相同，需要反覆練習。
