## 日文

可能動詞（かのうどうし）

### 文法情報

| 項目 | 內容 |
|------|------|
| 文法類型 | 動詞派生形態 |
| JLPT | N3 |
| 使用場面 | 日常會話、書面語 |
| 形成 | 五段動詞語尾う段→え段＋る |

### 形成規則

**基本形式**

五段動詞的可能動詞變化規則：將語尾的う段假名改為相應的え段假名，再加上「る」。

**變化對照表**

| 語尾 | 變化 | 例 |
|------|------|------|
| う | える | 買う → 買える |
| く | ける | 書く → 書ける |
| ぐ | げる | 泳ぐ → 泳げる |
| す | せる | 話す → 話せる |
| つ | てる | 待つ → 待てる |
| ぬ | ねる | 死ぬ → 死ねる |
| ぶ | べる | 飛ぶ → 飛べる |
| む | める | 読む → 読める |
| る | れる | 帰る → 帰れる |

## 日文解釋

可能動詞とは、五段動詞から派生した動詞で、「〜することができる」という可能の意味を表す。五段動詞の語尾をエ段に変え、「る」を付けることで形成される。例えば、「書く」は「書ける」、「読む」は「読める」となる。

形成された可能動詞は、文法的には一段動詞（下一段活用）として扱われる。つまり、「書ける」は「書け・ます」「書け・ない」「書け・て」のように活用する。

可能動詞は五段動詞にのみ適用される概念であり、一段動詞の可能形「食べられる」や、不規則動詞の可能形「できる」「来られる」は、可能動詞とは呼ばない。これらは受身形と同形になるため、区別が必要である。

現代日本語では、可能動詞は非常に自然で使用頻度が高い。「〜ことができる」より簡潔で、日常会話でよく用いられる。

## 英文解釋

A potential verb (可能動詞) is a derived verb form exclusive to godan (five-grade) verbs that expresses the ability or possibility to perform an action. It is formed by changing the final u-vowel syllable of a godan verb to its corresponding e-vowel form and adding る (ru).

Once formed, potential verbs function grammatically as ichidan (one-grade) verbs. For example, 書ける (kakeru, "can write") conjugates as 書けます, 書けない, 書けて, following the ichidan pattern.

This concept applies only to godan verbs. The potential forms of ichidan verbs (e.g., 食べられる) and irregular verbs (できる, 来られる) are not called potential verbs, as they share their form with the passive voice.

## 中文解釋

可能動詞是專屬於五段動詞的派生形態，用於表達「能夠做某事」的意思。形成方式是將五段動詞語尾的う段假名改為相應的え段假名，再加上「る」。

形成後的可能動詞在文法上視為一段動詞（下一段活用），例如「書ける」的活用為「書けます」「書けない」「書けて」等。

需要注意的是，可能動詞這個術語只適用於五段動詞。一段動詞的可能形「食べられる」和不規則動詞的可能形「できる」「来られる」不稱為可能動詞，因為它們與被動形同形，需要加以區別。

可能動詞比「〜ことができる」更簡潔自然，在日常會話中使用頻率極高。

## 核心用法

### 用法 1：表達能力

表示主體具有完成某動作的能力。

**例句 1**
```
彼は三か国語が話せる。
He can speak three languages.
他會說三國語言。
```

**例句 2**
```
この子はもう一人で歩ける。
This child can already walk by himself/herself.
這個孩子已經能自己走路了。
```

### 用法 2：表達可能性

表示某件事在條件上是可能的。

**例句 3**
```
今日の会議は午後三時から始められる。
Today's meeting can start from 3 PM.
今天的會議可以從下午三點開始。
```

### 用法 3：表達許可（間接）

透過可能表達來委婉詢問或表示許可。

**例句 4**
```
ここで写真が撮れますか。
Can I take photos here?
這裡可以拍照嗎？
```

**例句 5**
```
図書館では飲み物が飲めません。
You cannot drink beverages in the library.
圖書館裡不能喝飲料。
```

## 文法規則

### 規則 1：助詞變化

使用可能動詞時，原本標示對象的助詞「を」經常變為「が」。但現代日語中，「を」的使用也逐漸被接受。

- 本を読む → 本が読める（傳統用法）
- 本を読む → 本を読める（現代用法，也可接受）

### 規則 2：活用方式

可能動詞作為一段動詞活用：

| 活用形 | 書ける | 読める |
|--------|--------|--------|
| 未然形 | 書け | 読め |
| 連用形 | 書け | 読め |
| 終止形 | 書ける | 読める |
| 連体形 | 書ける | 読める |
| 仮定形 | 書けれ | 読めれ |
| 命令形 | 書けろ | 読めろ |

### 規則 3：與可能形的關係

「可能動詞」和「可能形」常被混用，但嚴格來說：
- **可能動詞**：僅指五段動詞派生出的形態（書ける、読める等）
- **可能形**：泛指所有表達可能意義的動詞變化（包括食べられる、できる等）

## 常見錯誤

### 錯誤 1：將一段動詞可能形誤稱為可能動詞

❌ 誤：「食べられる」は可能動詞です。
✅ 正：「食べられる」は一段動詞「食べる」の可能形です。可能動詞とは呼びません。
說明：可能動詞專指五段動詞的派生形態。一段動詞的可能形與被動形同形，不稱為可能動詞。

### 錯誤 2：可能動詞的活用錯誤

❌ 誤：日本語が話さない。（用五段活用）
✅ 正：日本語が話せない。（用一段活用）
說明：可能動詞「話せる」是一段動詞，否定形應為「話せない」，不是「話さない」。

### 錯誤 3：混淆「する」的可能形

❌ 誤：彼は運動がすれる。
✅ 正：彼は運動ができる。
說明：「する」是不規則動詞，其可能形是「できる」，不是「すれる」。

## 學習要點

1. **僅適用五段動詞**：可能動詞這個術語專指五段動詞的派生形態，一段動詞和不規則動詞不適用。

2. **活用為一段動詞**：形成後的可能動詞按照一段動詞規則活用，這是學習者常犯錯的地方。

3. **助詞「が」優先**：使用可能動詞時，對象通常用「が」標示，雖然「を」也逐漸被接受。

4. **與被動形區別**：五段動詞的可能動詞（書ける）和被動形（書かれる）形態不同，容易區分；但一段動詞兩者同形（食べられる），需要靠語境判斷。

5. **日常使用頻繁**：可能動詞比「〜ことができる」更常用於日常會話，掌握後能讓日語表達更自然。
