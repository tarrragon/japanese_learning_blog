## 日文解釋

「いる」は「人・動物がその場所に存在する」という意味です。[^dict]

日本語の存在動詞は、主語の「生命性」によって使い分けられる。「いる」は人間や動物など、自らの意思で移動できる存在に対して使用する。一方、「ある」は物品、植物、抽象的な概念など、自らの意思で移動しない存在に対して使用する。

この区別は日本語学習において非常に重要であり、英語の "be" や中国語の「有」「在」とは異なる体系を持つ。特に、植物は生物であるが「ある」を使用する点、またロボットや人形など人間に似た形態のものでも「ある」を使用する点に注意が必要である。

ただし、ペットや愛着のある物に対しては「いる」を使用することもあり、話者の心理的な認識が反映される場合もある。

[^dict]: 定義參考 スーパー大辞林（macOS 內建字典）

## 英文解釋

Japanese has two existence verbs: **いる** (iru) for animate beings and **ある** (aru) for inanimate objects. This distinction, based on animacy, is fundamental to Japanese grammar.

**いる** is used for:
- Humans (people, characters)
- Animals (pets, wild animals, insects)
- Any entity perceived as having agency or consciousness

**ある** is used for:
- Objects (books, tables, buildings)
- Plants (trees, flowers)
- Abstract concepts (time, opportunities, problems)
- Events (meetings, parties)

Unlike English "there is/are" or "exist," Japanese requires speakers to make an animacy distinction with every usage.

## 中文解釋

日文的存在動詞分為兩個：**いる**（用於有生命的主體）和 **ある**（用於無生命的事物）。這個區別是日文文法的基礎。

**いる** 適用於：
- 人類（人、角色）
- 動物（寵物、野生動物、昆蟲）
- 任何被視為有意識或能自主移動的存在

**ある** 適用於：
- 物品（書本、桌子、建築物）
- 植物（樹木、花朵）
- 抽象概念（時間、機會、問題）
- 事件（會議、派對）

與中文的「有」「在」不同，日文要求說話者在每次使用時都必須區分主語的生命性。

## 核心用法

### 用法 1：表示有生命的存在（いる）

用於描述人或動物在某個地點存在。

**例句 1**
```
教室に学生がいます。
There are students in the classroom.
教室裡有學生。
```

**例句 2**
```
公園で犬が遊んでいる。
A dog is playing in the park.
狗在公園裡玩。
```

### 用法 2：表示無生命的存在（ある）

用於描述物品、植物或抽象事物的存在。

**例句 3**
```
机の上に本があります。
There is a book on the desk.
桌上有一本書。
```

**例句 4**
```
庭に大きな桜の木がある。
There is a large cherry tree in the garden.
庭院裡有一棵大櫻花樹。
```

### 用法 3：表示所有（ある）

「ある」也可表示擁有某物。

**例句 5**
```
私には時間がありません。
I don't have time.
我沒有時間。
```

## 文法規則

### 規則 1：生命性判斷

判斷主語是否具有「自主移動能力」：
- 能自主移動 → いる（人、動物）
- 不能自主移動 → ある（物品、植物）

### 規則 2：否定形的特殊變化

「ある」的否定形是「ない」，而非「あらない」。這是日文中的不規則變化。

| 肯定 | 否定 |
|------|------|
| いる | いない |
| ある | ない（特殊） |

### 規則 3：場所標記助詞

存在的場所通常用「に」標記：
- 場所 + に + 主語 + が + いる／ある

## 常見錯誤

### 錯誤 1：植物使用「いる」

❌ 誤：庭に花がいます。
✅ 正：庭に花があります。
說明：植物雖然是生物，但不具自主移動能力，因此使用「ある」。

### 錯誤 2：「ある」的否定形錯誤

❌ 誤：本があらない。
✅ 正：本がない。
說明：「ある」的否定形是不規則的「ない」。

### 錯誤 3：機器人使用「いる」

❌ 誤：ロボットがいます。
✅ 正：ロボットがあります。
說明：機器人雖有人形，但不是生物，使用「ある」。不過，在擬人化的語境中，有時也可使用「いる」。

## 學習要點

1. **生命性是關鍵**：判斷主語是否能自主移動來選擇動詞
2. **植物用「ある」**：雖然植物是生物，但因不能自主移動，使用「ある」
3. **「ない」是特殊形**：「ある」的否定形是「ない」，這是重要的不規則變化
4. **心理認知可影響選擇**：對寵物或擬人化的事物，有時可使用「いる」
5. **與中英文不同**：不能直接對應「有」「在」或 "there is"，需要理解日文獨特的生命性區別
