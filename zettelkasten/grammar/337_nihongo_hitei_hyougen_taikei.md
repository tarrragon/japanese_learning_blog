## 日文解釋

日本語の否定表現は、動詞、形容詞、名詞のそれぞれに固有の変化規則を持つ体系的なシステムである。否定を表す最も基本的な形は助動詞「ない」であり、これは動詞の未然形に接続する。

動詞の否定形には、普通体（ない形）と丁寧体（ません形）の二つの主要なスタイルがある。五段動詞は語尾が「ア段」に変化してから「ない」が付き、一段動詞は語幹に直接「ない」が付く。例えば、「書く」→「書かない」、「食べる」→「食べない」となる。

形容詞の場合、イ形容詞は「い」を「く」に変えて「ない」を付ける（例：高い→高くない）。ナ形容詞と名詞は「では（じゃ）ない」の形を取る（例：静か→静かではない、学生→学生ではない）。

否定表現は単なる肯定の反対ではなく、部分否定、二重否定、婉曲否定など様々なニュアンスを表現できる豊かな文法体系である。

## 英文解釋

The Japanese negation system encompasses distinct conjugation patterns for verbs, adjectives, and nouns, forming a comprehensive grammatical framework for expressing denial, absence, or contrary states.

For verbs, negation is primarily achieved through the auxiliary "nai" attached to the negative stem (mizenkei). Godan verbs undergo vowel changes (e.g., kaku → kakanai), while ichidan verbs simply drop the final "ru" before adding "nai" (e.g., taberu → tabenai). The polite negative form uses "masen" instead.

Adjectives follow different patterns: i-adjectives replace the final "i" with "kunai" (e.g., takai → takakunai), while na-adjectives and nouns use "dewa nai" or the colloquial "ja nai" construction (e.g., shizuka → shizuka dewa nai).

Understanding Japanese negation requires mastering not just the morphological changes but also the pragmatic nuances, including partial negation, double negatives for emphasis, and euphemistic negative expressions commonly used in polite discourse.

## 中文解釋

日文否定表現體系是一套完整的語法系統，涵蓋動詞、形容詞、名詞等各類詞性的否定變化規則。

動詞否定主要透過助動詞「ない」來實現，接續於動詞的未然形。五段動詞需要先將詞尾變成「ア段」音再加「ない」（如：書く→書かない），一段動詞則去掉「る」直接加「ない」（如：食べる→食べない）。丁寧體否定使用「ません」形式。

イ形容詞的否定是將詞尾「い」變成「く」再加「ない」（如：高い→高くない）。ナ形容詞和名詞則使用「ではない」或口語的「じゃない」形式（如：静か→静かではない、学生→学生ではない）。

日文否定不僅僅是肯定的反面，還包含部分否定、雙重否定、婉轉否定等豐富的表達方式。掌握否定體系是理解日文語法結構和表達細膩語意的重要基礎。

## 核心用法

### 用法 1：動詞普通體否定（ない形）

動詞的基本否定形式，用於非正式場合或修飾其他詞語時。

**例句 1**
```
明日は会社に行かない。
I won't go to the office tomorrow.
我明天不去公司。
```

**例句 2**
```
この本はまだ読んでいない。
I haven't read this book yet.
這本書我還沒讀。
```

### 用法 2：動詞丁寧體否定（ません形）

正式場合或對上位者使用的禮貌否定形式。

**例句 3**
```
申し訳ありませんが、その件については存じません。
I'm sorry, but I don't know about that matter.
非常抱歉，關於那件事我不清楚。
```

### 用法 3：形容詞否定

イ形容詞和ナ形容詞各有不同的否定變化方式。

**例句 4**
```
今日の天気はあまり良くない。
Today's weather isn't very good.
今天的天氣不太好。
```

**例句 5**
```
この部屋は静かではないので、集中できない。
This room isn't quiet, so I can't concentrate.
這個房間不安靜，所以無法集中精神。
```

## 常見錯誤

### 錯誤 1：五段動詞否定形變化錯誤

❌ 誤：書くない
✅ 正：書かない
說明：五段動詞需要將詞尾變成「ア段」後再加「ない」。「書く」的未然形是「書か」，不是「書く」本身。

### 錯誤 2：イ形容詞否定形錯誤

❌ 誤：高いくない / 高いじゃない
✅ 正：高くない
說明：イ形容詞否定時要去掉「い」變成「く」，再加「ない」。不能直接加「くない」或用「じゃない」。

### 錯誤 3：ナ形容詞與名詞否定混淆

❌ 誤：静かくない
✅ 正：静かではない / 静かじゃない
說明：ナ形容詞不使用「くない」形式，而是使用「ではない」或口語的「じゃない」。

## 學習要點

1. **動詞分類是關鍵**：五段動詞和一段動詞的否定變化規則不同，必須先正確識別動詞類型。

2. **普通體與丁寧體的選擇**：根據場合和對話對象選擇適當的否定形式，正式場合使用「ません」，日常會話使用「ない」。

3. **形容詞詞性判斷**：イ形容詞和ナ形容詞的否定方式完全不同，需要記住常見的ナ形容詞以避免混淆。

4. **過去否定的形成**：「ない」變成「なかった」，「ません」變成「ませんでした」，這是否定過去式的基本規則。

5. **口語與書面語的差異**：「ではない」是書面語，「じゃない」是口語形式，兩者在不同場合使用。
