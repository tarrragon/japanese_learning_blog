## 日文

つもりだ

### 文法情報

| 項目 | 內容 |
|------|------|
| 文法類型 | 意圖表達 |
| JLPT | N4 |
| 使用場面 | 日常、正式 |
| 形成 | 動詞辭書形 / ない形 + つもりだ |

### 形成規則

**基本形式**
```
動詞辭書形 + つもりだ
動詞ない形 + つもりだ
```

**範例**
```
行く → 行くつもりだ（打算去）
行かない → 行かないつもりだ（不打算去）
勉強する → 勉強するつもりだ（打算學習）
```

## 日文解釋

「つもりだ」は、話し手の意図や計画を表す文法表現です。将来の行動について、すでに心の中で決めていることを述べるときに使います。「つもり」は名詞で、「意図」や「予定」という意味を持ちます。

この表現は、確定した予定ではなく、あくまでも話し手の意志や考えを表します。予定を表す「予定だ」よりも主観的で、話し手の決意や意図を強調します。また、否定形「〜ないつもりだ」を使うことで、「〜するつもりはない」という意思の不在も表現できます。

丁寧な場面では「つもりです」、過去の意図を説明する場合は「つもりでした」「つもりだった」という形も使われます。日常会話でも正式な場面でも広く使用される、意図表現の基本的な文法パターンです。

## 英文解釋

"Tsumori da" is a grammatical expression used to convey the speaker's intention or plan. It indicates that the speaker has already decided on a future action in their mind. "Tsumori" is a noun meaning "intention" or "plan."

This expression represents the speaker's will or thoughts rather than a confirmed schedule. It is more subjective than "yotei da" (scheduled to), emphasizing the speaker's determination or intention. The negative form "~nai tsumori da" can express the absence of intention, meaning "don't intend to do."

In polite situations, it becomes "tsumori desu." To explain past intentions, forms like "tsumori deshita" or "tsumori datta" are used. This is a fundamental grammatical pattern for expressing intentions, widely used in both casual and formal contexts.

## 中文解釋

「つもりだ」是表達說話者意圖或計劃的文法表達。它表示說話者已經在心中決定了未來的行動。「つもり」是名詞，意思是「意圖」或「計劃」。

這個表達方式呈現的是說話者的意志或想法，而不是確定的行程安排。相比「予定だ」（預定要），它更主觀，強調說話者的決心或意圖。否定形「〜ないつもりだ」可以表達沒有意圖，意思是「不打算做」。

在禮貌的場合使用「つもりです」，說明過去的意圖時使用「つもりでした」或「つもりだった」。這是表達意圖的基本文法模式，在日常會話和正式場合都廣泛使用。

## 核心用法

### 用法 1：表達肯定的意圖

說明未來打算做的事情，強調說話者的主觀意志。

**例句 1**
```
来年日本へ留学するつもりです。
I intend to study abroad in Japan next year.
明年我打算去日本留學。
```

**例句 2**
```
大学を卒業したら、教師になるつもりだ。
After graduating from university, I plan to become a teacher.
大學畢業後，我打算當老師。
```

### 用法 2：表達否定的意圖

使用「ない形 + つもりだ」表示不打算做某事。

**例句 3**
```
今日は残業しないつもりです。
I don't intend to work overtime today.
今天我不打算加班。
```

**例句 4**
```
彼には何も言わないつもりだった。
I didn't intend to say anything to him.
我本來不打算對他說什麼。
```

### 用法 3：說明過去的意圖

使用過去式「つもりでした/つもりだった」說明當時的打算。

**例句 5**
```
本当は昨日連絡するつもりでしたが、忘れてしまいました。
I actually intended to contact you yesterday, but I forgot.
我其實打算昨天聯絡你，但忘記了。
```

## 文法規則

### 規則 1：接續形式

- **動詞辭書形 + つもりだ**：行くつもりだ、食べるつもりだ
- **動詞ない形 + つもりだ**：行かないつもりだ、食べないつもりだ
- **過去式**：つもりでした（禮貌）、つもりだった（普通）

### 規則 2：禮貌程度

- **普通體**：つもりだ
- **丁寧體**：つもりです
- **過去丁寧體**：つもりでした

### 規則 3：與其他意圖表達的區別

- **つもりだ**：主觀意圖，強調說話者的決心
- **予定だ**：客觀預定，已安排的行程
- **〜ようと思う**：剛形成的想法，較不確定

## 常見錯誤

### 錯誤 1：使用た形接續

❌ 誤：行ったつもりだ
✅ 正：行くつもりだ
說明：つもりだ只能接辭書形或ない形，不能接た形。如果要表達過去的意圖，應該使用「つもりでした」。

### 錯誤 2：混淆つもりだ和予定だ

❌ 誤：会議は3時に始まるつもりです。（會議主觀意圖不合理）
✅ 正：会議は3時に始まる予定です。
說明：會議時間是客觀安排，應該使用「予定だ」。つもりだ只用於表達說話者或主語的主觀意圖。

### 錯誤 3：否定位置錯誤

❌ 誤：行くつもりではない
✅ 正：行かないつもりだ
說明：否定意圖應該在動詞部分使用ない形，而不是否定つもりだ本身。

## 學習要點

1. **主觀意圖**：つもりだ表達的是說話者的主觀意圖和決心，不是客觀事實或安排
2. **接續規則**：只能接動詞辭書形或ない形，不接た形或其他形式
3. **否定表達**：表達「不打算」時使用「ない形 + つもりだ」，不是「つもりではない」
4. **時態變化**：現在的意圖用「つもりだ/です」，過去的意圖用「つもりでした/だった」
5. **與予定だ的區別**：つもりだ強調主觀意志，予定だ表達客觀安排
