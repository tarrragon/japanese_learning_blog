## 日文
五段動詞（ごだんどうし）/ う動詞

**羅馬拼音**: godan doushi / u-doushi

**別名**: Group I verbs（第一類動詞）、u-verbs

**類型**：動詞分類・活用文法

## 日文解釋

五段動詞（う動詞）は、日本語動詞の三大分類の一つで、最も数が多い動詞グループです。語幹[^gokan]の最後の音が五十音図[^gojuuon]の五段（あ・い・う・え・お）を使って活用[^katsuyou]するため「五段動詞」と呼ばれます。

### 基本特徴

**語幹の最後の音が変化する**
五段動詞の特徴は、活用するときに語幹の最後の音が変化することです。例えば「書く」の場合、「書」の部分は同じですが、「く」の部分が「か・き・く・け・こ」と変化します。

**見分け方**
1. 辞書形[^jisho-kei]の語尾が以下のいずれかで終わる：
   - う、く、ぐ、す、つ、ぬ、ぶ、む、る

2. 特に注意が必要な「〜る」：
   - 「る」の前が「あ・う・お段」→ 五段動詞
   - 例：入る（hairu）、帰る（kaeru）、走る（hashiru）

**活用の複雑さ**
五段動詞は[一段動詞](003_ichidan_verb.md)と比べて活用が複雑です。特に[て形](001_te_form.md)は6つの異なる変化パターンがあります。

### 五段活用の仕組み

五段動詞は語尾が五十音図の五段を使って変化します：

| 段 | 用途 | 例（書く） |
|----|------|----------|
| あ段 | ない形 | 書かない |
| い段 | ます形 | 書きます |
| う段 | 辞書形 | 書く |
| え段 | 命令形・仮定形 | 書け、書けば |
| お段 | 意志形 | 書こう |

### て形の6つのパターン

て形は特に重要で、以下の6つのパターンがあります：

| 辞書形語尾 | 変化 | て形 | 例 |
|-----------|------|------|-----|
| う、つ、る | → って | 買って | 買う → 買って |
| む、ぶ、ぬ | → んで | 飲んで | 飲む → 飲んで |
| く | → いて | 書いて | 書く → 書いて |
| ぐ | → いで | 泳いで | 泳ぐ → 泳いで |
| す | → して | 話して | 話す → 話して |
| 行く（例外）| → って | 行って | 行く → 行って（例外）|

## 英文解釋

Godan verbs (also called u-verbs or Group I verbs) are one of the three major verb categories in Japanese and constitute the largest group. They are called "five-step" verbs because their final stem sound conjugates using all five rows (a, i, u, e, o) of the Japanese syllabary.

### Key Characteristics

**Changing Final Stem Sound**
The defining feature of godan verbs is that the final sound of the stem changes during conjugation. For example, with "kaku" (to write), the "ka" part stays the same, but the "ku" changes to "ka, ki, ku, ke, ko."

**Identification Rules**
1. Dictionary form ends in one of:
   - u, ku, gu, su, tsu, nu, bu, mu, ru

2. Special attention needed for "~ru":
   - If the syllable before "ru" is in the a, u, or o row → godan verb
   - Examples: hairu (enter), kaeru (return), hashiru (run)

**Conjugation Complexity**
Godan verbs have more complex conjugation than [ichidan verbs](003_ichidan_verb.md). The [te-form](001_te_form.md) in particular has 6 different transformation patterns.

### Five-Step Conjugation System

Godan verbs use all five rows of the syllabary:

| Row | Usage | Example (kaku) |
|-----|-------|---------------|
| a-row | Negative form | kakanai |
| i-row | Masu form | kakimasu |
| u-row | Dictionary form | kaku |
| e-row | Imperative/Conditional | kake, kakeba |
| o-row | Volitional form | kakou |

### Six Te-form Patterns

The te-form is particularly important, with these 6 patterns:

| Dictionary Ending | Change | Te-form | Example |
|------------------|--------|---------|---------|
| u, tsu, ru | → tte | katte | kau → katte |
| mu, bu, nu | → nde | nonde | nomu → nonde |
| ku | → ite | kaite | kaku → kaite |
| gu | → ide | oyoide | oyogu → oyoide |
| su | → shite | hanashite | hanasu → hanashite |
| iku (exception) | → tte | itte | iku → itte (exception) |

## 中文解釋

五段動詞（又稱う動詞或第一類動詞）是日語動詞三大分類之一，也是數量最多的一組。因為詞幹的最後一個音會使用五十音圖的五段（あ・い・う・え・お）來變化，所以稱為「五段動詞」。

### 基本特徵

**詞幹最後的音會變化**
五段動詞的特點是變化時詞幹的最後一個音會改變。例如「書く」（寫），「書」的部分不變，但「く」會變成「か・き・く・け・こ」。

**識別方法**
1. 辭書形以下列任一假名結尾：
   - う、く、ぐ、す、つ、ぬ、ぶ、む、る

2. 特別需要注意的「〜る」：
   - 「る」前面是「あ・う・お段」→ 五段動詞
   - 例：入る（hairu）、帰る（kaeru）、走る（hashiru）

**變化的複雜性**
五段動詞比[一段動詞](003_ichidan_verb.md)的變化更複雜。特別是[て形](001_te_form.md)有 6 種不同的變化模式。

### 五段活用的機制

五段動詞的詞尾使用五十音圖的五段來變化：

| 段 | 用途 | 例（書く） |
|----|------|----------|
| あ段 | ない形 | 書かない |
| い段 | ます形 | 書きます |
| う段 | 辭書形 | 書く |
| え段 | 命令形・假定形 | 書け、書けば |
| お段 | 意志形 | 書こう |

### て形的 6 種模式

て形特別重要，有以下 6 種模式：

| 辭書形結尾 | 變化 | て形 | 例子 |
|-----------|------|------|------|
| う、つ、る | → って | 買って | 買う → 買って |
| む、ぶ、ぬ | → んで | 飲んで | 飲む → 飲んで |
| く | → いて | 書いて | 書く → 書いて |
| ぐ | → いで | 泳いで | 泳ぐ → 泳いで |
| す | → して | 話して | 話す → 話して |
| 行く（例外）| → って | 行って | 行く → 行って（例外）|

## 例句

**例句1：〜く型動詞**
```
毎日日記を書きます。
I write in my diary every day.
我每天寫日記。
```

**例句2：〜む型動詞（んで）**
```
水を飲んでから、運動します。
After drinking water, I exercise.
喝完水之後運動。
```

**例句3：〜す型動詞**
```
友達と日本語で話します。
I speak in Japanese with my friends.
我和朋友用日語說話。
```

**例句4：〜る型（五段）**
```
毎日6時に家に帰ります。
I return home at 6 o'clock every day.
我每天6點回家。
```

## 補充說明

### 對比：五段動詞 vs [一段動詞](003_ichidan_verb.md)

| 特徵 | 五段動詞 | 一段動詞 |
|------|---------|---------|
| 數量 | 約4000個（多） | 約200個（少） |
| 詞幹 | 最後的音會變化 | 詞幹不變 |
| 辭書形結尾 | う、く、ぐ、す、つ、ぬ、ぶ、む、る | 〜る（前面是e/i段） |
| ます形變化 | 語尾變い段 + ます | 去る + ます |
| て形規則 | 複雜（6種） | 簡單（去る + て） |
| ない形規則 | 語尾變あ段 + ない | 去る + ない |
| 學習難度 | 中等（規則多） | 簡單（規則單一） |
| 例子 | 書く、話す、飲む | 食べる、見る、寝る |

### 完整活用表（以「書く」為例）

| 活用形 | 規則 | 例：書く | 用途 |
|--------|------|---------|------|
| 辞書形 | 原形 | 書く | 基本形 |
| ます形 | い段 + ます | 書きます | 禮貌形 |
| て形 | いて（く型） | 書いて | 連接、進行 |
| た形 | いた（く型） | 書いた | 過去 |
| ない形 | あ段 + ない | 書かない | 否定 |
| 可能形 | え段 + る | 書ける | 能力 |
| 受身形 | あ段 + れる | 書かれる | 被動 |
| 使役形 | あ段 + せる | 書かせる | 使役 |
| 命令形 | え段 | 書け | 命令 |
| 意志形 | お段 + う | 書こう | 意志 |

### 特殊例外：「行く」

「行く」是唯一的例外，雖然是く結尾但て形是「行って」（不是「行いて」）：
- 辞書形：行く（iku）
- て形：行って（itte）← 例外
- た形：行った（itta）← 例外

### 看起來像五段動詞的一段動詞

注意：不是所有「〜る」結尾的動詞都是五段動詞。

| 動詞 | 類型 | 理由 |
|------|------|------|
| 見る (miru) | 一段 | 「る」前是「い段」且實際是一段 |
| 食べる (taberu) | 一段 | 「る」前是「え段」且實際是一段 |
| 寝る (neru) | 一段 | 「る」前是「え段」且實際是一段 |

**判斷技巧**：
- 看ます形：
  - 五段：帰る → 帰ります（り出現）
  - 一段：食べる → 食べます（直接去る）

### 學習要點

1. **先記住て形的6種模式** - 這是最常用且最重要的
2. **練習分辨る結尾的動詞** - 容易與一段動詞混淆
3. **記住「行く」的例外** - 唯一的特殊情況
4. **理解五段原理** - 幫助記住所有活用形

---

[^gokan]: 語幹（ごかん）- 動詞中不變化的部分。例如「書く」中的「書」就是語幹，而「く」是活用語尾。

[^gojuuon]: 五十音図（ごじゅうおんず）- 日語假名的排列表，按照發音規律排成五行十段。五段指的是あ、い、う、え、お五個母音段。

[^katsuyou]: 活用（かつよう）- 動詞、形容詞等詞類根據文法功能而改變形態的現象。五段動詞的活用會使用五十音的五個段。

[^jisho-kei]: 辞書形（じしょけい）- 動詞的基本形態，也就是字典中查詢時使用的形式。日語辭書形通常是動詞的終止形（plain form）。
