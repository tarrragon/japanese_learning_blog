## 日文

〜そうだ（傳聞）

### 文法情報

| 項目 | 內容 |
|------|------|
| 文法類型 | 傳聞助動詞 |
| JLPT | N4 |
| 使用場面 | 正式、非正式皆可 |
| 形成 | 普通形 + そうだ / そうです |

### 形成規則

**基本形式**
```
普通形 + そうだ / そうです
```

**動詞**
```
食べるそうだ（聽說要吃）
食べたそうだ（聽說吃了）
食べないそうだ（聽說不吃）
```

**い形容詞**
```
高いそうだ（聽說很貴）
高かったそうだ（聽說很貴）
```

**な形容詞**
```
便利だそうだ（聽說很方便）
便利だったそうだ（聽說很方便）
```

**名詞**
```
学生だそうだ（聽說是學生）
学生だったそうだ（聽說是學生）
```

## 日文解釋

「〜そうだ」（伝聞）は、他の人から聞いた情報[^information-source]や、本、ニュースなどで得た情報を伝える時に使う表現です。話し手[^speaker]が直接経験したことではなく、間接的に得た情報[^indirect-information]であることを明示します。

この表現を使うことで、情報の信頼性[^reliability]や責任の所在[^responsibility]を明確にすることができます。特に、不確実な情報[^uncertain-information]を伝える時や、情報源[^information-source]を示したい時に重要です。

文末には「〜そうです」（丁寧形[^teineigo]）、「〜そうだ」（普通形[^futsukei]）のほか、「〜ということだ」「〜とのことだ」など、より明確な伝聞表現もあります。書き言葉[^written-language]では「〜という」もよく使われます。

## 英文解釋

"〜そうだ" (hearsay) is a grammatical pattern used to convey information[^information-source] that the speaker[^speaker] has heard from someone else or obtained from sources like books, news, or media. It explicitly indicates that the information is not based on the speaker's direct experience but is indirect information[^indirect-information].

This expression is crucial for indicating the reliability[^reliability] of information and clarifying where the responsibility[^responsibility] for the information lies. It is particularly important when conveying uncertain information[^uncertain-information] or when the speaker wants to cite a source.

The pattern can be expressed in polite form as "〜そうです" (teineigo[^teineigo]) or plain form as "〜そうだ" (futsukei[^futsukei]). More explicit forms include "〜ということだ" or "〜とのことだ". In written language[^written-language], "〜という" is also commonly used.

## 中文解釋

「〜そうだ」（傳聞）是用來傳達說話者[^speaker]從他人處聽說的資訊[^information-source]，或從書籍、新聞等管道獲得的資訊的文法表達。它明確表示這個資訊不是說話者直接經歷的，而是間接獲得的資訊[^indirect-information]。

使用這個表達可以明確資訊的可靠性[^reliability]和責任歸屬[^responsibility]。特別是在傳達不確定的資訊[^uncertain-information]或想要標明資訊來源[^information-source]時非常重要。

句尾可以使用「〜そうです」（丁寧形[^teineigo]）、「〜そうだ」（普通形[^futsukei]），或者使用「〜ということだ」、「〜とのことだ」等更明確的傳聞表達。在書面語[^written-language]中，也常使用「〜という」。

## 核心用法

### 用法 1：傳達聽說的資訊

表示從他人處聞接聽到的消息或情報。

**例句 1**
```
明日は雨が降るそうです。
I heard it will rain tomorrow.
聽說明天會下雨。
```

**例句 2**
```
田中さんは来月結婚するそうだ。
I heard that Tanaka-san is getting married next month.
聽說田中先生下個月要結婚。
```

### 用法 2：引述新聞或媒體資訊

引用新聞、報紙、網路等媒體報導的內容。

**例句 3**
```
ニュースによると、事故で三人が負傷したそうです。
According to the news, three people were injured in the accident.
根據新聞報導，聽說事故中有三人受傷。
```

**例句 4**
```
この店は安くておいしいそうだよ。
I heard this restaurant is cheap and delicious.
聽說這家店又便宜又好吃。
```

### 用法 3：轉述他人的說法

傳達別人說過的話或意見。

**例句 5**
```
先生によると、試験は難しくないそうです。
According to the teacher, the exam won't be difficult.
根據老師說的，聽說考試不會很難。
```

## 文法規則

### 規則 1：接續方式

「〜そうだ」接在句子的普通形後面，不接在丁寧形（ます形）後面。

**正確**：
- 行くそうだ（○）
- 行ったそうだ（○）

**錯誤**：
- 行きますそうだ（✗）
- 行きましたそうだ（✗）

### 規則 2：時態表現

時態（過去、現在、未來）在「そうだ」之前的動詞或形容詞中表示，而不是在「そうだ」本身。

- 明日雨が降るそうだ（聽說明天會下雨 - 未來）
- 昨日雨が降ったそうだ（聽說昨天下雨了 - 過去）
- 今雨が降っているそうだ（聽說現在在下雨 - 現在進行）

### 規則 3：疑問句的使用

疑問句也可以使用傳聞表達，通常表示確認聽到的資訊。

- 明日は休みだそうですか。（聽說明天休息嗎？）
- 田中さんは来ないそうですか。（聽說田中先生不來嗎？）

### 規則 4：否定形式

否定有兩種表達方式：

**否定內容**（聽說不...）：
- 行かないそうだ（聽說不去）

**否定傳聞**（不是聽說...）：
- 行くそうではない（並非聽說要去）

## 常見錯誤

### 錯誤 1：與樣態「〜そうだ」混淆

❌ 誤：雨が降りそうだそうだ
✅ 正：雨が降りそうだ（樣態：看起來要下雨）
✅ 正：雨が降るそうだ（傳聞：聽說會下雨）

說明：「降りそうだ」是樣態（看起來、好像），「降るそうだ」是傳聞（聽說）。兩者不能混用。樣態的「そうだ」接在動詞ます形後，傳聞的「そうだ」接在普通形後。

### 錯誤 2：使用丁寧形接續

❌ 誤：行きますそうです
✅ 正：行くそうです

說明：傳聞的「そうだ」前面必須使用普通形，不能用丁寧形（ます形）。丁寧程度在句尾的「そうです」/「そうだ」中表現。

### 錯誤 3：時態位置錯誤

❌ 誤：雨が降るそうだった（意思不清）
✅ 正：雨が降ったそうだ（聽說下雨了）
✅ 正：雨が降るそうだった（以前聽說的，但表達少見）

說明：時態應該在「そうだ」之前表示，表達聽說的事情發生在什麼時候。「そうだった」表示「以前聽說」，與「聽說過去發生的事」意思不同。

## 學習要點

1. **區別樣態與傳聞**：「動詞ます形＋そうだ」是樣態（看起來），「普通形＋そうだ」是傳聞（聽說）
2. **明確資訊來源**：使用「〜によると」、「〜の話では」等表達可以更明確指出資訊來源
3. **責任歸屬**：使用傳聞表達可以表明資訊不是自己的直接經驗，降低說話者的責任
4. **時態表達**：時態在「そうだ」前的動詞或形容詞中表示，不是在「そうだ」本身
5. **禮貌程度**：「そうです」比「そうだ」更有禮貌，根據場合選擇使用

## 註解

[^information-source]: **資訊來源** - 指資訊的出處，如他人的話語、新聞、書籍等。日文中明確標示資訊來源很重要。

[^speaker]: **說話者** - 發話者，指正在說話或寫作的人。

[^indirect-information]: **間接資訊** - 不是透過直接經驗，而是透過他人轉述或媒體獲得的資訊。

[^reliability]: **可靠性** - 資訊的可信程度。使用傳聞表達可以表明資訊的可靠性可能有限。

[^responsibility]: **責任歸屬** - 指對資訊正確性負責的主體。傳聞表達讓說話者不必為資訊的真實性負完全責任。

[^uncertain-information]: **不確定的資訊** - 真實性或準確性未經證實的資訊。

[^teineigo]: **丁寧形** - 禮貌的說話方式，使用「です・ます」形式。適用於正式場合和對長輩、上司等。

[^futsukei]: **普通形** - 非禮貌形式，也稱為常體。用於朋友、家人之間或書面語。

[^written-language]: **書面語** - 書寫時使用的語言形式，與口語有所不同，通常更正式。
