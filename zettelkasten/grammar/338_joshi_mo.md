助詞「も」表示「也、也是」的意思，用於表達類同或累加的概念。

## 日文解釋

「も」は副助詞（とりたて助詞）の一つで、ある事柄が他の事柄と同様であることを示す。基本的な意味は「同様に」「その上」という添加・類同の意味である。

主な用法として、（1）「AもBも」のように複数の事物を並列に挙げる用法、（2）疑問詞と結びついて「誰も」「何も」「どこも」のように全称を表す用法、（3）数量詞と結びついて「3時間も待った」のように数量の多さを強調する用法がある。

格助詞「が」「を」は「も」に置き換えられるが、「に」「で」「から」などは「にも」「でも」「からも」のように「も」を後接させる点に注意が必要である。

## 英文解釋

The particle "mo" is an adverbial particle (toritate-joshi) that indicates similarity, addition, or emphasis. Its primary function is to express that something shares the same quality or state as another thing, similar to English "also," "too," or "as well."

Key usages include: (1) listing multiple items with similar treatment (A mo B mo = both A and B), (2) combining with question words to create universal expressions (dare mo = everyone/no one, nani mo = everything/nothing), and (3) emphasizing quantity when attached to numbers (san-jikan mo = as much as three hours).

Note that "mo" replaces case particles "ga" and "wo" directly, but combines with other particles like "ni," "de," and "kara" (becoming "ni mo," "de mo," "kara mo").

## 中文解釋

「も」是一個副助詞（提示助詞），用於表示某事物與另一事物具有相同的性質或狀態，相當於中文的「也」、「也是」、「連...都」。

主要用法包括：（1）並列用法——「AもBも」表示「A和B都」；（2）全稱用法——與疑問詞結合，如「誰も」（任何人都）、「何も」（任何東西都）；（3）強調用法——與數量詞結合，如「3時間も」（竟然三小時），強調數量之多或程度之深。

需注意的是，「も」可以直接替換格助詞「が」和「を」，但與「に」、「で」、「から」等助詞連用時，需要放在後面形成「にも」、「でも」、「からも」。

## 核心用法

### 用法 1：類同・添加（也、也是）

表示某事物與先前提及的事物具有相同的性質或狀態。

**例句 1**
```
私は学生です。田中さんも学生です。
I am a student. Tanaka is also a student.
我是學生。田中也是學生。
```

**例句 2**
```
コーヒーも紅茶も好きです。
I like both coffee and tea.
咖啡和紅茶我都喜歡。
```

### 用法 2：疑問詞 + も（全稱表現）

與疑問詞結合，表示「全部都」（肯定句）或「全部都不」（否定句）。

**例句 3**
```
日曜日はどこも混んでいます。
Everywhere is crowded on Sundays.
週日到處都很擁擠。
```

**例句 4**
```
昨日は何も食べませんでした。
I didn't eat anything yesterday.
昨天什麼都沒吃。
```

### 用法 3：數量詞 + も（強調數量）

與數量詞結合，強調數量之多或程度之深，帶有說話者的驚訝或感嘆。

**例句 5**
```
この本を読むのに一週間もかかりました。
It took as long as a week to read this book.
讀這本書竟然花了一整個星期。
```

## 文法規則

### 規則 1：與格助詞的關係

「も」可以直接替換「が」和「を」：
- 田中さん**が**来た → 田中さん**も**来た
- りんご**を**食べた → りんご**も**食べた

但與其他助詞連用時，「も」要放在後面：
- 学校**に**行く → 学校**にも**行く
- 公園**で**遊ぶ → 公園**でも**遊ぶ
- 駅**から**歩く → 駅**からも**歩く

### 規則 2：「AもBも」的並列結構

當並列兩個以上的項目時，每個項目後都要加「も」：
- ○ 兄**も**姉**も**大学生です。
- × 兄も姉は大学生です。

### 規則 3：疑問詞 + も + 述語的極性

疑問詞 + も 的意義取決於述語是肯定還是否定：
- 誰**も**来た（肯定）→ 大家都來了
- 誰**も**来なかった（否定）→ 沒有人來

## 常見錯誤

### 錯誤 1：「も」的位置錯誤

❌ 誤：私は日本にも行きました。（想表達「我也去了日本」）
✅ 正：私も日本に行きました。
說明：「も」的位置決定強調的對象。「にも」強調「日本」這個地點，而「私も」強調「我」這個主體。

### 錯誤 2：「AもBも」結構不完整

❌ 誤：りんごも、みかんが好きです。
✅ 正：りんごも、みかんも好きです。
說明：並列結構中，所有項目都需要加「も」。

### 錯誤 3：與「は」混淆

❌ 誤：A：私は日本人です。B：私は日本人です。
✅ 正：A：私は日本人です。B：私も日本人です。
說明：當表達「也是」時，應使用「も」而非「は」。

## 學習要點

1. **位置決定意義**：「も」接在哪個詞後面，就是強調那個詞與其他事物相同。
2. **替換 vs 附加**：「も」替換「が」「を」，但附加在「に」「で」「から」等助詞後面。
3. **疑問詞結合**：疑問詞 + も 的意義取決於句子是肯定還是否定。
4. **強調數量**：數量詞 + も 帶有說話者的主觀評價（多、長、遠等）。
5. **並列完整性**：「AもBも」結構中，每個項目都必須加「も」。
