## 日文
疑問形式（ぎもんけいしき）

**羅馬拼音**：gimon keishiki

## 日文解釋
疑問形式とは、相手に質問をするための文の形です。日本語では、文末に「か」を付けることで疑問文を作ります。話し言葉では、上昇調（語尾を上げる）だけで疑問の意味を表すこともできますが、書き言葉や丁寧な場面では「か」を使うのが一般的です。疑問文には、「はい」「いいえ」で答える一般疑問文と、疑問詞（何、誰、いつ、どこ、など）を使う特殊疑問文があります。

## 英文解釋
Interrogative form (疑問形式) refers to the sentence structure used to ask questions in Japanese. Questions are formed by adding the particle "か" (ka) at the end of a sentence. In spoken language, questions can also be indicated simply by rising intonation without "か", but in written or polite contexts, "か" is typically used. There are two types of questions: yes/no questions (general questions) and wh-questions (special questions using interrogative words like 何 what, 誰 who, いつ when, どこ where, etc.).

## 中文解釋
疑問形式是指在日文中用來提問的句子結構。疑問句的形成是在句尾加上助詞「か」。在口語中，也可以僅透過上揚語調（句尾上揚）來表示疑問，但在書面語或正式場合中，通常會使用「か」。疑問句有兩種類型：可用「是」「否」回答的一般疑問句，以及使用疑問詞（何、誰、いつ、どこ等）的特殊疑問句。

## 基本構造

### 1. 一般疑問文（是非問句）
動詞/形容詞/名詞 + です/ます + か

**例**：
```
これは本ですか。
Is this a book?
這是書嗎？
```

### 2. 特殊疑問文（疑問詞疑問句）
疑問詞 + 動詞/形容詞 + です/ます + か

**例**：
```
これは何ですか。
What is this?
這是什麼？
```

## 主要用法

### 1. 丁寧形疑問文（ですます形 + か）
```
明日、学校に行きますか。
Will you go to school tomorrow?
明天會去學校嗎？
```

### 2. 普通形疑問文（口語省略か）
```
明日、学校に行く？
Going to school tomorrow?
明天去學校嗎？
```

### 3. 疑問詞疑問文
```
誰がこの本を読みましたか。
Who read this book?
誰讀了這本書？
```

### 4. 選擇疑問文
```
コーヒーですか、紅茶ですか。
Coffee or tea?
咖啡還是紅茶？
```

## 例句

**例句1**
```
あなたは学生ですか。
Are you a student?
你是學生嗎？
```

**例句2**
```
今日の天気はどうですか。
How is the weather today?
今天天氣如何？
```

**例句3**
```
どこで昼ごはんを食べますか。
Where will you eat lunch?
你會在哪裡吃午餐？
```

## 學習重點

1. **書面語**：必須使用「か」
2. **口語**：可省略「か」，用上揚語調
3. **丁寧度**：ですます形較為正式
4. **疑問詞位置**：通常放在句子中需要詢問的資訊位置
