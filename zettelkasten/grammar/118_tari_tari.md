## 日文

〜たり〜たり（する）

### 文法情報

| 項目 | 內容 |
|------|------|
| 文法類型 | 動詞・形容詞變化 |
| JLPT | N4 |
| 使用場面 | 日常會話、書面 |
| 形成 | 動詞た形 + り、動詞た形 + り（+ する） |

### 形成規則

**基本形式**
```
動詞た形 + り、動詞た形 + り（+ する）
い形容詞 → かったり、かったり（+ する）
な形容詞 → だったり、だったり（+ する）
```

**範例**
```
食べる → 食べたり、飲んだり（する）
行く → 行ったり、来たり（する）
高い → 高かったり、安かったり（する）
静か → 静かだったり、にぎやかだったり（する）
```

## 日文解釋

「〜たり〜たり（する）」は、複数の動作や状態を列挙する文法[^enumeration-grammar]です。

この表現は、いくつかの例を挙げて、それらが交互に行われたり、同時期に起こったりすることを表します。必ずしも全ての動作を網羅するわけではなく、代表的な例を示すニュアンス[^representative-examples]があります。

動詞の場合は「た形[^ta-form]」に「り」を付け、文末に「する」を付けることが多いです。「する」を省略すると、やや文語的[^written-style]な印象になります。日常会話[^casual-context]では「する」を付けるのが一般的です。

形容詞にも使えますが、動詞ほど頻繁には使われません。複数の状態が変化したり、両方の特徴を持っていたりすることを表現する際に用います。

## 英文解釋

The pattern "〜たり〜たり (suru)" is a grammatical structure[^enumeration-grammar] used to list multiple actions or states in Japanese.

This construction indicates that several actions occur alternately, repeatedly, or during the same period. It doesn't necessarily enumerate all possible actions, but rather presents representative examples[^representative-examples] to give an overall impression.

For verbs, "ri" is attached to the ta-form[^ta-form], and "suru" is typically added at the end. Omitting "suru" gives a slightly literary[^written-style] tone. In casual conversation[^casual-context], it's more natural to include "suru".

This pattern can also be used with adjectives to describe fluctuating states or multiple characteristics, though this usage is less common than with verbs.

## 中文解釋

「〜たり〜たり（する）」是用來列舉多個動作或狀態的文法結構[^enumeration-grammar]。

這個表達方式表示幾個動作或狀態交替進行、反覆發生，或在同一時期內發生。它不一定要列舉所有的動作，而是提出具代表性的例子[^representative-examples]來呈現整體的情況。

對於動詞，在「た形[^ta-form]」後面加「り」，句尾通常會加「する」。省略「する」會給人較文語[^written-style]的感覺。在日常會話[^casual-context]中，加上「する」比較自然。

形容詞也可以使用這個句型，用來表達狀態的變化或同時具有多種特徵，但使用頻率不如動詞高。

## 核心用法

### 用法 1：列舉日常活動

表示在某段時間內進行的多個活動。

**例句 1**
```
週末は映画を見たり、本を読んだりします。
On weekends, I watch movies, read books, and so on.
週末我會看電影、看書等等。
```

**例句 2**
```
昨日は掃除をしたり、洗濯をしたりしました。
Yesterday I did things like cleaning and laundry.
昨天我做了打掃、洗衣服等事情。
```

### 用法 2：表示反覆或交替的動作

表示動作反覆進行或交替出現。

**例句 3**
```
彼は泣いたり笑ったりして、忙しい人だ。
He's a busy person, crying and laughing alternately.
他一會兒哭一會兒笑，真是個忙碌的人。
```

**例句 4**
```
天気が晴れたり曇ったりしています。
The weather is alternating between sunny and cloudy.
天氣時晴時陰。
```

### 用法 3：列舉狀態或特徵（形容詞）

用形容詞表示多種狀態或特徵的變化。

**例句 5**
```
この店の料理は高かったり安かったりで、よく分からない。
The prices at this restaurant vary - sometimes expensive, sometimes cheap - it's hard to figure out.
這家店的料理有時貴有時便宜，讓人搞不清楚。
```

## 文法規則

### 規則 1：至少列舉兩個項目

「〜たり〜たり」必須列舉至少兩個動作或狀態，不能只用一個「たり」。

❌ 誤：週末は映画を見たりします。
✅ 正：週末は映画を見たり、本を読んだりします。

### 規則 2：文末通常加「する」

在日常會話中，句尾通常要加「する」。

✅ 推薦：散歩したり、買い物したりします。
△ 較文語：散歩したり、買い物したり。

### 規則 3：動詞必須用た形

「たり」前面的動詞必須是た形（過去式），不是ます形或辞書形。

❌ 誤：食べり、飲みり
❌ 誤：食べるり、飲むり
✅ 正：食べたり、飲んだり

### 規則 4：時態由「する」決定

整個句子的時態由最後的「する」決定，前面的「た形」不表示過去。

```
現在：食べたり飲んだりします（經常做）
過去：食べたり飲んだりしました（過去做過）
```

## 常見錯誤

### 錯誤 1：只用一個「たり」

❌ 誤：昨日は映画を見たりしました。
✅ 正：昨日は映画を見たり、音楽を聞いたりしました。
說明：「たり」必須至少出現兩次，列舉兩個以上的動作。只用一次會讓句子不完整。

### 錯誤 2：混用不同的活用形

❌ 誤：食べて、飲んだりします。
✅ 正：食べたり、飲んだりします。
說明：不能混用「て形」和「たり」。列舉時要統一使用「たり」的形式。

### 錯誤 3：時態理解錯誤

❌ 誤：昨日、公園を散歩しました、買い物したりしました。
✅ 正：昨日、公園を散歩したり、買い物したりしました。
說明：「た形」在這個句型中不表示過去，時態由「する/した」決定。所有列舉項目都要用「たり」。

## 學習要點

1. **至少兩次**：「たり」必須至少出現兩次，表示列舉的概念
2. **使用た形**：動詞要變成た形再加「り」，不是ます形或辞書形
3. **加する**：日常會話中通常在句尾加「する」，省略顯得文語化
4. **時態統一**：時態由最後的「する/した」決定，前面的た形不表示時態
5. **代表性列舉**：不需列舉所有動作，只需舉出代表性的例子

## 註解

[^enumeration-grammar]: **列舉文法** - 用來列舉多個動作或狀態的文法表達方式。日文中常見的列舉方式包括「〜たり〜たり」、「〜し」、「〜や」等。詳見 [列舉的概念](../concept/enumeration.md)

[^representative-examples]: **代表性例示** - 不完整列舉所有項目，只舉出具代表性的例子來暗示還有其他類似的項目。「〜たり」具有「等等」的含意。

[^ta-form]: **た形（過去式）** - 日文動詞的過去式形態，也稱為過去形或完了形。在「〜たり」句型中，た形不表示過去，而是作為活用基礎。詳見 [た形](../grammar/ta_form.md)

[^written-style]: **文語的** - 較正式、書面化的語言風格，與口語（話し言葉）相對。省略「する」的「〜たり〜たり」給人文語的印象。

[^casual-context]: **日常會話場景** - 朋友、家人之間或日常生活中使用的非正式對話。在此情境中「〜たり〜たりする」比省略「する」的形式更自然。詳見 [日常會話](../context/casual-conversation.md)
