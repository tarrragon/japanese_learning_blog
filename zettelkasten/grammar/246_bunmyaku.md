## 日文
文脈（ぶんみゃく）

**羅馬拼音**：bunmyaku

**品詞分類**：名詞（言語学・語用論用語）

## 日文解釋
「文脈」とは、文章や発話の前後関係から生じる意味や状況のことです。言葉の意味は単独では決まらず、前後の文や発話状況、話し手と聞き手の関係、共有知識などによって決定されます。「文脈を読む」とは、明示されていない意図や暗黙の了解を理解することを指します。語用論において、文脈は言語表現の解釈に不可欠な要素です。

## 英文解釋
"Bunmyaku" (文脈) refers to context—the meaning or situation that arises from the surrounding text or utterances. The meaning of words is not determined in isolation but is shaped by preceding and following sentences, the circumstances of speech, the relationship between speaker and listener, and shared knowledge. "Reading the context" (文脈を読む) means understanding implicit intentions or unspoken agreements. In pragmatics, context is an essential element for interpreting linguistic expressions.

## 中文解釋
「文脈」（ぶんみゃく）指的是上下文、語境——即從文章或言談的前後關係中產生的意義或情境。語言的意義並非孤立決定，而是由前後的句子、說話的情境、說話者與聽話者的關係、以及共享的知識等因素所決定。「讀懂文脈」（文脈を読む）意指理解未明說的意圖或默契。在語用學中，文脈是解讀語言表達不可或缺的要素。

## 例句

**例句1**
```
この文の意味は文脈によって変わります。
The meaning of this sentence changes depending on the context.
這個句子的意思會隨著上下文而改變。
```

**例句2**
```
文脈を無視して単語だけ訳すと、誤解が生じます。
If you translate only the words while ignoring the context, misunderstandings will arise.
如果忽略語境只翻譯單詞，會產生誤解。
```

**例句3**
```
日本では文脈を読む能力が重要視されます。
In Japan, the ability to read context is highly valued.
在日本，讀懂語境的能力受到重視。
```

**例句4**
```
彼の発言は文脈から外れていた。
His statement was out of context.
他的發言脫離了語境。
```

**例句5**
```
文脈情報がなければ、代名詞の指示対象を特定できません。
Without contextual information, we cannot identify what pronouns refer to.
沒有語境信息，就無法確定代詞所指的對象。
```

## 補充說明

### 文脈の種類
1. **言語的文脈**：前後の文章や発話の内容
2. **状況的文脈**：発話が行われる時間・場所・状況
3. **社会的文脈**：話し手と聞き手の関係・社会的立場
4. **文化的文脈**：共有される文化的知識・価値観

### 日本語における文脈依存性
日本語は文脈依存度が非常に高い言語です：
- 主語・目的語の省略が頻繁
- 敬語の選択が状況に依存
- 曖昧な表現が好まれる
- 「空気を読む」文化と連動

### 関連概念
- **高文脈文化（high-context culture）**：暗黙の了解に頼る文化（日本など）
- **低文脈文化（low-context culture）**：明示的表現を好む文化（欧米など）

### 語用論における文脈
- **指示表現**：「これ」「あれ」「ここ」などの解釈
- **省略**：省略された要素の復元
- **含意**：言外の意味の推論
- **前提**：発話の前提条件
