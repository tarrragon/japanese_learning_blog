表示動作完成或遺憾的文法結構「〜てしまう」，口語常縮約為「〜ちゃう／〜じゃう」。

<!--more-->

## 日文

〜てしまう（〜ちゃう／〜じゃう）

### 文法情報

| 項目 | 內容 |
|------|------|
| 文法類型 | 補助動詞 |
| JLPT | N4 |
| 使用場面 | 日常會話、書面語 |
| 形成 | 動詞て形 + しまう |

### 形成規則

**基本形式**
```
動詞て形 + しまう
```

**口語縮約形**
```
〜てしまう → 〜ちゃう
〜でしまう → 〜じゃう
```

**範例**
```
食べる → 食べてしまう → 食べちゃう
飲む → 飲んでしまう → 飲んじゃう
```

## 日文解釋

「〜てしまう」は動詞のて形に補助動詞「しまう」が接続した形で、大きく分けて二つの意味を持つ。[^dict]

一つ目は**完了**の意味で、動作や状態が完全に終わることを表す。「食べてしまう」は「全部食べ終える」という意味になり、動作の完結を強調する。

二つ目は**残念・後悔**の意味で、望ましくない結果や予期しない出来事に対する話者の心情を表す。「失くしてしまった」「壊れてしまった」のように、困った状況や後悔の気持ちを込めて使う。

日常会話では「〜ちゃう」「〜じゃう」という縮約形がよく使われる。「食べちゃった」「飲んじゃった」のように、カジュアルな場面で頻繁に耳にする表現である。書き言葉や改まった場面では「〜てしまう」の形を使う。

[^dict]: 定義參考 スーパー大辞林（macOS 內建字典）

## 英文解釋

The grammar pattern "~te shimau" is formed by attaching the auxiliary verb "shimau" to the te-form of verbs. It carries two primary meanings.

First, it expresses **completion** of an action, emphasizing that something has been done completely or thoroughly. For example, "tabete shimau" means "to eat up completely."

Second, it conveys **regret or unintended outcomes**, expressing the speaker's feeling that something unfortunate or unexpected has happened. This usage often implies that the result was not desired.

In casual speech, "~te shimau" contracts to "~chau" (after te-form) or "~jau" (after de-form), making it one of the most commonly heard expressions in everyday Japanese conversation.

## 中文解釋

「〜てしまう」是將補助動詞「しまう」接在動詞的て形後面所形成的文法結構，主要有兩個含義。

第一個含義是表示**動作完成**，強調動作或狀態已經徹底結束。例如「食べてしまう」表示「完全吃完」，著重於動作的完結性。

第二個含義是表示**遺憾或後悔**，用來表達不希望發生的結果或意外的狀況。例如「失くしてしまった」（弄丟了）、「壊れてしまった」（壞掉了）等，帶有說話者困擾或後悔的情緒。

在日常口語中，「〜てしまう」常縮約為「〜ちゃう」，「〜でしまう」縮約為「〜じゃう」。這是日語會話中極為常見的表達方式。正式場合或書面語則使用完整形式。

## 核心用法

### 用法 1：動作完成

表示動作徹底完成，強調完結性。

**例句 1**
```
宿題を全部やってしまった。
I've finished all my homework completely.
我把作業全部做完了。
```

**例句 2**
```
このケーキ、一人で食べちゃった。
I ate this whole cake by myself.
這個蛋糕我一個人吃掉了。
```

### 用法 2：遺憾・後悔

表示不希望發生的結果或意外狀況，帶有遺憾的語氣。

**例句 3**
```
大切な写真を消してしまいました。
I accidentally deleted important photos.
我不小心把重要的照片刪掉了。
```

**例句 4**
```
電車に傘を忘れちゃった。
I left my umbrella on the train.
我把傘忘在電車上了。
```

### 用法 3：不可控的變化

表示自然發生或無法控制的變化。

**例句 5**
```
雨が降ってきて、洗濯物が濡れてしまった。
It started raining and the laundry got wet.
下起雨來，晾的衣服都濕了。
```

## 常見錯誤

### 錯誤 1：語域混用

❌ 誤：先生、レポートを出しちゃいました。
✅ 正：先生、レポートを出してしまいました。
說明：對老師等長輩或正式場合應使用完整形式「〜てしまいました」，不用縮約形「〜ちゃいました」。

### 錯誤 2：混淆て形接續

❌ 誤：読むてしまう
✅ 正：読んでしまう
說明：五段動詞的て形要正確變化。「読む」的て形是「読んで」，所以是「読んでしまう」。

## 學習要點

1. **兩大核心語義**：掌握「完成」和「遺憾」兩種基本含義，從語境判斷說話者的意圖。

2. **縮約形的使用場合**：「〜ちゃう／〜じゃう」用於口語和親近關係；正式場合用完整形式。

3. **て形的正確接續**：確認動詞て形的正確變化，特別注意五段動詞的音便變化。

4. **語氣的細微差異**：完成用法較中性，遺憾用法帶有情緒色彩，注意區分使用。

5. **敬體形式**：丁寧體為「〜てしまいます」，過去式為「〜てしまいました」。
