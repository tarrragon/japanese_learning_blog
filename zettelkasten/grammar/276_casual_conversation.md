## 日文
カジュアル会話（かじゅあるかいわ）

**羅馬拼音**：kajuaru kaiwa

**品詞分類**：語用論（言語運用）

## 日文解釋
カジュアル会話とは、親しい間柄で使う砕けた話し方のことです。敬語を使わず、文末を短くしたり、助詞を省略したりすることが特徴です。「だ・である」調を基本とし、「～じゃん」「～っけ」「～かな」などの話し言葉特有の表現を多用します。友人同士、家族間、同年代の仲間との日常会話で使われます。

## 英文解釋
Casual conversation (カジュアル会話) refers to informal speech used among close acquaintances in Japanese. It is characterized by the absence of keigo (honorific language), shortened sentence endings, and frequent omission of particles. Based on the plain form (だ・である), it incorporates colloquial expressions such as ～じゃん (isn't it), ～っけ (was it?), and ～かな (I wonder). This register is used in everyday conversations among friends, family members, and peers of similar age or status.

## 中文解釋
隨意會話（カジュアル会話）是指日語中在親近關係間使用的非正式說話方式。其特徵包括：不使用敬語、句末縮短、經常省略助詞。以常體「だ・である」為基礎，大量使用口語特有的表達方式，如「～じゃん」（不是嗎）、「～っけ」（是...來著？）、「～かな」（我想...）等。主要用於朋友之間、家人之間、同年齡層的夥伴之間的日常對話。

## 例句

**例句1**
```
明日、暇？映画でも見に行かない？
Are you free tomorrow? Want to go see a movie or something?
明天有空嗎？要不要去看電影什麼的？
```

**例句2**
```
えっ、マジで？知らなかったわ。
Seriously? I had no idea.
咦，真的假的？我都不知道。
```

**例句3**
```
あの店、美味しいじゃん。また行こうよ。
That place is delicious, isn't it? Let's go again.
那家店很好吃不是嗎？再去吧。
```

**例句4**
```
昨日のテスト、難しくなかった？
Wasn't yesterday's test hard?
昨天的考試不是很難嗎？
```

**例句5**
```
ごめん、ちょっと遅れる。先に行っててくれる？
Sorry, I'll be a bit late. Can you go ahead?
抱歉，我會晚一點到。你可以先過去嗎？
```

## 補充說明

### 隨意會話的主要特徵

1. **句末表現的簡化**
   - 丁寧形「～です」→ 常體「～だ」或省略
   - 「～ますか」→「～の？」「～？」
   - 「～ません」→「～ない」

2. **助詞省略**[^1]
   - 「映画を見に行く」→「映画見に行く」
   - 「学校に行く」→「学校行く」

3. **口語特有表現**
   - **～じゃん**：確認或強調（「いいじゃん」→ 很好啊）
   - **～っけ**：回想確認（「明日だっけ？」→ 是明天來著？）
   - **～かな**：自言自語式疑問（「行けるかな」→ 能去嗎...）
   - **～よね**：尋求同意（「美味しいよね」→ 很好吃對吧）

### 與敬語的對比

| 敬語表現 | 隨意會話 |
|---------|---------|
| 何をしていますか | 何してる？ |
| 食べましたか | 食べた？ |
| 行きましょう | 行こう / 行こうよ |
| すみません | ごめん |
| ありがとうございます | ありがとう |

### 使用場合

1. **適用場景**：朋友、家人、同輩、親密同事
2. **不適用場景**：上司、長輩、初次見面、正式場合

### 常見錯誤
❌ 對上司或長輩使用隨意會話
⭕ 根據關係和場合選擇適當的說話方式

❌ 在商務郵件中使用口語表現
⭕ 書面語和口語需要區分使用
