## 日文

使役形（しえきけい）

### 文法情報

| 項目 | 內容 |
|------|------|
| 文法類型 | 動詞變化形式 |
| JLPT | N4 |
| 使用場面 | 正式、命令、許可表達 |
| 形成 | 動詞語幹 + 使役語尾 |

### 形成規則

**一段動詞（る動詞）**
```
る → させる
食べる → 食べさせる（讓吃）
見る → 見させる（讓看）
起きる → 起きさせる（讓起床）
```

**五段動詞（う動詞）**
```
語尾改為「あ」段 + せる
書く → 書かせる（讓寫）
飲む → 飲ませる（讓喝）
話す → 話させる（讓說）
行く → 行かせる（讓去）
```

**不規則動詞**
```
する → させる（讓做）
来る（くる）→ 来させる（こさせる）（讓來）
```

## 日文解釋

使役形は、**誰かに何かをさせる**、または**誰かが何かをすることを許可する**という意味を表す動詞の活用形です。日本語の使役表現には、大きく分けて二つの意味があります。

一つ目は**強制的な使役**です。これは、立場や権力を持つ者が他者に何かを強制的にさせる場合に使います。例えば、「先生が生徒に宿題をさせる」「上司が部下に残業させる」といった表現です。この場合、される側は必ずしも自発的にその行為をするわけではありません。

二つ目は**許可的な使役**です。これは、誰かが何かをすることを許可したり、させてあげたりする場合に使います。例えば、「子供を公園で遊ばせる」「学生に好きな本を選ばせる」などです。この場合、される側の意思を尊重し、行動の自由を与えるというニュアンスがあります。

使役形の重要な特徴として、**助詞の使い方**があります。他動詞の場合は「AがBに～をさせる」というパターンを使い、自動詞の場合は「AがBを～させる」または「AがBに～させる」というパターンを使います。「に」と「を」の使い分けは、文脈や強制の度合いによって変わることがあります。

また、使役形は**「〜させてあげる」「〜させてください」「〜させていただく」**などの表現と組み合わせることで、より細かいニュアンスを表現できます。「させてあげる」は好意的な許可を、「させてください」は許可の依頼を、「させていただく」は謙譲的な許可の依頼を表します。

使役形は日常会話でも頻繁に使われますが、特にビジネスシーンや教育現場、親子の関係など、明確な上下関係や立場の違いがある場面でよく使われます。

## 英文解釋

The causative form (使役形, shieki-kei) is a verb conjugation pattern that expresses **making or letting someone do something**. This form has two primary meanings in Japanese usage.

The first meaning is **compulsive causative**, where someone in a position of authority or power makes another person do something. For example, "先生が生徒に宿題をさせる" (The teacher makes students do homework) or "上司が部下に残業させる" (The boss makes subordinates work overtime). In these cases, the person being caused to act may not be doing so voluntarily.

The second meaning is **permissive causative**, where someone allows or lets another person do something. For example, "子供を公園で遊ばせる" (Let children play in the park) or "学生に好きな本を選ばせる" (Let students choose their favorite books). This usage respects the will of the person being allowed to act and grants them freedom of action.

An important feature of the causative form is its **particle usage**. With transitive verbs, the pattern is "A が B に～を させる" (A makes B do something to object), while with intransitive verbs, the pattern can be either "A が B を～させる" or "A が B に～させる" (A makes/lets B do something). The choice between "に" and "を" can vary depending on context and the degree of compulsion.

The causative form can be combined with expressions like **"〜させてあげる" (let someone do as a favor), "〜させてください" (please let me do), or "〜させていただく" (humbly request permission to do)**. These combinations allow for more nuanced expressions: "させてあげる" expresses benevolent permission, "させてください" requests permission, and "させていただく" is a humble request for permission.

While commonly used in everyday conversation, the causative form is particularly frequent in business settings, educational contexts, and parent-child relationships where clear hierarchical or positional differences exist.

## 中文解釋

使役形是一種動詞變化形式，表示**讓某人做某事**或**允許某人做某事**的意思。日語的使役表達主要有兩種含義。

第一種是**強制性使役**，指有權力或地位的人強制他人做某事。例如「先生が生徒に宿題をさせる」（老師讓學生做作業）或「上司が部下に残業させる」（上司讓下屬加班）。在這些情況下，被使役的人不一定是自願做這件事的。

第二種是**許可性使役**，指允許或讓某人做某事。例如「子供を公園で遊ばせる」（讓孩子在公園玩）或「學生に好きな本を選ばせる」（讓學生選擇喜歡的書）。這種用法尊重被允許者的意願，給予他們行動的自由。

使役形的一個重要特徵是**助詞的使用方式**。他動詞使用「A が B に～を させる」（A 讓 B 對某物做某事）的句型，而自動詞可以使用「A が B を～させる」或「A が B に～させる」（A 讓 B 做某事）的句型。「に」和「を」的選擇會根據語境和強制程度而有所不同。

使役形可以與**「〜させてあげる」（好意地讓某人做）、「〜させてください」（請讓我做）、「〜させていただく」（謙虛地請求許可做某事）**等表達組合，呈現更細緻的語感。「させてあげる」表達善意的許可，「させてください」請求許可，「させていただく」則是謙遜的許可請求。

使役形在日常會話中經常使用，特別是在商務場合、教育現場、親子關係等有明確上下關係或立場差異的情境中更為常見。

## 核心用法

### 用法 1：表達強制或命令

用於上位者命令下位者做某事的場景。

**例句 1**
```
母は子供に野菜を食べさせる。
The mother makes her child eat vegetables.
媽媽讓孩子吃蔬菜。
```

**例句 2**
```
先生は学生に作文を書かせた。
The teacher made students write compositions.
老師讓學生寫作文。
```

### 用法 2：表達許可或允許

用於允許某人做某事的場景，通常帶有善意或尊重的語氣。

**例句 3**
```
子供を自由に遊ばせてあげたい。
I want to let children play freely.
我想讓孩子自由地玩。
```

**例句 4**
```
学生に好きな席を選ばせる。
Let students choose their favorite seats.
讓學生選擇喜歡的座位。
```

### 用法 3：請求許可（させてください）

用於謙虛地請求對方允許自己做某事。

**例句 5**
```
もう一度説明させてください。
Please let me explain one more time.
請讓我再解釋一次。
```

## 文法規則

### 規則 1：一段動詞的變化

一段動詞（る動詞）的使役形變化規則是將「る」去掉，加上「させる」。

```
食べる → 食べさせる
見る → 見させる
起きる → 起きさせる
寝る → 寝させる
```

### 規則 2：五段動詞的變化

五段動詞（う動詞）的使役形變化是將語尾改為「あ」段，再加上「せる」。

```
書く → 書かせる（く → か）
飲む → 飲ませる（む → ま）
話す → 話させる（す → さ）
読む → 読ませる（む → ま）
行く → 行かせる（く → か）
```

### 規則 3：助詞的使用

**他動詞**：使用「に」標記被使役者，「を」標記動作對象
```
先生が学生に本を読ませる。
（老師讓學生讀書）
```

**自動詞**：使用「を」或「に」標記被使役者
```
母が子供を遊ばせる。（較強制）
母が子供に遊ばせる。（較溫和）
```

### 規則 4：與其他表達的組合

**させてあげる**：好意地給予許可
```
子供を外で遊ばせてあげる。
```

**させてください**：請求許可
```
質問させてください。
```

**させていただく**：謙遜的許可請求
```
説明させていただきます。
```

## 常見錯誤

### 錯誤 1：混淆使役形與可能形

❌ 誤：私は子供を食べさせられる。（我能讓孩子吃）
✅ 正：私は子供に食べさせる。（我讓孩子吃）
說明：使役形的意思是「讓某人做某事」，不要與可能形「～られる」混淆。

### 錯誤 2：助詞使用錯誤

❌ 誤：先生が学生を本を読ませる。
✅ 正：先生が学生に本を読ませる。
說明：他動詞的使役句型中，被使役者用「に」標記，動作對象用「を」標記。

### 錯誤 3：不規則動詞變化錯誤

❌ 誤：来る → 来させる（きさせる）
✅ 正：来る → 来させる（こさせる）
說明：「来る」的使役形讀音是「こさせる」，不是「きさせる」。

## 學習要點

1. **兩種意義**：使役形有「強制」和「許可」兩種意思，需從語境判斷
2. **變化規則**：一段動詞用「させる」，五段動詞用「あ段 + せる」
3. **助詞選擇**：他動詞用「に + を」，自動詞用「を」或「に」
4. **組合表達**：與「あげる」「ください」「いただく」組合產生不同語感
5. **使用場景**：常見於有上下關係或立場差異的場合
