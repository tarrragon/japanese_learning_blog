## 概要

**謙讓語（けんじょうご）**：自分の動作をへりくだって述べることで、相手や話題の人物に対して敬意を表す敬語の一種。

- **中文**：自謙語，通過降低自己的動作來表示對對方的尊敬
- **English**: Humble language; Japanese honorific expressions that show respect by humbling one's own actions

## 詳細說明

### 謙讓語の基本原理

謙讓語は自分側の動作を低めることで、相対的に相手を高める表現方法である。主に以下の特徴がある：

1. **動作主が自分側**：話し手や話し手の身内が動作主
2. **恩恵の方向**：動作が相手に向かう場合に使用
3. **相対的な敬意**：自己を低めることで相手を高める

### 謙讓語の種類

#### 謙讓語Ⅰ（基本形）

相手や第三者に向かう動作をへりくだって表現：

| 一般動詞 | 謙讓語Ⅰ | 意味 |
|---------|---------|------|
| 言う | 申し上げる | 說（謙讓形） |
| 行く/来る | 伺う、参る | 去/來（謙讓形） |
| 聞く | 伺う、承る | 聽（謙讓形） |
| 見る | 拝見する | 看（謙讓形） |
| 会う | お目にかかる | 見面（謙讓形） |
| 知る | 存じる | 知道（謙讓形） |
| もらう | いただく | 收到（謙讓形） |
| 食べる/飲む | いただく | 吃/喝（謙讓形） |
| する | いたす | 做（謙讓形） |
| あげる | 差し上げる | 給予（謙讓形） |

#### 謙讓語Ⅱ（丁重語）

話し手が聞き手に対して丁重に述べる表現：

| 一般動詞 | 謙讓語Ⅱ | 使用場面 |
|---------|---------|---------|
| 行く | 参る | 自己の移動を述べる |
| いる | おる | 自己の存在を述べる |
| 言う | 申す | 自己の発言を述べる |
| 食べる | いただく | 自己の飲食を述べる |
| する | いたす | 自己の行為を述べる |

### 謙讓語の作り方

#### お/ご～する/いたす

```
お + 和語動詞の連用形 + する/いたす
ご + 漢語名詞 + する/いたす
```

**例**：
- お伝えする（伝える → お伝えする）
- ご案内する（案内 → ご案内する）
- お持ちする（持つ → お持ちする）
- ご説明する（説明 → ご説明する）

#### お/ご～いただく

受益表現で相手の行為に対する感謝を表す：

- お教えいただく（教わる謙讓表現）
- ご指導いただく（指導を受ける謙讓表現）

## 使用例

### ビジネスシーン

**会議での発言**：
```
私からご説明いたします。
（中文：由我來說明）
（English: I will explain）
```

**訪問の申し出**：
```
明日、御社に伺います。
（中文：明天我將前往貴公司拜訪）
（English: I will visit your company tomorrow）
```

**資料の受領**：
```
資料を拝見いたしました。
（中文：我已拜讀資料）
（English: I have reviewed the materials）
```

### メールでの使用

**依頼**：
```
ご検討いただけますでしょうか。
（中文：能否請您考慮？）
（English: Could you please consider this?）
```

**報告**：
```
結果を申し上げます。
（中文：報告結果）
（English: I will report the results）
```

### 電話応対

**取次ぎ**：
```
担当者に申し伝えます。
（中文：我會轉告負責人）
（English: I will inform the person in charge）
```

**確認**：
```
お名前を伺ってもよろしいでしょうか。
（中文：可以請教您的姓名嗎？）
（English: May I ask your name?）
```

## よくある間違い

### 二重敬語

❌ **誤り**：お伺いいたします
- 「伺う」自体が謙讓語なので「お」は不要

✅ **正しい**：伺います

### 尊敬語との混同

❌ **誤り**：部長が申されました（自分の上司について外部に話す場合）
- 外部に対しては身内を低める必要がある

✅ **正しい**：部長が申しました

### 過剰な謙讓

❌ **誤り**：お客様が申されました（お客様の動作）
- お客様の動作には尊敬語を使う

✅ **正しい**：お客様がおっしゃいました

## 学習のポイント

1. **動作の向き**：自分→相手の動作に謙讓語を使う
2. **身内の扱い**：外部の人には身内を謙遜する
3. **固有表現の暗記**：特殊な謙讓語は暗記が必要
4. **場面に応じた使い分け**：形式度に応じて適切な表現を選ぶ

## 文法ノート

### 謙讓語Ⅰとの違い

- **謙讓語Ⅰ**：相手や第三者への動作（伺う、申し上げる）
- **謙讓語Ⅱ**：聞き手への丁重表現（参る、申す）

### レベル別使用

- **初級**：いただく、伺う、申す等の基本形
- **中級**：お～する/いたす等の構文
- **上級**：状況に応じた適切な使い分け
