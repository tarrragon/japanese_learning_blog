## 日文

です形（ですけい）

**羅馬拼音**: desu-kei

## 日文解釋

「です」は日本語の丁寧体[^teineigo]を作る基本的な助動詞[^jodoushi]で、名詞[^noun-predicate]や形容詞[^adjective]の後に付けて丁寧な表現を作ります。「だ」の丁寧形に相当し、英語の"be動詞"に似た機能を持ちますが、日本語では述語[^predicate]を完成させる文法要素です。

です形は日常会話[^daily-conversation]からビジネス場面[^business-context]まで、広く使われる最も基本的な丁寧表現です。初対面の人、目上の人、公式な場面では必ず使います。一方、親しい友人や家族との会話では普通体[^futsutai]（だ・である）を使うことが多いです。

ます形[^masu-form]が動詞の丁寧表現であるのに対し、です形は名詞文[^noun-sentence]や形容詞文[^adjective-sentence]の丁寧表現として機能し、日本語の丁寧体システム[^politeness-system]の中核を成しています。

## 英文解釋

"Desu" (です) is the fundamental auxiliary verb in Japanese that creates the polite form (teineigo) for sentences with noun predicates and adjectives. It is the polite equivalent of "da" (だ) in plain form, and functions similarly to the English "to be" verb, though in Japanese it serves as a grammatical element that completes predicates.

The desu-form is the most basic polite expression used across a wide range of situations, from daily conversation to business contexts. It is mandatory when speaking to people you've just met, superiors, or in formal settings. In contrast, the plain form (da/de aru) is often used with close friends and family.

While the masu-form provides polite expressions for verbs, the desu-form serves the same function for noun and adjective sentences, forming the core of Japanese politeness levels.

## 中文解釋

「です」是日文中用來構成禮貌體（丁寧體）的基本助動詞，接在名詞或形容詞後面形成禮貌的表達。它是「だ」的禮貌形式，功能類似英文的 be 動詞，但在日文中它是完成述語的文法要素。

です形是從日常會話到商務場合都廣泛使用的最基本禮貌表達。與初次見面的人、長輩或正式場合中必須使用です形。相對地，與親密朋友或家人對話時常使用普通體（だ・である）。

ます形是動詞的禮貌表達，而です形則是名詞句和形容詞句的禮貌表達，兩者共同構成日文禮貌體系統的核心。

## 形成規則

### 與名詞的結合

**基本形式**
```
名詞 + です
```

**範例**
```
学生 → 学生です（是學生）
日本人 → 日本人です（是日本人）
先生 → 先生です（是老師）
```

### 與形容詞的結合

#### な形容詞（形容動詞）
```
な形容詞語幹 + です
```

**範例**
```
きれい → きれいです（漂亮）
静か → 静かです（安靜）
便利 → 便利です（方便）
```

#### い形容詞
```
い形容詞 + です
```

**範例**
```
大きい → 大きいです（大的）
美味しい → 美味しいです（好吃）
高い → 高いです（高的、貴的）
```

**注意**: い形容詞本身已經包含述語功能，加「です」只是增加禮貌度，文法上「大きい」單獨也可作述語。

### 時態變化

| 時態 | 肯定 | 否定 |
|------|------|------|
| 現在 | です | ではありません／じゃありません |
| 過去 | でした | ではありませんでした／じゃありませんでした |

## 核心用法

### 用法 1：名詞述語（判斷句）

表示「A是B」的判斷關係。

**例句 1**
```
私は学生です。
I am a student.
我是學生。
```

**例句 2**
```
これは辞書です。
This is a dictionary.
這是字典。
```

### 用法 2：形容詞述語（性質描述）

描述事物的性質、狀態。

**例句 3**
```
この部屋は静かです。
This room is quiet.
這個房間很安靜。
```

**例句 4**
```
富士山は高いです。
Mt. Fuji is high.
富士山很高。
```

### 用法 3：過去時態

表示過去的狀態或判斷。

**例句 5**
```
昨日は暑かったです。
It was hot yesterday.
昨天很熱。
```

### 用法 4：否定表達

表示否定判斷或性質。

**例句 6**
```
私は日本人ではありません。
I am not Japanese.
我不是日本人。
```

## 文法規則

### 規則 1：です vs だ

「です」是「だ」的禮貌形式，用於需要禮貌表達的場合。

| 普通體（だ） | 禮貌體（です） |
|------------|--------------|
| 学生だ | 学生です |
| きれいだ | きれいです |
| 大きいだ | 大きいです |

**使用區別**：
- **です**: 對外人、長輩、正式場合
- **だ**: 對親友、家人、自言自語、書面語

### 規則 2：否定形的兩種表達

日文的否定有正式和口語兩種表達：

| 正式 | 口語 |
|------|------|
| ではありません | じゃありません |
| ではありませんでした | じゃありませんでした |

**注意**: 「じゃ」是「では」的口語縮約形，兩者意思相同，「では」更正式。

### 規則 3：い形容詞的特殊性

い形容詞本身已有述語功能，「です」的添加只增加禮貌度：

```
✅ 大きい（普通體，述語完整）
✅ 大きいです（禮貌體）
❌ 大きいだ（錯誤，い形容詞不接「だ」）
```

### 規則 4：な形容詞需要連接詞

な形容詞修飾名詞時需要「な」，作述語時直接加「です」：

```
修飾名詞: きれいな花（漂亮的花）
作述語: 花がきれいです（花很漂亮）
```

## 常見錯誤

### 錯誤 1：い形容詞接「だ」

❌ 誤: 大きいだ
✅ 正: 大きい（普通體）／大きいです（禮貌體）
說明：い形容詞不能接「だ」，因為い形容詞本身已經包含述語功能。普通體直接用「大きい」，禮貌體加「です」。

### 錯誤 2：な形容詞忘記去「な」

❌ 誤: きれいなです
✅ 正: きれいです
說明：な形容詞作述語時，不用「な」，直接接「です」。「な」只在修飾名詞時使用。

### 錯誤 3：混淆「ではありません」和「じゃありません」的使用場合

❌ 誤: （在正式商務簡報中）これは新製品じゃありません。
✅ 正: これは新製品ではありません。
說明：正式場合應使用「ではありません」，「じゃありません」適合日常口語。

### 錯誤 4：過去否定形的混亂

❌ 誤: 昨日学生じゃなかったです
✅ 正: 昨日学生じゃありませんでした
說明：過去否定應用「じゃありませんでした」或「ではありませんでした」，不能用「じゃなかった＋です」的形式。

## 學習要點

1. **です是禮貌體的核心**：與ます形共同構成日文禮貌表達系統，初學者必須掌握
2. **名詞和形容詞都需要です**：名詞句和形容詞句的述語都用です來表達禮貌
3. **區分い形容詞和な形容詞**：い形容詞直接加です，な形容詞去な後加です
4. **掌握時態變化**：現在・過去、肯定・否定的四種組合都要熟練
5. **理解使用場合**：正式場合用です，親密場合可用だ，根據對象選擇合適的形式
6. **です不單獨使用**：です必須接在名詞或形容詞後面，不能單獨成句
7. **注意口語變化**：「では」可縮約為「じゃ」，但要注意正式度的差異

## 註解

[^teineigo]: **丁寧語（ていねいご）** - 日語敬語系統中的禮貌語，使用「です・ます」等形式表達禮貌，不涉及提高對方或降低自己的地位。詳見 [036_teineigo_concept.md](../concept/036_teineigo_concept.md)（待建立）

[^jodoushi]: **助動詞（じょどうし）** - 附加在動詞、形容詞或名詞後面，表達時態、否定、推測、使役等文法意義的詞類。詳見 [auxiliary_verb.md](../concept/auxiliary_verb.md)（待建立）

[^noun-predicate]: **名詞述語（めいしじゅつご）** - 以名詞為核心的述語，表達判斷或分類關係，如「私は学生です」。詳見 [predicate_types.md](../concept/predicate_types.md)（待建立）

[^adjective]: **形容詞（けいようし）** - 描述性質、狀態的詞類，日文分為い形容詞和な形容詞兩種。詳見 [adjective_types.md](../concept/adjective_types.md)（待建立）

[^predicate]: **述語（じゅつご）** - 句子中表達動作、狀態或判斷的部分，日文述語位於句尾。詳見 [sentence_structure.md](../concept/sentence_structure.md)（待建立）

[^daily-conversation]: **日常會話（にちじょうかいわ）** - 日常生活中的對話，通常使用禮貌體（です・ます），但對親友可用普通體。詳見 [daily-conversation.md](../context/daily-conversation.md)（待建立）

[^business-context]: **ビジネス場面（ビジネスばめん）** - 商務、工作相關的正式場合，必須使用禮貌體和敬語。詳見 [business.md](../context/business.md)（待建立）

[^futsutai]: **普通體（ふつうたい）** - 日語的普通形式，使用「だ・である」等，對親友、家人或書面語使用。詳見 [040_teineitai_vs_futsutai.md](../concept/040_teineitai_vs_futsutai.md)（待建立）

[^masu-form]: **ます形（ますけい）** - 動詞的禮貌形式，與です形共同構成日文禮貌體系統。詳見 [007_masu_form.md](007_masu_form.md)（待建立）

[^noun-sentence]: **名詞文（めいしぶん）** - 以名詞為述語的句子，如「私は学生です」。詳見 [sentence_types.md](../concept/sentence_types.md)（待建立）

[^adjective-sentence]: **形容詞文（けいようしぶん）** - 以形容詞為述語的句子，如「花がきれいです」。詳見 [sentence_types.md](../concept/sentence_types.md)（待建立）

[^politeness-system]: **丁寧体システム（ていねいたいシステム）** - 日語的禮貌體系統，包括です・ます形以及敬語，根據對象和場合選擇合適的表達。詳見 [047_polite_expression.md](../concept/047_polite_expression.md)（待建立）
