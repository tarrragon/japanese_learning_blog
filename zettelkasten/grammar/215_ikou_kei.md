意向形（普通形）是日文動詞變化的重要形式，用於表達說話者的意志、決心或提議。

<!--more-->

## 日文

意向形（いこうけい）/ 動詞の意向形

**羅馬拼音**：ikou-kei / doushi no ikou-kei

### 文法情報

| 項目 | 內容 |
|------|------|
| 文法類型 | 動詞變化 |
| JLPT | N4 |
| 使用場面 | 非正式、日常會話 |
| 形成 | 根據動詞類型變化詞尾 |

### 形成規則

**基本形式**

**る動詞（一段動詞）**
```
辞書形：る → よう
```

**う動詞（五段動詞）**
```
辞書形：う段 → おう段
```

**不規則動詞**
```
する → しよう
来る（くる）→ 来よう（こよう）
```

### 變化範例

**一段動詞**
- 食べる → 食べよう
- 見る → 見よう
- 起きる → 起きよう
- 寝る → 寝よう

**五段動詞**
- 行く → 行こう（いこう）
- 買う → 買おう（かおう）
- 話す → 話そう（はなそう）
- 読む → 読もう（よもう）
- 遊ぶ → 遊ぼう（あそぼう）
- 書く → 書こう（かこう）
- 泳ぐ → 泳ごう（およごう）
- 待つ → 待とう（まとう）
- 死ぬ → 死のう（しのう）
- 帰る → 帰ろう（かえろう）

**不規則動詞**
- する → しよう
- 来る（くる）→ 来よう（こよう）
- 勉強する → 勉強しよう

## 日文解釋

動詞の意向形は、話し手の意志や決意、または相手への提案を表す活用形です。「〜しよう」という形で、「〜するつもりだ」「一緒に〜しませんか」という意味を表します。

意向形には普通形（意向形）と丁寧形（〜ましょう）の二種類があります。普通形の意向形は、親しい友人や家族との会話で使われることが多く、カジュアルな雰囲気を持っています。相手への提案として使う場合は、「〜しようか」や「〜しようよ」という形で使われることもあります。

この形は、自分の意志を表明したり、相手を誘ったりする際に欠かせない表現です。丁寧形の「〜ましょう」と使い分けることで、場面に応じた適切なコミュニケーションができます。

日本語学習者にとって、意向形は中級レベルで習得すべき重要な文法項目の一つです。動詞の種類によって変化の仕方が異なるため、規則的な練習が必要です。

## 英文解釋

The volitional form (意向形, ikou-kei) is a verb conjugation that expresses the speaker's will, intention, or suggestion. In its plain form, it conveys meanings such as "let's do...", "I will do...", or "shall we do...?".

There are two types of volitional forms: the plain volitional form (普通形) and the polite volitional form (〜ましょう). The plain volitional form is commonly used in casual conversations with close friends and family members. When making suggestions to others, it's often combined with particles like か or よ, forming patterns like 〜しようか or 〜しようよ.

The formation rules vary depending on the verb type:
- For ru-verbs (ichidan): replace る with よう
- For u-verbs (godan): change the final う-sound to the corresponding おう-sound
- For irregular verbs: する becomes しよう, and 来る becomes 来よう

This form is essential for expressing personal intentions and making informal invitations, making it a crucial grammatical structure for intermediate-level Japanese learners.

## 中文解釋

意向形（いこうけい）是日文動詞變化，用來表達說話者的意志、意圖或提議。普通形的意向形傳達「讓我們做...」、「我要做...」或「我們做...好嗎？」等意思。

意向形有兩種：普通形（意向形）和禮貌形（〜ましょう）。普通形的意向形主要用於與親近的朋友和家人的日常對話中，氛圍較為隨意。當作為對他人的提議時，常與助詞か或よ結合，形成〜しようか或〜しようよ等句型。

變化規則根據動詞類型而不同：
- 一段動詞（る動詞）：將る替換為よう
- 五段動詞（う動詞）：將詞尾的う段音變為對應的おう段音
- 不規則動詞：する變為しよう，来る變為来よう

這個形式對於表達個人意圖和發出非正式邀請至關重要，是中級日語學習者必須掌握的重要文法結構。

## 核心用法

### 用法 1：表達說話者的意志或決心

用於表明自己的決定或打算做某事。

**例句 1**
```
明日から毎日運動しよう。
I'll exercise every day starting tomorrow.
從明天開始我要每天運動。
```

**例句 2**
```
今年こそ日本語を頑張って勉強しよう。
This year I'll really study Japanese hard.
今年一定要努力學習日語。
```

### 用法 2：向對方提出建議或邀請（非正式）

用於向親近的人提議一起做某事。

**例句 3**
```
今度の週末、一緒に映画を見ようよ。
Let's watch a movie together this weekend.
這週末一起看電影吧。
```

**例句 4**
```
お腹が空いたね。何か食べよう。
We're hungry, aren't we? Let's eat something.
肚子餓了呢，吃點東西吧。
```

### 用法 3：徵求對方意見（〜しようか）

加上「か」形成疑問句，詢問對方的意見。

**例句 5**
```
窓を開けようか。
Shall I open the window?
要打開窗戶嗎？
```

**例句 6**
```
もう帰ろうか。
Shall we go home now?
我們該回家了嗎？
```

### 用法 4：表示推測（〜しようとする）

與「とする」結合，表示正要做某事的狀態。

**例句 7**
```
彼女は何か言おうとしている。
She is about to say something.
她正要說什麼。
```

## 文法規則

### 規則 1：一段動詞（る動詞）的變化

去掉詞尾的「る」，加上「よう」。

**範例**
```
食べる → 食べよう
起きる → 起きよう
見る → 見よう
教える → 教えよう
```

**注意事項**
所有一段動詞的變化都遵循這個規則，沒有例外。

### 規則 2：五段動詞（う動詞）的變化

將詞尾的う段音（う、く、ぐ、す、つ、ぬ、ぶ、む、る）變為對應的おう段音（おう、こう、ごう、そう、とう、のう、ぼう、もう、ろう）。

**範例**
```
う → おう：買う → 買おう
く → こう：書く → 書こう
ぐ → ごう：泳ぐ → 泳ごう
す → そう：話す → 話そう
つ → とう：待つ → 待とう
ぬ → のう：死ぬ → 死のう
ぶ → ぼう：遊ぶ → 遊ぼう
む → もう：読む → 読もう
る → ろう：帰る → 帰ろう
```

### 規則 3：不規則動詞的變化

只有兩個不規則動詞需要特別記憶。

**範例**
```
する → しよう
来る（くる）→ 来よう（こよう）
```

**複合動詞**
```
勉強する → 勉強しよう
電話する → 電話しよう
```

### 規則 4：與丁寧形的對應關係

意向形的普通形和丁寧形（ましょう形）在意思上相同，但語氣和使用場合不同。

**對應範例**
```
食べよう（普通形）↔ 食べましょう（丁寧形）
行こう（普通形）↔ 行きましょう（丁寧形）
しよう（普通形）↔ しましょう（丁寧形）
```

**使用場合**
- 普通形：親しい友人、家族、同僚との会話
- 丁寧形：目上の人、初対面の人、フォーマルな場面

## 常見錯誤

### 錯誤 1：一段動詞變化錯誤

學習者容易將一段動詞按照五段動詞的規則變化。

❌ 誤：食べる → 食べろう
✅ 正：食べる → 食べよう

說明：一段動詞只需去掉「る」加「よう」，不需要音變。「食べろう」是不存在的形式。

### 錯誤 2：混淆動詞類型

某些結尾為「る」的動詞實際上是五段動詞，不是一段動詞。

❌ 誤：帰る → 帰よう
✅ 正：帰る → 帰ろう

說明：「帰る」雖然以「る」結尾，但它是五段動詞，需要將「る」變為「ろう」。區分一段動詞和五段動詞的「る」結尾動詞是學習的重點。

### 錯誤 3：不規則動詞記憶錯誤

不規則動詞的變化需要特別記憶。

❌ 誤：来る → 来ろう（くろう）
✅ 正：来る → 来よう（こよう）

說明：「来る」是不規則動詞，其意向形是「来よう（こよう）」，不能按照五段動詞的規則變化。

### 錯誤 4：在正式場合使用普通形

普通形的意向形不適合正式場合或對上級使用。

❌ 誤：（對老師說）明日、一緒に昼ご飯を食べよう。
✅ 正：（對老師說）明日、一緒に昼ご飯を食べましょうか。

說明：對老師、上司等需要使用敬語的對象，應該使用丁寧形「〜ましょう」或更禮貌的表達方式。

## 學習要點

1. **掌握三種動詞類型的變化規則**：一段動詞、五段動詞、不規則動詞各有不同的變化方式，需要分別練習記憶。

2. **注意使用場合的適當性**：普通形的意向形主要用於非正式場合，與親近的人交談時使用。正式場合應使用「〜ましょう」。

3. **理解「〜しようか」的用法**：加上「か」可以徵求對方意見，語氣更加委婉，常用於提供幫助或詢問意願。

4. **區分意志和提議**：根據上下文，意向形可以表達說話者自己的意志（「我要做...」），也可以是對他人的提議（「我們一起做...吧」）。

5. **與「〜しようとする」的搭配**：意向形加上「とする」可以表示「正要做某事」或「試圖做某事」，這是常見的文法組合。
