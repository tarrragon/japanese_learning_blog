# しかし

## 解釋

### 日文
文章または文の始まりに用いて、前述の内容に対する逆接や対比を表す接続詞。「けれども」「だが」より改まった表現で、書き言葉でよく使われる。

### 英文
A formal conjunction used at the beginning of a sentence to express contrast or opposition to the preceding statement; more formal than "keredo" or "daga", commonly used in written language.

### 中文
放在句首或段落開頭的正式接續詞，表示與前述內容相反或對比的意思；比「けれども」、「だが」更正式，常用於書面語。

## 用法

### 基本句型
```
［文A］。しかし、［文B］。
```

### 說明
- 接續兩個句子，表達對比或轉折關係
- 通常放在句首，後面加逗號
- 語氣正式，適合書面語或正式場合
- 口語中較常使用「でも」「だけど」等

## 例句

### 例句 1
```
今日は忙しかった。しかし、充実した一日だった。
きょうはいそがしかった。しかし、じゅうじつしたいちにちだった。
Today was busy. However, it was a fulfilling day.
今天很忙。但是，是充實的一天。
```

### 例句 2
```
計画は完璧だった。しかし、実行には至らなかった。
けいかくはかんぺきだった。しかし、じっこうにはいたらなかった。
The plan was perfect. However, it was never executed.
計畫很完美。但是，沒能實行。
```

### 例句 3
```
彼は優秀な学生だ。しかし、試験には失敗した。
かれはゆうしゅうながくせいだ。しかし、しけんにはしっぱいした。
He is an excellent student. However, he failed the exam.
他是優秀的學生。但是，考試失敗了。
```

## 對比

### しかし vs でも
| 項目 | しかし | でも |
|------|--------|------|
| 正式度 | 正式 | 隨意 |
| 使用場合 | 書面語、正式場合 | 口語、日常對話 |
| 位置 | 句首 | 句首 |

### しかし vs けれども
| 項目 | しかし | けれども |
|------|--------|----------|
| 獨立性 | 獨立句子 | 可接續同一句 |
| 正式度 | 較正式 | 中等 |
| 語感 | 轉折感強 | 轉折感較柔和 |
