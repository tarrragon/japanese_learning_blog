# ません（Masen / Polite Negative Form）

## 日文

ません（マセン）

### 文法情報

| 項目 | 內容 |
|------|------|
| 文法類型 | 動詞活用（否定形） |
| JLPT | N5 |
| 使用場面 | 禮貌表達、正式場合、日常會話 |
| 形成 | 動詞ます形語幹 + ません |

### 形成規則

**基本形式**
```
動詞ます形語幹 + ません
```

**變化步驟**
1. 動詞變為ます形[^masukei]
2. 將「ます」替換為「ません」

**範例**
```
食べます → 食べません（不吃）
行きます → 行きません（不去）
します → しません（不做）
```

## 日文解釋

「ません」は、動詞のます形[^masukei]の否定形で、丁寧語[^teineigo]における基本的な否定表現です。動詞の語幹に「ません」を接続することで、聞き手に対して礼儀正しく「〜しない」という意味を伝えます。

現在形と未来形の両方に使用でき、「今〜しない」または「これから〜しない」という意味を表現します。普通体[^futsutai]の「ない形」に対応する丁寧体[^teineigo]の表現として、ビジネス場面、初対面の人との会話、年上の人への発話など、幅広い場面で使用されます。

「ません」は、断りや拒否を表現する際にも使用されますが、日本語の敬語システム[^keigo]において基本的な礼儀を示す形式であるため、直接的な拒否を和らげる効果があります。また、習慣や状態を否定する際にも使用され、「毎日〜ません」のように頻度を表す副詞[^hindo]と組み合わせて使うことも多いです。

## 英文解釋

"Masen" (ません) is the negative form of the masu-form[^masukei] verb conjugation in Japanese, serving as the basic polite negation pattern. It is formed by replacing "masu" (ます) with "masen" (ません) in the verb stem, politely conveying the meaning of "not doing" or "won't do" something.

This form can express both present and future negation, meaning "don't do" or "won't do" something. As the polite counterpart to the plain negative form[^naikei] (ない形), "masen" is widely used in business settings, conversations with strangers, and when speaking to elders or superiors.

When used to decline or refuse something, "masen" softens the directness of the refusal due to its polite nature within the Japanese honorific system[^keigo]. It is also commonly used with frequency adverbs[^hindo] like "毎日" (every day) to express habitual negation, such as "I don't do [something] every day."

## 中文解釋

「ません」是日文動詞ます形[^masukei]的否定形式，是禮貌體[^teineigo]中基本的否定表達方式。通過將動詞語幹後的「ます」替換為「ません」，可以禮貌地表達「不做某事」的意思。

這個形式可以表達現在和未來的否定意義，意思是「現在不做」或「將來不做」某事。作為普通體否定形式[^naikei]（ない形）對應的禮貌表達，「ません」廣泛用於商務場合、與陌生人對話、對長輩說話等各種情境。

在表達拒絕或謝絕時也使用「ません」，但由於它在日語敬語系統[^keigo]中屬於基本禮貌形式，因此能夠緩和直接拒絕的語氣。此外，在否定習慣或狀態時也經常使用，常與頻率副詞[^hindo]如「毎日」（每天）搭配使用，例如「每天不做某事」。

## 核心用法

### 用法 1：禮貌地表達不做某事（現在）

用於禮貌地說明現在不做某事或不具備某種習慣。

**例句 1**
```
すみません、お酒は飲みません。
Sumimasen, osake wa nomimasen.
Sorry, I don't drink alcohol.
不好意思，我不喝酒。
```

**例句 2**
```
私は肉を食べません。
Watashi wa niku o tabemasen.
I don't eat meat.
我不吃肉。
```

### 用法 2：禮貌地表達不會做某事（未來）

用於禮貌地表達將來不打算做某事或拒絕某項建議。

**例句 3**
```
明日は出かけません。
Ashita wa dekakemasen.
I won't go out tomorrow.
我明天不會出門。
```

**例句 4**
```
今日は残業しません。
Kyō wa zangyō shimasen.
I won't work overtime today.
我今天不加班。
```

### 用法 3：禮貌地拒絕邀請或提議

用於委婉地謝絕邀請、拒絕提議或表達無法參與。

**例句 5**
```
申し訳ありませんが、パーティーには行きません。
Mōshiwake arimasen ga, pātī ni wa ikimasen.
I'm sorry, but I won't go to the party.
很抱歉，我不會去參加派對。
```

## 文法規則

### 規則 1：時態表達

「ません」本身表達現在或未來的否定，不表達過去。

**現在/未來否定**：
- 食べません（現在不吃 / 將來不吃）
- 行きません（現在不去 / 將來不去）

**過去否定**（需使用「ませんでした」）：
- 食べませんでした（過去沒吃）
- 行きませんでした（過去沒去）

### 規則 2：動詞類型與變化

所有動詞（一段動詞[^ichidan]、五段動詞[^godan]、不規則動詞[^fukisoku]）都遵循相同規則：將ます形的「ます」替換為「ません」。

**一段動詞**：
- 見ます → 見ません（不看）
- 寝ます → 寝ません（不睡）

**五段動詞**：
- 書きます → 書きません（不寫）
- 読みます → 読みません（不讀）

**不規則動詞**：
- します → しません（不做）
- 来ます（きます）→ 来ません（きません）（不來）

### 規則 3：與助詞的搭配

「ません」前的助詞「は」常用於強調對比或主題。

**使用「は」強調對比**：
- コーヒーは飲みますが、お茶は飲みません。
  （咖啡我喝，但茶我不喝。）

**使用「を」表示賓語**：
- 宿題をしません。
  （不做作業。）

## 常見錯誤

### 錯誤 1：混淆「ません」與「ないです」

❌ 誤：行かないです。（不夠禮貌或不自然）
✅ 正：行きません。（正確的禮貌否定形式）
說明：雖然「ないです」在口語中有時可見，但標準的禮貌否定形式是「ません」。

### 錯誤 2：過去時使用錯誤

❌ 誤：昨日は行きません。（用現在否定表達過去）
✅ 正：昨日は行きませんでした。（正確的過去否定）
說明：表達過去的否定必須使用「ませんでした」，而非「ません」。

### 錯誤 3：動詞活用錯誤

❌ 誤：食べるません。（直接接在辭書形後）
✅ 正：食べません。（先變ます形，再替換為ません）
說明：必須先將動詞變為ます形（去掉「ます」後的語幹），再加「ません」。

### 錯誤 4：不規則動詞活用錯誤

❌ 誤：来りません（きりません）。（誤以為是一段動詞）
✅ 正：来ません（きません）。（不規則動詞的正確變化）
說明：「来る」是不規則動詞，其ます形是「来ます（きます）」，否定是「来ません（きません）」。

## 學習要點

1. **形成方式簡單**：只需將ます形的「ます」替換為「ません」，所有動詞類型都適用這個規則。

2. **時態區分**：「ません」表達現在/未來否定，過去否定使用「ませんでした」。

3. **禮貌程度**：「ません」是禮貌體否定，對應普通體的「ない」。正式場合、對長輩、對陌生人都應使用「ません」。

4. **與「ないです」的區別**：雖然口語中有時聽到「ないです」，但標準且更禮貌的表達是「ません」。

5. **拒絕時的禮貌性**：使用「ません」拒絕邀請或提議時，語氣比普通體「ない」更委婉、更有禮貌。

## 註解

[^masukei]: **ます形** - 日文動詞的基本禮貌形式，通過在動詞語幹後接續「ます」形成。詳見 [ます形](./007_masu_form.md)

[^teineigo]: **丁寧語** - 日文三大敬語類型之一，透過「です・ます」等形式表達對聽者的基本禮貌。詳見 [丁寧語概念](../concept/036_teineigo_concept.md)

[^futsutai]: **普通體** - 與禮貌體相對的語體，使用辭書形、ない形、た形等，常用於親密關係或書面語。詳見 [普通體與禮貌體](../concept/040_teineitai_vs_futsutai.md)

[^naikei]: **ない形** - 動詞的普通體否定形式，表達「不做某事」，是普通體的基本否定形。詳見 [ない形](./044_nai-kei.md)

[^keigo]: **敬語系統** - 日語透過尊敬語、謙讓語、丁寧語三大類表達對聽者或話題人物的敬意。

[^hindo]: **頻率副詞** - 表示動作發生頻率的副詞，如「毎日」（每天）、「よく」（經常）、「時々」（有時）等。

[^ichidan]: **一段動詞** - 動詞語幹不變，只變化詞尾「る」的動詞類型。例：食べる → 食べます、食べて、食べた。

[^godan]: **五段動詞** - 語尾在五十音圖的五個段（あ・い・う・え・お）變化的動詞類型。例：書く → 書きます、書いて、書いた。

[^fukisoku]: **不規則動詞** - 不遵循一般變化規則的動詞，日文中主要是「する」和「来る」。詳見 [不規則活用](../concept/002_fukisoku-katsuyou.md)
