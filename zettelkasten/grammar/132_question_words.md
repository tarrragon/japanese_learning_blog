## 日文

疑問詞（ぎもんし）

**羅馬拼音**：gimonshi

## 日文解釋

疑問詞とは、物事や状況について尋ねる時に使う言葉のことです[^question-concept]。日本語では「何」「誰」「どこ」「いつ」「なぜ」「どう」などがあり、それぞれ異なる種類の情報を求めます。疑問詞は文の中で名詞や副詞として機能し、疑問文[^question-sentence]を作る際の核心要素となります。

日本語の疑問詞の特徴として、疑問助詞「か」と組み合わせて不定語[^indefinite]を作ることができます（例：「誰か」=誰か分からない人、「何か」=何か分からない物）。また、疑問詞を文末に置いても疑問文になりますが、これは会話体[^conversational-style]でよく見られる特徴です。

日常会話から書き言葉まで、あらゆる場面で使われる基本的な表現であり、日本語学習の初期段階で必ず習得すべき重要な要素です。

## 英文解釋

Question words (疑問詞, gimonshi) are interrogative expressions used to ask about things, people, places, times, reasons, or manners in Japanese. The main question words include 何 (what), 誰 (who), どこ (where), いつ (when), なぜ (why), and どう (how). Each serves a specific function in forming questions about different types of information.

A unique feature of Japanese question words is their ability to combine with the particle か (ka) to form indefinite pronouns[^indefinite] (e.g., 誰か = someone, 何か = something). In spoken Japanese, question words can also appear at the end of sentences to create questions, reflecting the flexible word order of conversational Japanese[^conversational-style].

Mastering question words is fundamental to Japanese communication, as they are essential for both asking and understanding questions in all contexts, from casual conversation to formal written discourse.

## 中文解釋

疑問詞是用來詢問事物、人物、地點、時間、原因或方式的詞語。日文的主要疑問詞包括何（什麼）、誰（誰）、どこ（哪裡）、いつ（什麼時候）、なぜ（為什麼）和どう（怎麼樣），每個疑問詞都用於詢問不同類型的資訊。

日文疑問詞的一個獨特特徵是可以與助詞「か」結合，形成不定代詞[^indefinite]（例如：誰か = 某人，何か = 某物）。在口語中，疑問詞也可以出現在句尾來構成疑問句，這反映了日語口語[^conversational-style]靈活的語序特點。

掌握疑問詞是日語溝通的基礎，因為它們對於提問和理解問題至關重要，適用於從日常對話到正式書面語的所有情境。

## 核心用法

### 用法 1：基本疑問句（何・誰・どこ）

疑問詞放在詢問的部分，句尾加「か」形成疑問句。

**例句 1**
```
これは何ですか。
What is this?
這是什麼？
```

**例句 2**
```
誰がこの本を書きましたか。
Who wrote this book?
誰寫了這本書？
```

**例句 3**
```
図書館はどこにありますか。
Where is the library?
圖書館在哪裡？
```

### 用法 2：時間與理由（いつ・なぜ）

詢問時間點、時間範圍或原因。

**例句 4**
```
日本にいつ来ましたか。
When did you come to Japan?
你什麼時候來日本的？
```

**例句 5**
```
なぜ日本語を勉強していますか。
Why are you studying Japanese?
你為什麼學日文？
```

### 用法 3：方式與狀態（どう・どのように）

詢問做事的方法或事物的狀態。

**例句 6**
```
週末はどうでしたか。
How was your weekend?
你的週末過得如何？
```

**例句 7**
```
駅までどのように行きますか。
How do you get to the station?
你怎麼去車站？
```

### 用法 4：選擇與數量（どれ・いくら）

用於在選項中選擇或詢問價格、數量。

**例句 8**
```
このケーキはいくらですか。
How much is this cake?
這個蛋糕多少錢？
```

**例句 9**
```
この中でどれが一番好きですか。
Which one do you like best among these?
這些當中你最喜歡哪一個？
```

## 文法規則

### 規則 1：疑問詞的位置

疑問詞通常放在要詢問的部分（句中位置），不像英文會移到句首。

```
正式疑問句：疑問詞 + ... + か
會話體疑問句：疑問詞 + ... + (語氣上揚)
```

### 規則 2：「何」的讀音變化

「何」根據後接詞語的發音有兩種讀法：
- **なに**：後接格助詞（を、が、に等）或直接作主詞、賓語
  - 例：何を、何が、何に
- **なん**：後接量詞、「です」或其他特定詞
  - 例：何時（なんじ）、何歳（なんさい）、何ですか

### 規則 3：不定詞的形成

疑問詞 + か = 不定代詞（某個~）
疑問詞 + も = 全稱（任何~都）+ 否定動詞 = 全稱否定（什麼都不~）

```
誰か → 某人（someone）
誰も + 否定 → 任何人都不（no one）

何か → 某物（something）
何も + 否定 → 什麼都不（nothing）

どこか → 某處（somewhere）
どこも + 否定 → 哪裡都不（nowhere）
```

### 規則 4：禮貌程度的調整

疑問詞本身是中性的，禮貌程度由句末表達決定：
- カジュアル：疑問詞 + の？
- 普通體：疑問詞 + か
- 丁寧體：疑問詞 + ですか / ますか
- 更禮貌：疑問詞 + でしょうか / ましょうか

## 常見錯誤

### 錯誤 1：疑問詞位置錯誤

❌ 誤：何これですか。（What this is?）
✅ 正：これは何ですか。（This is what? → What is this?）
說明：日文疑問詞不移到句首，保持在要詢問的部分。

### 錯誤 2：「何」的讀音混淆

❌ 誤：何時（なにじ）、何歳（なにさい）
✅ 正：何時（なんじ）、何歳（なんさい）
說明：接量詞或「です」時讀「なん」。

### 錯誤 3：不定詞的否定用法錯誤

❌ 誤：誰もが来ません。（Everyone doesn't come - 歧義）
✅ 正：誰も来ません。（No one comes）
說明：「疑問詞 + も + 否定」表示全稱否定，不需要「が」。

### 錯誤 4：混淆「どう」和「どのように」

❌ 誤：味はどのようですか。（過度正式）
✅ 正：味はどうですか。（How is the taste?）
說明：詢問狀態、感受時用「どう」較自然；「どのように」用於詢問具體方法。

## 學習要點

1. **位置固定**：日文疑問詞不移到句首，停留在被詢問的部分（主語、賓語、狀語位置）
2. **何的雙讀**：「何」有「なに」和「なん」兩種讀音，需根據後續詞語判斷
3. **不定表達**：疑問詞 + か 形成不定代詞，疑問詞 + も + 否定 形成全稱否定
4. **禮貌層級**：通過句末表達（の、か、ですか、でしょうか）調整禮貌程度
5. **會話省略**：口語中常省略「か」，用語氣上揚表示疑問
