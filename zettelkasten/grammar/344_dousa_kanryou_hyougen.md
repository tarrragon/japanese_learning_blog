動作完了表現是日文中表達動作已經完成的文法形式。

<!--more-->

## 日文

動作完了表現（どうさかんりょうひょうげん）

### 文法情報

| 項目 | 內容 |
|------|------|
| 文法類型 | アスペクト表現 |
| JLPT | N4 |
| 使用場面 | 口語、書面 |
| 形成 | 動詞て形 + しまう／た形 |

### 形成規則

**1. ～てしまう（完了・遺憾）**
```
動詞て形 + しまう → 動作完了または遺憾の表現
食べる → 食べてしまう
読む → 読んでしまう
```

**2. ～た形（単純完了）**
```
動詞辞書形 → た形
食べる → 食べた
読む → 読んだ
```

**3. ～ちゃう・～じゃう（口語縮約形）**
```
てしまう → ちゃう / ちまう
でしまう → じゃう / じまう
食べてしまう → 食べちゃう
読んでしまう → 読んじゃう
```

## 日文解釋

動作完了表現とは、ある動作や行為が完全に終わったことを表す文法形式である。日本語では「～てしまう」と「～た」が代表的な完了表現として使われる。

「～てしまう」は単なる完了だけでなく、その結果に対する話者の感情（残念、後悔、意外など）を含むことが多い。例えば「お菓子を全部食べてしまった」には「食べ終わった」という事実と「食べすぎた」という後悔のニュアンスが含まれる。

一方、「～た」形は客観的な完了を表し、感情的なニュアンスは薄い。「本を読んだ」は単に読み終わったという事実を述べているだけである。

口語では「～ちゃう」「～じゃう」という縮約形がよく使われ、カジュアルな会話では標準的な表現となっている。

## 英文解釋

Completion expressions in Japanese indicate that an action has been fully finished or accomplished. The two main forms are "~te shimau" and the "~ta" form.

"~Te shimau" expresses not only completion but often carries an emotional nuance—regret, disappointment, or surprise about the completed action. For example, "Okashi wo zenbu tabete shimatta" (I ended up eating all the sweets) implies both the completion of eating and a sense of regret.

The "~ta" form simply indicates objective completion without emotional coloring. "Hon wo yonda" (I read the book) is a neutral statement of fact.

In casual speech, the contracted forms "~chau" and "~jau" are commonly used instead of "~te shimau" and "~de shimau" respectively.

## 中文解釋

動作完了表現是日文中表達動作已經完全結束的文法形式。主要有「～てしまう」和「～た」兩種形式。

「～てしまう」不僅表示動作完成，還常常帶有說話者的情感色彩——遺憾、後悔或意外。例如「お菓子を全部食べてしまった」（把點心全吃掉了）既表達了吃完的事實，也隱含著吃太多的後悔。

相對地，「～た」形單純表示客觀的完成，情感色彩較淡。「本を読んだ」（看完書了）只是陳述事實。

口語中常使用「～ちゃう」「～じゃう」等縮約形式，在日常對話中非常普遍。

## 核心用法

### 用法 1：純粹完了（動作結束）

表示動作徹底完成，強調動作的完結。

**例句 1**
```
宿題を全部終わらせてしまった。
I finished all my homework.
我把作業全部做完了。
```

**例句 2**
```
この本、もう読んでしまいましたか。
Have you already finished reading this book?
這本書你已經看完了嗎？
```

### 用法 2：遺憾・後悔（不本意的結果）

表示對完成的動作感到遺憾或後悔。

**例句 3**
```
うっかり大事なメールを削除してしまった。
I accidentally deleted an important email.
我不小心把重要的郵件刪掉了。
```

**例句 4**
```
彼女に秘密を言ってしまった。
I ended up telling her the secret.
我把秘密說出去了。
```

### 用法 3：口語縮約形

日常會話中使用的縮略形式。

**例句 5**
```
あ、牛乳飲んじゃった。ごめん。
Oops, I drank the milk. Sorry.
啊，我把牛奶喝掉了。抱歉。
```

## 文法規則

### 規則 1：動詞接續規則

「～てしまう」接續遵循て形變化規則：
- 五段動詞：音便變化後加「しまう」
- 一段動詞：去「る」加「てしまう」
- カ変・サ変：「きてしまう」「してしまう」

### 規則 2：時態變化

| 形式 | 非過去 | 過去 |
|------|--------|------|
| 丁寧形 | てしまいます | てしまいました |
| 普通形 | てしまう | てしまった |
| 縮約形（非過去） | ちゃう/じゃう | ちゃった/じゃった |

### 規則 3：否定形式

「～てしまう」的否定較少使用，通常用「～ないでおく」或「～ないようにする」表達。

## 常見錯誤

### 錯誤 1：混淆完了與進行

❌ 誤：今、本を読んでしまっている。
✅ 正：今、本を読んでいる。／本を読んでしまった。
說明：「～てしまう」表示完了，不與進行式「～ている」同時使用表示正在進行。

### 錯誤 2：過度使用遺憾語氣

❌ 誤：合格してしまった。（本意是高興）
✅ 正：合格した！
說明：「～てしまう」帶有遺憾語氣，正面結果通常不用此形式。

### 錯誤 3：正式場合使用縮約形

❌ 誤：報告書を提出しちゃいました。（向上司報告）
✅ 正：報告書を提出してしまいました。
說明：「～ちゃう」是口語形式，正式場合應使用「～てしまう」。

## 學習要點

1. **區分完了與遺憾**：「～てしまう」根據語境可表示純粹完了或帶有遺憾，需依情況判斷。

2. **掌握縮約形**：「～ちゃう」「～じゃう」在口語中極為常見，是自然對話的必備知識。

3. **注意語域**：縮約形僅限口語，書面語和正式場合使用完整形式。

4. **與「～ておく」對比**：「～てしまう」強調完結，「～ておく」強調為未來做準備。

5. **感情表達**：善用「～てしまう」表達意外、遺憾、後悔等微妙情感。
