## 日文

れる・られる（尊敬語用法）

### 文法情報

| 項目 | 內容 |
|------|------|
| 文法類型 | 助動詞 |
| 用途 | 尊敬語表達 |
| JLPT | N4 |
| 使用場面 | 正式、商務、對長輩 |
| 形成 | 動詞未然形 + れる／られる |

### 形成規則

**基本形式**
```
五段動詞：動詞未然形（あ段）+ れる
一段動詞：動詞未然形（語幹）+ られる
不規則動詞：する → される、来る → 来られる
```

**範例**
```
書く（五段） → 書かれる
読む（五段） → 読まれる
食べる（一段） → 食べられる
見る（一段） → 見られる
する（不規則） → される
来る（不規則） → 来られる
```

## 日文解釋

「れる・られる」は、動詞に接続して尊敬の意味を表す助動詞[^jodoushi]です。相手や第三者の動作を高めて表現する際に用いられ、日本語の敬語体系[^keigo-system]において重要な役割を果たします。

五段動詞[^godan]には「れる」、一段動詞[^ichidan]には「られる」が接続します。ただし、同じ形が可能形[^kanoukei]や受身形[^ukemi]と重なるため、文脈[^context]によって意味を判断する必要があります。

ビジネス場面[^business]や目上の人[^meue]に対する会話では必須の表現であり、適切に使用できることは社会人[^shakaijin]として重要なコミュニケーション能力[^communication]の一つとされています。「お〜になる」という別の尊敬語表現[^sonkeigo-pattern]と比べると、やや硬い印象を与えることがあります。

## 英文解釋

The auxiliary verbs "reru" and "rareru" are used to express respect for the subject's actions in Japanese. They attach to the mizenkei (negative stem) of verbs: "reru" for godan verbs and "rareru" for ichidan verbs.

This construction is a key component of Japanese honorific language (keigo). It elevates the status of the person performing the action, showing respect and politeness. The same forms can also indicate potential or passive meanings, so context is crucial for proper interpretation.

In business settings and when speaking with superiors, this respectful form is essential. While it overlaps in function with the "o-~ni naru" pattern, the "reru/rareru" form tends to sound slightly more formal or stiff in certain contexts.

## 中文解釋

「れる・られる」是日文中表達尊敬語的助動詞形式，接續在動詞的未然形（否定語幹）之後。五段動詞使用「れる」，一段動詞使用「られる」。

這種表達方式是日文敬語系統中的核心組成部分，用於提高動作執行者的地位，表達尊敬和禮貌。由於這個形式與可能形和受身形相同，因此需要依據上下文來判斷實際意思。

在商務場合和對長輩說話時，這種尊敬語表達是必需的。與「お〜になる」這種尊敬語模式相比，「れる・られる」在某些情境下聽起來更為正式或較為生硬。

## 核心用法

### 用法 1：對長輩或上司的動作表達尊敬

對上司、長輩、客戶等地位較高者的動作使用尊敬語表達。

**例句 1**
```
部長は毎日７時に会社に来られます。
The department manager comes to the company at 7 o'clock every day.
部長每天七點來公司。
```

**例句 2**
```
先生はこの本を書かれました。
The professor wrote this book.
老師寫了這本書。
```

### 用法 2：詢問對方的意見或行動

禮貌地詢問對方的想法、意見或行動計劃。

**例句 3**
```
課長はどう思われますか。
What do you think, Section Chief?
課長您怎麼想？
```

**例句 4**
```
明日のパーティーに参加されますか。
Will you attend tomorrow's party?
您明天會參加派對嗎？
```

### 用法 3：描述第三者的動作（在正式場合）

在正式場合中，對不在場的第三者（如公司高層、客戶等）的動作也使用尊敬語。

**例句 5**
```
社長は来週アメリカに行かれるそうです。
I heard the president is going to America next week.
聽說社長下週要去美國。
```

## 文法規則

### 規則 1：動詞類型與接續形式

- **五段動詞**：將語尾變成あ段 + れる（例：書く → 書かれる）
- **一段動詞**：去掉る + られる（例：食べる → 食べられる）
- **不規則動詞**：する → される、来る（くる） → 来られる（こられる）

### 規則 2：與可能形、受身形的區別

「れる・られる」的形式與可能形和受身形完全相同，需要根據上下文判斷：

- **尊敬語**：主語是需要尊敬的人物（上司、客戶、長輩等）
- **可能形**：表達能力或可能性（主語通常是說話者或一般人）
- **受身形**：主語是動作的承受者（被動的對象）

判斷關鍵在於**主語的身分和句子的邏輯關係**。

### 規則 3：不適用於謙讓語的動詞

某些動詞已經有專門的尊敬語形式，通常不使用「れる・られる」：

- 言う → おっしゃる（較少用「言われる」）
- 食べる → 召し上がる（較少用「食べられる」，因會與可能形混淆）
- 見る → ご覧になる（較少用「見られる」）
- 行く → いらっしゃる（較少用「行かれる」）

### 規則 4：與「お〜になる」的比較

「れる・られる」和「お〜になる」都表達尊敬，但有細微差異：

- **お〜になる**：更自然、更常用，語感較柔和
- **れる・られる**：較正式、較硬，書面語色彩較重

範例：
- お書きになる vs 書かれる（都是「寫」的尊敬語）
- お読みになる vs 読まれる（都是「讀」的尊敬語）

## 常見錯誤

### 錯誤 1：與可能形混淆

可能形和尊敬語形式相同，但意思完全不同。

❌ 誤：私は日本語が話されます。（錯誤地使用尊敬語表達自己的能力）
✅ 正：私は日本語が話せます。（可能形，表達能力）
✅ 正：先生は日本語が話されます。（尊敬語，指老師會說日文）

說明：對自己的動作不能使用尊敬語。表達自己的能力應該使用可能形「話せます」。

### 錯誤 2：對自己的動作使用尊敬語

尊敬語只用於他人，不能用於自己的動作。

❌ 誤：私は明日会社に行かれます。
✅ 正：私は明日会社に行きます。
✅ 正：部長は明日会社に行かれます。

說明：尊敬語是提高對方地位的表達方式，對自己使用尊敬語會顯得不自然且不符合日文敬語邏輯。

### 錯誤 3：過度使用導致不自然

在日常對話中過度使用「れる・られる」會顯得過於生硬。

❌ 誤：田中さんは昨日本を読まれて、映画を見られて、友達と話されました。（過度使用）
✅ 正：田中さんは昨日本を読んで、映画を見て、友達と話しました。（一般朋友不需如此正式）
✅ 正：社長は昨日本を読まれて、会議に出席されました。（對社長才適合使用）

說明：尊敬語的使用需要考慮對象和場合。對一般朋友或同事不需要使用過度正式的尊敬語。

## 學習要點

1. **形式識別**：五段動詞用「れる」，一段動詞用「られる」，形成規則與受身形、可能形相同。

2. **語境判斷**：相同形式可能是尊敬語、可能形或受身形，需根據主語身分和句子邏輯判斷。

3. **使用對象**：只用於上司、長輩、客戶等需要尊敬的對象，不能用於自己或地位較低的人。

4. **場合選擇**：商務場合和正式場合必須使用，但日常對話中對朋友或家人不需使用。

5. **與「お〜になる」對比**：「れる・られる」較正式、較硬，「お〜になる」較自然、較常用。

## 註解

[^jodoushi]: **助動詞** - 接續在用言或助動詞後面，增加文法意義的詞類。詳見 [overview.md](../auxiliary/overview.md.md)

[^keigo-system]: **敬語體系** - 日語的敬語分為尊敬語、謙讓語、丁寧語三大類，是日本社會階層和禮儀文化的語言體現。詳見 [keigo_system.md](../concept/keigo_system.md.md)

[^godan]: **五段動詞** - 語尾變化會經過五個段（あいうえお）的動詞，又稱「う動詞」。詳見 [041_verb_classification.md](../concept/041_verb_classification.md.md)

[^ichidan]: **一段動詞** - 語幹不變，只變化語尾「る」的動詞，分為上一段和下一段，又稱「る動詞」。詳見 [041_verb_classification.md](../concept/041_verb_classification.md.md)

[^kanoukei]: **可能形** - 表達「能夠做某事」的動詞變化形式。形式與尊敬語相同，需由語境判斷。詳見 [002_kanoukei.md](../grammar/002_kanoukei.md.md)

[^ukemi]: **受身形** - 表達被動意義的動詞變化形式。形式與尊敬語相同，需由主語和邏輯關係判斷。詳見 [001_ukemi.md](../grammar/001_ukemi.md.md)

[^context]: **文脈（上下文）** - 語言使用的情境脈絡，包括前後文、說話對象、場合等因素。日語是高語境語言，許多意義需從文脈推斷。詳見 [038_given_new_information.md](../concept/038_given_new_information.md.md)

[^business]: **ビジネス場面** - 商務環境，包括公司內部溝通、與客戶交涉等正式職場情境。在這些場合中，適當使用敬語是基本禮儀。詳見 [business_conversation.md](../context/business_conversation.md.md)

[^meue]: **目上の人** - 地位、年齡、社會階層較高的人，如上司、長輩、客戶等。對目上的人必須使用尊敬語。詳見 [social_hierarchy.md](../concept/social_hierarchy.md.md)

[^shakaijin]: **社会人** - 已就業、有工作的成年人。在日本社會中，成為社會人意味著需要掌握商務禮儀和敬語使用。詳見 [shakaijin.md](../concept/shakaijin.md.md)

[^communication]: **コミュニケーション能力** - 溝通交流的能力。在日本職場中，適當使用敬語被視為重要的溝通能力。詳見 [communication](../domain/communication.md)

[^sonkeigo-pattern]: **尊敬語表現** - 日語中提高對方地位的表達方式，包括「れる・られる」、「お〜になる」、特殊尊敬語動詞等多種模式。詳見 [001_sonkeigo_overview.md](../honorific/001_sonkeigo_overview.md.md)
