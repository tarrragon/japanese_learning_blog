## 日文解釋

**状態動詞（じょうたいどうし）**

状態動詞とは、動作ではなく状態を表す動詞のことである。動作動詞が「行為の進行」を表すのに対し、状態動詞は「存在・所有・認知・感情」などの静的な状態を表す。

### 状態動詞の特徴

1. **進行形「〜ている」を取らない**：状態動詞は既に状態を表しているため、「〜ている」をつけると意味が重複するか不自然になる
   - ○ いる（exist）→ ✗ いている
   - ○ ある（exist）→ ✗ あっている

2. **辞書形で現在の状態を表す**：動作動詞が辞書形で習慣や未来を表すのに対し、状態動詞は辞書形で現在の状態を表す
   - 猫がいる（The cat is here - 現在）
   - 時間がある（I have time - 現在）

3. **瞬間的な変化がない**：状態動詞は持続的な状態を表し、動作のように開始点と終了点が明確でない

### 主要な状態動詞

**存在系**：
- ある[^aru] - 無生物の存在
- いる[^iru] - 有生物の存在

**能力・可能系**：
- できる[^dekiru] - 能力・可能
- 分かる - 理解している状態
- 見える - 見える状態（能力ではなく状態）
- 聞こえる - 聞こえる状態

**認知・思考系**：
- 要る - 必要である状態
- 足りる - 足りている状態

**感情・感覚系**：
- 好む - 好んでいる状態（ただし「好んでいる」も使われる）

### 動作動詞との対比

| 動作動詞 | 状態動詞 |
|---------|---------|
| 食べる（eating） | ある（existing） |
| 走る（running） | いる（being） |
| 作る（making） | できる（being able） |
| 聞く（listening） | 聞こえる（being audible） |
| 見る（looking） | 見える（being visible） |

## English Explanation

**State Verbs (Stative Verbs)**

State verbs are verbs that express states rather than actions. While action verbs describe "actions in progress," state verbs describe static conditions such as "existence, possession, cognition, and emotion."

### Characteristics of State Verbs

1. **Cannot take progressive 〜ている**: Since state verbs already express a state, adding 〜ている creates redundancy or sounds unnatural
   - ○ いる (exist) → ✗ いている
   - ○ ある (exist) → ✗ あっている

2. **Dictionary form expresses present state**: While action verbs use dictionary form for habits or future, state verbs use it for present state
   - 猫がいる (The cat is here - present)
   - 時間がある (I have time - present)

3. **No momentary change**: State verbs express continuous states without clear starting and ending points like actions

### Main State Verbs

**Existence**:
- ある - existence of inanimate objects
- いる - existence of animate beings

**Ability/Possibility**:
- できる - ability/possibility
- 分かる - state of understanding
- 見える - state of being visible (not the action of looking)
- 聞こえる - state of being audible

**Cognition/Thought**:
- 要る - state of being needed
- 足りる - state of being sufficient

### Comparison with Action Verbs

The key distinction is that action verbs can answer "What are you doing now?" while state verbs describe conditions that simply exist.

## 中文解釋

**狀態動詞**

狀態動詞是表示狀態而非動作的動詞。動作動詞描述「進行中的行為」，而狀態動詞則描述「存在、所有、認知、感情」等靜態狀態。

### 狀態動詞的特徵

1. **不能接續進行式「〜ている」**：狀態動詞已經表示狀態，加上「〜ている」會語意重複或不自然
   - ○ いる（存在）→ ✗ いている
   - ○ ある（存在）→ ✗ あっている

2. **辭書形表示現在狀態**：動作動詞的辭書形表示習慣或未來，而狀態動詞的辭書形表示現在狀態
   - 猫がいる（貓在這裡 - 現在）
   - 時間がある（我有時間 - 現在）

3. **沒有瞬間變化**：狀態動詞表示持續狀態，不像動作有明確的開始點和結束點

### 主要的狀態動詞

**存在類**：
- ある - 無生命物的存在
- いる - 有生命體的存在

**能力・可能類**：
- できる - 能力・可能
- 分かる - 理解的狀態
- 見える - 可見的狀態（不是看的動作）
- 聞こえる - 可聽見的狀態

**認知・思考類**：
- 要る - 需要的狀態
- 足りる - 足夠的狀態

### 與動作動詞的對比

關鍵區別在於：動作動詞可以回答「你現在在做什麼？」，而狀態動詞描述的是單純存在的條件。

## 核心例句

### 例句 1：ある（存在 - 無生物）

```
机の上に本がある。
Tsukue no ue ni hon ga aru.
There is a book on the desk.
桌上有書。
```

**解說**：「ある」是狀態動詞，辭書形直接表示現在狀態「有」，不能說「あっている」。

### 例句 2：いる（存在 - 有生物）

```
庭に猫がいる。
Niwa ni neko ga iru.
There is a cat in the garden.
庭院裡有貓。
```

**解說**：「いる」是狀態動詞，表示有生命體的存在狀態，不能說「いている」。

### 例句 3：できる（能力）

```
彼は英語ができる。
Kare wa eigo ga dekiru.
He can speak English.
他會英語。
```

**解說**：「できる」表示能力狀態，不是「正在能夠」的動作，因此不用「できている」表示能力。

### 例句 4：分かる（理解狀態）

```
意味が分かる。
Imi ga wakaru.
I understand the meaning.
我明白意思。
```

**解說**：「分かる」是狀態動詞，表示理解的狀態。「分かっている」有時使用，但意思變成「已經知道」（結果狀態）。

### 例句 5：見える（可見狀態）

```
ここから富士山が見える。
Koko kara Fujisan ga mieru.
Mt. Fuji is visible from here.
從這裡可以看到富士山。
```

**解說**：「見える」是狀態動詞，表示「可見」的狀態，與動作動詞「見る」（看）不同。

### 例句 6：聞こえる（可聽狀態）

```
遠くで音楽が聞こえる。
Tooku de ongaku ga kikoeru.
Music can be heard in the distance.
遠處可以聽到音樂。
```

**解說**：「聞こえる」表示「可聽見」的狀態，與動作動詞「聞く」（聽）不同。

## 動作動詞 vs 狀態動詞對比

### 對比 1：見る vs 見える

**見る（動作動詞）**
```
映画を見ている。
Eiga wo mite iru.
I am watching a movie.
我正在看電影。
```

**見える（狀態動詞）**
```
星が見える。
Hoshi ga mieru.
The stars are visible.
可以看到星星。
```

### 對比 2：聞く vs 聞こえる

**聞く（動作動詞）**
```
音楽を聞いている。
Ongaku wo kiite iru.
I am listening to music.
我正在聽音樂。
```

**聞こえる（狀態動詞）**
```
声が聞こえる。
Koe ga kikoeru.
A voice can be heard.
可以聽到聲音。
```

### 對比 3：作る vs できる

**作る（動作動詞）**
```
ケーキを作っている。
Keeki wo tsukutte iru.
I am making a cake.
我正在做蛋糕。
```

**できる（狀態動詞 - 能力）**
```
日本語ができる。
Nihongo ga dekiru.
I can speak Japanese.
我會日語。
```

## 常見錯誤

### 錯誤 1：對狀態動詞使用進行式

❌ 誤：猫がいている。
✅ 正：猫がいる。
說明：「いる」已經表示存在狀態，不需要也不能加「〜ている」。

### 錯誤 2：對狀態動詞使用進行式表示能力

❌ 誤：彼は英語ができている。（表示能力時）
✅ 正：彼は英語ができる。
說明：表示能力時使用辭書形。「できている」只用於表示「已完成」的結果狀態。

### 錯誤 3：混淆見る和見える

❌ 誤：星を見る。（想表達「看得見」時）
✅ 正：星が見える。
說明：「見る」是意志動作（看），「見える」是自發狀態（可見）。助詞也不同：見る用「を」，見える用「が」。

### 錯誤 4：混淆聞く和聞こえる

❌ 誤：音楽を聞こえる。
✅ 正：音楽が聞こえる。
說明：「聞こえる」是狀態動詞，主語用「が」標示，不用「を」。

## 學習要點

1. **識別狀態動詞**：掌握常見的狀態動詞清單（ある、いる、できる、分かる、見える、聞こえる等），這些動詞表示狀態而非動作。

2. **進行式限制**：狀態動詞通常不接「〜ている」，因為辭書形已經表示現在狀態。例外：部分動詞如「分かる」可以用「分かっている」表示結果狀態。

3. **與動作動詞對比**：
   - 見る（看的動作）↔ 見える（可見的狀態）
   - 聞く（聽的動作）↔ 聞こえる（可聽的狀態）

4. **助詞使用**：狀態動詞常用「が」標示主語，而非「を」。

5. **與英語比較**：日語的狀態動詞概念與英語的 stative verbs 類似，但分類和用法有差異。

## 註解

[^aru]: **ある** - 無生物存在動詞，是日語中最基本的狀態動詞之一。表示物品、場所、抽象概念的存在。否定形為不規則的「ない」。詳見 [ある/いる](099_aru_iru.md)

[^iru]: **いる** - 有生物存在動詞，用於人、動物等能自主移動的生命體。與「ある」形成日語存在表達的核心對比。詳見 [ある/いる](099_aru_iru.md)

[^dekiru]: **できる** - 表示能力或可能性的狀態動詞，是「する」的可能形。作為狀態動詞使用時，辭書形表示現在具備的能力。詳見 [できる](063_dekiru.md)
