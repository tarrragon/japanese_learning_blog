## 日文

たら条件形（たらじょうけんけい）

### 文法情報

| 項目 | 內容 |
|------|------|
| 文法類型 | 條件表達 |
| JLPT | N4 |
| 使用場面 | 日常、正式、書面 |
| 形成 | 動詞・形容詞た形 + ら |

### 形成規則

**基本形式**
```
動詞た形 + ら
い形容詞た形 + ら
な形容詞だった + ら
名詞だった + ら
```

**活用範例**
```
行く → 行った → 行ったら
食べる → 食べた → 食べたら
高い → 高かった → 高かったら
便利 → 便利だった → 便利だったら
雨 → 雨だった → 雨だったら
```

## 日文解釋

「たら」は日本語の条件表現[^conditional-form]の一つで、仮定条件[^hypothetical]や時間的順序[^temporal-sequence]を表します。動詞や形容詞のた形[^ta-form]に「ら」を付けて作ります。

この表現には主に三つの用法があります。第一に、仮定条件を表す用法で「もし〜なら」という意味です。第二に、確定条件[^definite-condition]を表す用法で、前件が実現した後に後件が起こることを示します。第三に、発見や気づき[^discovery]を表す用法で、「〜したところ、〜だった」という意味になります。

「たら」は話し言葉[^spoken-language]でも書き言葉[^written-language]でも使われる汎用性[^versatility]の高い条件表現です。他の条件表現である「と」「ば」「なら」と比べて、使える場面が広く、後件に意志表現[^volitional-expression]や命令形[^imperative]も使えるという特徴があります。

## 英文解釋

"Tara" is one of the conditional expressions[^conditional-form] in Japanese, used to express hypothetical conditions[^hypothetical] or temporal sequences[^temporal-sequence]. It is formed by adding "ra" to the ta-form[^ta-form] of verbs and adjectives.

This expression has three main usages. First, it expresses hypothetical conditions with the meaning "if" or "when." Second, it expresses definite conditions[^definite-condition], indicating that the consequence occurs after the condition is fulfilled. Third, it expresses discovery[^discovery], with the meaning "when I did X, I found that Y."

"Tara" is a highly versatile[^versatility] conditional expression used in both spoken[^spoken-language] and written[^written-language] language. Compared to other conditional forms like "to," "ba," and "nara," it has broader applications and can be followed by volitional expressions[^volitional-expression] and imperative forms[^imperative].

## 中文解釋

「たら」是日文的條件表達[^conditional-form]之一，用來表示假定條件[^hypothetical]或時間順序[^temporal-sequence]。它的形成方式是在動詞或形容詞的た形[^ta-form]後加「ら」。

這個表達方式主要有三種用法。第一種是表示假定條件，意思是「如果～的話」。第二種是表示確定條件[^definite-condition]，表示前項實現後，後項就會發生。第三種是表示發現或察覺[^discovery]，意思是「做了～之後，發現～」。

「たら」是口語[^spoken-language]和書面語[^written-language]都能使用的多功能[^versatility]條件表達。相較於其他條件表達如「と」「ば」「なら」，它的適用範圍更廣，後句可以接意志表達[^volitional-expression]和命令形[^imperative]。

## 核心用法

### 用法 1：假定條件（如果～的話）

表示假設的情況，相當於中文的「如果～的話」或「要是～就」。

**例句 1**
```
雨が降ったら、試合は中止になります。
If it rains, the match will be canceled.
如果下雨的話，比賽就會取消。
```

**例句 2**
```
時間があったら、一緒に映画を見に行きませんか。
If you have time, would you like to go see a movie together?
如果有時間的話，要不要一起去看電影？
```

### 用法 2：確定條件（做了～之後）

表示前項動作完成後，後項動作或狀態會發生。強調時間順序。

**例句 3**
```
家に帰ったら、すぐ電話してください。
When you get home, please call me right away.
回到家之後，請馬上打電話給我。
```

**例句 4**
```
この本を読んだら、貸してください。
When you finish reading this book, please lend it to me.
看完這本書之後，請借給我。
```

### 用法 3：發現・意外（做了～發現）

表示做了某事之後發現意外的結果或狀況。

**例句 5**
```
窓を開けたら、猫が入ってきた。
When I opened the window, a cat came in.
打開窗戶後，有隻貓跑進來了。
```

## 文法規則

### 規則 1：後件的限制較少

與「と」條件形不同，「たら」的後件可以是意志表達、命令、請求等。

- ✅ 春になったら、桜を見に行きましょう。（意志）
- ✅ 駅に着いたら、電話してください。（請求）
- ✅ 暇だったら、手伝って。（命令）

### 規則 2：可表示過去的發現

「たら」可以用來描述過去發現的事實。

- 家に帰ったら、誰もいなかった。（過去的發現）
- 起きたら、もう昼だった。（過去的發現）

### 規則 3：可與「もし」搭配

表示假定條件時，常與「もし」（如果）一起使用，加強假設語氣。

- もし時間があったら、連絡してください。
- もし分からなかったら、聞いてください。

## 常見錯誤

### 錯誤 1：混淆「たら」與「と」

「と」的後件不能是意志表達或命令，而「たら」可以。

❌ 誤：春になると、花見に行きましょう。
✅ 正：春になったら、花見に行きましょう。
說明：「行きましょう」是意志表達，應該用「たら」而不是「と」。

### 錯誤 2：時態混用

「たら」本身已經包含了た形，前件不需要再加時態標記。

❌ 誤：明日雨が降るだったら、行きません。
✅ 正：明日雨が降ったら、行きません。
說明：「降る」變成た形是「降った」，不是「降るだった」。

### 錯誤 3：過度使用於恆常條件

表示恆常真理或自然規律時，用「と」比「たら」更自然。

❌ 不自然：春になったら、花が咲きます。（每年都是）
✅ 自然：春になると、花が咲きます。
說明：表示自然規律時，「と」比「たら」更恰當。

## 學習要點

1. **形成方式**：た形 + ら，適用於動詞、形容詞、名詞
2. **三大用法**：假定條件、確定條件、發現・意外
3. **後件自由**：可接意志、命令、請求等表達
4. **時間順序**：強調前項完成後才發生後項
5. **與「と」「ば」「なら」的區別**：「たら」最為靈活多用
