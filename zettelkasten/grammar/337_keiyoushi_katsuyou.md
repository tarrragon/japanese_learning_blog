## 日文解釋

日本語における形容詞の活用とは、形容詞が文中の役割や接続する語によって形を変える現象を指します。日本語の形容詞は大きく分けてイ形容詞（形容詞）とナ形容詞（形容動詞）の二種類があり、それぞれ異なる活用パターンを持っています。

イ形容詞は語尾が「い」で終わり、この「い」を取り除いて様々な活用語尾を付けることで、否定、過去、連用、て形などの形を作ります。例えば「高い」であれば、「高くない」（否定）、「高かった」（過去）、「高く」（連用）、「高くて」（て形）のように変化します。

一方、ナ形容詞は名詞を修飾する際に「な」が必要なことからその名前が付けられています。ナ形容詞の活用は名詞に近く、「だ」「で」「に」などの助動詞や助詞と組み合わせて形を変えます。例えば「静か」であれば、「静かではない」（否定）、「静かだった」（過去）、「静かに」（連用）、「静かで」（て形）となります。

両者の活用パターンを正確に理解し使い分けることは、文法的に正しい日本語を話すための基礎となります。

## 英文解釋

Adjective conjugation (形容詞活用) refers to the systematic changes that Japanese adjectives undergo depending on their grammatical function and the words they connect to. Japanese has two main types of adjectives, each with distinct conjugation patterns.

**I-adjectives (イ形容詞)** end in い and conjugate by removing this ending and adding various suffixes:
- Negative: 高い → 高くない (takai → takakunai, "not expensive")
- Past: 高い → 高かった (takai → takakatta, "was expensive")
- Adverbial: 高い → 高く (takai → takaku, "expensively")
- Te-form: 高い → 高くて (takai → takakute, connecting form)

**Na-adjectives (ナ形容詞)** conjugate more like nouns, using auxiliary words:
- Negative: 静か → 静かではない (shizuka → shizuka de wa nai, "not quiet")
- Past: 静か → 静かだった (shizuka → shizuka datta, "was quiet")
- Adverbial: 静か → 静かに (shizuka → shizuka ni, "quietly")
- Te-form: 静か → 静かで (shizuka → shizuka de, connecting form)

Mastering these patterns is essential for constructing grammatically correct sentences in Japanese.

## 中文解釋

形容詞活用是指日語形容詞根據其在句中的語法功能和所連接的詞彙而發生的系統性變化。日語有兩大類形容詞，各自擁有獨特的活用規則。

**イ形容詞**以「い」結尾，活用時去掉「い」後加上各種語尾：
- 否定形：高い → 高くない（不貴）
- 過去形：高い → 高かった（以前很貴）
- 連用形：高い → 高く（貴地，修飾動詞）
- て形：高い → 高くて（連接後續內容）

**ナ形容詞**的活用方式類似名詞，需搭配助動詞：
- 否定形：静か → 静かではない（不安靜）
- 過去形：静か → 静かだった（以前很安靜）
- 連用形：静か → 静かに（安靜地）
- て形：静か → 静かで（連接後續內容）

學習者需要特別注意：某些以「い」結尾的詞彙實際上是ナ形容詞（如「きれい」「嫌い」），必須按照ナ形容詞的規則活用，不能誤用イ形容詞的活用方式。正確掌握這兩類形容詞的活用模式，是建構正確日語句子的基礎能力。

## 核心用法

### 用法 1：述語用法（作為謂語）

形容詞直接放在句末，描述主語的性質或狀態。這是形容詞最基本的用法。

**例句 1**
```
この部屋は広い。
This room is spacious.
這個房間很寬敞。
```

**例句 2**
```
あの映画は有名だ。
That movie is famous.
那部電影很有名。
```

### 用法 2：連體修飾（修飾名詞）

形容詞放在名詞前面，描述該名詞的特徵。イ形容詞直接接名詞，ナ形容詞需加「な」。

**例句 3**
```
新しいスマホを買いたい。
I want to buy a new smartphone.
我想買一台新手機。
```

**例句 4**
```
便利な場所に住んでいます。
I live in a convenient location.
我住在交通便利的地方。
```

### 用法 3：連用修飾（修飾動詞）

形容詞變成連用形後可以修飾動詞。イ形容詞用「く」，ナ形容詞用「に」。

**例句 5**
```
彼女は日本語を上手に話す。彼は速く走ることができる。
She speaks Japanese skillfully. He can run fast.
她日語說得很好。他跑得很快。
```

## 常見錯誤

### 錯誤 1：將ナ形容詞誤用為イ形容詞

❌ 誤：きれいくない
✅ 正：きれいではない／きれいじゃない
說明：「きれい」雖然以「い」結尾，但它是ナ形容詞，不能使用イ形容詞的「くない」否定形。

### 錯誤 2：イ形容詞過去形的錯誤變化

❌ 誤：高いだった
✅ 正：高かった
說明：イ形容詞的過去形是將「い」變成「かった」，不是加「だった」。「だった」是ナ形容詞和名詞的過去形標記。

### 錯誤 3：混淆連體形和連用形

❌ 誤：静かく話す
✅ 正：静かに話す
說明：ナ形容詞修飾動詞時用「に」形成連用形，不是「く」。「く」是イ形容詞的連用形語尾。

## 學習要點

1. **識別形容詞類型**：大多數以「い」結尾的是イ形容詞，但「きれい」「嫌い」「有名」等是ナ形容詞，需要特別記憶。

2. **否定形的形成**：イ形容詞去「い」加「くない」；ナ形容詞加「ではない」或口語的「じゃない」。

3. **過去形的形成**：イ形容詞去「い」加「かった」；ナ形容詞加「だった」。

4. **連用形的區別**：イ形容詞用「く」形，ナ形容詞用「に」形來修飾動詞。

5. **て形的連接功能**：イ形容詞「くて」、ナ形容詞「で」用於連接多個形容詞或句子成分。
