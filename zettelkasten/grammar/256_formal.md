## 日文

正式表現（せいしきひょうげん）

### 文法情報

| 項目 | 內容 |
|------|------|
| 文法類型 | 語體・待遇表現 |
| JLPT | N4-N3 |
| 使用場面 | 商務、正式場合、書面語 |
| 特徵 | です・ます形、敬語、書面表達 |

## 日文解釋

正式表現とは、ビジネスや公式な場面で使用される丁寧な話し方です[^formal-context]。敬語[^keigo]やです・ます形[^desu-masu]を用いることで、相手への敬意を表します。

日本語の正式表現には以下の特徴があります：

1. **丁寧体の使用**：動詞の「〜ます」形、形容詞・名詞の「〜です」形
2. **敬語の使用**：尊敬語[^sonkeigo]・謙譲語[^kenjougo]による待遇表現
3. **書面語的表現**：「〜において」「〜に関しまして」などの硬い表現
4. **婉曲表現**：直接的な表現を避け、遠回しに伝える

正式表現は単に文法形式だけでなく、相手との関係性や場面に応じた適切な言葉遣いを選ぶことが重要です。

## 英文解釋

Formal expressions in Japanese refer to polite language styles used in business and official settings[^formal-context]. By using honorific language[^keigo] and polite forms[^desu-masu], speakers show respect to their interlocutors.

Key characteristics of formal Japanese include:

1. **Polite forms**: The "〜ます" form for verbs, "〜です" form for adjectives and nouns
2. **Honorific language**: Respectful language[^sonkeigo] and humble language[^kenjougo]
3. **Written-style expressions**: Formal phrases like "において" (in/at) and "に関しまして" (regarding)
4. **Indirect expressions**: Avoiding direct statements in favor of circumlocution

Mastering formal expressions requires not just grammatical knowledge, but understanding when and how to appropriately adjust language based on context and relationships.

## 中文解釋

正式表達是日語中用於商務和正式場合的禮貌說話方式[^formal-context]。透過使用敬語[^keigo]和です・ます形[^desu-masu]來表達對對方的敬意。

日語正式表達的特徵包括：

1. **丁寧體的使用**：動詞「〜ます」形、形容詞・名詞「〜です」形
2. **敬語的使用**：尊敬語[^sonkeigo]・謙讓語[^kenjougo]的待遇表現
3. **書面語表達**：「〜において」「〜に関しまして」等正式表達
4. **婉轉表達**：避免直接表達，以迂迴方式傳達

掌握正式表達不僅需要了解語法形式，更重要的是根據場合和人際關係選擇適當的用語。

## 核心用法

### 用法 1：丁寧體（です・ます形）

最基本的正式表達，適用於一般正式場合。

**例句 1**
```
私は田中と申します。よろしくお願いいたします。
I am Tanaka. It's nice to meet you.
我叫田中。請多多指教。
```

**例句 2**
```
こちらの資料をご覧ください。
Please look at these materials.
請看這份資料。
```

### 用法 2：敬語表達

根據對象使用尊敬語或謙讓語。

**例句 3（尊敬語）**
```
社長はいつお帰りになりますか。
When will the president return?
社長什麼時候回來？
```

**例句 4（謙讓語）**
```
明日、お伺いしてもよろしいでしょうか。
Would it be all right if I visit tomorrow?
明天可以前去拜訪嗎？
```

### 用法 3：書面語表達

用於正式文書、報告等書面場合。

**例句 5**
```
本件につきまして、ご検討のほどよろしくお願い申し上げます。
We kindly request your consideration regarding this matter.
關於此事，懇請您審查考慮。
```

**例句 6**
```
上記の件に関しまして、ご報告申し上げます。
I would like to report regarding the above matter.
關於上述事項，向您報告。
```

## 文法規則

### 規則 1：動詞的正式形

| 普通形 | 正式形 | 更正式 |
|--------|--------|--------|
| 行く | 行きます | 参ります |
| 食べる | 食べます | いただきます |
| する | します | いたします |
| 見る | 見ます | 拝見します |
| 言う | 言います | 申します |

### 規則 2：形容詞・名詞的正式形

| 普通形 | 正式形 |
|--------|--------|
| 高い | 高いです |
| きれいだ | きれいです |
| 学生だ | 学生です |
| ない | ありません / ございません |

### 規則 3：婉轉表達

| 直接表達 | 婉轉表達 |
|----------|----------|
| できません | できかねます |
| 知りません | 存じません |
| いません | いらっしゃいません |
| わかりません | わかりかねます |

## 使用場合

### 適用場合

1. **商務環境**：會議、商談、電話、電子郵件
2. **正式演講**：發表會、致詞、報告
3. **書面文件**：申請書、報告書、公文
4. **對上級**：與上司、老師、長輩交流
5. **對陌生人**：初次見面、客戶服務

### 不適用場合

- 朋友間的日常對話
- 家人之間的交流
- 非正式的社交場合

## 常見錯誤

### 錯誤 1：敬語過度使用

❌ 誤：お電話をおかけになられました。
✅ 正：お電話をおかけになりました。
說明：敬語不可重複使用（二重敬語）。

### 錯誤 2：自他混淆

❌ 誤：社長がおっしゃられました。
✅ 正：社長がおっしゃいました。
說明：「おっしゃる」已是尊敬語，不需再加「られる」。

### 錯誤 3：場合不適當

❌ 誤：（對朋友）本日はお忙しい中、お越しいただきありがとうございます。
✅ 正：今日は来てくれてありがとう。
說明：過度正式會造成距離感，需根據場合調整。

## 學習要點

1. **從基礎開始**：先掌握です・ます形，再學習敬語
2. **理解場合**：同樣的內容，不同場合用不同程度的正式度
3. **注意一致性**：一段話中保持相同的語體
4. **適度使用**：過度正式反而顯得不自然
5. **多觀察學習**：從日劇、新聞、商務場合學習實際用法

## 註解

[^formal-context]: **正式場合** - 商務會議、正式演講、書面文件、與上級或陌生人交流等需要使用禮貌語言的情境。

[^keigo]: **敬語** - 日語中表示敬意的語言系統，包含尊敬語、謙讓語、丁寧語。詳見 [keigo_system](008_keigo_system.md)

[^desu-masu]: **です・ます形** - 日語丁寧體的基本形式，是正式表達的基礎。

[^sonkeigo]: **尊敬語** - 用於抬高對方地位的敬語，如「いらっしゃる」「おっしゃる」。

[^kenjougo]: **謙讓語** - 用於降低自己地位的敬語，如「参る」「申す」。詳見 [kenjougo_system](160_kenjougo_system.md)
