## 日文解釋

否定表現とは、動作・状態・性質が存在しないこと、または成立しないことを表す文法体系である。

日本語の否定表現は、単に「ない」を付けるだけでなく、様々な形式と微妙なニュアンスを持つ。基本的な「〜ない」形式から、丁寧な「〜ません」、書き言葉的な「〜ぬ」「〜ず」、さらに二重否定や部分否定まで、豊かな表現体系を形成している。

特に注目すべき点は、日本語では否定が文末に来るため、話者は文の最後まで聞かないと肯定か否定かわからないという特徴がある。これは英語のように文頭近くで否定が現れる言語とは大きく異なる。

また、日本語の否定表現は単純な否定だけでなく、婉曲表現や推測、可能性の表現など、多様な語用論的機能を果たす。例えば「よくないですか」は否定疑問形式でありながら、実際には勧誘や確認の意味で使われることが多い。

## 英文解釋

Negative expressions in Japanese (hitei hyougen) form a comprehensive grammatical system that indicates the absence, non-existence, or negation of actions, states, and qualities.

Unlike English where negation typically appears near the beginning of a sentence (e.g., "I don't..."), Japanese places negation at the end of sentences. This fundamental structural difference means listeners must wait until the sentence conclusion to determine whether it is affirmative or negative.

The Japanese negation system includes:
1. Basic negation with -nai for verbs and adjectives
2. Polite negation with -masen
3. Classical/literary forms like -nu and -zu
4. Double negatives and partial negation
5. Pragmatic uses such as softening requests or expressing uncertainty

Understanding these various forms and their appropriate contexts is essential for natural Japanese communication.

## 中文解釋

否定表現是日語中表達「不、沒有、不是」等否定意義的文法系統。

日語否定表達的特點在於否定詞位於句尾，這與中文「不」、「沒」放在動詞前面的結構完全不同。因此，聽日語時必須等到句子結束才能確定是肯定句還是否定句。

日語否定系統包含多個層次：
- **基本否定**：動詞、形容詞加上「ない」
- **禮貌否定**：使用「ません」形式
- **書面否定**：使用「ぬ」「ず」等古典形式
- **雙重否定**：如「行かないわけではない」（並非不去）
- **部分否定**：如「全部ではない」（並非全部）

此外，日語否定還有許多語用功能，如委婉表達、推測、可能性暗示等，需要根據語境靈活理解和使用。

## 核心用法

### 用法 1：基本動詞否定（〜ない）

最基本的否定形式，表示動作不發生或不進行。

**例句 1**
```
今日は会社に行かない。
I'm not going to the office today.
今天不去公司。
```

**例句 2**
```
朝ごはんはあまり食べない。
I don't really eat breakfast.
早餐不太吃。
```

### 用法 2：形容詞否定（〜くない／〜ではない）

い形容詞用「くない」，な形容詞用「ではない」。

**例句 3**
```
この部屋は広くない。
This room is not spacious.
這個房間不寬敞。
```

**例句 4**
```
彼女は有名ではない。
She is not famous.
她不有名。
```

### 用法 3：丁寧體否定（〜ません）

禮貌場合使用的否定形式。

**例句 5**
```
申し訳ございませんが、そちらの商品は取り扱っておりません。
I'm sorry, but we don't carry that product.
非常抱歉，我們不販售那項商品。
```

### 用法 4：部分否定（全て〜ない vs 〜全てが〜わけではない）

區分「完全不」和「不是全部」的差異。

**例句 6**
```
全ての学生が合格したわけではない。
Not all students passed.
並非所有學生都合格了。
```

### 用法 5：雙重否定（〜ないわけではない）

表示「並非不…」，暗示某種程度的肯定。

**例句 7**
```
行きたくないわけではないが、時間がない。
It's not that I don't want to go, but I don't have time.
並非不想去，只是沒有時間。
```

## 文法規則

### 規則 1：動詞否定形的形成

**五段動詞**：將語尾假名變為「あ段」+ ない
- 書く → 書か + ない → 書かない
- 読む → 読ま + ない → 読まない
- 行く → 行か + ない → 行かない

**一段動詞**：去掉「る」+ ない
- 食べる → 食べ + ない → 食べない
- 見る → 見 + ない → 見ない

**不規則動詞**：
- する → しない
- 来る → 来ない（こない）

### 規則 2：形容詞否定形的形成

**い形容詞**：去掉「い」+ くない
- 高い → 高くない
- 美しい → 美しくない

**な形容詞**：詞幹 + ではない（口語：じゃない）
- 静か → 静かではない / 静かじゃない
- 便利 → 便利ではない / 便利じゃない

### 規則 3：時態與否定的結合

| 時態 | 動詞例（行く） | い形容詞例（高い） |
|------|---------------|-----------------|
| 現在否定 | 行かない | 高くない |
| 過去否定 | 行かなかった | 高くなかった |
| 現在否定丁寧 | 行きません | 高くありません |
| 過去否定丁寧 | 行きませんでした | 高くありませんでした |

## 常見錯誤

### 錯誤 1：い形容詞否定形式錯誤

❌ 誤：この料理は美味しいじゃない。
✅ 正：この料理は美味しくない。
說明：い形容詞否定必須用「〜くない」，不能用「〜じゃない」。「じゃない」只用於な形容詞和名詞。

### 錯誤 2：混淆全體否定與部分否定

❌ 誤：みんなが来なかった。（想表達「並非所有人都來」）
✅ 正：みんなが来たわけではない。
說明：「みんなが来なかった」意為「所有人都沒來」，若要表達「並非所有人都來」，應使用「〜わけではない」。

### 錯誤 3：ない形與ません的混用

❌ 誤：私は行かないです。（過於口語化的混搭）
✅ 正：私は行きません。（正式）/ 私は行かない。（普通）
說明：雖然「〜ないです」在現代日語中逐漸被接受，但在正式場合仍建議使用「〜ません」。

## 學習要點

1. **否定位置**：日語否定詞在句尾，與中文、英文不同，需要聽完整句才能判斷肯否。

2. **形式區分**：い形容詞用「くない」，な形容詞和名詞用「ではない」，不可混用。

3. **語域選擇**：日常對話用「ない」形，正式場合用「ません」形，書面語有時用「ぬ」「ず」。

4. **部分否定**：使用「〜わけではない」「〜とは限らない」表達「並非…」的部分否定。

5. **雙重否定**：「〜ないわけではない」不等於完全肯定，而是帶有保留或委婉的語氣。
