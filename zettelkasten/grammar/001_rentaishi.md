## 日文
連体詞（れんたいし）

**羅馬拼音**：rentaishi

**品詞分類**：連体詞（修飾専用品詞）

## 日文解釋
「連体詞」は、常に名詞を修飾する機能だけを持つ品詞です。形容詞と異なり、活用がなく、述語として単独で使うことができません。「この」「その」「あの」などの指示連体詞、「大きな」「小さな」などの性質を表す連体詞、「ある」「とんだ」などの特殊な連体詞があります。名詞の前に置いて、その名詞の性質や状態を限定する役割を果たします。

## 英文解釋
Rentaishi (連体詞) are a class of words that function exclusively to modify nouns and cannot be used independently as predicates. Unlike adjectives, rentaishi do not conjugate. They include demonstrative rentaishi like この (kono, "this"), その (sono, "that"), and あの (ano, "that over there"), as well as descriptive rentaishi like 大きな (ookina, "big") and 小さな (chiisana, "small"). Rentaishi always appear before the noun they modify and serve to specify or limit the characteristics of that noun.

## 中文解釋
連體詞（れんたいし）是專門用於修飾名詞的品詞，無法單獨作為述語使用。與形容詞不同，連體詞沒有活用變化。連體詞包括指示連體詞（如：この「這個」、その「那個」、あの「那個（遠處）」）、性質連體詞（如：大きな「大的」、小さな「小的」）、以及特殊連體詞（如：ある「某個」、とんだ「意外的」）。連體詞總是放在名詞前面，用於限定或說明該名詞的性質或狀態。

## 例句

**例句1**
```

```

```

Let's rest under the big tree.
我們在大樹下休息吧。
```

**例句3**
```

那個人是我的老師。
```

```
ある日、不思議なことが起きました。

某一天，發生了不可思議的事情。
```

```
小さな子供が公園で遊んでいます。

小孩子在公園裡玩耍。
```

## 補充說明

### 連体詞の主要分類
1. **指示連体詞**：この・その・あの・どの（指示物品或位置）
2. **性質連体詞**：大きな・小さな・おかしな（描述性質特徵）
3. **特殊連体詞**：ある・いわゆる・とんだ・さる（特定語境使用）

### 連体詞 vs. い形容詞的重要區別[^1]
- **[大きい](../adj-i/001_大きい.md)**（形容詞）：可單獨作述語「この木は大きい」、可活用「大きくない」
- **大きな**（連体詞）：❌ 不可單獨作述語、❌ 不可活用、✅ 只能修飾名詞

### 使用注意事項
1. **固定形態**：連体詞沒有活用變化，形態固定
2. **修飾專用**：必須接在名詞之前，不能單獨使用
3. **語感差異**：「大きな」比「[大きい](../adj-i/001_大きい.md)」更強調主觀感受或意外性
4. **慣用搭配**：某些連体詞有固定搭配（如：ある日、いわゆる専門家）

### 常見錯誤
❌ この、その等連体詞不能單獨使用，必須接名詞
⭕ このペン（這支筆）、その人（那個人）

❌ 大きなは使えません。（連体詞不能作述語）
⭕ 大きな犬を見ました。（連体詞修飾名詞）
