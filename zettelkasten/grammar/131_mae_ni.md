## 日文

〜前に（〜まえに）

**羅馬拼音**：mae ni

**文法類型**：時間表現

### 文法情報

| 項目 | 內容 |
|------|------|
| 文法類型 | 時間表現 |
| JLPT | N5 |
| 使用場面 | 日常、正式、書面 |
| 形成 | 動詞辭書形 / 名詞 + の + 前に |

### 形成規則

**基本形式**
```
動詞辭書形 + 前に
名詞 + の + 前に
```

**範例**
```
行く → 行く前に
食べる → 食べる前に
会議 → 会議の前に
授業 → 授業の前に
```

## 日文解釋

「〜前に」は、ある動作や出来事が起こる前の時点を表す時間表現[^jikan-hyougen]です。主節[^shusetsu]の動作が従属節[^juuzokusetsu]の動作よりも先に行われることを示します。

動詞[^doushi]の辞書形[^jishokei]に接続し、「その動作の前に」という意味を表します。名詞[^meishi]の場合は「の」を挟んで「名詞の前に」となります。日常会話[^casual]でも正式な場面[^formal]でも広く使われる基本的な文法です。

時間的順序[^jikanteki-junjo]を明確に示す表現で、計画や準備、予定などを説明する際に頻繁に用いられます。「〜後で」[^ato-de]と対をなす表現として、日本語学習の初期段階で習得する重要な文法項目です。

## 英文解釋

"〜前に" (mae ni) is a temporal expression[^jikan-hyougen] meaning "before" or "prior to." It indicates that the action in the main clause[^shusetsu] occurs before the action or event in the subordinate clause[^juuzokusetsu].

It connects to the dictionary form[^jishokei] of verbs[^doushi], meaning "before doing (something)." For nouns[^meishi], it takes the form "noun + の + 前に." This pattern is widely used in both casual[^casual] and formal[^formal] contexts as a fundamental grammatical structure.

This expression clearly indicates temporal sequence[^jikanteki-junjo] and is frequently used when explaining plans, preparations, or schedules. As a counterpart to "〜後で"[^ato-de] (after), it is an essential grammar point learned in the early stages of Japanese language study.

## 中文解釋

「〜前に」是表示「在...之前」的時間表現[^jikan-hyougen]。它表示主句[^shusetsu]的動作發生在從屬子句[^juuzokusetsu]的動作或事件之前。

連接動詞[^doushi]的辭書形[^jishokei]，表示「在做某事之前」的意思。如果是名詞[^meishi]，則使用「名詞 + の + 前に」的形式。這個句型在日常會話[^casual]和正式場合[^formal]都廣泛使用，是基本的文法結構。

這個表達方式能清楚地表示時間順序[^jikanteki-junjo]，在說明計畫、準備或時間安排時經常使用。作為「〜後で」[^ato-de]（之後）的對應表達，是日語學習初期階段需要掌握的重要文法項目。

## 核心用法

### 用法 1：動詞辭書形 + 前に

表示在做某個動作之前。

**例句 1**
```
寝る前に歯を磨きます。
I brush my teeth before sleeping.
睡覺前刷牙。
```

**例句 2**
```
日本へ行く前に、日本語を勉強しました。
I studied Japanese before going to Japan.
去日本之前，我學習了日語。
```

### 用法 2：名詞 + の + 前に

表示在某個時間點或事件之前。

**例句 3**
```
会議の前に資料を準備してください。
Please prepare the materials before the meeting.
請在會議前準備資料。
```

**例句 4**
```
試験の前に、しっかり復習しましょう。
Let's review thoroughly before the exam.
考試前，讓我們好好複習吧。
```

### 用法 3：時間點 + 前に

表示在某個具體時間之前。

**例句 5**
```
5時の前に家に帰ります。
I'll go home before 5 o'clock.
我會在五點前回家。
```

## 文法規則

### 規則 1：動詞形態

動詞接續時，必須使用辭書形（基本形），不能使用ます形或其他活用形。

```
✅ 正：食べる前に手を洗う
❌ 誤：食べます前に手を洗う
❌ 誤：食べて前に手を洗う
```

### 規則 2：時態表現

「前に」本身不表示時態，句子的時態由主句的動詞決定。

```
行く前に電話します。（未來）
行く前に電話しました。（過去）
```

### 規則 3：否定形的特殊用法

表示「在...之前就...」時，使用「ない形 + うちに」，而非「ない形 + 前に」。

```
✅ 推薦：忘れないうちにメモします。（趁還沒忘記就記下來）
△ 可用：忘れる前にメモします。（在忘記之前記下來）
```

### 規則 4：與「〜後で」的對應

「前に」和「後で」是一對相反的時間表達。

```
食べる前に → 在吃之前
食べた後で → 在吃之後
```

## 常見錯誤

### 錯誤 1：使用ます形連接

❌ 誤：行きます前に準備します。
✅ 正：行く前に準備します。
說明：「前に」前面必須接辭書形，不能接ます形。這是初學者最常犯的錯誤。

### 錯誤 2：名詞直接連接

❌ 誤：会議前に資料を準備する。
✅ 正：会議の前に資料を準備する。
說明：名詞和「前に」之間必須加「の」。但在某些固定表達中可以省略，如「食事前」。

### 錯誤 3：與「〜てから」混淆

❌ 誤：宿題をしてから、遊びに行く前に。（邏輯錯誤）
✅ 正：遊びに行く前に、宿題をします。
✅ 正：宿題をしてから、遊びに行きます。
說明：「前に」表示之前，「てから」表示之後。不要在同一個句子中混用造成邏輯混亂。

## 學習要點

1. **基本接續**：動詞辭書形 + 前に，名詞 + の + 前に
2. **時態由主句決定**：「前に」本身不表時態，看主句動詞的形態
3. **廣泛使用**：從日常對話到正式書面都適用的基本文法
4. **與後で對應**：前に（之前）↔ 後で（之後）是一對相反的時間表達
5. **表達順序**：清楚表達動作的先後順序，常用於說明準備、計畫等

## 註解

[^jikan-hyougen]: **時間表現** - 表示時間點、時間段、時間順序等的語法結構。日語有豐富的時間表達方式，是基本語法的重要組成部分。詳見 [jikan_hyougen](../concept/jikan_hyougen.md)

[^shusetsu]: **主節（主句）** - 複句中語法上獨立、能單獨成句的部分。在「AしてからB」這樣的結構中，B是主節。詳見 [shusetsu](../concept/shusetsu.md)

[^juuzokusetsu]: **從屬節（從屬子句）** - 複句中依附於主節、提供補充說明的部分。在「AしてからB」中，A是從屬節。詳見 [juuzoku_setsu](../concept/juuzoku_setsu.md)

[^doushi]: **動詞** - 表示動作、存在、狀態變化的詞類。日語動詞有三大類：一段動詞、五段動詞、不規則動詞。詳見 [verb_classification](../concept/verb_classification.md)

[^jishokei]: **辭書形（基本形）** - 動詞的基本形式，也是在字典中查詢的形式。如「食べる」「行く」「来る」等。詳見 [verb_forms](verb_forms.md)

[^meishi]: **名詞** - 表示人、事物、場所、概念等的詞類。可以做主語、賓語，需要助詞連接其他成分。詳見 [meishi](../concept/meishi.md)

[^casual]: **日常會話場景** - 朋友、家人之間的非正式對話，或輕鬆的日常交流情境。詳見 [casual-conversation](../context/casual-conversation.md)

[^formal]: **正式場合** - 商務、學術、公開演講等需要使用正式語言的情境。詳見 [formal-context](../context/formal-context.md)

[^jikanteki-junjo]: **時間順序** - 事件或動作發生的先後關係。日語通過「前に」「後で」「てから」等文法明確表示時間順序。詳見 [jikanteki_junjo](../concept/jikanteki_junjo.md)

[^ato-de]: **〜後で** - 表示「在...之後」的時間表達，與「前に」相對。例：「食べた後で」（吃了之後）。詳見 [ato_de](ato_de.md)
