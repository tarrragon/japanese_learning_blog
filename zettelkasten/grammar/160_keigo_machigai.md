# 敬語常見錯誤

## Japanese
### Definition
日本語学習者が敬語を使う際によく犯す間違いのパターンと、その正しい使い方。適切な敬語使用は、ビジネス場面や目上の人とのコミュニケーションで不可欠である。

### Common Patterns

#### 1. 二重敬語
誤：お召し上がりになられる
正：召し上がる / お召し上がりになる

#### 2. 謙譲語と尊敬語の混同
誤：先生が申されました
正：先生がおっしゃいました

#### 3. 「させていただく」の濫用
誤：資料を読ませていただきます
正：資料を拝読いたします

#### 4. 「お」「ご」の誤用
誤：お電話します
正：お電話いたします / 電話いたします

#### 5. 身内に尊敬語を使う
誤：弊社の山田部長がおっしゃっていました
正：弊社の山田が申しておりました

### Examples
1. 「社長が参られます」→「社長がいらっしゃいます」
2. 「お客様が来られました」→「お客様がお見えになりました」
3. 「先生に伺いました」→「先生にお聞きしました」
4. 「部長、お疲れ様でございます」→「部長、お疲れ様でした」

## English
### Definition
Common mistake patterns that Japanese learners make when using keigo (honorific language), along with correct usage. Proper keigo usage is essential in business settings and communication with superiors.

### Key Error Types

#### 1. Double Honorifics
Layering multiple honorific forms unnecessarily.

#### 2. Confusing Humble and Respectful Forms
Using humble verbs (謙譲語) when respectful verbs (尊敬語) are needed, or vice versa.

#### 3. Overusing させていただく
Excessive use of this humble expression when simpler forms suffice.

#### 4. Incorrect Prefix Usage
Misapplying お or ご to verbs inappropriately.

#### 5. Using Respectful Language for One's Own Group
Applying honorifics to one's company members when speaking to outsiders.

## 中文
### Definition
日語學習者在使用敬語時常犯的錯誤模式及其正確用法。適當使用敬語在商務場合和與上級溝通時至關重要。

### 主要錯誤類型

#### 1. 雙重敬語
不必要地疊加多個敬語形式。

#### 2. 謙讓語與尊敬語混淆
該用尊敬語時用謙讓語，反之亦然。

#### 3. 濫用「させていただく」
過度使用這個謙讓表達，其實簡單形式即可。

#### 4. 前綴誤用
不恰當地在動詞前加上「お」或「ご」。

#### 5. 對自己公司成員使用尊敬語
對外說話時對自己公司成員使用敬語。
