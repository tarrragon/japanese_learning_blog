## 日文

〜ながら

### 文法情報

| 項目 | 內容 |
|------|------|
| 文法類型 | 動詞變化・接續助詞 |
| JLPT | N4 |
| 使用場面 | 日常會話、書面語 |
| 形成 | 動詞ます形語幹 + ながら |

### 形成規則

**基本形式**
```
動詞ます形語幹 + ながら
```

**範例**
```
食べる → 食べます → 食べ + ながら → 食べながら
聞く → 聞きます → 聞き + ながら → 聞きながら
見る → 見ます → 見 + ながら → 見ながら
歩く → 歩きます → 歩き + ながら → 歩きながら
```

## 日文解釋

「〜ながら」は、二つの動作や状態が同時に進行していることを表す接続助詞[^setsuzoku-joshi]です。動詞のます形[^masu-form]の語幹に接続します。

主な用法として、主動作[^main-action]とそれに付随する従動作[^sub-action]を表現します。従動作は「〜ながら」で表し、主動作は後続の文で述べます。例えば、「音楽を聞きながら勉強する」では、「音楽を聞く」が従動作で、「勉強する」が主動作です。

また、相反する二つの事柄を対比的に述べる用法もあります。この場合は「〜ながら（も）」の形で、「知りながら言わない」（知っているのに言わない）のように使われます。

日常会話[^daily-conversation]でも書き言葉[^written-language]でも頻繁に使用される重要な文法パターンです。自然な日本語表現において、同時動作を効率的に伝えるために不可欠な表現です。

## 英文解釋

"〜ながら" is a conjunctive particle[^setsuzoku-joshi] that expresses simultaneous actions or states. It attaches to the stem of the masu-form[^masu-form] of verbs.

The primary usage is to express a main action[^main-action] with an accompanying sub-action[^sub-action]. The sub-action is marked with "〜ながら," while the main action follows in the subsequent clause. For example, in "音楽を聞きながら勉強する" (study while listening to music), "listening to music" is the sub-action and "studying" is the main action.

There is also a contrastive usage where "〜ながら(も)" expresses two contrasting facts. This is used in phrases like "知りながら言わない" (not saying despite knowing), indicating a contradiction between awareness and action.

This pattern is frequently used in both casual conversation[^daily-conversation] and written language[^written-language], making it an essential construction for expressing simultaneous actions naturally in Japanese.

## 中文解釋

「〜ながら」是接續助詞[^setsuzoku-joshi]，表示兩個動作或狀態同時進行。接在動詞ます形[^masu-form]的語幹之後。

主要用法是表達主要動作[^main-action]和伴隨的次要動作[^sub-action]。次要動作用「〜ながら」表示，主要動作在後面的句子中陳述。例如「音楽を聞きながら勉強する」（一邊聽音樂一邊學習），「聽音樂」是次要動作，「學習」是主要動作。

另外還有對比用法，用「〜ながら（も）」形式表達兩個相反的事實。如「知りながら言わない」（明明知道卻不說），表示認知與行為之間的矛盾。

這個句型在日常會話[^daily-conversation]和書面語[^written-language]中都很常用，是表達同時動作的重要文法結構，對自然地表達日語不可或缺。

## 核心用法

### 用法 1：同時進行的兩個動作

表示一邊做某事，一邊做另一件事。次要動作用「〜ながら」，主要動作在後面。

**例句 1**
```
音楽を聞きながら勉強します。
I study while listening to music.
我一邊聽音樂一邊學習。
```

**例句 2**
```
スマホを見ながら歩くのは危ないです。
Walking while looking at your smartphone is dangerous.
一邊看手機一邊走路很危險。
```

### 用法 2：伴隨的持續狀態

表示在某種持續的狀態下進行主要動作。

**例句 3**
```
コーヒーを飲みながら話しましょう。
Let's talk while drinking coffee.
我們一邊喝咖啡一邊聊吧。
```

**例句 4**
```
笑いながら「大丈夫だよ」と言った。
She said "It's okay" while smiling.
她一邊笑一邊說「沒關係啦」。
```

### 用法 3：對比用法（逆接）

用「〜ながら（も）」表示明知...卻...，強調矛盾或對比。

**例句 5**
```
悪いと知りながら、やってしまった。
I did it even though I knew it was wrong.
明知不對，還是做了。
```

## 文法規則

### 規則 1：動詞的接續方式

「〜ながら」只接在動詞ます形的語幹後面，不能接形容詞或名詞。

- ✅ 食べ**ながら**（食べます → 食べ）
- ✅ 見**ながら**（見ます → 見）
- ❌ 静か**ながら**（形容詞不能直接接）
- ❌ 学生**ながら**（名詞不能接）

### 規則 2：主從關係

「〜ながら」表示的動作是次要的、伴隨的動作，主要動作在後面的句子中表達。主語通常相同。

```
［次要動作］ながら ［主要動作］
```

### 規則 3：對比用法的特徴

對比用法時，常用「〜ながら（も）」或「〜ながらに」的形式，表示逆接關係。

```
残念ながら、行けません。
Unfortunately, I can't go.
很遺憾，我不能去。
```

### 規則 4：時態的處理

「〜ながら」本身沒有時態標記，時態由後面的主要動作決定。

- 現在：歩き**ながら**話します
- 過去：歩き**ながら**話しました
- 未來：歩き**ながら**話すでしょう

## 常見錯誤

### 錯誤 1：使用辞書形接續

❌ 誤：食べる**ながら**本を読む
✅ 正：食べ**ながら**本を読む
說明：必須使用ます形的語幹，不能直接接辞書形。「食べる」→「食べます」→「食べ」+ ながら。

### 錯誤 2：主從關係顛倒

❌ 誤：勉強しながら音楽を聞く（想表達「主要是聽音樂」時）
✅ 正：音楽を聞きながら勉強する
說明：「〜ながら」表示的是次要動作。如果想強調「勉強」是主要的，就應該把「音楽を聞く」放在「〜ながら」前面。

### 錯誤 3：主語不一致

❌ 誤：私がテレビを見ながら、弟は勉強した
✅ 正：私がテレビを見ている間に、弟は勉強した
說明：「〜ながら」的兩個動作通常需要同一個主語。主語不同時，應該使用「〜間に」等其他表達方式。

### 錯誤 4：與形容詞混淆

❌ 誤：小さい**ながら**強い
✅ 正：小さい**けれど**強い
說明：形容詞表示逆接時，不能用「ながら」，應該使用「けれど」「が」等接續詞。

## 學習要點

1. **接續規則**：只能接在動詞ます形的語幹後面，是最基本的規則。

2. **主從關係**：「〜ながら」前面的動作是次要的、伴隨的，主要動作在後面。選擇時要考慮哪個是重點。

3. **同一主語**：兩個動作通常由同一個人做。主語不同時要使用其他表達方式（如「〜間に」）。

4. **對比用法**：「〜ながら（も）」可以表示逆接，相當於「雖然...但是...」的意思，用於表達矛盾或遺憾。

5. **自然性**：選擇「〜ながら」連接的兩個動作時，要考慮實際上能否同時進行。不合理的組合會顯得不自然。

## 註解

[^setsuzoku-joshi]: **接續助詞** - 連接兩個句子或詞組的助詞，表示各種邏輯關係（順接、逆接、並列等）。詳見 [setsuzoku_joshi.md](../particle/setsuzoku_joshi.md)

[^masu-form]: **ます形** - 動詞的丁寧形，由動詞語幹加「ます」組成。是日語禮貌表達的基本形式。詳見 [masu_form.md](masu_form.md)

[^main-action]: **主要動作** - 在「〜ながら」結構中，後面句子表達的主要動作或焦點動作。決定整個句子的時態和語氣。詳見 [main_sub_action.md](../concept/main_sub_action.md)

[^sub-action]: **次要動作** - 在「〜ながら」結構中，用「〜ながら」標記的伴隨動作或背景動作。通常是持續性的動作。詳見 [main_sub_action.md](../concept/main_sub_action.md)

[^daily-conversation]: **日常會話** - 朋友、家人之間的非正式對話情境。詳見 [casual_conversation.md](../context/casual_conversation.md)

[^written-language]: **書面語** - 正式的書寫語言，與口語相對。文法完整，用詞較為正式。詳見 [written_language.md](../concept/written_language.md)
