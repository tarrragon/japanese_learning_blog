## 日文

程度副詞（ていどふくし）

**羅馬拼音**：teido-fukushi

**詞性**：副詞

### 文法情報

| 項目 | 內容 |
|------|------|
| 文法類型 | 副詞分類 |
| JLPT | N4 |
| 使用場面 | 書面、口語、正式、非正式 |
| 功能 | 修飾形容詞、動詞、其他副詞 |

### 常見程度副詞

**極高程度**
```
とても - 非常
大変（たいへん）- 非常（較正式）
非常に（ひじょうに）- 極其（書面語）
すごく - 非常（口語）
```

**較高程度**
```
かなり - 相當
結構（けっこう）- 相當
だいぶ - 相當
なかなか - 相當
```

**中等程度**
```
まあまあ - 馬馬虎虎
まずまず - 尚可
そこそこ - 差不多
```

**較低程度**
```
少し（すこし）- 稍微
ちょっと - 稍微（口語）
やや - 略微（書面語）
多少（たしょう）- 稍許
```

**否定搭配**
```
あまり - 不太（與否定呼應）
全然（ぜんぜん）- 完全不（與否定呼應）
ほとんど - 幾乎不（與否定呼應）
全く（まったく）- 完全不（與否定呼應）
```

## 日文解釋

程度副詞（ていどふくし）とは、形容詞や動詞、他の副詞を修飾して、その程度や度合いを表す副詞のことです。日本語において、程度副詞は文の意味を正確に伝えるために重要な役割を果たします。

程度副詞の特徴として、まず位置の柔軟性があります。基本的には修飾する語の直前に置きますが、文脈によって位置が変わることもあります。また、程度副詞は肯定文と否定文で使い分けが必要です。「とても」「すごく」などは主に肯定文で使用し、「あまり」「全然」などは否定文と呼応して使用します。

さらに、程度副詞には正式度の違いがあります。「非常に」「大変」は正式な場面で使用され、「すごく」「ちょっと」は日常会話で使用されます。話し手は場面や相手に応じて、適切な程度副詞を選ぶ必要があります。

## 英文解釋

Degree adverbs (teido-fukushi) are adverbs that modify adjectives, verbs, or other adverbs to express the extent or degree of something. In Japanese, degree adverbs play a crucial role in conveying the precise meaning of sentences.

A key characteristic of degree adverbs is their positional flexibility. They are generally placed immediately before the word they modify, but the position can vary depending on context. Additionally, degree adverbs must be used differently in affirmative and negative sentences. Words like "totemo" and "sugoku" are mainly used in affirmative sentences, while "amari" and "zenzen" are used in correlation with negative forms.

Furthermore, degree adverbs vary in formality levels. "Hijou ni" and "taihen" are used in formal situations, while "sugoku" and "chotto" are used in everyday conversation. Speakers need to choose appropriate degree adverbs according to the situation and the listener.

## 中文解釋

程度副詞是指修飾形容詞、動詞或其他副詞，用來表達程度或度合的副詞。在日語中，程度副詞對於準確傳達句子意義扮演著重要角色。

程度副詞的特徵之一是位置的靈活性。基本上放在被修飾詞語的前面，但根據語境位置可能會有所變化。此外，程度副詞在肯定句和否定句中的使用需要區分。「とても」「すごく」等主要用於肯定句，而「あまり」「全然」等則與否定形式呼應使用。

程度副詞還有正式程度的差異。「非常に」「大変」用於正式場合，「すごく」「ちょっと」用於日常對話。說話者需要根據場合和對象選擇適當的程度副詞。

## 核心用法

### 用法 1：極高程度（肯定句）

表達非常高的程度，常用於強調。

**例句 1**
```
この料理はとてもおいしいです。
This dish is very delicious.
這道菜非常好吃。
```

**例句 2**
```
今日は大変寒いですね。
It's extremely cold today, isn't it?
今天非常冷呢。
```

### 用法 2：較低程度

表達輕微或稍微的程度。

**例句 3**
```
少し疲れました。
I'm a little tired.
稍微有點累。
```

**例句 4**
```
ちょっと難しいかもしれません。
It might be a little difficult.
可能有點難。
```

### 用法 3：否定呼應

與否定形式搭配使用，表達程度不足。

**例句 5**
```
あまり時間がありません。
I don't have much time.
沒太多時間。
```

**例句 6**
```
全然分かりませんでした。
I didn't understand at all.
完全不懂。
```

## 文法規則

### 規則 1：基本位置

程度副詞通常放在被修飾詞語的前面。

**修飾形容詞**
- とても**高い** - 非常高
- 少し**寒い** - 稍微冷

**修飾動詞**
- かなり**疲れた** - 相當累
- あまり**食べない** - 不太吃

### 規則 2：肯定與否定的區分

**肯定句專用**
- とても、すごく、大変、非常に

**否定句呼應**
- あまり〜ない - 不太〜
- 全然〜ない - 完全不〜
- ほとんど〜ない - 幾乎不〜

### 規則 3：正式度區分

**正式/書面**
- 非常に、大変、極めて、やや

**非正式/口語**
- すごく、ちょっと、めっちゃ（關西方言）

## 常見錯誤

### 錯誤 1：肯定否定混用

❌ 誤：あまりおいしいです。
✅ 正：とてもおいしいです。
說明：「あまり」必須與否定形式搭配使用。

### 錯誤 2：正式場合使用口語

❌ 誤：すごく感謝しております。（商務場合）
✅ 正：大変感謝しております。
說明：商務或正式場合應使用「大変」「非常に」等正式詞彙。

### 錯誤 3：雙重程度副詞

❌ 誤：とてもすごく面白い。
✅ 正：とても面白い。/すごく面白い。
說明：通常不疊加使用同類型的程度副詞。

## 學習要點

1. **肯定否定區分**：掌握哪些副詞用於肯定句，哪些用於否定句。

2. **正式度意識**：根據場合選擇適當正式度的程度副詞。

3. **語感差異**：同樣表示「非常」，「とても」「大変」「すごく」的語感和使用場合不同。

4. **位置固定**：程度副詞基本上放在被修飾詞語前面。

5. **口語表達**：日常對話中常用「ちょっと」「すごく」等口語形式。
