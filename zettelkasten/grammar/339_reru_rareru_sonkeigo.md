## 日文

れる・られる（尊敬語助動詞）

### 文法情報

| 項目 | 内容 |
|------|------|
| 文法類型 | 助動詞（尊敬語） |
| JLPT | N3 |
| 使用場面 | 正式場合、商務、書面語 |
| 形成 | 動詞未然形 + れる/られる |

### 形成規則

**五段動詞（Group I）**
```
動詞語尾變成「あ段」+ れる
読む → 読まれる
書く → 書かれる
話す → 話される
行く → 行かれる
```

**一段動詞（Group II）**
```
動詞去掉「る」+ られる
食べる → 食べられる
見る → 見られる
起きる → 起きられる
```

**不規則動詞（Group III）**
```
する → される
来る → 来られる
```

## 日文解釋

「れる・られる」は尊敬を表す助動詞で、動作の主体に対して敬意を示す表現である。[^dict]

尊敬語としての「れる・られる」は、目上の人や敬意を払うべき相手の動作を述べる際に使用される。この形式は「お～になる」形式と比べてやや控えめな敬意を表し、日常的なビジネス会話や一般的な敬語表現として広く使われている。

注意すべき点として、「れる・られる」は受身・可能・自発を表す場合もあるため、文脈によって意味を判断する必要がある。尊敬の意味で使われる場合は、主語が人（特に目上の人）であり、その人が自発的に行う動作であることが多い。

[^dict]: 定義參考 スーパー大辞林（macOS 内建字典）

## 英文解釋

The auxiliary verbs "reru" and "rareru" are used to express respect toward the subject of an action. This is one of the most common ways to form honorific expressions in Japanese.

When attached to verbs, these auxiliaries elevate the status of the person performing the action. This form is considered slightly less formal than the "o-~ni naru" construction but is widely used in business and everyday polite speech.

It's important to note that "reru/rareru" can also indicate passive voice, potential form, or spontaneous occurrence, so context is crucial for determining the intended meaning.

## 中文解釋

「れる・られる」是表達尊敬的助動詞，用於對動作主體表示敬意。這是日文尊敬語的基本形式之一。

當這些助動詞接在動詞後面時，會提升執行動作者的地位。相較於「お〜になる」形式，這種表達稍微不那麼正式，但在商務場合和日常禮貌用語中被廣泛使用。

需要注意的是，「れる・られる」也可以表示被動、可能或自發的意思，因此需要根據上下文判斷其含義。判斷是否為尊敬語時，主要看主語是否為需要尊敬的對象。

## 核心用法

### 用法 1：表達對長輩或上司動作的敬意

這是最基本的尊敬語用法，用於描述需要敬重之人的行為。

**例句 1**
```
社長は毎朝早く出社されます。
The president comes to the office early every morning.
社長每天早上都很早到公司。
```

**例句 2**
```
先生はもう帰られましたか。
Has the teacher already gone home?
老師已經回去了嗎？
```

### 用法 2：詢問對方的動作或狀況

用於禮貌地詢問對方（需要敬重的人）的動作或意向。

**例句 3**
```
何を召し上がられますか。いえ、何を食べられますか。
What would you like to eat?
您想吃什麼？
```

**例句 4**
```
明日のパーティーには参加されますか。
Will you be attending tomorrow's party?
您會參加明天的派對嗎？
```

### 用法 3：書面語或正式公告中的尊敬表達

在書面語或正式場合中使用，表達對讀者或聽眾的敬意。

**例句 5**
```
ご不明な点がございましたら、担当者にお問い合わせされてください。
If you have any questions, please contact the person in charge.
如有任何疑問，請聯繫負責人。
```

## 文法規則

### 規則 1：接續方式依動詞類型而異

- 五段動詞：未然形（語尾變「あ段」）+ れる
- 一段動詞：去掉「る」+ られる
- 不規則動詞：する → される、来る → 来られる

### 規則 2：與「ます」連用形成禮貌尊敬語

尊敬語通常與「ます」形結合使用，形成更禮貌的表達：
- 読まれます、書かれます、話されます
- 食べられます、見られます
- されます、来られます

### 規則 3：區分尊敬語與被動/可能的方法

判斷是否為尊敬語的要點：
1. 主語是否為需要尊敬的對象（上司、長輩、客人等）
2. 動作是否為主語自發進行的
3. 文脈是否暗示敬意

## 常見錯誤

### 錯誤 1：對自己使用尊敬語

❌ 誤：私は明日会社に行かれます。
✅ 正：私は明日会社に参ります。／行きます。
說明：尊敬語只能用於他人，不能用於自己。對自己使用謙讓語或普通語。

### 錯誤 2：與「お〜になる」混淆或重複使用

❌ 誤：先生はお読まれになりました。
✅ 正：先生は読まれました。／先生はお読みになりました。
說明：「れる・られる」和「お〜になる」是兩種不同的尊敬語形式，不能混用。

### 錯誤 3：一段動詞誤用五段動詞的接續

❌ 誤：先生は食べれました。（口語省略形，正式場合不當）
✅ 正：先生は食べられました。
說明：一段動詞的尊敬語必須用「られる」，不可省略「ら」。

## 學習要點

1. **動詞分類是關鍵**：正確判斷動詞類型才能正確接續れる/られる
2. **區分多重意義**：れる・られる有受身、可能、自發、尊敬四種意義，需從文脈判斷
3. **不可用於自己**：尊敬語只用於描述他人的動作，描述自己時用謙讓語
4. **與其他尊敬語比較**：「お〜になる」比「れる・られる」更為鄭重
5. **商務場合常用**：這是職場最基本的敬語形式之一，需熟練掌握
