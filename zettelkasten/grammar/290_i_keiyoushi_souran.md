## 日文解釋

イ形容詞（い形容詞とも表記する）は、日本語の形容詞の一類で、終止形が「い」で終わる形容詞を指す。日本語の形容詞は「イ形容詞」と「ナ形容詞（形容動詞）」の二つに大別される。

イ形容詞の最大の特徴は、そのまま名詞を修飾できることである。「高い山」「新しい本」のように、「な」などの助詞を介さずに直接名詞に接続する。また、文末に置いて述語として使う場合も、「だ」を付ける必要がない（「この山は高い」）。

活用においては、語尾の「い」を規則的に変化させる。否定形は「〜くない」、過去形は「〜かった」、連用形は「〜く」となる。この規則性がイ形容詞の学習を容易にしている。

ただし、「きれい」「嫌い」などは「い」で終わるがナ形容詞であることに注意。これらは名詞修飾時に「きれいな花」のように「な」が必要となる。

## 英文解釋

**I-adjectives** (also written as イ形容詞 or い形容詞) are one of the two main categories of adjectives in Japanese, characterized by their dictionary form ending in the hiragana い (i). They contrast with な-adjectives (na-adjectives), which require the particle な when modifying nouns.

The defining feature of i-adjectives is their ability to directly modify nouns without any intervening particle. For example, 高い山 (takai yama, "tall mountain") and 新しい本 (atarashii hon, "new book") show the adjective immediately preceding the noun.

I-adjectives conjugate by changing their final い:
- Negative: 〜い → 〜くない (高い → 高くない)
- Past: 〜い → 〜かった (高い → 高かった)
- Te-form: 〜い → 〜くて (高い → 高くて)
- Adverbial: 〜い → 〜く (高い → 高く)

**Important exception**: Words like きれい (kirei, "pretty") and 嫌い (kirai, "dislike") end in い but are actually な-adjectives, requiring な when modifying nouns.

## 中文解釋

**イ形容詞**（也寫作い形容詞）是日語形容詞的主要類別之一，其特徵是辭書形以「い」結尾。日語形容詞主要分為「イ形容詞」和「ナ形容詞（形容動詞）」兩大類。

イ形容詞的核心特徵是可以直接修飾名詞，不需要任何連接詞。例如：
- 高い山（高的山）
- 新しい本（新的書）
- 美しい景色（美麗的景色）

作為謂語使用時，也不需要加「だ」：
- この山は高い。（這座山很高。）

活用規則相當規則，通過改變「い」來表達不同語意：
- 否定：高い → 高くない（不高）
- 過去：高い → 高かった（高的〔過去〕）
- て形：高い → 高くて（又高又...）
- 連用形：高い → 高く（高地...）

**重要例外**：「きれい（漂亮）」「嫌い（討厭）」雖然以「い」結尾，但實際上是ナ形容詞，修飾名詞時需要加「な」（如：きれいな花）。

## 核心用法

### 用法 1：直接修飾名詞

イ形容詞可以直接放在名詞前面進行修飾，這是與ナ形容詞最明顯的區別。

**例句 1**
```
大きい犬が公園を走っている。
A big dog is running in the park.
一隻大狗正在公園裡跑。
```

**例句 2**
```
彼女は長い髪が似合う。
Long hair suits her.
長髮很適合她。
```

### 用法 2：作為謂語結句

イ形容詞可直接放在句尾作為謂語，不需要加「だ」（但可加「です」表示禮貌）。

**例句 3**
```
今日の天気は暖かいですね。
The weather today is warm, isn't it?
今天的天氣很暖和呢。
```

### 用法 3：連用形（〜く）修飾動詞

將「い」變為「く」形成連用形，可以修飾動詞，相當於副詞功能。

**例句 4**
```
彼女は歌を上手く歌った。
She sang the song well.
她唱歌唱得很好。
```

### 用法 4：て形（〜くて）連接句子

將「い」變為「くて」可以連接多個形容詞或形成原因表達。

**例句 5**
```
この料理は安くておいしい。
This dish is cheap and delicious.
這道菜便宜又好吃。
```

## 常見錯誤

### 錯誤 1：把ナ形容詞當作イ形容詞

❌ 誤：きれい人
✅ 正：きれいな人
說明：「きれい」雖以「い」結尾，但它是ナ形容詞，修飾名詞必須加「な」。

### 錯誤 2：否定形變化錯誤

❌ 誤：高いじゃない / 高いない
✅ 正：高くない
說明：イ形容詞的否定形是把「い」變成「くない」，而不是加「じゃない」或「ない」。

### 錯誤 3：過去形使用ナ形容詞變化

❌ 誤：高いだった
✅ 正：高かった
說明：イ形容詞過去形是「〜かった」，而不是加「だった」。「だった」是ナ形容詞的過去形。

### 錯誤 4：「いい」的特殊活用

❌ 誤：いくない / いかった
✅ 正：よくない / よかった
說明：「いい（好的）」是特殊的イ形容詞，活用時使用「よい」的形式。

## 學習要點

1. **識別方法**：判斷形容詞是イ還是ナ，看修飾名詞時是否需要「な」。需要「な」的是ナ形容詞。

2. **活用公式**：記住「い」的變化模式——「くない」（否定）、「かった」（過去）、「くて」（て形）、「く」（連用形）。

3. **例外詞彙**：牢記「きれい」「嫌い」「有名」等看似イ形容詞但實為ナ形容詞的詞彙。

4. **「いい」的特殊性**：「いい/よい」活用時統一使用「よ-」形式（よくない、よかった、よくて）。

5. **敬體表達**：イ形容詞結句時加「です」表示禮貌（高いです），但這只是禮貌標記，不影響形容詞本身的詞性。
