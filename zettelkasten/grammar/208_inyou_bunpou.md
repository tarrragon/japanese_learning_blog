## 日文

引用文法

**羅馬拼音**：inyou bunpou

### 文法情報

| 項目 | 內容 |
|------|------|
| 文法類型 | 引用表達 |
| JLPT | N4 |
| 使用場面 | 正式、日常 |
| 形成 | 引用內容 + と + 引用動詞 |

### 形成規則

**基本形式**
```
普通形 + と + 言う/思う/聞く/書く etc.
```

**範例**
```
明日雨が降る → 明日雨が降ると言った（說明天會下雨）
彼は来ない → 彼は来ないと思う（我想他不會來）
どこですか → どこですかと聞いた（問了在哪裡）
```

## 日文解釋

「と」引用は、他人の発言や自分の思考、伝聞内容などを間接的に表現する文法構造です。引用する内容（被引用部分）をそのまま「と」で括り、その後に「言う」「思う」「聞く」「書く」などの引用動詞を続けます。

**主な特徴**：
1. **直接引用**：発言や思考の内容をそのまま引用する
2. **普通形使用**：引用部分は普通形（だ・である体）を使う
3. **多様な引用動詞**：言う、思う、聞く、書く、答える、尋ねる、伝える等
4. **疑問文も可能**：疑問詞や「か」を使った疑問文も引用できる

**引用動詞の種類**：
- **発言**：言う、話す、答える、尋ねる、伝える
- **思考**：思う、考える、信じる、感じる
- **伝聞**：聞く、読む、見る、書く
- **命令・依頼**：頼む、命じる、指示する

引用文法は会話文や小説、ニュース報道、学術文章など、あらゆる場面で使われる基本的な文法構造です。

## 英文解釋

The "to" quotation construction is a grammatical structure used to indirectly express others' speech, one's own thoughts, or reported information. The quoted content (quotation) is enclosed with "to" and followed by quotation verbs such as "iu" (say), "omou" (think), "kiku" (hear/ask), "kaku" (write), etc.

**Key characteristics**:
1. **Direct quotation**: Quotes speech or thoughts as they are
2. **Plain form usage**: The quoted part uses plain form (da/dearu style)
3. **Diverse quotation verbs**: iu, omou, kiku, kaku, kotaeru, tazuneru, tsutaeru, etc.
4. **Questions possible**: Can quote questions using question words or "ka"

**Types of quotation verbs**:
- **Speech**: iu (say), hanasu (speak), kotaeru (answer), tazuneru (ask), tsutaeru (convey)
- **Thought**: omou (think), kangaeru (consider), shinjiru (believe), kanjiru (feel)
- **Report**: kiku (hear), yomu (read), miru (see), kaku (write)
- **Command/Request**: tanomu (request), meijiru (order), shiji suru (instruct)

Quotation grammar is a fundamental structure used in all contexts including conversations, novels, news reports, and academic writing.

## 中文解釋

「と」引用是日文中用來間接表達他人發言、自己思考或傳聞內容的文法結構。將要引用的內容（被引用部分）原封不動地用「と」括起來，然後接續「言う」「思う」「聞く」「書く」等引用動詞。

**主要特徵**：
1. **直接引用**：原封不動地引用發言或思考的內容
2. **使用普通形**：引用部分使用普通形（だ・である體）
3. **多樣的引用動詞**：言う、思う、聞く、書く、答える、尋ねる、伝える等
4. **可引用疑問句**：可以引用使用疑問詞或「か」的疑問句

**引用動詞的種類**：
- **發言**：言う、話す、答える、尋ねる、伝える
- **思考**：思う、考える、信じる、感じる
- **傳聞**：聞く、読む、見る、書く
- **命令・請求**：頼む、命じる、指示する

引用文法是對話、小說、新聞報導、學術文章等所有場合都會使用的基本文法結構。

## 核心用法

### 用法 1：引用發言（と言う）

轉述他人的話語或自己說過的話。

**例句 1**
```
彼は「明日来る」と言いました。
He said, "I'll come tomorrow."
他說「明天會來」。
```

**例句 2**
```
先生は授業が難しいと言っていた。
The teacher said the class was difficult.
老師說課程很難。
```

### 用法 2：表達思考（と思う）

表達自己或他人的想法、意見、判斷。

**例句 3**
```
私はこの映画が面白いと思います。
I think this movie is interesting.
我覺得這部電影很有趣。
```

**例句 4**
```
彼女は試験に合格できると思っている。
She thinks she can pass the exam.
她覺得自己能通過考試。
```

### 用法 3：引用疑問（と聞く／尋ねる）

轉述問題或詢問的內容。

**例句 5**
```
駅はどこですかと聞きました。
I asked, "Where is the station?"
我問了「車站在哪裡？」
```

**例句 6**
```
彼は何時に来るのかと尋ねた。
He asked what time (someone) would come.
他問了幾點會來。
```

### 用法 4：引用書面內容（と書く）

轉述書寫或記載的內容。

**例句 7**
```
手紙には「元気です」と書いてあった。
The letter said, "I'm doing well."
信上寫著「我很好」。
```

## 文法規則

### 規則 1：引用部分使用普通形

引用的內容必須使用普通形（常体），不使用敬體（ます形）。

```
◯ 彼は忙しいと言った。（正確）
   He said he was busy.

✗ 彼は忙しいですと言った。（不自然）
```

### 規則 2：名詞・な形容詞的だ

名詞和な形容詞在現在肯定形時，「だ」可以省略或保留。

```
◯ 彼は学生だと言った。（保留だ）
◯ 彼は学生と言った。（省略だ）
   He said he was a student.
```

但過去形和否定形不能省略：
```
◯ 彼は学生だったと言った。（過去）
◯ 彼は学生ではないと言った。（否定）
```

### 規則 3：疑問文的引用

疑問文可以保留疑問詞，句末加「か」或直接接「と」。

```
◯ どこに行くのかと聞いた。（加か）
◯ どこに行くと聞いた。（不加か，較口語）
   (Someone) asked where (I) was going.
```

Yes/No疑問句通常保留「か」：
```
◯ 明日来るかと聞いた。
   (Someone) asked if (I) would come tomorrow.
```

### 規則 4：命令形的引用

命令形可以直接引用，但較常轉換為「～なさい」或「～てください」形式。

```
◯ 早く来いと言った。（直接命令，較粗魯）
◯ 早く来なさいと言った。（較禮貌）
◯ 早く来てくださいと言った。（最禮貌）
   (Someone) told (me) to come quickly.
```

## 常見錯誤

### 錯誤 1：引用部分使用敬體

❌ 誤：彼は忙しいですと言いました。
✅ 正：彼は忙しいと言いました。
說明：引用部分應使用普通形，不用ます形。

```
彼は忙しいと言いました。
He said he was busy.
他說他很忙。
```

### 錯誤 2：混淆「と」的用法

❌ 誤：雨が降ると思います。（條件形的誤用）
✅ 正：雨が降ると思います。（引用，正確）
說明：「と思う」是引用文法，表達想法。但要注意與「と條件形」的區別。

條件形：
```
雨が降ると、試合は中止です。
When it rains, the game is canceled.
下雨的話，比賽就會取消。
```

引用（思考）：
```
雨が降ると思います。
I think it will rain.
我想會下雨。
```

### 錯誤 3：省略不當

❌ 誤：彼は学生ではないと言った。→ 彼は学生ないと言った。
✅ 正：彼は学生ではないと言った。
說明：否定形的「では」不能省略。

```
彼は学生ではないと言った。
He said he wasn't a student.
他說他不是學生。
```

## 學習要點

1. **普通形原則**：引用部分必須使用普通形（常体），這是最重要的規則
2. **引用動詞多樣**：除了「言う」「思う」，還有「聞く」「書く」「答える」等多種動詞
3. **疑問文引用**：可以引用疑問句，保留疑問詞，「か」可加可不加（口語常省略）
4. **與條件形區別**：「と」既可表示引用，也可表示條件，需根據後續動詞判斷
5. **だ的處理**：名詞和な形容詞的現在肯定形，「だ」可省略，但過去形和否定形不可省略

## 註解

[^quotation-verbs]: **引用動詞** - 接在「と」後面的動詞，主要分為發言動詞（言う、話す）、思考動詞（思う、考える）、傳聞動詞（聞く、読む）等類別。

[^plain-form]: **普通形** - 也稱常体，是動詞、形容詞、名詞的基本形式，與敬體（ます形）相對。詳見相關文法卡片。

[^conditional-to]: **と條件形** - 「と」也可表示條件，與引用的「と」是不同用法。條件形表示「如果...就...」的因果關係。詳見 [と條件形](055_to_jouken.md)
