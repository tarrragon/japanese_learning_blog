て形 + 来る 表示動作朝向說話者方向進行，或表示狀態變化的開始與持續。

<!--more-->

## 日文

〜て形 + 来る（てけい + くる）

### 文法情報

| 項目 | 內容 |
|------|------|
| 文法類型 | 補助動詞（方向性） |
| JLPT | N3 |
| 使用場面 | 日常會話、書面語 |
| 形成 | 動詞て形 + 来る |

### 形成規則

**基本形式**
```
動詞て形 + 来る（くる）
```

**範例**
```
走る → 走って来る
持つ → 持って来る
食べる → 食べて来る
する → して来る
```

## 日文解釋

「〜て来る」は動詞のて形に補助動詞「来る」を接続させた文型で、主に三つの用法がある。

第一に、**物理的な移動方向**を表す。動作をしながら話し手の方へ移動することを示す。「走って来る」「歩いて来る」など、移動の様態と方向を同時に表現できる。

第二に、**時間軸上の変化**を表す。過去から現在に至るまでの継続や変化の過程を示す。「〜てきた」の形で、ある状態や動作が現在まで続いてきたことを表す。

第三に、**変化の開始**を表す。新たな状態や感覚が生じ始めることを示す。「寒くなって来た」「眠くなって来た」のように、話し手が変化を感じ始めた時点を表現する。

「〜ていく」とは対照的に、話し手を基準点として「こちらへ向かう」動きや変化を表すのが特徴である。

## 英文解釋

The grammar pattern "~te kuru" combines the te-form of a verb with the auxiliary verb "kuru" (to come). This construction conveys three primary meanings.

First, it indicates **physical direction of movement** toward the speaker. When someone "hashitte kuru" (comes running), the action of running is performed while moving toward the speaker's location.

Second, it expresses **temporal continuation** from past to present. In the form "~te kita," it shows that an action or state has continued up until now, emphasizing the journey or process leading to the present moment.

Third, it marks the **onset of change**. Expressions like "samuku natte kita" (it has started getting cold) indicate that the speaker has begun to notice or experience a change in state.

This pattern contrasts with "~te iku," which indicates movement or change away from the speaker's reference point.

## 中文解釋

「〜て来る」是將動詞て形與補助動詞「来る」結合的文型，主要有三種用法。

第一，表示**物理移動方向**。表達某人邊做某動作邊朝說話者移動。例如「走って来る」表示「跑過來」，同時傳達移動方式和方向。

第二，表示**時間軸上的延續**。用「〜てきた」的形式，表達某動作或狀態從過去持續到現在。強調過程和累積。

第三，表示**變化的開始**。當說話者開始感受到某種狀態變化時使用。如「寒くなって来た」表示「開始變冷了」，強調變化剛開始被察覺的時間點。

與「〜ていく」相對，此文型以說話者為基準點，表達「朝這邊」的動作或變化方向。

## 核心用法

### 用法 1：物理移動方向（朝說話者移動）

表示動作主體邊進行某動作邊朝說話者方向移動。

**例句 1**
```
子供たちが走って来る。
Children are running toward us.
孩子們跑過來了。
```

**例句 2**
```
彼女は手を振りながら歩いて来た。
She came walking while waving her hand.
她邊揮手邊走過來。
```

### 用法 2：過去到現在的持續（〜てきた）

表示某動作或狀態從過去持續到現在的過程。

**例句 3**
```
この会社は30年間成長して来た。
This company has been growing for 30 years.
這家公司已經成長了三十年。
```

**例句 4**
```
私は長年この問題について考えて来ました。
I have been thinking about this issue for many years.
我多年來一直在思考這個問題。
```

### 用法 3：變化的開始（開始感受到變化）

表示說話者開始察覺或感受到某種狀態變化。

**例句 5**
```
だんだん暖かくなって来た。
It has started getting warmer gradually.
天氣漸漸開始變暖了。
```

## 文法規則

### 規則 1：接續方式

「来る」前面必須接動詞的て形。形容詞變化時需加「なる」變成動詞後再接續。

```
動詞て形 + 来る
形容詞 + なる → なって来る
```

### 規則 2：與「〜ていく」的對比

| 文型 | 方向 | 時間軸 |
|------|------|--------|
| 〜て来る | 朝說話者 | 過去→現在 |
| 〜ていく | 離開說話者 | 現在→未來 |

### 規則 3：敬語形式

正式場合使用「〜て参る」（謙讓語）或「〜ていらっしゃる」（尊敬語）。

```
普通：持って来る
謙讓：持って参る
尊敬：お持ちになっていらっしゃる
```

## 常見錯誤

### 錯誤 1：方向誤用

❌ 誤：私は友達の家に走って来た。（自己離開的動作）
✅ 正：私は友達の家に走って行った。
說明：說話者離開自己位置前往他處時，應使用「〜ていく」而非「〜て来る」。

### 錯誤 2：形容詞直接接續

❌ 誤：暖かくて来た。
✅ 正：暖かくなって来た。
說明：形容詞不能直接接「来る」，必須先變成「〜くなる」的形式。

### 錯誤 3：過度使用於未來時態

❌ 誤：明日から寒くなって来るでしょう。
✅ 正：明日から寒くなっていくでしょう。
說明：表示未來的持續變化時，通常使用「〜ていく」而非「〜て来る」。

## 學習要點

1. **方向基準點**：「〜て来る」以說話者為基準點，表示動作或變化朝向說話者。
2. **時間軸理解**：過去→現在的累積用「〜てきた」，現在→未來的發展用「〜ていく」。
3. **變化的察覺**：用於表示說話者「開始感受到」某種變化的時刻。
4. **敬語轉換**：商務場合需使用「〜て参る」等謙讓表達。
5. **與移動動詞的區別**：「来る」本身就有移動意義，但作為補助動詞時主要表示方向性。
