## 日文解釋

日本語の形容詞には、イ形容詞（形容詞）とナ形容詞（形容動詞）の二種類があり、それぞれ異なる活用パターンを持っています。

イ形容詞は語尾が「い」で終わり、活用する際には「い」を取って各活用語尾を付けます。例えば、「高い」の場合、「高く」（連用形）、「高くて」（て形）、「高かった」（過去形）のように変化します。

一方、ナ形容詞は名詞を修飾する際に「な」が付くことからその名前が付いています。ナ形容詞の活用は名詞に似ており、「だ」「で」「に」などの助動詞や助詞と組み合わせて様々な意味を表現します。

両者の活用を正しく使い分けることは、自然な日本語を話すために非常に重要です。特に、否定形や過去形での変化パターンの違いに注意が必要です。

## 英文解釋

Japanese adjectives are divided into two main categories: i-adjectives (adjectives ending in い) and na-adjectives (adjectival nouns). Each category follows distinct conjugation patterns.

**I-adjectives** conjugate by dropping the final い and adding various endings:
- Negative: い → くない (takai → takakunai)
- Past: い → かった (takai → takakatta)
- Te-form: い → くて (takai → takakute)
- Adverbial: い → く (takai → takaku)

**Na-adjectives** conjugate more like nouns, using auxiliary words:
- Attributive: 語幹 + な (shizuka na heya = quiet room)
- Adverbial: 語幹 + に (shizuka ni naru = become quiet)
- Negative: 語幹 + ではない (shizuka de wa nai)
- Past: 語幹 + だった (shizuka datta)

Understanding these conjugation patterns is essential for forming grammatically correct sentences and expressing various tenses, moods, and connections in Japanese.

## 中文解釋

日語形容詞分為兩大類：イ形容詞（以「い」結尾的形容詞）和ナ形容詞（形容動詞）。每種類型都有其獨特的活用規則。

**イ形容詞的活用**是透過去掉詞尾的「い」，然後加上不同的活用語尾來完成：
- 否定形：い → くない（高い → 高くない）
- 過去形：い → かった（高い → 高かった）
- て形：い → くて（高い → 高くて）
- 連用形：い → く（高い → 高く）

**ナ形容詞的活用**則類似名詞，需要搭配助動詞或助詞：
- 連體形：語幹 + な（靜かな部屋）
- 連用形：語幹 + に（靜かになる）
- 否定形：語幹 + ではない（靜かではない）
- 過去形：語幹 + だった（靜かだった）

掌握這兩種形容詞的活用模式，是正確表達日語時態、語氣和句子連接的基礎。需要特別注意的是，某些以「い」結尾的詞彙實際上是ナ形容詞（如「きれい」「嫌い」），不能按照イ形容詞的規則活用。

## 核心用法

### 用法 1：述語用法（作為謂語）

形容詞直接放在句末，用來描述主語的性質或狀態。

**例句 1**
```
この料理はおいしい。
This dish is delicious.
這道菜很好吃。
```

**例句 2**
```
彼女は親切だ。
She is kind.
她很親切。
```

### 用法 2：連體修飾（修飾名詞）

形容詞放在名詞前面，用來描述該名詞的特徵。

**例句 3**
```
新しいパソコンを買いました。
I bought a new computer.
我買了一台新電腦。
```

**例句 4**
```
静かな場所で勉強したい。
I want to study in a quiet place.
我想在安靜的地方讀書。
```

### 用法 3：連用形（修飾動詞或連接句子）

形容詞變成連用形後，可以修飾動詞或連接句子。

**例句 5**
```
彼は速く走る。そして、この部屋は広くて、明るい。
He runs fast. And this room is spacious and bright.
他跑得很快。而且這個房間又寬敞又明亮。
```

## 常見錯誤

### 錯誤 1：將ナ形容詞誤用為イ形容詞

❌ 誤：きれいくない
✅ 正：きれいではない／きれいじゃない
說明：「きれい」雖然以「い」結尾，但它是ナ形容詞，不能按照イ形容詞的規則變化。

### 錯誤 2：イ形容詞過去形的錯誤變化

❌ 誤：高いだった
✅ 正：高かった
說明：イ形容詞的過去形是將「い」變成「かった」，不需要加「だった」。

### 錯誤 3：混淆連體形和連用形

❌ 誤：静かく話す
✅ 正：静かに話す
說明：ナ形容詞修飾動詞時要用「に」，而不是「く」。「く」是イ形容詞的連用形語尾。

## 學習要點

1. **區分イ形容詞和ナ形容詞**：雖然大多數以「い」結尾的詞是イ形容詞，但「きれい」「嫌い」「有名」等是ナ形容詞，需要特別記憶。

2. **掌握否定形的變化**：イ形容詞的否定形是「〜くない」，ナ形容詞的否定形是「〜ではない」或口語的「〜じゃない」。

3. **理解過去形的形成**：イ形容詞用「〜かった」，ナ形容詞用「〜だった」。

4. **活用連用形**：イ形容詞的連用形「〜く」用於修飾動詞或接續助動詞；ナ形容詞的連用形「〜に」用於修飾動詞。

5. **注意て形的連接功能**：形容詞的て形（イ形容詞「〜くて」、ナ形容詞「〜で」）用於連接多個形容詞或形容詞與其他成分。
