## 日文

動詞活用表（どうしかつようひょう）

**羅馬拼音**: doushi katsuyou hyou

**類型**：文法參考・動詞活用

## 日文解釋

動詞活用表は、日本語動詞の全ての活用形を一覧にまとめた文法資料です。日本語には三種類の動詞があり、それぞれ異なる活用パターンを持っています。

### 動詞の三分類

日本語の動詞は以下の三つに分類されます：

1. **五段動詞（Group I / u-verbs）**：語尾が「う・く・ぐ・す・つ・ぬ・ぶ・む・る」で終わる動詞
2. **一段動詞（Group II / ru-verbs）**：語尾が「いる・える」で終わる動詞
3. **不規則動詞（Group III）**：「する」と「来る」の二つのみ

### 活用形の種類

主要な活用形は以下の通りです：

| 活用形 | 日本語名 | 用途 |
|--------|----------|------|
| 辞書形 | じしょけい | 基本形・文末・連体修飾 |
| ます形 | ますけい | 丁寧な表現 |
| て形 | てけい | 接続・依頼・進行 |
| た形 | たけい | 過去・完了 |
| ない形 | ないけい | 否定 |
| 可能形 | かのうけい | 能力・可能性 |
| 受身形 | うけみけい | 受動態 |
| 使役形 | しえきけい | 使役表現 |
| 意志形 | いしけい | 意志・勧誘 |
| 命令形 | めいれいけい | 命令 |
| 仮定形 | かていけい | 条件 |

活用表を理解することで、どんな動詞でも正しく変化させることができるようになります。特に五段動詞の活用は複雑なため、パターンを覚えることが重要です。

## 英文解釋

A verb conjugation chart is a comprehensive reference that organizes all conjugation patterns for Japanese verbs. Japanese has three verb categories, each with distinct conjugation rules.

### The Three Verb Categories

1. **Godan Verbs (Group I / u-verbs)**: Verbs ending in u, ku, gu, su, tsu, nu, bu, mu, or ru
2. **Ichidan Verbs (Group II / ru-verbs)**: Verbs ending in -iru or -eru
3. **Irregular Verbs (Group III)**: Only "suru" (to do) and "kuru" (to come)

### Main Conjugation Forms

| Form | Japanese Name | Usage |
|------|---------------|-------|
| Dictionary | jishokei | Base form, sentence-final, noun modification |
| Masu | masu-kei | Polite expressions |
| Te | te-kei | Connection, requests, progressive |
| Ta | ta-kei | Past, completion |
| Nai | nai-kei | Negative |
| Potential | kanou-kei | Ability, possibility |
| Passive | ukemi-kei | Passive voice |
| Causative | shieki-kei | Causative expressions |
| Volitional | ishi-kei | Intention, invitation |
| Imperative | meirei-kei | Commands |
| Conditional | katei-kei | Conditions |

Understanding conjugation charts enables learners to correctly transform any verb. Godan verbs are particularly complex, so memorizing their patterns is essential.

## 核心用法

### 一覽表：五段動詞活用

以「書く（かく）」為例：

| 活用形 | 變化 | 例 |
|--------|------|-----|
| 辞書形 | - | 書く |
| ます形 | く→き+ます | 書きます |
| て形 | く→いて | 書いて |
| た形 | く→いた | 書いた |
| ない形 | く→か+ない | 書かない |
| 可能形 | く→ける | 書ける |
| 受身形 | く→かれる | 書かれる |
| 使役形 | く→かせる | 書かせる |
| 意志形 | く→こう | 書こう |
| 命令形 | く→け | 書け |
| 仮定形 | く→けば | 書けば |

**例句 1**
```
毎日日記を書く習慣があります。
I have a habit of writing in my diary every day.
我有每天寫日記的習慣。
```

### 一覽表：一段動詞活用

以「食べる（たべる）」為例：

| 活用形 | 變化 | 例 |
|--------|------|-----|
| 辞書形 | - | 食べる |
| ます形 | る→ます | 食べます |
| て形 | る→て | 食べて |
| た形 | る→た | 食べた |
| ない形 | る→ない | 食べない |
| 可能形 | る→られる | 食べられる |
| 受身形 | る→られる | 食べられる |
| 使役形 | る→させる | 食べさせる |
| 意志形 | る→よう | 食べよう |
| 命令形 | る→ろ | 食べろ |
| 仮定形 | る→れば | 食べれば |

**例句 2**
```
朝ごはんを食べてから学校に行きます。
After eating breakfast, I go to school.
吃完早餐後去學校。
```

### て形變化規則（五段動詞）

五段動詞的て形變化較複雜，需要記住以下規則：

| 辭書形詞尾 | て形變化 | 例 |
|-----------|----------|-----|
| う・つ・る | → って | 買う→買って、待つ→待って、帰る→帰って |
| む・ぶ・ぬ | → んで | 読む→読んで、遊ぶ→遊んで、死ぬ→死んで |
| く | → いて | 書く→書いて、聞く→聞いて |
| ぐ | → いで | 泳ぐ→泳いで、急ぐ→急いで |
| す | → して | 話す→話して、消す→消して |
| 行く（例外）| → 行って | 行く→行って（特殊變化）|

**例句 5**
```
駅まで歩いて行きました。
I walked to the station.
走路去了車站。
```

## 常見錯誤

### 錯誤 1：一段動詞與五段動詞混淆

❌ 誤：帰る → 帰て（誤以為是一段動詞）
✅ 正：帰る → 帰って（五段動詞，因為「帰」的讀音是「かえ」，不是「かえい」或「かええ」）
說明：「帰る」雖然以「る」結尾，但「る」前面是「え段」的「え」，不是「い段」或「え段」的假名加上「る」。判斷方法是看「る」前面的假名在哪一段。

### 錯誤 2：「行く」的て形

❌ 誤：行く → 行いて
✅ 正：行く → 行って
說明：「行く」是唯一的例外，不遵循「く→いて」的規則，而是變成「行って」。這需要特別記住。

### 錯誤 3：可能形的混淆

❌ 誤：食べる → 食べれる（ら抜き言葉）
✅ 正：食べる → 食べられる
說明：正式場合應使用「食べられる」。「食べれる」雖然在口語中常見，但在書面語和正式場合應避免使用。

## 學習要點

1. **先掌握動詞分類**：判斷動詞屬於五段、一段還是不規則動詞是活用的第一步。

2. **記住五段動詞的て形規則**：六種變化模式需要反覆練習，特別是「行く」的例外變化。

3. **一段動詞活用簡單**：只需去掉「る」再加上相應的詞尾，規則統一。

4. **不規則動詞數量有限**：只有「する」和「来る」兩個，變化模式需要單獨記憶。

5. **透過例句記憶活用**：單純背誦活用表效果有限，配合例句使用更容易記住。
