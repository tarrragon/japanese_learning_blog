## 概念

### 日文解釋

他動詞（たどうし）は、目的語を必要とし、「〜を」という助詞と一緒に使われる動詞です。動作が対象物に直接働きかけることを表します。一方、自動詞（じどうし）は目的語を必要とせず、「〜が」という助詞と一緒に使われ、主語自身の状態変化や動作を表します。

日本語の多くの動詞には他動詞と自動詞の対（ペア）があり、同じ状況を異なる視点から表現できます。

### 英文解釋

Transitive verbs (tadoushi) require a direct object and are used with the particle "を (wo)". They express actions that directly affect an object. Intransitive verbs (jidoushi) do not require a direct object and are used with the particle "が (ga)". They express the subject's own state change or action.

Many Japanese verbs come in transitive-intransitive pairs, allowing the same situation to be expressed from different perspectives.

### 中文解釋

他動詞需要受詞，與助詞「を」一起使用，表示動作直接作用於對象物。自動詞不需要受詞，與助詞「が」一起使用，表示主語自身的狀態變化或動作。

日語中許多動詞都有他動詞與自動詞的成對形式，可以從不同角度表達相同的情況。

## 文法結構

### 他動詞結構

```
[主語]が [對象]を [他動詞]
```

### 自動詞結構

```
[主語]が [自動詞]
```

## 例句

### 基本例句

**他動詞（有意圖的動作）**：
```
母がドアを開けました。
媽媽打開了門。
```

**自動詞（自然發生）**：
```
風でドアが開きました。
門被風吹開了。
```

---

### 進階例句

**強調動作主體**：
```
誰が窓を壊しましたか。
是誰打破了窗戶？
```

**強調結果狀態**：
```
窓が壊れています。
窗戶破了。
```

**避免指責**：
```
コップが割れてしまいました。（自動詞）
杯子破掉了。（比「コップを割ってしまいました」更委婉）
```

---

## 使用場景

1. **責任歸屬**：他動詞強調動作者，自動詞淡化責任
2. **自然現象**：自動詞常用於自然發生的事件
3. **被動語態替代**：自動詞可避免使用被動形
4. **禮貌表達**：自動詞可用於委婉表達錯誤或意外

## 學習要點

### 辨別技巧

1. **助詞判斷**：
   - 他動詞：〜を + 動詞
   - 自動詞：〜が + 動詞

2. **詞尾模式**（常見但非絕對）：
   - 自動詞：-u 結尾（開く、並ぶ、集まる）
   - 他動詞：-eru 結尾（開ける、並べる、集める）

   - 自動詞：-eru 結尾（壊れる、割れる）
   - 他動詞：-u 或 -su 結尾（壊す、割る）

### 常見錯誤

1. ❌ `電気を消えた`
   ✅ `電気が消えた`（自動詞用「が」）

2. ❌ `ドアが開けた`
   ✅ `ドアを開けた`（他動詞用「を」）

3. ❌ `会議は始めます`
   ✅ `会議を始めます`（他動詞需要受詞標記）

## 腳註

[^1]: 助詞「を」用於標記他動詞的直接受詞，是區分他動詞與自動詞的關鍵助詞。
[^2]: 助詞「が」常與自動詞搭配，標記發生變化或動作的主語。
[^3]: 被動語態與自動詞在語意上有相似之處，都強調動作的承受者或結果狀態。
[^4]: 使役形與他動詞都表達使對象產生變化，但使役形更強調迫使或允許的意味。
[^5]: 「開ける」是「開く」的他動詞對應，表示主動打開的動作。
[^6]: 「開く」作為自動詞時表示自然開啟，不需要動作主體。
[^7]: 「始める」是他動詞，表示主動開始某事。
[^8]: 「始まる」是自動詞，表示某事自然開始。
