## 日文
謙遜表現（けんそんひょうげん）

**羅馬拼音**: kenson hyougen

**詞性**：語用概念・社會語言學

## 日文解釋

謙遜表現とは、自分や自分側のものを控えめに表現することで、相手への配慮を示す言葉遣いです。敬語システムにおける謙譲語とは異なり、より広い意味での「控えめな態度」を表す表現全般を指します。

### 主な謙遜表現の種類

**1. 自己卑下表現**

自分の能力や成果を低く評価して表現します。

- つまらないものですが... （贈り物をする時）
- まだまだ未熟ですが... （自分の能力について）
- 私なんかが... （自分を指して）

**2. 否定的控えめ表現**

肯定的な内容を否定形で控えめに表します。

- 悪くないですね（良いの意味）
- できなくはないです（できるの意味）
- お役に立てなくて...（謝罪として）

**3. 曖昧化表現**

断定を避け、曖昧に表現します。

- ...かもしれません
- ...ではないかと思います
- ...のような気がします

**4. 否定疑問形**

肯定的な意見を否定疑問で表現します。

- これでよろしいでしょうか（確認求め）
- お時間よろしいですか（許可求め）

### 謙譲語との違い

| 項目 | 謙遜表現 | 謙譲語 |
|------|---------|-------|
| 範囲 | 態度・姿勢全般 | 動詞の敬語形式 |
| 機能 | 社会的配慮の表示 | 行為者を低める |
| 例 | つまらないものですが | 参る、申す |
| 場面 | 広い場面で使用 | 敬語が必要な場面 |

## 英文解釋

Kenson hyougen (modest expression) refers to speech patterns that show consideration for others by expressing oneself or one's belongings in a humble manner. Unlike kenjougo (humble language) in the keigo system, this concept encompasses a broader range of expressions that convey a "modest attitude."

### Types of Modest Expressions

**1. Self-deprecating Expressions**

Expressing one's abilities or achievements in a lower manner.

- Tsumaranai mono desu ga... (When giving a gift: "It's just a small thing...")
- Mada mada mijuku desu ga... (About one's ability: "I'm still inexperienced...")
- Watashi nanka ga... (Referring to oneself: "Someone like me...")

**2. Negative Understatement**

Expressing positive content in negative forms.

- Waruku nai desu ne (Meaning "good")
- Dekinaku wa nai desu (Meaning "I can do it")
- Oyaku ni tatenakute... (As an apology)

**3. Ambiguity Expressions**

Avoiding definitive statements and expressing vaguely.

- ...kamoshiremasen (perhaps)
- ...dewa nai ka to omoimasu (I think it might be...)
- ...no you na ki ga shimasu (I feel like...)

**4. Negative Interrogative Form**

Expressing positive opinions through negative questions.

- Kore de yoroshii deshou ka (Is this acceptable?)
- Ojikan yoroshii desu ka (Do you have time?)

### Difference from Kenjougo

| Aspect | Modest Expression | Kenjougo |
|--------|------------------|----------|
| Scope | General attitude | Verb honorific forms |
| Function | Showing social consideration | Lowering the actor |
| Example | tsumaranai mono desu ga | mairu, mousu |
| Usage | Wide range of situations | Situations requiring keigo |

## 中文解釋

謙遜表現是通過謙虛地表達自己或自己方面的事物來展示對他人關懷的說話方式。與[敬語系統](008_keigo_system.md)中的謙讓語不同，這個概念涵蓋了更廣泛的表達「謙虛態度」的說法。

### 謙遜表現的主要類型

**1. 自我貶抑表現**

將自己的能力或成就以較低的方式表達。

- つまらないものですが...（送禮物時：「這是不值一提的東西...」）
- まだまだ未熟ですが...（關於自己的能力：「我還很不成熟...」）
- 私なんかが...（指自己：「像我這樣的人...」）

**2. 否定式低調表現**

用否定形式表達肯定的內容。

- 悪くないですね（意思是「好」）
- できなくはないです（意思是「我能做到」）
- お役に立てなくて...（作為道歉）

**3. 模糊化表現**

避免斷言，以模糊的方式表達。

- ...かもしれません（也許）
- ...ではないかと思います（我認為可能是...）
- ...のような気がします（我感覺好像...）

**4. 否定疑問形式**

用否定疑問來表達肯定的意見。

- これでよろしいでしょうか（這樣可以嗎？）
- お時間よろしいですか（您有時間嗎？）

### 與謙讓語的區別

| 項目 | 謙遜表現 | 謙讓語 |
|------|---------|-------|
| 範圍 | 一般態度與姿態 | 動詞敬語形式 |
| 功能 | 表示社會性的關懷 | 降低行為者 |
| 例子 | つまらないものですが | 参る、申す |
| 場合 | 廣泛場合使用 | 需要敬語的場合 |

## 例句

**例句1：送禮物時的謙遜**
```
つまらないものですが、よろしければどうぞ。
This is just a small thing, but please accept it if you'd like.
這是不值一提的東西，如果您不嫌棄的話請收下。
```

**例句2：對自己能力的謙虛**
```
まだまだ勉強中ですので、至らない点もあるかと思います。
I'm still learning, so there may be some shortcomings.
我還在學習中，可能有不足之處。
```

**例句3：避免直接肯定**
```
この案でよろしいのではないでしょうか。
I wonder if this plan might be acceptable.
這個方案應該可以吧。
```

**例句4：請求許可的謙遜表達**
```
お忙しいところ恐れ入りますが、少々お時間よろしいでしょうか。
I'm sorry to bother you when you're busy, but would you have a moment?
打擾您百忙之中非常抱歉，能否佔用您一點時間？
```

## 補充說明

### 文化背景

日本社會重視「和」的概念，謙遜表現是維持社會和諧的重要語用策略。過度自誇被視為不禮貌，而適度的謙遜則被認為是成熟的社會表現。

### 使用場景

| 場景 | 常用表現 | 說明 |
|------|---------|------|
| 送禮 | つまらないものですが | 降低禮物價值 |
| 自我介紹 | 未熟者ですが | 表現謙虛 |
| 請求幫助 | 恐れ入りますが | 表示不好意思 |
| 表達意見 | ...と思うのですが | 避免斷言 |
| 接受讚美 | いえいえ、まだまだです | 否定讚美 |

### 與西方文化的差異

| 項目 | 日本文化 | 西方文化 |
|------|---------|---------|
| 接受讚美 | 否定或轉移 | 直接感謝 |
| 送禮表達 | 貶低禮物 | 強調精心挑選 |
| 自我評價 | 謙虛低調 | 適度自信 |
| 意見表達 | 間接委婉 | 直接明確 |

### 學習建議

1. **N4-N3階段**：學習基本的謙遜表現詞彙和句型
2. **N3-N2階段**：理解不同場景下的適當謙遜程度

### 注意事項

- **避免過度謙遜**：過度謙遜可能讓對方感到不自在或不真誠
- **場合判斷**：朋友之間過於謙遜反而顯得疏遠
- **文化差異**：在國際場合可能需要調整謙遜的程度
