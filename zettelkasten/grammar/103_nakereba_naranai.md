## 日文

〜なければならない

### 文法情報

| 項目 | 內容 |
|------|------|
| 文法類型 | 義務表達 |
| JLPT | N4 |
| 使用場面 | 正式、書面、口語 |
| 形成 | 動詞ない形 + ければならない |

### 形成規則

**基本形式**
```
動詞ない形（去掉「ない」）+ なければならない
```

**範例**
```
行く → 行かない → 行かなければならない
食べる → 食べない → 食べなければならない
する → しない → しなければならない
来る → 来ない → 来なければならない
```

**口語縮約形**
```
〜なければならない → 〜なきゃならない → 〜なきゃ
〜なくてはならない → 〜なくちゃならない → 〜なくちゃ
```

## 日文解釋

「〜なければならない」は、何かをする必要性[^hitsuyousei]や義務[^gimu]を表す文法表現[^bunpou-hyougen]です。条件形[^joukenkeL]「〜なければ」と否定[^hitei]「ならない」を組み合わせた形で、「〜しないと、いけない状態になる」という意味を持ちます。

この表現は、社会的規則[^shakai-kisoku]、法律[^houritsu]、約束[^yakusoku]、または個人的な責任[^sekinin]など、様々な文脈[^bunmyaku]で使われます。話者[^wasya]が何かをしなければならないという強い必要性を示し、選択の余地[^sentaku-yochi]がないことを暗示します。

正式な場面[^seishiki-bamen]では「〜なければなりません」の丁寧形[^teineikei]を使い、日常会話[^nichijou-kaiwa]では「〜なきゃならない」や「〜なくちゃならない」といった縮約形[^shukuyakukei]がよく使われます。

文法的には、動詞のない形[^nai-kei]から「ない」を取り除き、「なければならない」を接続します。この構造は、仮定条件[^katei-jouken]「もし〜しなければ」と、その結果「ならない（いけない状態になる）」を組み合わせた論理構造[^ronri-kouzou]を持っています。

## 英文解釋

"〜なければならない" (nakereba naranai) is a grammatical expression[^bunpou-hyougen] that indicates necessity[^hitsuyousei] or obligation[^gimu]. It combines the conditional form[^joukenkeL] "〜なければ" (if not) with the negative[^hitei] "ならない" (won't do), literally meaning "if one doesn't do something, it won't do."

This expression is used in various contexts[^bunmyaku], including social rules[^shakai-kisoku], laws[^houritsu], promises[^yakusoku], or personal responsibilities[^sekinin]. It indicates a strong necessity, implying that there is no choice[^sentaku-yochi] or alternative.

In formal situations[^seishiki-bamen], the polite form[^teineikei] "〜なければなりません" is used. In casual conversation[^nichijou-kaiwa], contracted forms[^shukuyakukei] like "〜なきゃならない" or "〜なくちゃならない" are common.

Grammatically, it's formed by removing "ない" from the negative form[^nai-kei] of a verb and adding "なければならない." This structure has a logical construction[^ronri-kouzou] combining a hypothetical condition[^katei-jouken] "if one doesn't do" with the consequence "it won't do (will be problematic)."

## 中文解釋

「〜なければならない」是表示必要性[^hitsuyousei]或義務[^gimu]的文法表達[^bunpou-hyougen]。它結合了條件形[^joukenkeL]「〜なければ」（如果不）和否定[^hitei]「ならない」（不行），字面意思是「如果不做某事，就不行」。

這個表達用於各種情境[^bunmyaku]，包括社會規則[^shakai-kisoku]、法律[^houritsu]、約定[^yakusoku]或個人責任[^sekinin]等。它表示強烈的必要性，暗示沒有選擇[^sentaku-yochi]的餘地。

在正式場合[^seishiki-bamen]使用禮貌形[^teineikei]「〜なければなりません」，在日常會話[^nichijou-kaiwa]中常用縮約形[^shukuyakukei]「〜なきゃならない」或「〜なくちゃならない」。

文法上，將動詞的否定形[^nai-kei]去掉「ない」後加上「なければならない」。這個結構具有邏輯構造[^ronri-kouzou]，結合假設條件[^katei-jouken]「如果不做」和結果「不行（會有問題）」。

## 核心用法

### 用法 1：表示社會規則或法律義務

表達必須遵守的社會規範或法律要求。

**例句 1**
```
学生は毎日学校に行かなければなりません。
Students must go to school every day.
學生必須每天上學。
```

**例句 2**
```
18歳未満の人はお酒を飲んではいけません。20歳になるまで待たなければなりません。
People under 18 must not drink alcohol. They must wait until they turn 20.
未滿 18 歲的人不能喝酒。必須等到 20 歲。
```

### 用法 2：表示個人責任或承諾

說明個人必須履行的責任或承諾。

**例句 3**
```
明日までにレポートを提出しなければならない。
I have to submit the report by tomorrow.
我必須在明天之前提交報告。
```

**例句 4**
```
約束したから、絶対に行かなければならない。
Since I promised, I absolutely must go.
因為我答應了，所以絕對要去。
```

### 用法 3：表示必要條件（口語縮約形）

在日常對話中使用縮約形表達必須做的事。

**例句 5**
```
もう寝なきゃ。明日早いから。
I gotta sleep now. I have to wake up early tomorrow.
得睡了。因為明天要早起。
```

## 文法規則

### 規則 1：形成方式

將動詞ない形去掉「ない」，加上「なければならない」。

| 動詞類型 | 辭書形 | ない形 | 〜なければならない |
|---------|-------|-------|------------------|
| 五段動詞 | 書く | 書かない | 書かなければならない |
| 五段動詞 | 読む | 読まない | 読まなければならない |
| 一段動詞 | 見る | 見ない | 見なければならない |
| 一段動詞 | 食べる | 食べない | 食べなければならない |
| 不規則 | する | しない | しなければならない |
| 不規則 | 来る | 来ない | 来なければならない |

### 規則 2：禮貌程度

根據場合選擇適當的形式。

| 形式 | 禮貌度 | 使用場合 |
|------|-------|---------|
| 〜なければならない | 普通 | 一般說明、書面 |
| 〜なければなりません | 禮貌 | 正式場合、對長輩 |
| 〜なきゃならない | 隨意 | 朋友之間 |
| 〜なきゃ | 非常隨意 | 親密朋友、自言自語 |
| 〜なくちゃならない | 隨意 | 日常會話 |
| 〜なくちゃ | 非常隨意 | 口語 |

### 規則 3：否定形式

表示「不必」使用「〜なくてもいい」，而非「〜なければならなくない」。

```
❌ 行かなければならなくない（錯誤）
✅ 行かなくてもいい（正確）
   You don't have to go.
   不必去。
```

## 常見錯誤

### 錯誤 1：否定形式的誤用

❌ 誤：明日は来なければならなくないです。
✅ 正：明日は来なくてもいいです。
說明：表示「不必」時，不能用「〜なければならない」的否定形，而要用「〜なくてもいい」。這是初學者常犯的錯誤，因為直接否定「必須」在邏輯上會變得複雜。

### 錯誤 2：動詞形式錯誤

❌ 誤：食べるなければならない。
✅ 正：食べなければならない。
說明：必須使用動詞的ない形（去掉「ない」），而不是辭書形。「食べる」→「食べない」→「食べなければならない」。

### 錯誤 3：縮約形的過度使用

❌ 誤：お客様は9時までに来なきゃならないです。（對客人使用過於隨意的表達）
✅ 正：お客様は9時までにお越しになければなりません。
說明：對長輩、客人或正式場合，不應使用縮約形「〜なきゃ」，應使用禮貌形「〜なければなりません」或更恭敬的表達。

### 錯誤 4：混淆相似文法

❌ 誤：宿題をしなければならなくてはいけない。（混用兩種表達）
✅ 正：宿題をしなければならない。/ 宿題をしなくてはいけない。
說明：「〜なければならない」和「〜なくてはいけない」意思相同，不應混在一起使用。選擇其中一種即可。

## 學習要點

1. **基本意義**：表示義務、必須性，暗示沒有選擇的餘地。

2. **形成方式**：動詞ない形（去掉「ない」）+ なければならない。記住這個公式是掌握這個文法的關鍵。

3. **禮貌程度變化**：從正式的「〜なければなりません」到隨意的「〜なきゃ」，要根據場合選擇適當形式。

4. **否定表達**：表示「不必」時用「〜なくてもいい」，不是「〜なければならない」的否定形。

5. **同義表達**：「〜なくてはならない」、「〜なくてはいけない」、「〜ないといけない」意思相同，可互換使用，但語感略有不同。

[^bunpou-hyougen]: **文法表現** - 由多個文法要素組合而成的固定句型，用於表達特定的意義或功能。詳見 [grammar_pattern](../concept/grammar_pattern.md)（待建立）

[^joukenkeL]: **條件形** - 日語動詞、形容詞的假定條件形式，表示「如果〜的話」。動詞ば形（如：行けば）和假定形（如：行ったら）都屬於條件形。詳見 [conditional_forms](conditional_forms.md)（待建立）

[^hitei]: **否定** - 表示否定、不存在或不做某事的文法形式。日語的否定形式豐富，包括動詞ない形、形容詞的否定等。詳見 [negation](../concept/negation.md)（待建立）

[^shakai-kisoku]: **社會規則** - 社會成員普遍遵守的行為規範，包括法律、習俗、禮儀等。詳見 [shakai_kisoku](../concept/shakai_kisoku.md)（待建立）

[^houritsu]: **法律** - 國家制定的強制性規範，違反將受到處罰。日語中常用「法律」、「法令」、「規則」等詞。詳見 [houritsu](../noun/houritsu.md)（待建立）

[^yakusoku]: **約束** - 與他人達成的承諾或協議，是日本文化中非常重視的概念。破壞約束被視為失信的行為。詳見 [yakusoku](../noun/yakusoku.md)（待建立）

[^sekinin]: **責任** - 對某事負有義務或承擔後果的狀態。日本社會特別強調個人和集體責任。詳見 [sekinin](../noun/sekinin.md)（待建立）

[^bunmyaku]: **文脈** - 語言使用的情境和背景，包括前後文、說話場合、人際關係等。理解文脈對正確使用日語極為重要。詳見 [bunmyaku](../concept/bunmyaku.md)（待建立）

[^wasya]: **話者** - 說話的人、發話者。在語言學中，話者的身份、立場和意圖會影響語言選擇。詳見 [speaker](../concept/speaker.md)（待建立）

[^sentaku-yochi]: **選擇的餘地** - 能夠在多個選項中做出選擇的自由。「〜なければならない」暗示沒有選擇，必須執行。詳見 [choice](../concept/choice.md)（待建立）
