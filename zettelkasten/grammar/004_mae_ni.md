## 日文
〜前に（〜まえに）

**羅馬拼音**: ~mae ni

**詞性**：時間文法結構・時間接續詞

## 日文解釋

「〜前に」は、ある動作や出来事が行われる前の時点を示す文法表現です。「前に」の前に動作や出来事を置くことで、「その動作や出来事が起こる前に」という時間的な順序関係を表します。

### 接続方法

**1. 動詞 + 前に**
- 動詞辞書形 + 前に
- 例：食べる前に（吃之前）、寝る前に（睡之前）

**2. 名詞 + の + 前に**
- 名詞 + の + 前に
- 例：朝ごはんの前に（早餐之前）、会議の前に（會議之前）

**3. 時間名詞 + 前（に）**
- 数量 + 時間名詞 + 前（に）
- 例：3日前（三天前）、1時間前（一小時前）
- 注意：此用法的「に」常省略

### 文法特徴

- 「前に」の前の動詞は必ず辞書形を使う（た形ではない）
- 主文と従属文の主語は同じでも違ってもよい
- 主文は過去形でも現在形でもよい
- 意志・計画を表す動作と相性がよい

### 使用場面

日常生活の習慣、予定、順序立てた行動を説明する時に頻繁に使用されます。特に時間管理や作業手順を説明する際に重要な表現です。

## 英文解釋

"~Mae ni" is a grammar pattern that expresses "before (doing something)" or "before (something happens)". It indicates the temporal sequence where one action or event occurs before another.

### Connection Patterns

**1. Verb + mae ni**
- Dictionary form + mae ni
- Example: taberu mae ni (before eating), neru mae ni (before sleeping)

**2. Noun + no + mae ni**
- Noun + no + mae ni
- Example: asagohan no mae ni (before breakfast), kaigi no mae ni (before the meeting)

**3. Time noun + mae (ni)**
- Quantity + time noun + mae (ni)
- Example: 3-nichi mae (3 days ago), 1-jikan mae (1 hour ago)
- Note: "ni" is often omitted in this usage

### Grammatical Features

- The verb before "mae ni" must be in dictionary form (not ta-form)
- The subjects of the main and subordinate clauses can be the same or different
- The main clause can be in past or present tense
- Works well with actions expressing intention or plans

### Common Usage

Frequently used to describe daily habits, schedules, and sequential actions, especially when explaining time management or procedural steps.

## 中文解釋

「〜前に」是表示「在...之前」的文法結構，用來表達一個動作或事件發生在另一個動作或事件之前的時間順序關係。

### 接續方法

**1. 動詞 + 前に**
- 動詞辭書形[^dictionary-form] + 前に
- 例：食べる前に（吃之前）、寝る前に（睡之前）

**2. 名詞 + の + 前に**
- 名詞 + の + 前に
- 例：朝ごはんの前に（早餐之前）、会議の前に（會議之前）

**3. 時間名詞 + 前（に）**
- 數量 + 時間名詞 + 前（に）
- 例：3日前（三天前）、1時間前（一小時前）
- 注意：此用法的「に」常省略

### 文法特徵

- 「前に」前的動詞必須用辭書形（不是た形）
- 主句和從屬句的主語可以相同或不同
- 主句可以是過去式也可以是現在式
- 適合表達意志或計畫的動作

### 常見使用場合

常用於描述日常習慣、計畫和依序進行的動作，特別是在說明時間管理或作業流程時非常重要。

## 例句

**例句1：日常習慣（洗臉）**
```
[朝ごはん](../noun/001_asagohan.md)を食べる前に、まず顔を洗います。
Before eating breakfast, I first wash my face.
在吃早餐之前，我先洗臉。
```

**例句2：睡前習慣（刷牙）**
```
寝る前に歯を磨く習慣があります。
I have a habit of brushing my teeth before sleeping.
我有睡前刷牙的習慣。
```

**例句3：出門準備（確認天氣）**
```
出かける前に天気を確認しましょう。
Let's check the weather before going out.
出門前確認一下天氣吧。
```

**例句4：工作前準備（喝咖啡）**
```
仕事を始める前に、コーヒーを飲みます。
I drink coffee before starting work.
開始工作前我會喝咖啡。
```

**例句5：時間表達（三天前）**
```
彼は3日前に日本へ来ました。
He came to Japan 3 days ago.
他三天前來日本了。
```

## 補充說明

### 對比：〜前に vs 〜後で/〜てから

| 文法 | 意思 | 動詞接續 | 範例 | 時間關係 |
|------|------|---------|------|----------|
| 〜前に | 在...之前 | 辭書形 | 食べる前に | A之前做B |
| 〜後で | 在...之後 | た形 | 食べた後で | A之後做B |
| 〜てから | ...之後 | て形 | 食べてから | A之後做B（更強調順序） |

**重要區別**：
- 「前に」用辭書形：食べる前に（吃之前）
- 「後で」用た形[^ta-form]：食べた後で（吃之後）
- 「てから」用[て形](001_te_form.md)：食べてから（吃完之後）

### 時態注意事項

主句的時態不影響「前に」的使用：

| 主句時態 | 例句 | 說明 |
|---------|------|------|
| 現在式 | [朝ごはん](../noun/001_asagohan.md)を食べる前に、顔を洗います | 習慣 |
| 過去式 | [朝ごはん](../noun/001_asagohan.md)を食べる前に、顔を洗いました | 過去的事實 |
| 未來/意志 | [朝ごはん](../noun/001_asagohan.md)を食べる前に、顔を洗います | 計畫 |

注意：「前に」前的動詞永遠是辭書形，不隨主句時態變化。

### 常見動詞搭配

與「前に」常搭配使用的動詞：
- **日常動作**：食べる、飲む、寝る、起きる、出る
- **準備動作**：確認する、チェックする、用意する
- **開始動作**：始める、出発する、出かける
- **考慮動作**：考える、決める、相談する

### 省略「に」的情況

在時間表達中，「に」常被省略：
- 3日前に → 3日前（更常見）
- 1時間前に → 1時間前（更常見）
- 5分前に → 5分前（更常見）

但在動詞或名詞之後的「前に」，「に」通常不省略：
- 食べる前に（不省略）
- [朝ごはん](../noun/001_asagohan.md)の前に（不省略）

## 註解

[^dictionary-form]: **辞書形（辭書形）** - 動詞的基本形式，也稱為「原形」或「plain form」。是在字典中查詢時使用的形式。例：食べる、寝る、行く。在「〜前に」句型中，動詞必須使用辭書形。相關：[一段動詞](003_ichidan_verb.md)、[五段動詞](006_godan_verb.md)。

[^ta-form]: **た形** - 動詞的過去式形式，用於表達過去的動作或完成的動作。例：食べる → 食べた、寝る → 寝た。與「辭書形」不同，「た形」用於「〜後で」（之後）的句型中，強調前一個動作已經完成。
