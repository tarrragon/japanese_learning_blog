## 日文解釋

イ形容詞とは、日本語の形容詞の一種で、終止形（文を終える形）が「い」で終わるものを指します。日本語の伝統文法では単に「形容詞」と呼ばれ、ナ形容詞（形容動詞）と対比される品詞です。

イ形容詞は物事の性質・状態・感情を表し、名詞を直接修飾したり、述語として機能したりします。活用語であるため、時制・否定・丁寧さなどに応じて語形が変化します。

### 主な特徴

1. **語尾が「い」**：辞書形が必ず「い」で終わる（例：高い、美しい、楽しい）
2. **自立的に活用**：助動詞を使わず、語尾変化だけで否定・過去を表せる
3. **名詞を直接修飾**：「大きい家」のように、そのまま名詞の前に置ける
4. **感情・状態を表す**：主観的な評価や客観的な性質を描写する

### 活用パターン

| 活用形 | 語尾変化 | 例（高い） |
|--------|----------|------------|
| 終止形 | ～い | 高い |
| 連体形 | ～い | 高い（名詞） |
| 連用形 | ～く | 高く |
| 否定形 | ～くない | 高くない |
| 過去形 | ～かった | 高かった |
| 否定過去 | ～くなかった | 高くなかった |
| て形 | ～くて | 高くて |
| 仮定形 | ～ければ | 高ければ |

## 英文解釋

I-adjectives (イ形容詞) are one of the two main adjective categories in Japanese, characterized by their dictionary form ending in 「い」. In traditional Japanese grammar, they are simply called "adjectives" (形容詞), distinguished from na-adjectives (形容動詞).

### Key Characteristics

1. **Inflecting Words**: Unlike English adjectives, i-adjectives conjugate for tense, negation, and politeness without auxiliary verbs
2. **Direct Modification**: They can directly precede nouns without any connecting particle (e.g., 大きい家 "big house")
3. **Predicate Function**: They can serve as the main predicate of a sentence (e.g., この本は面白い "This book is interesting")
4. **Uniform Conjugation**: All i-adjectives follow the same conjugation pattern (with one exception: いい/良い)

### Conjugation Overview

- **Affirmative Present**: 高い (takai) - "is tall/expensive"
- **Negative Present**: 高くない (takakunai) - "is not tall/expensive"
- **Affirmative Past**: 高かった (takakatta) - "was tall/expensive"
- **Negative Past**: 高くなかった (takakunakatta) - "was not tall/expensive"
- **Te-form**: 高くて (takakute) - "being tall/expensive, and..."
- **Adverbial**: 高く (takaku) - "highly, expensively"

### The Exception: いい/良い

The adjective いい (good) is irregular. When conjugating, it reverts to its original form 良い (yoi):
- いい → 良くない (yokunai), 良かった (yokatta), 良くなかった (yokunakatta)

## 中文解釋

イ形容詞（い形容詞）是日語兩大形容詞類型之一，其特徵是辭書形以「い」結尾。在傳統日語文法中稱為「形容詞」，與ナ形容詞（形容動詞）形成對比。

### 核心特徵

1. **可自行變化**：不需要助動詞，透過語尾變化即可表達否定、過去式
2. **直接修飾名詞**：無需連接詞，直接放在名詞前面（如：大きい家＝大房子）
3. **可作謂語**：可單獨作為句子的謂語（如：この本は面白い＝這本書很有趣）
4. **統一變化規則**：所有イ形容詞遵循相同的活用規則（僅「いい/良い」例外）

### 活用規則

將語尾「い」去掉後，加上相應的語尾：

| 形式 | 變化 | 例（高い＝貴、高） |
|------|------|-------------------|
| 肯定現在 | ～い | 高い |
| 否定現在 | ～くない | 高くない |
| 肯定過去 | ～かった | 高かった |
| 否定過去 | ～くなかった | 高くなかった |
| て形 | ～くて | 高くて |
| 副詞形 | ～く | 高く |
| 假定形 | ～ければ | 高ければ |

### 特殊情況：いい（好）

「いい」是唯一的不規則イ形容詞。變化時需回到原形「良い（よい）」：
- いい → 良くない、良かった、良くなかった

## 核心用法

### 用法 1：述語（謂語用法）

イ形容詞可直接作為句子的謂語，表達主語的性質或狀態。

**例句 1**
```
今日は暑いですね。
It's hot today, isn't it?
今天很熱呢。
```

**例句 2**
```
この映画は面白かった。
This movie was interesting.
這部電影很有趣。
```

### 用法 2：連體修飾（修飾名詞）

イ形容詞直接放在名詞前面修飾，不需要任何連接詞。

**例句 3**
```
小さい犬が好きです。
I like small dogs.
我喜歡小狗。
```

### 用法 3：副詞用法（～く形）

將「い」變成「く」，可修飾動詞或其他形容詞。

**例句 4**
```
もっと早く起きなさい。
Wake up earlier.
起得再早一點。
```

### 用法 4：接續用法（～くて形）

連接多個形容詞或句子時，使用て形。

**例句 5**
```
このレストランは安くておいしい。
This restaurant is cheap and delicious.
這家餐廳便宜又好吃。
```

## 常見錯誤

### 錯誤 1：否定形混用ナ形容詞規則

❌ 誤：このスープは熱いじゃない。
✅ 正：このスープは熱くない。
說明：イ形容詞否定形用「～くない」，不用「じゃない」。

### 錯誤 2：修飾名詞時加「な」

❌ 誤：高いな山
✅ 正：高い山
說明：イ形容詞直接修飾名詞，不需要「な」。

### 錯誤 3：「いい」的活用錯誤

❌ 誤：この本はいくない。/ いかった。
✅ 正：この本は良くない。/ 良かった。
說明：「いい」變化時需回到「良い」的形式。

## 學習要點

1. **辨別イ形容詞與ナ形容詞**：雖然都以「い」音結尾，但如「きれい（な）」「嫌い（な）」屬於ナ形容詞
2. **掌握基本活用**：熟記「～い → ～くない → ～かった → ～くなかった」的變化
3. **注意「いい」的特殊性**：這是唯一的不規則變化，必須特別記憶
4. **理解語感差異**：同義的イ形容詞和ナ形容詞常有微妙的語感差異（如：うれしい vs 幸せな）
5. **活用て形連接**：用「～くて」連接多個形容詞是常見的表達方式
