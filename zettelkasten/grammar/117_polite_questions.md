## 日文

丁寧な質問（ていねいなしつもん）

**羅馬拼音**: teinei na shitsumon

**文法類型**: 疑問表達・語用模式

## 日文解釋

丁寧な質問は、相手に対して敬意を示しながら情報を尋ねる表現方法です。日本語では、単に疑問詞を使うだけでなく、相手との関係性[^social-distance]や場面の改まり度[^formality]に応じて、適切な語尾や表現を選ぶ必要があります。

| 項目 | 內容 |
|------|------|
| 文法類型 | 疑問文パターン |
| JLPT | N4-N3 |
| 使用場面 | 正式・商務・社交場合 |
| 形成 | 疑問詞 + 丁寧語[^teineigo] + か/でしょうか |

### 形成規則

**基本形式**

1. **〜ますか** - 基本的な丁寧な疑問形
   ```
   動詞ます形 + か
   ```

2. **〜でしょうか** - より丁寧で婉曲的な表現
   ```
   動詞辞書形/名詞/形容詞 + でしょうか
   ```

3. **〜ませんか** - 誘いや提案を含む疑問形
   ```
   動詞ません形 + か
   ```

4. **疑問詞 + ですか** - 基本的な情報確認
   ```
   疑問詞（何・誰・いつ・どこなど）+ ですか
   ```

### 丁寧さのレベル

日本語の疑問文には、丁寧さのレベルがあります：

**レベル1：カジュアル** - 友人・家族
```
行く？
何食べる？
```

**レベル2：丁寧** - 一般的な礼儀
```
行きますか。
何を食べますか。
```

**レベル3：より丁寧** - ビジネス・初対面
```
行かれますか。（尊敬語[^sonkeigo]）
いらっしゃいますか。
```

**レベル4：最も丁寧** - 正式な場面
```
お出かけになりますか。
お食事はいかがでしょうか。
```

丁寧な質問では、疑問詞の選択も重要です。例えば、「何」よりも「どれ」「どちら」、「誰」よりも「どなた」を使うことで、より丁寧な印象を与えることができます。

また、質問の前に前置き表現[^maegaki]を加えることで、唐突さを和らげ、より丁寧な印象を作ることができます。「恐れ入りますが」「失礼ですが」「お忙しいところ申し訳ございませんが」などの表現が代表的です。

## 英文解釋

Polite questions in Japanese are expressions used to ask for information while showing respect to the listener. In Japanese, it's not just about using question words; speakers must choose appropriate sentence endings and expressions based on their relationship with the listener[^social-distance] and the formality of the situation[^formality].

### Formation Rules

**Basic Patterns**

1. **〜masu ka** - Basic polite question form
   ```
   Verb masu-form + ka
   ```

2. **〜deshou ka** - More polite and indirect expression
   ```
   Verb dictionary form/Noun/Adjective + deshou ka
   ```

3. **〜masen ka** - Question form implying invitation or suggestion
   ```
   Verb negative form + ka
   ```

4. **Question word + desu ka** - Basic information confirmation
   ```
   Question word (nani, dare, itsu, doko, etc.) + desu ka
   ```

### Levels of Politeness

Japanese questions have different levels of politeness:

**Level 1: Casual** - Friends, Family
```
Iku? (Going?)
Nani taberu? (What to eat?)
```

**Level 2: Polite** - General courtesy
```
Ikimasu ka. (Will you go?)
Nani wo tabemasu ka. (What will you eat?)
```

**Level 3: More Polite** - Business, First Meetings
```
Ikaremasu ka. (Honorific[^sonkeigo])
Irasshaimasu ka. (Will you go/come?)
```

**Level 4: Most Polite** - Formal Situations
```
Odekake ni narimasu ka.
Oshokuji wa ikaga deshou ka. (How about a meal?)
```

The choice of question words is also important in polite questions. Using "dore" or "dochira" instead of "nani," or "donata" instead of "dare" creates a more polite impression.

Additionally, adding introductory phrases[^maegaki] before questions softens the directness and creates a more polite impression. Common expressions include "osore irimasu ga" (excuse me but), "shitsurei desu ga" (pardon me but), and "oisogashii tokoro moushiwake gozaimasen ga" (I'm sorry to bother you when you're busy).

## 中文解釋

日文的禮貌提問是在向對方詢問資訊時，同時表達敬意的表達方式。在日語中，不僅要使用疑問詞，還需要根據與對方的關係[^social-distance]和場合的正式程度[^formality]，選擇適當的語尾和表達方式。

### 形成規則

**基本模式**

1. **〜ますか** - 基本的禮貌疑問形式
   ```
   動詞ます形 + か
   ```

2. **〜でしょうか** - 更禮貌且委婉的表達
   ```
   動詞辭書形/名詞/形容詞 + でしょうか
   ```

3. **〜ませんか** - 含有邀請或建議的疑問形式
   ```
   動詞ません形 + か
   ```

4. **疑問詞 + ですか** - 基本的資訊確認
   ```
   疑問詞（什麼、誰、何時、哪裡等）+ ですか
   ```

### 禮貌程度的層次

日語疑問句有不同的禮貌程度：

**層次1：隨意** - 朋友、家人
```
行く？（去嗎？）
何食べる？（吃什麼？）
```

**層次2：禮貌** - 一般禮節
```
行きますか。（您去嗎？）
何を食べますか。（您吃什麼？）
```

**層次3：更禮貌** - 商務、初次見面
```
行かれますか。（尊敬語[^sonkeigo]）
いらっしゃいますか。（您去/來嗎？）
```

**層次4：最禮貌** - 正式場合
```
お出かけになりますか。（您要外出嗎？）
お食事はいかがでしょうか。（用餐如何？）
```

在禮貌提問中，疑問詞的選擇也很重要。例如，使用「どれ」或「どちら」而不是「何」，使用「どなた」而不是「誰」，可以給人更禮貌的印象。

此外，在問題前加上開場白表達[^maegaki]可以緩和突兀感，營造更禮貌的印象。常見的表達包括「恐れ入りますが」（不好意思）、「失礼ですが」（抱歉）、「お忙しいところ申し訳ございませんが」（很抱歉在您忙碌時打擾）等。

## 核心用法

### 用法 1：基本禮貌提問（〜ますか）

最常用的禮貌疑問形式，適用於日常禮貌場合。

**例句 1**
```
明日、会議に参加しますか。
Will you attend the meeting tomorrow?
您明天會參加會議嗎？
```

**例句 2**
```
コーヒーと紅茶、どちらがよろしいですか。
Which would you prefer, coffee or tea?
咖啡和紅茶，您想要哪一種？
```

### 用法 2：委婉詢問（〜でしょうか）

比「〜ますか」更委婉，常用於商務和正式場合，降低問題的直接性。

**例句 3**
```
この書類は明日までに提出すればよろしいでしょうか。
Would it be acceptable to submit these documents by tomorrow?
這份文件明天之前提交可以嗎？
```

**例句 4**
```
田中様はいつお戻りでしょうか。
When might Mr. Tanaka return?
田中先生什麼時候會回來呢？
```

### 用法 3：邀請或建議（〜ませんか）

表示邀請或建議，比直接邀請更委婉禮貌。

**例句 5**
```
一緒に昼食を食べませんか。
Would you like to have lunch together?
要不要一起吃午餐？
```

### 用法 4：前置禮貌語 + 疑問

在問題前加上前置禮貌語，使提問更加委婉。

**例句 6**
```
恐れ入りますが、お名前を教えていただけますか。
Excuse me, but could you tell me your name?
不好意思，可以請教您的大名嗎？
```

**例句 7**
```
お忙しいところ申し訳ございませんが、少しお時間をいただけますでしょうか。
I apologize for bothering you when you're busy, but could I have a moment of your time?
很抱歉在您忙碌時打擾，可以佔用您一點時間嗎？
```

## 文法規則

### 規則 1：疑問詞的禮貌度選擇

根據場合選擇適當禮貌度的疑問詞。

| 普通 | 禮貌 | 說明 |
|------|------|------|
| 何（なに） | どれ・どちら | 什麼、哪一個 |
| 誰（だれ） | どなた | 誰 |
| どこ | どちら | 哪裡 |
| いつ | いつ（相同） | 何時 |
| どう | いかが | 如何 |

### 規則 2：語尾的選擇

根據禮貌程度選擇適當的語尾。

**直接程度：高 → 低**
```
〜か？
〜ですか？
〜ますか？
〜でしょうか？
〜いただけますか？
〜いただけますでしょうか？
```

### 規則 3：尊敬語的使用

對方的行為使用尊敬語[^sonkeigo]形式提問。

```
普通：行きますか。
尊敬：いらっしゃいますか。
更尊敬：お出かけになりますか。
```

### 規則 4：前置表達的使用

**常用前置表達**：
- 恐れ入りますが（不好意思）
- 失礼ですが（抱歉）
- すみませんが（對不起）
- お尋ねしますが（請問）
- お聞きしたいのですが（想請教）

## 常見錯誤

### 錯誤 1：在正式場合使用過於直接的表達

❌ 誤：これ、何ですか。（在商務場合）
✅ 正：恐れ入りますが、こちらは何でしょうか。
說明：商務場合需要使用前置禮貌語和委婉表達「〜でしょうか」。

### 錯誤 2：混用禮貌程度

❌ 誤：明日来る？ですか。（混用普通形和丁寧形）
✅ 正：明日来ますか。或 明日来るの？（統一語域）
說明：不可在同一句中混用不同禮貌程度的表達。

### 錯誤 3：過度使用「〜か」導致生硬

❌ 誤：お名前は何ですか。お住まいはどこですか。お仕事は何ですか。（連續直接提問）
✅ 正：お名前を伺ってもよろしいでしょうか。差し支えなければ、ご住所を教えていただけますか。
說明：連續提問時應使用更委婉的表達方式，避免給人質問的感覺。

### 錯誤 4：忘記使用疑問詞的禮貌形式

❌ 誤：誰ですか。（對上司或客戶）
✅ 正：どなたですか。或 どちら様ですか。
說明：對地位較高或需要尊敬的對象，應使用禮貌疑問詞「どなた」或「どちら様」。

## 學習要點

1. **禮貌程度分層**：掌握從隨意到最禮貌的四個層次，根據場合和對象選擇適當程度。

2. **疑問詞選擇**：使用禮貌疑問詞（どなた、どちら、いかが）可提升整體禮貌度。

3. **前置表達必要**：在正式場合或對不熟悉的人提問時，必須使用前置禮貌語。

4. **委婉表達優先**：商務和正式場合優先使用「〜でしょうか」而非「〜ますか」。

5. **避免連續直問**：多個問題時應變換表達方式，避免給人質問或審問的感覺。

## 註解

[^social-distance]: **社會距離** - 指說話者與聽者之間的社會關係遠近。包括年齡差距、地位高低、熟悉程度等因素。社會距離越大，需要使用越禮貌的語言表達。在日本社會中，正確判斷社會距離是選擇適當語言表達的關鍵。

[^formality]: **場面的改まり度** - 指場合的正式程度。公開場合、商務場合、初次見面等屬於正式場合，需要使用更高程度的禮貌表達。家庭內部、朋友之間等私人場合則可使用較隨意的表達。判斷場合的正式度是日語語用能力的重要部分。

[^teineigo]: **丁寧語** - 日語三大敬語系統之一，是最基本的禮貌表達形式。主要通過「です・ます」形式表達對聽者的禮貌。與尊敬語（提高對方）、謙讓語（降低自己）不同，丁寧語只是提升整體談話的禮貌度。詳見 [敬語系統](005_keigo_system.md)

[^sonkeigo]: **尊敬語** - 日語敬語系統中用於提高對方或話題人物地位的表達方式。通過特殊動詞（いらっしゃる、召し上がる）、お〜になる形、或られる形來表達對對方的尊敬。在禮貌提問中，對方的行為應使用尊敬語形式。詳見 [敬語系統](005_keigo_system.md)

[^maegaki]: **前置き表現** - 在提問或請求前使用的開場白表達，用於緩和直接性、表達歉意或引起注意。常見表達包括「恐れ入りますが」（不好意思）、「失礼ですが」（抱歉）、「お忙しいところ申し訳ございませんが」（很抱歉打擾您）等。使用前置表達是日語禮貌溝通的重要技巧。
