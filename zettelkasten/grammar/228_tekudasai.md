禮貌地請求對方做某事的基本句型，是日文中最常用的請求表達之一。

<!--more-->

## 日文

〜てください

### 文法情報

| 項目 | 內容 |
|------|------|
| 文法類型 | 請求表達 |
| JLPT | N5 |
| 使用場面 | 日常會話、商務、服務業 |
| 形成 | 動詞て形 + ください |

### 形成規則

**基本形式**
```
動詞て形 + ください
```

**範例**
```
書く → 書いてください（請寫）
食べる → 食べてください（請吃）
来る → 来てください（請來）
する → してください（請做）
```

## 日文解釋

「〜てください」は、相手に何かをしてもらいたいときに使う丁寧な依頼表現です。動詞のて形に「ください」を付けることで、相手に対して礼儀正しく行動を求めることができます。

この表現は、JLPT N5レベルの基本的な文法項目であり、日常会話からビジネスシーン、サービス業まで幅広く使用されます。「〜てください」は命令形よりも柔らかく、「〜てくださいませんか」よりも直接的な依頼表現として位置づけられます。

使用場面は非常に多様で、道案内（「まっすぐ行ってください」）、依頼（「手伝ってください」）、指示（「ここに書いてください」）、勧誘（「どうぞ座ってください」）など、様々な文脈で活用されます。ただし、目上の人や正式な場面では、より丁寧な表現（「〜ていただけますか」など）を使用することが望ましい場合もあります。

## 英文解釋

"〜てください" is a polite request expression used when asking someone to do something. By attaching "ください" to the te-form of a verb, you can politely request an action from the listener.

This expression is a fundamental grammar point at the JLPT N5 level and is widely used in various contexts, from daily conversation to business settings and service industries. "〜てください" is softer than the imperative form but more direct than "〜てくださいませんか."

The usage scenarios are very diverse, including giving directions ("まっすぐ行ってください"), making requests ("手伝ってください"), giving instructions ("ここに書いてください"), and offering invitations ("どうぞ座ってください"). However, in formal situations or when speaking to superiors, more polite expressions such as "〜ていただけますか" may be more appropriate.

## 中文解釋

「〜てください」是請求對方做某事時使用的禮貌表達方式。在動詞て形後面加上「ください」，就能有禮貌地請求對方採取某個行動。

這個表達是 JLPT N5 等級的基本文法項目，從日常會話到商務場合、服務業等都被廣泛使用。「〜てください」比命令形柔和，但比「〜てくださいませんか」更直接。

使用場景非常多樣，包括指路（「まっすぐ行ってください」）、請求幫助（「手伝ってください」）、給予指示（「ここに書いてください」）、邀請（「どうぞ座ってください」）等各種情境。不過，對上級或在正式場合，使用更禮貌的表達（如「〜ていただけますか」）可能更為恰當。

## 核心用法

### 用法 1：基本請求

禮貌地請求對方做某事。

**例句 1**
```
少し待ってください。
Please wait a moment.
請稍等一下。
```

**例句 2**
```
ここに名前を書いてください。
Please write your name here.
請在這裡寫下名字。
```

### 用法 2：指示與說明

用於給予指示或說明步驟。

**例句 3**
```
この道をまっすぐ行ってください。
Please go straight on this road.
請沿著這條路直走。
```

**例句 4**
```
毎日三回、食後に飲んでください。
Please take it three times a day after meals.
請每天三次，飯後服用。
```

### 用法 3：邀請與建議

用「どうぞ」搭配使用，表示邀請或建議。

**例句 5**
```
どうぞ座ってください。
Please have a seat.
請坐。
```

**例句 6**
```
どうぞ召し上がってください。
Please help yourself. (polite)
請用（用餐）。
```

### 用法 4：商務與服務場景

在商務或服務業中的標準用語。

**例句 7**
```
こちらでお待ちください。
Please wait here.
請在這裡等候。
```

**例句 8**
```
お気をつけてお帰りください。
Please take care on your way home.
請小心回家。
```

## 文法規則

### 規則 1：て形變化

必須正確使用動詞的て形：
- 五段動詞：書く → 書いて、行く → 行って
- 一段動詞：食べる → 食べて、見る → 見て
- 不規則動詞：する → して、来る → 来て（きて）

### 規則 2：禮貌程度

「〜てください」是丁寧語（です・ます體），適用於一般禮貌場合，但不是最高級的敬語。

### 規則 3：否定形式

請求「不要做某事」時使用「〜ないでください」：
- 食べる → 食べないでください（請不要吃）
- 来る → 来ないでください（請不要來）

### 規則 4：更禮貌的變體

可以加上「ませんか」或「いただけますか」提高禮貌度：
- 〜てくださいませんか（能否請您...）
- 〜ていただけますか（能否麻煩您...，謙讓語）

## 常見錯誤

### 錯誤 1：使用辞書形

❌ 誤：書くください。
✅ 正：書いてください。
說明：必須使用て形，不能直接用辞書形。

### 錯誤 2：て形變化錯誤

❌ 誤：行ってください。（應該是「行ってください」但發音容易錯）
✅ 正：行ってください。（いってください）
說明：「行く」的て形是「行って」，雖然寫法相同但要注意發音。

### 錯誤 3：過度使用於上級

❌ 不當：社長、これを見てください。
✅ 改善：社長、こちらをご覧いただけますでしょうか。
說明：對上級使用時，應該使用更禮貌的表達方式。

### 錯誤 4：否定形式錯誤

❌ 誤：食べてくださいません。
✅ 正：食べないでください。
說明：否定請求使用「ないでください」，不是「てくださいません」。

## 學習要點

1. **基本結構**：動詞て形 + ください，是 N5 最重要的請求表達
2. **適用範圍廣**：從日常到商務都可使用，但要注意場合選擇禮貌度
3. **て形要正確**：務必熟練動詞て形變化規則
4. **否定請求**：使用「ないでください」表達「請不要...」
5. **禮貌階梯**：命令形 < 〜て < 〜てください < 〜てくださいませんか < 〜ていただけますか
