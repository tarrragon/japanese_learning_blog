## 日文解釋

日本語の形容詞は、物事の性質・状態・感情などを表す品詞であり、「い形容詞」と「な形容詞」の二種類に大別される。

い形容詞は語尾が「い」で終わり、活用語尾「い」が様々な形に変化することで、否定・過去・接続などの文法的意味を表す。例えば「高い」は「高くない」（否定）、「高かった」（過去）のように変化する。

な形容詞は、名詞を修飾する際に「な」を付けることからこの名がある。形容動詞とも呼ばれ、活用は名詞＋だ（copula）の変化に近い。「静かだ」は「静かではない」（否定）、「静かだった」（過去）のように変化する。

両者の活用パターンは異なるため、学習初期段階でしっかりと区別して覚えることが重要である。

## 英文解釋

Japanese adjectives are divided into two main categories: **i-adjectives** (い形容詞) and **na-adjectives** (な形容詞). Each type follows different conjugation patterns.

**I-adjectives** end with the hiragana い in their dictionary form. They conjugate by changing this い ending: replacing it with く for the adverbial form, くない for negation, かった for past tense, and くて for the te-form.

**Na-adjectives** (also called adjectival nouns or 形容動詞) behave more like nouns in their conjugation. They take な when modifying nouns directly, に for the adverbial form, and conjugate with the copula だ for various tenses and moods.

Understanding these two conjugation systems is fundamental to Japanese grammar, as they appear in virtually every sentence that describes qualities, states, or conditions.

## 中文解釋

日文形容詞分為兩大類：**い形容詞**和**な形容詞**，各自遵循不同的活用規則。

**い形容詞**（也稱為形容詞）以「い」結尾。活用時，語尾的「い」會發生變化：
- 否定：い → くない（高い → 高くない「不高」）
- 過去：い → かった（高い → 高かった「以前很高」）
- 連接：い → くて（高い → 高くて「又高...」）

**な形容詞**（也稱為形容動詞）修飾名詞時需要加「な」。其活用與名詞＋だ的變化類似：
- 否定：語幹 + ではない（静か → 静かではない「不安靜」）
- 過去：語幹 + だった（静か → 静かだった「以前很安靜」）
- 連接：語幹 + で（静か → 静かで「又安靜...」）

掌握這兩種活用系統是日文文法的基礎，因為描述性質、狀態的句子幾乎都會用到形容詞。

## 核心用法

### 用法 1：基本述語用法（い形容詞）

い形容詞直接作為述語，描述主語的性質或狀態。

**例句 1**
```
今日は暑いです。
Today is hot.
今天很熱。
```

**例句 2**
```
この本は面白かった。
This book was interesting.
這本書很有趣。
```

### 用法 2：基本述語用法（な形容詞）

な形容詞作為述語時，需要加上だ（非敬體）或です（敬體）。

**例句 3**
```
この公園は静かです。
This park is quiet.
這個公園很安靜。
```

### 用法 3：連体修飾用法

形容詞修飾名詞時的用法。い形容詞直接接名詞；な形容詞需加「な」。

**例句 4**
```
新しいパソコンを買いました。
I bought a new computer.
我買了一台新電腦。
```

**例句 5**
```
有名な作家に会いました。
I met a famous author.
我見到了一位有名的作家。
```

## 文法規則

### 規則 1：い形容詞的語尾變化

い形容詞的活用以語尾「い」的變化為核心。所有變化都發生在這個「い」上：

| 目標形式 | 變化規則 | 例 |
|----------|----------|-----|
| 副詞化 | い → く | 早い → 早く起きる |
| 否定 | い → くない | 高い → 高くない |
| 過去 | い → かった | 寒い → 寒かった |
| 接續 | い → くて | 安い → 安くて |
| 假定 | い → ければ | 良い → 良ければ |

### 規則 2：な形容詞的語幹與接續

な形容詞的活用圍繞語幹（去掉だ的部分）展開：

| 目標形式 | 接續方式 | 例 |
|----------|----------|-----|
| 修飾名詞 | 語幹 + な | 元気な子供 |
| 副詞化 | 語幹 + に | 静かに話す |
| 接續 | 語幹 + で | 便利で安い |
| 假定 | 語幹 + なら | 暇なら |

### 規則 3：「いい」的特殊活用

「いい」（好い/良い）是唯一的不規則い形容詞。活用時要用「よ-」系列：

| 活用形 | 形式 |
|--------|------|
| 辞書形 | いい / 良い |
| 否定形 | よくない |
| 過去形 | よかった |
| て形 | よくて |
| 仮定形 | よければ |

## 常見錯誤

### 錯誤 1：把「きれい」當作い形容詞

❌ 誤：きれいくない
✅ 正：きれいではない / きれいじゃない

說明：「きれい」雖然以「い」結尾，但它是な形容詞（語源為漢字「綺麗」）。類似的還有「嫌い」「有名」等。判斷依據是修飾名詞時是否需要「な」：きれいな花（正確）。

### 錯誤 2：な形容詞的過去形使用「だった」還是「でした」

❌ 誤：静かだでした（混用）
✅ 正：静かでした（敬體）/ 静かだった（常體）

說明：敬體過去形是「〜でした」，常體過去形是「〜だった」，不能混用。

### 錯誤 3：「いい」的活用使用「いかった」

❌ 誤：昨日の映画はいかった。
✅ 正：昨日の映画はよかった。

說明：「いい」是唯一的不規則い形容詞，活用時必須用「よ-」系列。辞書形可以說「いい」，但其他形式都要用「よ-」。

## 學習要點

1. **區分い形容詞和な形容詞**：修飾名詞時，い形容詞直接接續（高い山），な形容詞需加「な」（静かな夜）。

2. **掌握い形容詞的四個核心變化**：く（副詞/否定基礎）、くない（否定）、かった（過去）、くて（接續）。

3. **記住な形容詞近似名詞**：な形容詞的活用與「名詞＋だ」相似，理解這點有助於快速掌握。

4. **特別記憶「いい」的不規則活用**：除了辞書形，其他活用都用「よ-」系列。

5. **注意以「い」結尾的な形容詞**：「きれい」「嫌い」「幸い」等雖以「い」結尾，但都是な形容詞。
