## 日文

連体修飾（れんたいしゅうしょく）

### 文法情報

| 項目 | 內容 |
|------|------|
| 文法類型 | 名詞修飾結構 |
| JLPT | N5 |
| 使用場面 | 所有場景 |
| 形成 | 連體形 + 名詞 |

### 形成規則

**基本形式**
```
修飾語（連體形）+ 被修飾名詞
```

**連體形的變化**
```
動詞：辞書形即連體形
  食べる → 食べる人（吃的人）
  行った → 行った場所（去過的地方）

い形容詞：原形即連體形
  美しい → 美しい花（美麗的花）
  高かった → 高かった山（曾經很高的山）

な形容詞：加「な」
  静か → 静かな部屋（安靜的房間）
  便利 → 便利な店（便利的店）

名詞：加「の」
  私 → 私の本（我的書）
  日本 → 日本の文化（日本的文化）
```

## 日文解釋

連体修飾は、日本語の文法における最も基本的かつ重要な構造の一つです。動詞、形容詞、名詞などが、後続する名詞を修飾する形式で、英語の関係代名詞節や形容詞句に相当します。

この構造の最大の特徴は、修飾語が被修飾語の**前に置かれる**ことです。これは日本語が**左分岐構造**を持つ言語であることを示しています。つまり、情報が左から右へ、詳細から中心へと展開されます。

連体修飾には、単純な修飾から複雑な修飾まで様々なレベルがあります。単純な例では「赤い花」のように一語で修飾しますが、複雑な例では「昨日図書館で友達に借りた本」のように、時間、場所、動作など複数の要素を含む長い修飾節を形成できます。

特に重要なのは、日本語では**特別な接続詞を必要としない**という点です。英語の "who", "which", "that" のような関係代名詞は不要で、動詞や形容詞の連体形を直接名詞の前に置くだけで修飾関係が成立します。これは日本語の形態的な豊かさを示す特徴の一つです。

また、連体修飾は**時制や否定の情報**も含むことができます。「昨日食べた料理」「まだ読んでいない本」「行かなかった場所」など、修飾節内で時制や否定を表現することで、より正確で詳細な情報を伝えることができます。

## 英文解釋

Rentai shushoku (adnominal modification) is one of the most fundamental and important grammatical structures in Japanese. It refers to the pattern where verbs, adjectives, and nouns modify a following noun, functioning similarly to English relative clauses or adjectival phrases.

The key characteristic of this structure is that **modifiers precede the modified noun**. This demonstrates that Japanese is a **left-branching language**, where information flows from left to right, from details to the core element.

Rentai shushoku ranges from simple to complex modification. Simple examples include single-word modifiers like "赤い花" (red flower), while complex examples can involve lengthy modifying clauses such as "昨日図書館で友達に借りた本" (the book I borrowed from my friend at the library yesterday), containing multiple elements including time, place, and action.

Crucially, Japanese **requires no special conjunctions** for this structure. Unlike English relative pronouns such as "who", "which", or "that", Japanese simply places the rentaikei (adnominal form) of verbs or adjectives directly before the noun to establish the modification relationship. This reflects the morphological richness of Japanese.

Furthermore, rentai shushoku can incorporate **tense and negation information**. Examples like "昨日食べた料理" (the food I ate yesterday), "まだ読んでいない本" (the book I haven't read yet), and "行かなかった場所" (the place I didn't go) demonstrate how modifying clauses can express precise temporal and negative information.

## 中文解釋

連體修飾是日文文法中最基本也最重要的結構之一。它指的是動詞、形容詞、名詞等修飾後續名詞的形式，功能類似英文的關係子句或形容詞片語。

這個結構的最大特點是**修飾語放在被修飾名詞之前**。這顯示日文是一種**左分支語言**，資訊從左到右、從細節到核心展開。

連體修飾有從簡單到複雜的各種層次。簡單的例子如「赤い花」（紅色的花）用一個詞修飾，而複雜的例子如「昨日図書館で友達に借りた本」（昨天在圖書館向朋友借的書），可以包含時間、地點、動作等多個要素的長修飾節。

特別重要的是，日文**不需要特殊的連接詞**。不像英文的關係代名詞 "who"、"which"、"that"，日文只需將動詞或形容詞的連體形直接放在名詞前面，就能建立修飾關係。這體現了日文在形態上的豐富性。

此外，連體修飾可以包含**時態和否定資訊**。如「昨日食べた料理」（昨天吃的料理）、「まだ読んでいない本」（還沒讀的書）、「行かなかった場所」（沒去的地方）等，透過修飾節內的時態和否定表達，能傳達更精確、更詳細的資訊。

## 核心用法

### 用法 1：基本名詞修飾（單一修飾語）

最簡單的形式，用一個詞修飾名詞。

**例句 1**
```
美しい景色が広がっている。
Beautiful scenery spreads out before us.
美麗的景色展現在眼前。
```

**例句 2**
```
静かな環境で勉強したい。
I want to study in a quiet environment.
我想在安靜的環境學習。
```

### 用法 2：包含動作的複雜修飾

修飾節包含完整的動作和相關要素。

**例句 3**
```
昨日買った新しい靴がもう壊れた。
The new shoes I bought yesterday are already broken.
昨天買的新鞋子已經壞了。
```

**例句 4**
```
母が作ってくれる料理はいつも美味しい。
The food that my mother makes for me is always delicious.
媽媽做的料理總是很好吃。
```

### 用法 3：多層修飾（修飾節內再包含修飾）

在修飾節內部還有其他修飾關係，形成嵌套結構。

**例句 5**
```
友達が勧めてくれた、駅の近くにあるレストランに行った。
I went to the restaurant near the station that my friend recommended.
我去了朋友推薦的、在車站附近的餐廳。
```

## 文法規則

### 規則 1：修飾語的語順（前置修飾）

日文的修飾語**必須放在被修飾名詞之前**，這是固定語序。

**日文語序**
```
✅ 昨日読んだ本（修飾語在前）
❌ 本昨日読んだ（語序錯誤）
```

**對比英文語序**
```
The book that I read yesterday（名詞在前，修飾在後）
```

### 規則 2：連體形的選擇

根據修飾語的詞性，使用對應的連體形。

**動詞**：辞書形
```
食べる人、行った場所、来なかった友達
```

**い形容詞**：原形
```
高い山、美しい花、面白かった映画
```

**な形容詞**：加「な」
```
静かな部屋、便利な場所、元気な子供
```

**名詞**：加「の」
```
私の本、昨日の会議、東京の空
```

### 規則 3：修飾節內的時制獨立性

修飾節內的時制與主句的時制可以不同，各自獨立表達。

```
昨日買った本を明日読む。
I will read tomorrow the book I bought yesterday.
我明天要讀昨天買的書。

→ 修飾節「買った」是過去式
→ 主句「読む」是現在未來式
```

## 常見錯誤

### 錯誤 1：誤用丁寧形（ます形）作連體形

❌ 誤：毎日読みます本
✅ 正：毎日読む本
說明：連體形使用辞書形（普通形），不能使用丁寧形（ます形）。即使在敬語場合，修飾語也使用普通形。

### 錯誤 2：な形容詞忘記加「な」

❌ 誤：静か部屋
✅ 正：静かな部屋
說明：な形容詞修飾名詞時，必須加「な」連接。這是な形容詞的標誌性特徵。

### 錯誤 3：修飾節內助詞使用錯誤

❌ 誤：友達は作った料理
✅ 正：友達が作った料理
說明：修飾節內的主語通常用「が」標記，不用「は」。這是連體修飾的重要規則。

## 學習要點

1. **前置修飾原則**：日文修飾語永遠在被修飾名詞之前，形成左分支結構
2. **無需關係代名詞**：不像英文，日文不需要 who/which/that 等連接詞
3. **連體形規則**：動詞用辞書形、い形容詞用原形、な形容詞加な、名詞加の
4. **時制獨立性**：修飾節內可以有自己的時制，與主句時制獨立
5. **嵌套修飾**：可以形成多層嵌套的複雜修飾結構，表達豐富細節
