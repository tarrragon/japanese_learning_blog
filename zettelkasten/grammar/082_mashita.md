# ました（Mashita / Polite Past Form）

## 定義

### 日本語
「ました」は、日本語の動詞における丁寧体[^teineigo]の過去形です。ます形[^masu-kei]の過去時制を表す活用形[^katsuyou]で、過去に完了した動作や出来事を礼儀正しく述べる際に使用されます。日常会話、ビジネス場面、公式な場所など、幅広い場面で最も頻繁に使用される過去形表現です。

「ました」は「ます」の語幹に「た」が接続した形と理解できますが、実際の形成は「ます」を「ました」に置き換えるだけの規則的な変化です。この形式は、話し手が聞き手に対して敬意を示しながら、過去の事実や経験を伝える基本的な手段となります。

N5レベルで学ぶ最も基礎的な時制表現の一つであり、過去の動作・状態を表現するための不可欠な文法要素です。

### English
"Mashita" (ました) is the polite past tense form[^teineigo] of Japanese verbs. It is the past tense conjugation[^katsuyou] of the masu-form[^masu-kei], used to respectfully describe completed actions or events in the past. This is the most frequently used past tense expression across various situations including daily conversation, business settings, and formal occasions.

While "mashita" can be understood as the stem of "masu" with "ta" attached, the actual formation is a regular change from "masu" to "mashita." This form serves as the basic means for speakers to convey past facts or experiences while showing respect to the listener.

As one of the most fundamental tense expressions learned at the N5 level, it is an essential grammatical element for expressing past actions and states.

### 繁體中文
「ました」是日文動詞的禮貌體[^teineigo]過去式。它是ます形[^masu-kei]的過去時態活用形[^katsuyou]，用於禮貌地描述過去完成的動作或事件。這是日常對話、商務場合和正式場所等各種情境中最常用的過去式表達。

「ました」可以理解為「ます」的語幹加上「た」，但實際形成只需將「ます」替換為「ました」的規則變化。這種形式是說話者在向聽者表示敬意的同時，傳達過去事實或經驗的基本手段。

作為N5級別學習的最基礎時態表達之一，它是表達過去動作和狀態不可或缺的文法要素。

## 形成規則

### 基本規則
**規則**：動詞ます形 → ました

無論是哪一類動詞（一段動詞[^ichidan]、五段動詞[^godan]、不規則動詞[^fukisoku]），只要先變化為ます形，再將「ます」替換為「ました」即可。

### 各類動詞的變化示例

#### 1. る動詞（一段動詞[^ichidan]）
- 食べる → 食べます → 食べ**ました**（ate / 吃了）
- 見る → 見ます → 見**ました**（saw / 看了）
- 寝る → 寝ます → 寝**ました**（slept / 睡了）
- 起きる → 起きます → 起き**ました**（woke up / 起床了）
- 教える → 教えます → 教え**ました**（taught / 教了）

#### 2. う動詞（五段動詞[^godan]）
- 書く → 書きます → 書き**ました**（wrote / 寫了）
- 読む → 読みます → 読み**ました**（read / 讀了）
- 話す → 話します → 話し**ました**（spoke / 說了）
- 買う → 買います → 買い**ました**（bought / 買了）
- 待つ → 待ちます → 待ち**ました**（waited / 等了）
- 帰る → 帰ります → 帰り**ました**（returned / 回來了）
- 飲む → 飲みます → 飲み**ました**（drank / 喝了）
- 行く → 行きます → 行き**ました**（went / 去了）

#### 3. 不規則動詞[^fukisoku]
- する → します → し**ました**（did / 做了）
- 来る（くる）→ 来ます（きます）→ 来**ました**（きました）（came / 來了）

### 時制活用表

| 時態 | 肯定形 | 否定形 |
|------|--------|--------|
| **現在・未来** | ～ます | ～ません |
| **過去** | ～ました | ～ませんでした |

**例：食べる（吃）**
- 食べます（吃 / 現在・習慣或未來）
- 食べません（不吃）
- 食べ**ました**（吃了 / 過去）
- 食べませんでした（沒吃 / 過去否定）

## 核心用法

### 用法 1：過去完了動作

禮貌地敘述過去已經完成的動作或行為。

**例句 1**
```
昨日、友達と映画を見ました。
Kinō, tomodachi to eiga o mimashita.
I watched a movie with my friend yesterday.
我昨天和朋友看了電影。
```

**例句 2**
```
先週、日本語を勉強しました。
Senshū, Nihongo o benkyō shimashita.
I studied Japanese last week.
我上週學了日文。
```

### 用法 2：過去經驗

表達說話者過去的經驗或體驗。

**例句 3**
```
去年、日本へ行きました。
Kyonen, Nihon e ikimashita.
I went to Japan last year.
我去年去了日本。
```

**例句 4**
```
大学で経済学を学びました。
Daigaku de keizaigaku o manabimashita.
I studied economics at university.
我在大學學了經濟學。
```

### 用法 3：過去狀態描述

禮貌地描述過去的狀態或情況（通常搭配です的過去形「でした」）。

**例句 5**
```
会議は三時に終わりました。
Kaigi wa sanji ni owarimashita.
The meeting ended at three o'clock.
會議三點結束了。
```

## 文法規則

### 規則 1：時間標記搭配

「ました」通常與表示過去的時間詞搭配使用，明確表達動作發生的時間點。

**常見時間詞**：
- 昨日（きのう）- 昨天
- 先週（せんしゅう）- 上週
- 去年（きょねん）- 去年
- 先月（せんげつ）- 上個月
- さっき - 剛才
- 午前中（ごぜんちゅう）- 上午
- ～前（～まえ）- ～之前

**例**：
- 昨日、本を買いました。（昨天買了書。）
- さっき、電話しました。（剛才打了電話。）

### 規則 2：否定形「ませんでした」

過去的否定表達使用「ませんでした」，不是「ましたない」或其他形式。

**形成**：ます形 → ません → ませんでした

**例**：
- 食べます → 食べません → 食べ**ませんでした**（沒吃）
- 行きます → 行きません → 行き**ませんでした**（沒去）
- します → しません → し**ませんでした**（沒做）

### 規則 3：疑問形「ましたか」

在「ました」後加上疑問助詞[^ka]「か」，形成禮貌的過去疑問句。

**形成**：ました + か

**例**：
- もう食べましたか。（已經吃了嗎？）
- 昨日、何をしましたか。（昨天做了什麼？）
- 日本へ行ったことがありますか。→ 行きましたか。（去過日本嗎？）

## 常見錯誤

### 錯誤 1：與た形混淆

❌ 誤：昨日、友達と映画を見たです。
✅ 正：昨日、友達と映画を見ました。

**說明**：「た形」（見た）是普通體的過去形，不能直接加「です」。丁寧體應該使用「ました」。

### 錯誤 2：過去否定形錯誤

❌ 誤：食べましなかった。
❌ 誤：食べないました。
✅ 正：食べませんでした。

**說明**：過去否定形是「ませんでした」，不是「ましなかった」或「ないました」。必須先將「ます」變為「ません」，再加「でした」。

### 錯誤 3：動詞變化不完全

❌ 誤：食べるました。
❌ 誤：書くました。
✅ 正：食べました。
✅ 正：書きました。

**說明**：必須先將動詞變為ます形，再替換為「ました」。不能直接在辭書形後加「ました」。

## 學習要點

1. **形成簡單**：只要會變ます形，就會變ました形（ます → ました）
2. **適用廣泛**：所有動詞類型都遵循同一規則，無例外
3. **時間標記**：通常搭配過去時間詞使用（昨日、先週、去年等）
4. **禮貌程度**：與「ます」相同，適用於正式場合和日常禮貌對話
5. **與た形區別**：ました（禮貌體）vs た（普通體），使用場合不同

## 註解

[^teineigo]: **丁寧語** - 日文三大敬語類型之一，透過「です・ます」等形式表達對聽者的基本禮貌。詳見 [丁寧語概念](../concept/036_teineigo_concept.md)

[^masu-kei]: **ます形** - 日文動詞的禮貌體基本形式，表達現在或未來的動作。詳見 [ます形](./043_masu-kei.md)

[^katsuyou]: **動詞活用** - 動詞根據時態、禮貌程度、接續關係等改變形態的文法現象。詳見 [動詞分類](../concept/041_verb_classification.md)

[^ichidan]: **一段動詞** - 動詞語幹不變，只變化詞尾「る」的動詞類型。例：食べる → 食べます → 食べました。

[^godan]: **五段動詞** - 語尾在五十音圖的五個段（あ・い・う・え・お）變化的動詞類型。例：書く → 書きます → 書きました。

[^fukisoku]: **不規則動詞** - 不遵循一般變化規則的動詞，日文中主要是「する」和「来る」。詳見 [不規則活用](../concept/002_fukisoku-katsuyou.md)

[^ka]: **疑問助詞「か」** - 置於句末形成疑問句的助詞。在丁寧體中，「か」使句子成為禮貌的疑問表達。
