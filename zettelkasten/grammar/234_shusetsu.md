## 日文
主節（しゅせつ）

**羅馬拼音**：shusetsu

**品詞分類**：文法用語（統語論）

## 日文解釋
「主節」は、複文において中心となる節（clause）のことです。従属節（従属的な役割を果たす節）に対して、文全体の核となる述語を含み、独立して意味を成すことができます。日本語では主節が文末に来ることが多く、従属節は主節の前に置かれます。主節は文の最終的な判断、主張、または述べたいことを表します。

## 英文解釋
The main clause (主節, shusetsu) is the central clause in a complex sentence. In contrast to subordinate clauses (which play a dependent role), the main clause contains the core predicate of the sentence and can stand independently with complete meaning. In Japanese, the main clause typically appears at the end of the sentence, while subordinate clauses precede it. The main clause expresses the final judgment, assertion, or main point of the sentence.

## 中文解釋
主節（しゅせつ）是複句結構中的核心子句。相對於從屬節（扮演依附角色的子句），主節包含整個句子的核心述語，能夠獨立表達完整意思。在日文中，主節通常出現在句尾，從屬節則置於主節之前。主節表達句子最終的判斷、主張或想要表達的核心內容。

## 例句

**例句1**
```
雨が降っているから、傘を持って行きます。
Because it is raining, I will take an umbrella.
因為在下雨，所以我要帶傘去。
```
> 「傘を持って行きます」が主節、「雨が降っているから」が従属節

**例句2**
```
彼が来たら、会議を始めましょう。
When he comes, let us start the meeting.
他來的話，就開始開會吧。
```
> 「会議を始めましょう」が主節、「彼が来たら」が従属節

**例句3**
```
日本語を勉強しているので、漢字が読めるようになりました。
Because I am studying Japanese, I have become able to read kanji.
因為在學日文，所以變得會讀漢字了。
```
> 「漢字が読めるようになりました」が主節、「日本語を勉強しているので」が従属節

**例句4**
```
時間があれば、映画を見に行きたいです。
If I have time, I want to go see a movie.
如果有時間的話，我想去看電影。
```
> 「映画を見に行きたいです」が主節、「時間があれば」が従属節

**例句5**
```
母が作った料理はとてもおいしかったです。
The food that my mother made was very delicious.
媽媽做的料理非常好吃。
```
> 「料理はとてもおいしかったです」が主節、「母が作った」が連体修飾節

## 補充說明

### 主節與從屬節的關係
複文（複雑な文）は、主節と一つ以上の従属節から構成されます。
- **主節**：文の中心となる節、独立して意味を成す
- **従属節**：主節を修飾・補足する節、単独では不完全

### 日本語の語順と主節
日本語は「従属節 → 主節」の語順が基本です：
1. 条件節（〜たら、〜ば、〜と、〜なら）→ 主節
2. 理由節（〜から、〜ので）→ 主節
3. 時間節（〜とき、〜前に、〜後で）→ 主節
4. 連体修飾節 → 被修飾名詞 → 主節の述語

### 主節の特徴
1. **文末に位置**：日本語では主節が通常文末に来る
2. **述語を含む**：文の最終的な述語は主節にある
3. **独立可能**：従属節がなくても意味が通じる
4. **モダリティを担う**：話者の態度・判断を表す表現は主節に現れる

### 従属節との見分け方
- 従属節：接続助詞（から、ので、ば、たら、など）で終わる
- 主節：終止形または丁寧形で終わる
