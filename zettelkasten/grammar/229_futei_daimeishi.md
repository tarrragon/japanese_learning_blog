# 不定代詞（不定代名詞）

## 定義

### 中文
指稱不特定人、事、物、地點或時間的代詞。由疑問詞加上「か」或「も」構成，用於表達「某個」或「任何」的概念。

### 英文
Indefinite pronouns formed by adding か (someone/something) or も (anyone/anything) to interrogative words, used to refer to unspecified persons, things, places, or times.

### 日文
疑問詞に「か」や「も」を付けて、不特定の人・物・場所・時間などを指す代名詞。「ある～」や「どんな～でも」という意味を表す。

## 主要用法

### 1. 「～か」系列（肯定存在）

- **誰か**（だれか）：某人
- **何か**（なにか）：某物
- **どこか**：某處
- **いつか**：某時
- **どれか**：某個（複數中選一）

**例句：**
```
誰かが窓を開けました。
（だれかがまどをあけました。）
有人打開了窗戶。
Someone opened the window.
```

```
何か飲みたいですか。
（なにかのみたいですか。）
想喝點什麼嗎？
Would you like something to drink?
```

### 2. 「～も」系列（全面否定）

- **誰も**（だれも）：任何人都不（用於否定句）
- **何も**（なにも）：什麼都不（用於否定句）
- **どこも**：哪裡都不
- **いつも**：總是、隨時[^1]

**例句：**
```
誰も来ませんでした。
（だれもきませんでした。）
沒有任何人來。
Nobody came.
```

```
何も食べていません。
（なにもたべていません。）
什麼都沒吃。
I haven't eaten anything.
```

### 3. 「～でも」系列（任何都可）

- **誰でも**（だれでも）：任何人都可以
- **何でも**（なんでも）：什麼都可以
- **どこでも**：哪裡都可以
- **いつでも**：任何時候都可以

**例句：**
```
誰でも参加できます。
（だれでもさんかできます。）
任何人都可以參加。
Anyone can participate.
```

```
何でも聞いてください。
（なんでもきいてください。）
請隨便問。
Please ask me anything.
```

## 文法特徵

### 1. 助詞使用規則

**「～か」後接助詞「が」或「を」**：
```
誰かが電話をかけました。（某人打了電話）
何かを探していますか。（在找什麼嗎？）
```

**「～も」後接否定形式**：
```
どこも行きません。（哪裡都不去）
```

**「～でも」不需後接否定**：
```
いつでも大丈夫です。（任何時候都可以）
```

### 2. 疑問句中的使用

在疑問句中，「～か」表示「是否有某個～」：
```
誰か来ましたか。
（有人來了嗎？）
Did someone come?
```

## 特殊用法

### 1. いつも的特殊性

「いつも」不同於其他「～も」系列，主要表示「總是、經常」，不用於否定：
```
彼はいつも遅れます。
（かれはいつもおくれます。）
他總是遲到。
He is always late.
```

### 2. 疑問詞+でも與疑問詞+も的區別

**疑問詞+でも**（肯定）：
```
誰でも知っています。（任何人都知道）
```

**疑問詞+も+否定**：
```
誰も知りません。（沒人知道）
```

## 常見搭配

| 不定代詞 | 疑問詞基礎 | 「～か」 | 「～も」 | 「～でも」 |
|---------|----------|---------|---------|-----------|
| 人 | 誰 | 誰か | 誰も～ない | 誰でも |
| 物 | 何 | 何か | 何も～ない | 何でも |
| 場所 | どこ | どこか | どこも～ない | どこでも |
| 時間 | いつ | いつか | - | いつでも |
| 選擇 | どれ | どれか | どれも～ない | どれでも |
| 方式 | どう | - | - | どうでも |

## 學習重點

1. **「～か」表存在，「～も」配否定，「～でも」表任意**
2. **「誰も」「何も」等必須搭配否定形式**
3. **「いつも」是特例，表示習慣性**
4. **注意疑問句中「～か」的語氣差異**

---

[^1]: 「いつも」是不定代詞系列的特例，通常不與否定搭配，而是表達時間的恆常性。
