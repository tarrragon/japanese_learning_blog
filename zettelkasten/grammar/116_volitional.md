# 意向形 (Volitional Form)

## 日文

### 基本概念

**意向形**（いこうけい）是日文動詞的一種活用形式，用於表達說話者的意志、建議、推測等意涵。意向形有兩種形式：普通形（〜う/〜よう）和丁寧形（〜ましょう）。本卡片主要說明普通形的意向形。

### 形成規則

#### 一段動詞（る動詞）
**規則**：語幹 + よう

| 辞書形 | 語幹 | 意向形 |
|-------|------|--------|
| 食べる | 食べ | 食べよう |
| 見る | 見 | 見よう |
| 寝る | 寝 | 寝よう |
| 起きる | 起き | 起きよう |

#### 五段動詞（う動詞）
**規則**：語尾變為「お」段 + う

| 辞書形 | 語尾 | 意向形 | 說明 |
|-------|------|--------|------|
| 行く | く → こう | 行こう | ku → kou |
| 書く | く → こう | 書こう | ku → kou |
| 話す | す → そう | 話そう | su → sou |
| 待つ | つ → とう | 待とう | tsu → tou |
| 死ぬ | ぬ → のう | 死のう | nu → nou |
| 遊ぶ | ぶ → ぼう | 遊ぼう | bu → bou |
| 読む | む → もう | 読もう | mu → mou |
| 帰る | る → ろう | 帰ろう | ru → rou |
| 買う | う → おう | 買おう | u → ou |

#### 不規則動詞

| 辞書形 | 意向形 | 備考 |
|-------|--------|------|
| する | しよう | 不規則 |
| 来る | 来よう（こよう）| 不規則 |

## 日文解釋

意向形は話し手の意志や意図を表す動詞の活用形です。主に三つの機能があります：（1）話し手の意志や決意を表す、（2）聞き手に対する勧誘や提案を表す、（3）推量や不確実な意図を表す。

意向形には普通形（〜う/〜よう）と丁寧形（〜ましょう）の二種類があります。普通形は友人や家族など親しい関係で使われ、カジュアルな場面に適しています。丁寧形は一般的な礼儀正しい場面で使用されます。

文法的には、一段動詞は語幹に「よう」を付け、五段動詞は語尾を「お」段に変えて「う」を付けます。不規則動詞の「する」は「しよう」、「来る」は「来よう」となります。

意向形は単独で使うこともできますが、「〜と思う」「〜とする」などの表現と組み合わせることで、より複雑な意味を表現できます。また、疑問形「〜うか」「〜ようか」は自問自答や迷いを表します。

## 英文解釋

The volitional form (意向形, ikōkei) is a verb conjugation in Japanese that expresses the speaker's will, intention, or volition. It has three primary functions: (1) expressing the speaker's determination or decision, (2) making suggestions or invitations to the listener, and (3) expressing conjecture or uncertain intention.

There are two forms: the plain form (〜う/〜よう) and the polite form (〜ましょう). The plain form is used among friends and family in casual situations, while the polite form is used in general polite contexts.

Grammatically, ichidan verbs add よう to the stem, godan verbs change the final vowel to the "o" row and add う, and irregular verbs have unique forms (する → しよう, 来る → 来よう). The volitional form can be used alone or combined with expressions like 〜と思う or 〜とする for more complex meanings.

## 中文解釋

意向形是日文動詞的一種活用形式，用來表達說話者的意志、意圖或決心。它有三個主要功能：（1）表達說話者的決意或決定、（2）對聽者提出建議或邀請、（3）表達推測或不確定的意圖。

意向形有兩種形式：普通形（〜う/〜よう）和丁寧形（〜ましょう）。普通形用於朋友和家人等親密關係的隨意場合，丁寧形則用於一般禮貌場合。

從文法上來說，一段動詞在語幹後加「よう」，五段動詞將語尾改為「お」段再加「う」，不規則動詞有特殊形式（する → しよう、来る → 来よう）。意向形可以單獨使用，也可以與「〜と思う」「〜とする」等表達結合，表達更複雜的意義。

## 核心用法

### 用法 1：表達說話者的意志或決心

表達「我要做某事」「我打算做某事」的強烈意志。

**例句 1**
```
明日早く起きよう。
Ashita hayaku okiyou.
I'll wake up early tomorrow. / I'm going to wake up early tomorrow.
明天我要早起。
```

**例句 2**
```
今日こそ勉強しよう。
Kyou koso benkyou shiyou.
I'll study today for sure.
今天一定要讀書。
```

### 用法 2：建議或邀請（非正式場合）

在朋友、家人之間提出建議或邀請一起做某事。

**例句 3**
```
一緒に映画を見よう。
Issho ni eiga wo miyou.
Let's watch a movie together.
我們一起看電影吧。
```

**例句 4**
```
明日遊ぼう。
Ashita asobou.
Let's hang out tomorrow.
明天一起玩吧。
```

### 用法 3：自問自答或表達猶豫

使用疑問形「〜うか/〜ようか」表達內心的猶豫或思考。

**例句 5**
```
どうしようかな。
Dou shiyou kana.
What should I do? / I wonder what I should do.
該怎麼辦呢？
```

**例句 6**
```
本を読もうか、テレビを見ようか。
Hon wo yomou ka, terebi wo miyou ka.
Should I read a book or watch TV?
要看書還是看電視呢？
```

## 文法規則

### 規則 1：與「と思う」結合表達打算

「意向形 + と思う」表示「我想要做某事」「我打算做某事」。

**說明**：
- 肯定：〜ようと思う（我打算做某事）
- 否定：〜ないと思う（我不打算做某事）
- 過去：〜ようと思った（我曾經打算做某事）

**例句**：
```
来年留学しようと思います。
Rainen ryuugaku shiyou to omoimasu.
I'm thinking of studying abroad next year.
明年我打算留學。
```

### 規則 2：與「とする」結合表達嘗試或即將

「意向形 + とする」表示「正要做某事」「試圖做某事」。

**說明**：
- 表示動作即將發生
- 常用於敘述性文章或正式場合

**例句**：
```
家を出ようとしたとき、電話が鳴った。
Ie wo deyou to shita toki, denwa ga natta.
Just as I was about to leave home, the phone rang.
正要出門時，電話響了。
```

### 規則 3：疑問形的兩種用法

「〜うか/〜ようか」有兩種主要用法。

**用法 A：自問自答**
```
今日は何を食べようかな。
What should I eat today? (thinking to oneself)
今天要吃什麼呢？（自言自語）
```

**用法 B：提議（詢問對方意見）**
```
ちょっと休もうか。
Shall we take a break?
要不要休息一下？（詢問對方）
```

## 常見錯誤

### 錯誤 1：五段動詞的意向形變化錯誤

❌ 誤：行きよう（×）
✅ 正：行こう（○）

說明：五段動詞不是加「よう」，而是語尾變為「お」段 + う。「行く」的語尾是「く」，要變成「こう」，所以是「行こう」。

### 錯誤 2：混淆普通形和丁寧形的使用場合

❌ 誤：先生、一緒に行こう。（對老師說話時）
✅ 正：先生、一緒に行きましょう。

說明：對長輩或上司使用意向形時，應該使用丁寧形「ましょう」，而不是普通形「う/よう」。普通形僅用於親密關係或隨意場合。

### 錯誤 3：「〜と思う」前誤用丁寧形

❌ 誤：明日行きましょうと思います。（×）
✅ 正：明日行こうと思います。（○）

說明：「〜と思う」前面必須使用普通形，即使整個句子是禮貌體。這是日文的固定結構，不能使用「ましょう」。

## 學習要點

1. **兩種形式**：普通形（〜う/〜よう）用於隨意場合，丁寧形（〜ましょう）用於禮貌場合
2. **三大功能**：表達意志、提出建議、表達猶豫或推測
3. **變化規則**：一段動詞加「よう」，五段動詞語尾變「お」段 + う
4. **常見組合**：〜ようと思う（打算）、〜ようとする（試圖、正要）
5. **疑問形**：〜うか/〜ようか 表達自問自答或提議
