## 日文

〜てしまう

### 文法情報

| 項目 | 內容 |
|------|------|
| 文法類型 | 補助動詞 |
| JLPT | N4 |
| 使用場面 | 日常會話、書面語 |
| 形成 | 動詞て形 + しまう |

### 形成規則

**基本形式**
```
動詞て形 + しまう
```

**活用變化**
```
食べてしまう      (辞書形)
食べてしまいます  (ます形)
食べてしまった    (た形)
食べてしまって    (て形)
```

**口語縮約形**
```
食べてしまう → 食べちゃう
読んでしまう → 読んじゃう
買ってしまった → 買っちゃった
```

## 日文解釋

「〜てしまう」は、動詞の後に付いて、動作の完了や話し手の心的態度を表す補助動詞である。主に二つの用法がある。

第一の用法は、動作が完全に終わったことを強調する「完了」の意味である。「宿題を終わってしまった」のように、その動作がすっかり完了したことを表す。この用法では、動作の徹底性や完結性が強調される。

第二の用法は、好ましくない結果や予期せぬ事態に対する「残念」「後悔」「困惑」などの感情を表す。「財布を忘れてしまった」のように、望ましくない出来事が起こってしまったことに対する話し手の心情を込める。この場合、しばしば「〜ちゃう/〜じゃう」という口語形が使われる。

日常会話では縮約形が頻繁に用いられ、カジュアルな印象を与える。特に若者の会話や親しい間柄では「食べちゃった」「見ちゃった」という形が自然である。書き言葉では「〜てしまう」の形が使われるが、会話では縮約形が圧倒的に多い。

また、この文法形式は時制と組み合わせることで、「〜てしまった」(過去)、「〜てしまっている」(状態の継続)など、さまざまなニュアンスを表現できる。特に「〜てしまった」の形は、取り返しのつかない事態を表すことが多く、後悔の念を強く表す。

## 英文解釋

"〜てしまう" (te-shimau) is an auxiliary verb that attaches to the te-form of verbs, expressing either completion of an action or the speaker's emotional attitude towards an undesirable outcome.

In its completion sense, it emphasizes that an action has been fully completed or finished. For example, "読んでしまった" (I've finished reading it) stresses the thoroughness or finality of the action. This usage is similar to the English expressions "completely" or "entirely."

More commonly in everyday conversation, it expresses regret, disappointment, or dismay about an unwanted situation. When used this way, it conveys that something unfortunate has happened, often unexpectedly. For instance, "鍵を忘れてしまった" (I forgot my keys) implies not just the fact of forgetting, but also the speaker's regret or frustration about it.

In casual speech, this form frequently contracts to "〜ちゃう" or "〜じゃう," especially among younger speakers and in informal contexts. These contracted forms sound more natural and less formal than the full "〜てしまう" form. The choice between the full and contracted forms depends on the formality of the situation and the speaker's relationship with the listener.

This grammatical pattern is highly versatile and can combine with various tenses to express different temporal and aspectual meanings, making it an essential component of natural Japanese expression.

## 中文解釋

「〜てしまう」是一個補助動詞，接在動詞て形後面，表達兩種主要意義：動作的完了和說話者對不良結果的情感態度。

第一種用法表示「完了」，強調動作已經完全結束。例如「全部食べてしまった」(全部吃完了)，突出動作的徹底性和完結性。這種用法相當於中文的「完全...了」或「...完了」。

第二種也是更常見的用法，表達「遺憾」、「後悔」或「困擾」等負面情緒。當用於描述不希望發生的事情時，它傳達了說話者的懊惱心情。例如「財布を落としてしまった」(把錢包弄丟了)，不僅陳述了掉錢包這個事實，更表達了說話者的後悔和困擾。

在日常會話中，這個形式常常縮約為「〜ちゃう」或「〜じゃう」，聽起來更加自然和口語化。特別是在年輕人之間或親密關係中，「食べちゃった」、「見ちゃった」這樣的縮約形是標準用法。書面語則傾向使用完整的「〜てしまう」形式。

這個文法可以與不同時態結合，如「〜てしまった」(已經...了)、「〜てしまっている」(已經...的狀態持續中)，表達豐富的時間和情態意義。

## 核心用法

### 用法 1：完了・完成

表示動作已經完全結束、徹底完成。

**例句 1**
```
宿題を全部やってしまいました。
I've finished all my homework.
我已經把作業全部做完了。
```

**例句 2**
```
ケーキを一人で食べてしまった。
I ate the whole cake by myself.
我一個人把整個蛋糕吃完了。
```

### 用法 2：遺憾・後悔（望ましくない結果）

表示對不希望發生的事情感到遺憾、後悔或困擾。

**例句 3**
```
大事な書類を間違って捨ててしまった。
I accidentally threw away an important document.
我不小心把重要文件丟掉了。
```

**例句 4**
```
電車で寝ちゃって、終点まで行っちゃった。
I fell asleep on the train and ended up at the last stop.
我在電車上睡著了，結果坐到終點站了。
```

### 用法 3：口語縮約形（カジュアル）

日常會話中常用的縮約形式，更加自然和口語化。

**例句 5**
```
あ、スマホ家に忘れちゃった！
Oh no, I left my phone at home!
啊，我把手機忘在家裡了！
```

## 文法規則

### 規則 1：て形的接續

「しまう」必須接在動詞的て形後面，不能直接接辞書形或其他形式。

**正確接續**
```
食べる → 食べて + しまう → 食べてしまう
読む → 読んで + しまう → 読んでしまう
来る → 来て + しまう → 来てしまう
```

### 規則 2：縮約形的形成

在口語中，「てしまう」常縮約為「ちゃう」，「でしまう」縮約為「じゃう」。

**縮約規則**
```
〜てしまう → 〜ちゃう
食べてしまう → 食べちゃう
見てしまう → 見ちゃう

〜でしまう → 〜じゃう
読んでしまう → 読んじゃう
飲んでしまう → 飲んじゃう
```

### 規則 3：時制的變化

「しまう」本身可以活用，與其他時制結合表達不同意義。

**時制變化**
```
〜てしまう      (非過去) - 將會...完 / 恐怕會...
〜てしまった    (過去) - ...完了 / 已經...了
〜てしまっている (状態) - 已經...的狀態持續中
〜てしまいたい  (希望) - 想要...完
```

### 規則 4：語用選擇

完了意義和遺憾意義的區別主要通過上下文判斷。

**判斷原則**
- 正面或中性結果 → 完了的強調
- 負面或不期望的結果 → 遺憾・後悔
- 說話者的語氣和情境決定解讀

## 常見錯誤

### 錯誤 1：直接接辞書形

❌ 誤：食べるしまった。
✅ 正：食べてしまった。
說明：「しまう」必須接在て形後面，不能接辞書形。正確的形成順序是：食べる → 食べて → 食べてしまう。

### 錯誤 2：混淆縮約形的規則

❌ 誤：読んでちゃった。
✅ 正：読んじゃった。
說明：「でしまう」應該縮約為「じゃう」，不是「ちゃう」。て形以「で」結尾的動詞縮約為「じゃ」，以「て」結尾的縮約為「ちゃ」。

### 錯誤 3：過度使用於正面事件

❌ 不自然：合格してしまった！(おめでとう的場面)
✅ 自然：合格した！/ 合格できた！
說明：在慶祝正面成果時使用「〜てしまう」會帶來遺憾的語感，不適當。除非是「總算完成了」的釋然感，否則應避免用於純粹的好消息。

## 學習要點

1. **雙重意義**：「〜てしまう」既可表示完了，也可表示遺憾，具體意義取決於上下文和說話者的情感態度。

2. **口語縮約**：日常會話中「〜ちゃう/〜じゃう」比完整形式更常見，掌握縮約規則對自然表達至關重要。

3. **情感色彩**：用於負面事件時，這個形式強烈表達說話者的懊惱、後悔或無奈，是情感表達的重要工具。

4. **時制組合**：與不同時制結合可產生豐富的意義變化，特別是「〜てしまった」表示無法挽回的既成事實。

5. **語域選擇**：書面語傾向使用完整形式「〜てしまう」，口語則大量使用縮約形「〜ちゃう/〜じゃう」，根據場合選擇適當形式。
