# て形

## Japanese
動詞のて形は、日本語の基本的な活用形の一つで、複数の文法機能を持つ。動詞を連結したり、補助動詞と組み合わせたり、依頼や許可の表現に使われる。

## Traditional Chinese
て形是日文動詞的基本變化形式之一，具有多種文法功能。它用於連接動詞、與補助動詞結合，以及表達請求和許可等意思。

## English
The te-form is one of the basic conjugation patterns of Japanese verbs, serving multiple grammatical functions. It is used to connect verbs, combine with auxiliary verbs, and express requests and permissions.

---

## 活用規則（Conjugation Rules）

### 1. Group I (五段動詞/U-verbs)

| 語尾 | 變化 | 例 |
|------|------|-----|
| く | いて | 書く → 書いて |
| ぐ | いで | 泳ぐ → 泳いで |
| す | して | 話す → 話して |
| つ/る/う | って | 待つ → 待って、帰る → 帰って、買う → 買って |
| ぬ/ぶ/む | んで | 死ぬ → 死んで、遊ぶ → 遊んで、読む → 読んで |

**例外**：
- 行く → 行って（いって）

### 2. Group II (一段動詞/Ru-verbs)

| 規則 | 例 |
|------|-----|
| る → て | 食べる → 食べて、見る → 見て |

### 3. Group III (不規則動詞/Irregular verbs)

| 動詞 | て形 |
|------|------|
| する | して |
| 来る（くる） | 来て（きて） |

---

## 主要用途（Main Uses）

### 1. 動作の連結（Connecting actions）

連續動作を順番に述べる。

```
朝起きて、顔を洗って、朝ごはんを食べます。
I wake up in the morning, wash my face, and eat breakfast.
我早上起床、洗臉、吃早餐。
```

### 2. 補助動詞との組み合わせ（With auxiliary verbs）

- **～ている**（進行・状態）: 今勉強している（I'm studying now）
- **～てある**（結果状態）: ドアが開けてある（The door has been opened）
- **～ておく**（準備・事前）: 準備しておく（Prepare in advance）
- **～てみる**（試行）: 食べてみる（Try eating）
- **～てしまう**（完了・後悔）: 忘れてしまった（I forgot it）

### 3. 依頼・命令（Requests and commands）

- **～てください**（丁寧な依頼）: 教えてください（Please teach me）
- **～て**（カジュアルな依頼）: ちょっと待って（Wait a moment）
- **～てくれる/くださる**（恩恵）: 助けてくれた（He/She helped me）

### 4. 理由・原因（Reason/Cause）

```
雨が降って、試合が中止になった。
It rained, and the game was canceled.
因為下雨，比賽取消了。
```

### 5. 許可・禁止（Permission/Prohibition）

- **～てもいい**（許可）: 入ってもいいですか（May I come in?）
- **～てはいけない**（禁止）: ここで食べてはいけない（You must not eat here）

---

## 例文（Example Sentences）

### 基本例文

1. **連結**
   ```
   昨日友達に会って、映画を見て、食事をしました。
   Yesterday I met my friend, watched a movie, and had dinner.
   昨天我見了朋友、看了電影、吃了飯。
   ```

2. **進行形**
   ```
   今何をしていますか。
   What are you doing now?
   你現在在做什麼？
   ```

3. **依頼**
   ```
   窓を開けてください。
   Please open the window.
   請打開窗戶。
   ```

4. **試行**
   ```
   この料理を食べてみてください。
   Please try eating this dish.
   請試試看這道菜。
   ```

5. **完了**
   ```
   宿題を全部やってしまった。
   I finished all my homework.
   我把作業全部做完了。
   ```

## 注意點（Notes）

1. **音便規則**：五段動詞のて形は音便（音の変化）を伴う
2. **意味の多様性**：文脈によって、連結・理由・手段など複数の意味を表す
3. **話し言葉での省略**：カジュアルな会話では「～てる」のように短縮される
4. **方言差**：地域によってて形の発音や使い方に違いがある場合がある
