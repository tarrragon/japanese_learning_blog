## 日文

副詞の否定用法

**羅馬拼音**: fukushi no hitei youhou

**概要**: 副詞を使って否定の意味を表す各種表現パターン

## 日文解釋

副詞の否定用法は、副詞が否定的な意味を持つ日本語の表現方法である。主な否定副詞には「全然（ぜんぜん）」「全く（まったく）」「あまり」「ちっとも」「さっぱり」「少しも（すこしも）」などがある。

これらの副詞は通常、否定形の述語と共に使用され、「全く～ない」「あまり～ない」のように、否定の程度や質を強調する。副詞の選択によって、否定の強度や微妙なニュアンスが変わる。

**主要な否定副詞の特徴**:
- 「全然」：強い否定を表し、やや口語的
- 「全く」：強い否定を表し、書き言葉や丁寧な表現に適している
- 「あまり」：部分的な否定を表す（例：「あまり好きではない」は「完全には好きではない」という意味）
- 「ちっとも」：強い否定、「全然」と類似
- 「さっぱり」：全く分からない、全くない意味の強い否定
- 「少しも」：わずかでもない、という意味の強い否定

## 英文解釋

Adverbial negation in Japanese refers to using adverbs to express negative meanings. Common negation adverbs include "zenzen" (completely), "mattaku" (absolutely), "amari" (not much), "chitto mo" (not at all), "sappari" (completely/not at all), and "sukoshi mo" (not even a little).

These adverbs typically function with negative predicate forms, creating expressions like "zenzen~nai" (not at all~) and "amari~nai" (not much~). The choice of negation adverb affects the strength and nuance of the negation.

**Key characteristics of negation adverbs**:
- "Zenzen": Expresses strong negation, somewhat colloquial
- "Mattaku": Expresses strong negation, suitable for written or polite speech
- "Amari": Expresses partial negation (e.g., "amari suki de wa nai" means "not entirely likable")
- "Chitto mo": Strong negation, similar to "zenzen"
- "Sappari": Complete negation, especially regarding understanding or existence
- "Sukoshi mo": Not even a little, strong negation

## 中文解釋

副詞的否定用法是指用副詞來表達日文中的否定意義。主要的否定副詞包括「全然」「全く」「あまり」「ちっとも」「さっぱり」「少しも」等。

這些副詞通常與否定形式的述語一起使用，形成「全く～ない」「あまり～ない」這樣的表達方式。副詞的選擇會影響否定的強度和微妙的語氣。

**主要否定副詞的特點**：
- 「全然」：表示強烈的否定，略帶口語風格
- 「全く」：表示強烈的否定，適合書面語或正式表達
- 「あまり」：表示部分否定（例如：「あまり好きではない」意為「不完全地喜歡」）
- 「ちっとも」：強烈否定，與「全然」類似
- 「さっぱり」：完全否定，尤其是不明白或沒有的含義
- 「少しも」：連一點都不，強烈的否定

## 例句

**例句1 - 全然（全く）**
```
このプロジェクトは全然進んでいない。
This project hasn't progressed at all.
這個項目完全沒有進展。
```

**例句2 - 全く**
```
彼は全く返事をしてくれません。
He doesn't respond at all.
他完全不回應。
```

**例句3 - あまり（部分否定）**
```
最近、あまり映画を見ていません。
I haven't been watching movies much recently.
最近我不太看電影。
```

**例句4 - ちっとも**
```
ちっとも分かりません。
I don't understand at all.
我根本不明白。
```

**例句5 - さっぱり**
```
この説明はさっぱり分かりません。
I don't understand this explanation at all.
我完全不明白這個解釋。
```

**例句6 - 少しも**
```
少しも間違えていません。
There's not even a little mistake.
一點都沒有出錯。
```

**例句7 - 複合使用**
```
全然予想していなかった出来事が起こった。
A completely unexpected event occurred.
發生了一件我完全沒有預想的事件。
```

**例句8 - 文語的使用**
```
その計画は全く実現する可能性がない。
That plan has absolutely no possibility of being realized.
那個計劃完全沒有實現的可能性。
```

## 對比與關聯

**否定副詞的強度比較**:
- 最強：「全然」「全く」「ちっとも」「さっぱり」「少しも」
- 中等：「あまり」（表示部分否定而非完全否定）

**與否定形式的關係**:
- 否定副詞は通常、[否定形](../grammar/047_nai_kei.md)の述語と共に使用される
- 「全然＋ない」「あまり＋ない」のような組み合わせが標準的

**「全然」と「全く」の違い**:
- 「全然」：より日常的、やや口語的なニュアンス
- 「全く」：より書き言葉的、正式な印象

**「あまり」の特殊性**:
- [あまり](../grammar/amari.md)は部分否定を表す（完全否定ではない）
- 「あまり好きではない」= 「完全には好きではないが、嫌いというわけではない」

**関連する表現**:
- [ない形](047_nai_kei.md)：基本的な否定形
- [ても](../grammar/temo_ii.md)：譲歩表現
- [～ほうがいい](107_hou_ga_ii.md)：推奨表現

## 文法筆記

**使用パターン**:
```
否定副詞 + V（ない形）
否定副詞 + Adj（否定形）
否定副詞 + N（ではない）
```

**例**:
- 全く + 理解できない = 全く理解できない（completely cannot understand）
- あまり + 好きではない = あまり好きではない（not much like）
- さっぱり + 分からない = さっぱり分からない（don't understand at all）

**強度レベル**:
```
完全否定: 全然、全く、ちっとも、さっぱり、少しも
↓
部分否定: あまり、それほど、大して
```

**文体による使い分け**:
- 書き言葉・正式：「全く」「全然」
- 口語・日常：「全然」「ちっとも」「さっぱり」
- 文学的：「少しも」
