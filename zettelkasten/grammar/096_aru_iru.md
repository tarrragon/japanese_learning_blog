## 日文

**ある/いる**（存在表達）

### 文法情報

| 項目 | 內容 |
|------|------|
| 文法類型 | 存在動詞句型 |
| JLPT | N5 |
| 使用場面 | 日常、正式、書面皆可 |
| 形成 | 場所+に+名詞+が+ある/いる |

### 基本句型

**句型 1：〜に〜がある/いる**（在某處有某物）
```
場所 + に + 名詞 + が + ある/いる
```

**句型 2：〜は〜にある/いる**（某物在某處）
```
名詞 + は + 場所 + に + ある/いる
```

**句型 3：〜は〜がある/いる**（某人/某處有某物）
```
主體 + は + 名詞 + が + ある/いる
```

## 日文解釋

日本語の存在表現は「ある」と「いる」という二つの動詞を中心に構成される。これらは英語の "there is/are" や "to exist" に相当するが、日本語では対象の**生命性（animacy）**[^animacy]によって使い分けが必須となる。

**基本原則**：
- **いる**：有生物（人間、動物、昆虫など、自力で移動できる生き物）
- **ある**：無生物（物体、植物、抽象概念など、自力で移動できないもの）

この区別は日本語の最も基本的な文法規則の一つであり、N5レベルで完全に習得すべき項目である。存在文は「〜に〜がある/いる」という形で場所と存在物を結びつけ、「何がどこにあるか」を表現する。助詞「に」[^particle-ni]は場所を示し、「が」[^particle-ga]は存在するものを標示する。

**文法的特徴**：
1. 存在文では主語に「が」を使うのが基本
2. 場所には助詞「に」を使う（「で」ではない）
3. 所有を表す時は「〜は〜がある/いる」の形を使う
4. 否定形は「ない」「いない」となる

## 英文解釋

Japanese existence expressions are built around two verbs: **ある (aru)** and **いる (iru)**. While both can be translated as "to be" or "to exist" in English, Japanese requires a strict distinction based on **animacy**[^animacy] - whether the subject is capable of independent movement.

**Core Distinction**:
- **いる (iru)**: For animate beings (people, animals, insects - living things that can move)
- **ある (aru)**: For inanimate objects (things, plants, abstract concepts - things that cannot move)

This distinction is one of the most fundamental grammar rules in Japanese and must be mastered at the N5 level. The basic existence sentence pattern "〜に〜がある/いる" connects a location with what exists there, expressing "what is where." The particle に[^particle-ni] marks the location, while が[^particle-ga] marks the thing that exists.

**Key Grammar Points**:
1. Existence sentences typically use が for the subject
2. Locations are marked with に (not で)
3. Possession is expressed with "〜は〜がある/いる"
4. Negative forms: ない (for ある), いない (for いる)

## 中文解釋

日文的存在表達建立在兩個動詞上：**ある（aru）**和**いる（iru）**。雖然兩者都可以翻譯為「有」或「在」，但日文必須根據**生命性（animacy）**[^animacy]嚴格區分使用 - 即主語是否能夠自主移動。

**核心區別**：
- **いる（iru）**：用於有生命體（人、動物、昆蟲 - 能自主移動的生物）
- **ある（aru）**：用於無生命物（物品、植物、抽象概念 - 不能自主移動的事物）

這個區別是日文最基本的文法規則之一，必須在 N5 階段完全掌握。基本存在句型「〜に〜がある/いる」將地點與存在物連接起來，表達「什麼在哪裡」。助詞「に」[^particle-ni]標示地點，「が」[^particle-ga]標示存在的事物。

**重要文法點**：
1. 存在句通常使用「が」標示主語
2. 地點使用助詞「に」標示（不是「で」）
3. 表示所有時使用「〜は〜がある/いる」
4. 否定形：ない（ある的否定）、いない（いる的否定）

## 核心用法

### 用法 1：表示某處有某物（〜に〜がある/いる）

這是最基本的存在句型，用來陳述某個地點存在某物或某人。

**例句 1**
```
テーブルの上に本があります。
There is a book on the table.
桌上有一本書。
```

**例句 2**
```
公園に子供がたくさんいます。
There are many children in the park.
公園裡有很多小孩。
```

### 用法 2：表示某物在某處（〜は〜にある/いる）

強調特定事物的位置，回答「在哪裡」的問題。

**例句 3**
```
田中さんは会議室にいます。
Tanaka-san is in the conference room.
田中先生在會議室。
```

**例句 4**
```
駅は北口にあります。
The station is at the north exit.
車站在北出口。
```

### 用法 3：表示擁有（〜は〜がある/いる）

表示某人擁有或具備某物、某種能力或特徵。

**例句 5**
```
私は時間がありません。
I don't have time.
我沒有時間。
```

**例句 6**
```
この会社は優秀な社員がたくさんいます。
This company has many excellent employees.
這家公司有很多優秀的員工。
```

## 文法規則

### 規則 1：生命性區分（Animacy Distinction）

**核心原則**：對象能否自主移動

| 分類 | 使用動詞 | 範例 |
|------|---------|------|
| 人類 | いる | 人、子供、先生 |
| 動物 | いる | 犬、猫、鳥、魚 |
| 昆蟲 | いる | 蝶、蜂、蟻 |
| 植物 | ある | 木、花、草 |
| 物品 | ある | 本、机、車 |
| 抽象概念 | ある | 時間、問題、理由 |

**判斷方法**：「它能自己走動嗎？」
- 能 → いる
- 不能 → ある

### 規則 2：助詞的使用

**に（場所標示）**
```
場所 + に + ある/いる
例：家に猫がいる（家裡有貓）
```

**が（主語標示）**
```
名詞 + が + ある/いる
例：本がある（有書）
```

**は（主題標示）**
```
名詞 + は + 場所 + に + ある/いる
例：猫は家にいる（貓在家裡）
```

**重要**：存在句的地點用「に」，不用「で」
- ✅ 机の上に本がある
- ❌ 机の上で本がある

### 規則 3：否定形

**ある的否定**：
- 現在否定：ない / ありません
- 過去否定：なかった / ありませんでした

**いる的否定**：
- 現在否定：いない / いません
- 過去否定：いなかった / いませんでした

**例句**：
```
時間がない。（沒有時間）
誰もいない。（沒有人）
```

## 常見錯誤

### 錯誤 1：動物使用「ある」

❌ 誤：公園に犬があります。
✅ 正：公園に犬がいます。
說明：狗是有生命的動物，能自主移動，必須使用「いる」。這是最常見的錯誤，學習者常將「ある」用於所有「存在」的情況。

### 錯誤 2：植物使用「いる」

❌ 誤：庭に木がいます。
✅ 正：庭に木があります。
說明：植物雖然是生物，但不能自主移動，因此使用「ある」。這是生命性區分的重要特例。

### 錯誤 3：地點助詞使用「で」

❌ 誤：机の上で本があります。
✅ 正：机の上に本があります。
說明：存在句的地點必須用「に」標示，「で」用於動作發生的地點（如「図書館で勉強する」）。這是助詞使用的重要區別。

### 錯誤 4：主語助詞誤用「は」

❌ 誤：机の上は本があります。（不自然）
✅ 正：机の上に本があります。
說明：在存在句中，存在物通常用「が」標示。「は」用於對比或強調時（如「机の上には本があります」表示「桌上（有別於其他地方）有書」）。

## 學習要點

1. **掌握生命性區分**：人和動物用「いる」，其他用「ある」是最基本的原則，必須牢記。

2. **理解句型差異**：「〜に〜がある/いる」和「〜は〜にある/いる」表達的焦點不同，前者強調「有什麼」，後者強調「在哪裡」。

3. **助詞正確使用**：存在句用「に」標示地點，用「が」標示存在物，這是固定搭配，不可混淆。

4. **植物的特殊性**：雖然植物在生物學上是活的，但在日文文法中視為無生命物，使用「ある」。

5. **否定形的不規則**：「ある」的否定形是「ない」（不是「あらない」），這是不規則變化，需特別記憶。

## 註解

[^animacy]: **生命性（Animacy）** - 語言學中用來區分有生命和無生命事物的概念。在日文中，生命性主要根據「是否能自主移動」來判斷，而非生物學定義。詳見 [生命性概念](../concept/022_animacy.md)

[^particle-ni]: **助詞「に」** - 在存在句中標示地點或位置。與「で」不同，「に」表示靜態的存在位置，而「で」表示動作發生的場所。詳見 [に](../particle/008_ni.md)

[^particle-ga]: **助詞「が」** - 在存在句中標示存在的主語。存在句通常用「が」而非「は」，因為存在句是陳述「有什麼」的新資訊。詳見 [が](../particle/001_ga.md)
