## 日文

〜後で（あとで）

### 文法情報

| 項目 | 內容 |
|------|------|
| 文法類型 | 時間表達 |
| JLPT | N5 |
| 使用場面 | 日常會話、書面語 |
| 形成 | 動詞辞書形 / 名詞 + の + 後で |

### 形成規則

**基本形式**
```
動詞辞書形 + 後で
名詞 + の + 後で
```

**範例**
```
食べる → 食べた後で
会議 → 会議の後で
```

## 日文解釋

「〜後で」は、ある動作や出来事が終わってから、別の動作を行うことを表す時間表現[^time-expression]です。動詞の辞書形[^dictionary-form]や名詞に接続して使用します。

この表現は日常会話[^casual-context]でも書面語[^formal-writing]でも広く使われ、順序立てて行動を説明する際に便利です。「〜てから」[^te-kara]と似ていますが、「後で」の方がやや時間的な間隔[^time-gap]があることを暗示する場合があります。

「〜後」単独では「そのあと」という意味で使われることもありますが、「〜後で」の形で使うことで、明確に「その後に何かをする」という意図を表現できます。また、過去形「〜た後で」を使うことで、前の動作が完全に終了したことを強調[^completion-emphasis]することができます。

## 英文解釋

"〜後で" (ato de) is a time expression[^time-expression] that indicates performing one action after another action or event has been completed. It connects to the dictionary form[^dictionary-form] of verbs or to nouns (with の).

This expression is widely used in both casual conversation[^casual-context] and formal writing[^formal-writing], making it convenient for explaining actions in sequence. While similar to "〜てから"[^te-kara], "後で" often implies a slightly longer time gap[^time-gap] between the two actions.

When used alone as "〜後," it can mean "afterwards" or "later," but using the full form "〜後で" clearly expresses the intention of "doing something after that." Using the past tense form "〜た後で" emphasizes[^completion-emphasis] that the previous action has been completely finished.

## 中文解釋

「〜後で」是一種時間表達[^time-expression]，表示在某個動作或事件完成之後進行另一個動作。它接續於動詞的辞書形[^dictionary-form]或名詞（加の）之後。

這個表達在日常會話[^casual-context]和正式書面語[^formal-writing]中都廣泛使用，適合用來順序說明行動。雖然與「〜てから」[^te-kara]相似，但「後で」通常暗示兩個動作之間有稍長的時間間隔[^time-gap]。

單獨使用「〜後」時可表示「之後」的意思，但使用完整形式「〜後で」能明確表達「在那之後做某事」的意圖。使用過去式「〜た後で」可以強調[^completion-emphasis]前一個動作已完全結束。

## 核心用法

### 用法 1：動詞辞書形 + 後で

表示在某動作之後進行另一動作。

**例句 1**
```
宿題をした後で、テレビを見ます。
I will watch TV after doing my homework.
做完作業後會看電視。
```

**例句 2**
```
昼ごはんを食べた後で、散歩しましょう。
Let's take a walk after eating lunch.
吃完午飯後去散步吧。
```

### 用法 2：名詞 + の + 後で

表示在某事件或時間點之後。

**例句 3**
```
会議の後で、資料を確認してください。
Please check the documents after the meeting.
會議結束後請確認資料。
```

**例句 4**
```
仕事の後で、一緒に飲みに行きませんか。
Would you like to go for a drink after work?
下班後要不要一起去喝一杯？
```

### 用法 3：表示將來的計劃

用於說明之後要做的事情。

**例句 5**
```
この映画を見た後で、感想を教えてください。
Please tell me your thoughts after watching this movie.
看完這部電影後請告訴我你的感想。
```

## 文法規則

### 規則 1：動詞形態的選擇

- 辞書形 + 後で：強調動作的一般順序
- た形 + 後で：強調前一動作的完成

```
食べる後で → 較不自然
食べた後で → 自然、常用
```

### 規則 2：與「〜てから」的區別

- 「〜後で」：時間間隔可長可短，較中性
- 「〜てから」：強調緊接著的順序，時間間隔較短

```
ご飯を食べた後で、勉強します。（可能間隔一段時間）
ご飯を食べてから、勉強します。（吃完馬上學習）
```

### 規則 3：省略「で」的情況

在某些情況下，「で」可以省略，但意思稍有不同：

```
後で電話します。（之後會打電話）
後、電話します。（之後，打電話）- 較生硬
```

## 常見錯誤

### 錯誤 1：使用ます形

❌ 誤：食べます後で、勉強します。
✅ 正：食べた後で、勉強します。
說明：「後で」前面應使用辞書形或た形，不能使用ます形。

### 錯誤 2：名詞後忘記加「の」

❌ 誤：会議後で、帰ります。
✅ 正：会議の後で、帰ります。
說明：名詞後面必須加「の」才能接續「後で」。

### 錯誤 3：混淆時間點

❌ 誤：3時後で、会いましょう。
✅ 正：3時の後で、会いましょう。/ 3時に会いましょう。
說明：具體時間點後要加「の」，或直接使用「に」表示時間。

## 學習要點

1. **基本結構**：動詞辞書形/た形 + 後で，名詞 + の + 後で
2. **時間順序**：明確表示兩個動作的先後關係
3. **靈活運用**：可用於日常會話和正式場合
4. **與てから比較**：「後で」時間間隔較彈性
5. **過去形強調**：使用「た後で」強調前一動作已完成
