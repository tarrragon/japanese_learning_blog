サ変動詞是日文動詞的特殊類型，由名詞加「する」構成，具有獨特的活用形式。

## 日文

サ変動詞（サへんどうし）

### 文法情報

| 項目 | 內容 |
|------|------|
| 文法類型 | 動詞活用分類 |
| JLPT | N3 |
| 使用場面 | 書面、口語 |
| 形成 | 名詞 + する |

### 形成規則

**基本形式**
```
漢語名詞 + する → サ変動詞
外來語名詞 + する → サ変動詞
和語名詞 + する → サ変動詞（部分）
```

**範例**
```
勉強 + する → 勉強する
電話 + する → 電話する
コピー + する → コピーする
```

## 日文解釋

「サ変動詞」とは、「サ行変格活用動詞」の略称で、日本語の動詞活用の一種である。名詞に「する」を付けて動詞化したもので、「勉強する」「運動する」「電話する」などが代表的な例である。

活用形は不規則であり、五段活用や一段活用とは異なるパターンを持つ。特に「し」「せ」「さ」「すれ」「しろ・せよ」という活用語尾が特徴的である。漢語由来の名詞と結びつくことが多いが、外来語（「コピーする」「キャンセルする」）や一部の和語（「電話する」）とも結合する。

日本語学習において、サ変動詞は非常に生産的なパターンであり、新しい語彙を動詞化する際によく用いられる。

## 英文解釋

"Sahen doushi" (サ変動詞) refers to suru-verbs or verbal nouns in Japanese grammar. These are irregular verbs formed by combining a noun with the verb "suru" (to do). They constitute one of the most productive verb-forming patterns in Japanese.

The conjugation pattern of sahen verbs is irregular and differs from both godan (five-step) and ichidan (one-step) verb conjugations. They are particularly common with Sino-Japanese compounds (kango) but also appear with loanwords and some native Japanese words.

## 中文解釋

「サ変動詞」是「サ行變格活用動詞」的簡稱，是日文動詞中的一個特殊類別。這類動詞由名詞加上「する」構成，形成複合動詞。

サ變動詞的活用形式不規則，與五段動詞和一段動詞的活用模式不同。這類動詞在日文中數量龐大，尤其是漢語詞彙轉化為動詞時，大多採用這種形式。外來語和部分和語也可以透過加「する」變成動詞。

對於學習者來說，掌握サ變動詞的活用規則是理解日文動詞系統的重要環節。

## 核心用法

### 用法 1：漢語名詞 + する

最常見的サ變動詞形式，將漢語詞彙動詞化。

**例句 1**
```
毎日三時間勉強しています。
I study for three hours every day.
我每天學習三個小時。
```

**例句 2**
```
来週、友達と旅行する予定です。
I plan to travel with friends next week.
我計畫下週和朋友去旅行。
```

### 用法 2：外來語 + する

將外來語動詞化的形式。

**例句 3**
```
この書類をコピーしてください。
Please copy this document.
請影印這份文件。
```

### 用法 3：サ變動詞的活用形

展示サ變動詞的各種活用形式。

**例句 4**
```
予約をキャンセルしなければなりません。
I have to cancel the reservation.
我必須取消預約。
```

**例句 5**
```
もっと運動すれば、健康になれますよ。
If you exercise more, you can become healthy.
如果多運動的話，可以變得更健康喔。
```

## 文法規則

### 規則 1：活用表

| 活用形 | サ変動詞（する） | 例（勉強する） |
|--------|-----------------|---------------|
| 未然形 | し・せ・さ | 勉強し・せ・さ |
| 連用形 | し | 勉強し |
| 終止形 | する | 勉強する |
| 連体形 | する | 勉強する |
| 仮定形 | すれ | 勉強すれ |
| 命令形 | しろ・せよ | 勉強しろ・せよ |

### 規則 2：否定形的形成

```
する → しない
勉強する → 勉強しない
```

### 規則 3：敬體形的形成

```
する → します
勉強する → 勉強します
```

### 規則 4：可能形的形成

```
する → できる（或 せる/しうる）
勉強する → 勉強できる
```

注意：サ變動詞的可能形通常使用「できる」而非「せる」。

## 常見錯誤

### 錯誤 1：與名詞混淆

❌ 誤：勉強をします（冗餘的を）
✅ 正：勉強します

說明：サ變動詞作為一個複合動詞，名詞部分與「する」直接結合，中間不需要助詞。但若要強調動作對象，可用「勉強をする」。

### 錯誤 2：可能形的誤用

❌ 誤：日本語が話することができます
✅ 正：日本語を話すことができます 或 日本語が話せます

說明：「話する」不是サ變動詞，「話す」是五段動詞。

### 錯誤 3：活用形混淆

❌ 誤：勉強せない（錯誤的否定形）
✅ 正：勉強しない

說明：否定形使用「し」而非「せ」。

## 學習要點

1. **識別サ變動詞**：名詞 + する 的結構是サ變動詞的基本形式
2. **掌握活用規則**：サ變動詞的活用不規則，需要特別記憶
3. **注意可能形**：サ變動詞的可能形通常用「できる」表示
4. **區分名詞和動詞**：「勉強」是名詞，「勉強する」是動詞
5. **外來語動詞化**：外來語加「する」是創造新動詞的常見方式
