## 日文

〜た時（〜たとき）

**羅馬拼音**: ~ta toki

**詞性**: 時間文法結構・時間接續詞

## 日文解釋

「〜た時」は、過去のある時点で何かが起こったことを表す文法表現です。「時（とき）」は「時点」や「タイミング」を意味し、「〜た時」という形で「その動作が完了した時点で」という意味を表します。

### 文法情報

| 項目 | 內容 |
|------|------|
| 文法類型 | 時間表達 |
| JLPT | N4 |
| 使用場面 | 日常會話、書面語 |
| 形成 | 動詞た形 + 時 |

### 形成規則

**基本形式**
```
動詞た形 + 時
名詞 + の + 時
い形容詞 + 時
な形容詞 + な + 時
```

**範例**
```
行く → 行った時（去的時候）
雨 → 雨の時（下雨的時候）
暑い → 暑い時（熱的時候）
暇 → 暇な時（有空的時候）
```

### 文法特徵

「〜た時」の前には、た形を使うことで「その動作が完了した時点」を明確に示します。これは「〜時」（辞書形 + 時）とは異なり、動作の完了を強調する表現です。

**重要な違い**：
- **辞書形 + 時**：「する時」→ 習慣的・一般的な時点
- **た形 + 時**：「した時」→ 過去の特定の時点、完了した時点

主文の時制は過去・現在・未来のいずれでもよく、従属節の「〜た時」は時点を示すだけです。会話でも書き言葉でも頻繁に使用され、過去の経験や出来事を説明する際に不可欠な表現です。

## 英文解釋

"~Ta toki" is a grammar pattern that expresses "when (something happened)" or "at the time when (an action was completed)". The word "toki" (時) means "time" or "moment", and when combined with the ta-form of a verb, it indicates "at the moment when that action was completed".

### Grammar Information

| Item | Content |
|------|---------|
| Grammar Type | Time Expression |
| JLPT | N4 |
| Usage Context | Daily conversation, written language |
| Formation | Verb ta-form + toki |

### Formation Rules

**Basic Pattern**
```
Verb ta-form + toki
Noun + no + toki
I-adjective + toki
Na-adjective + na + toki
```

**Examples**
```
iku → itta toki (when I went)
ame → ame no toki (when it rains)
atsui → atsui toki (when it's hot)
hima → hima na toki (when I'm free)
```

### Grammatical Features

By using the ta-form before "~toki", you explicitly indicate "at the moment when that action was completed". This differs from "~toki" (dictionary form + toki), which emphasizes the completion of the action.

**Key Difference**:
- **Dictionary form + toki**: "suru toki" → habitual/general time point
- **Ta-form + toki**: "shita toki" → specific past moment, completed time point

The tense of the main clause can be past, present, or future, while the subordinate clause "~ta toki" simply indicates the time point. This expression is frequently used in both conversation and written language, and is essential when explaining past experiences or events.

## 中文解釋

「〜た時」是表示「...的時候」的文法結構，用來表達某個動作完成時的時間點。「時（とき）」的意思是「時間」或「時刻」，與動詞た形結合後，表示「那個動作完成的時候」。

**基本形式**
```
動詞た形[^ta-form] + 時
```

**範例**
```

```

### 文法特徵

在「〜た時」前使用た形，可以明確表示「那個動作完成的時點」。這與「〜時」（辭書形 + 時）不同，強調動作的完成性。

**重要區別**：
- **辭書形 + 時**：「する時」→ 習慣性、一般性的時間點
- **た形 + 時**：「した時」→ 過去的特定時點、完成的時點

主句的時態可以是過去、現在或未來，而從屬句的「〜た時」只是表示時間點。這個表達在對話和書面語中都很常用，是說明過去經驗或事件時不可或缺的表達方式。

## 核心用法

### 用法 1：描述過去特定時點發生的事

表達在過去某個特定時刻發生的動作或狀態。

**例句 1**
```
日本に行った時、富士山を見ました。
When I went to Japan, I saw Mt. Fuji.
去日本的時候，看到了富士山。
```

**例句 2**
```
家に帰った時[^ta-form]、雨が降っていました。
When I got home, it was raining.
回到家的時候，正在下雨。
```

### 用法 2：回憶過去的經驗或情況

用於回顧過去的經驗，描述當時的情況或感受。

**例句 3**
```
子供だった時、毎日公園で遊びました。
When I was a child, I played in the park every day.
小時候，每天都在公園玩。
```

**例句 4**
```
試験が終わった時[^ta-form]、とても嬉しかったです。
When the exam ended, I was very happy.
考試結束的時候，非常開心。
```

### 用法 3：條件性的時間表達

表達「當...發生時」的條件性時間關係，主句可以是未來的動作。

**例句 5**
```
準備ができた時、連絡してください。
When you're ready, please contact me.
準備好的時候，請聯絡我。
```

## 文法規則

### 規則 1：必須使用た形

「〜た時」前的動詞必須使用た形[^ta-form]，不能使用辭書形[^dictionary-form]。

**正確**：
- 食べた時（○）- 吃的時候
- 見た時（○）- 看的時候
- 行った時（○）- 去的時候

**錯誤**：
- 食べる時（×）- 應該用た形
- 見る時（×）- 應該用た形

### 規則 2：主句時態不受限制

主句可以使用過去式、現在式或未來式，「〜た時」只表示時間點。

**過去式主句**：
```
家に帰った時、手紙が届いていた。
When I got home, a letter had arrived.
回家的時候，信已經到了。
```

**現在式主句**：
```
困った時、いつも友達に相談します。
When I'm in trouble, I always consult my friends.
遇到困難的時候，總是跟朋友商量。
```

**未來式主句**：
```
卒業した時、旅行に行くつもりです。
When I graduate, I plan to go on a trip.
畢業的時候，打算去旅行。
```

### 規則 3：與名詞、形容詞的接續

不同詞性的接續方式各有不同。

**名詞**：名詞 + の + 時
```
学生の時（當學生的時候）
子供の時（小時候）
```

**い形容詞**：い形容詞 + 時
```
暑い時（熱的時候）
寒い時（冷的時候）
```

**な形容詞**：な形容詞 + な + 時
```
暇な時（有空的時候）
元気な時（精神好的時候）
```

## 常見錯誤

### 錯誤 1：使用辭書形而非た形

「〜た時」必須用た形，不能用辭書形。

❌ 誤：日本に行く時、富士山を見ました。
✅ 正：日本に行った時、富士山を見ました。
說明：「行く時」表示習慣性或一般時間，「行った時」表示特定過去時點。這裡描述的是過去的特定經驗，必須用た形。

### 錯誤 2：與「〜時」（辭書形 + 時）混淆

辭書形 + 時和た形 + 時有不同的語感和用法。

❌ 誤：毎朝起きた時、運動します。（語感奇怪）
✅ 正：毎朝起きる時、運動します。
說明：描述習慣性動作時，通常用辭書形 + 時。「起きた時」強調完成，「起きる時」表示習慣。

### 錯誤 3：名詞接續忘記加「の」

名詞後面必須加「の」才能接「時」。

❌ 誤：学生時、よく勉強しました。
✅ 正：学生の時、よく勉強しました。
說明：名詞和「時」之間需要用「の」連接。

## 學習要點

1. **た形表示完成性**：「〜た時」強調動作完成的時點，與「〜時」（辭書形）的一般性時間不同
2. **主句時態自由**：主句可以用任何時態，「〜た時」本身只標示時間點
3. **過去經驗的回憶**：常用於描述過去的經驗或特定時刻發生的事
4. **與條件句的差異**：「〜た時」是時間表達，「〜たら」[^tara]是條件表達，語感不同
5. **口語和書面均可**：這是日常對話和書面語都很常用的基本文法

## 註解

[^ta-form]: **た形（過去形）** - 動詞的過去式或完了形。表示動作已經完成或過去發生的事。例：食べる → 食べた、行く → 行った。在「〜た時」句型中使用，強調動作完成的時點。詳見 [た形](009_ta_form.md)

[^dictionary-form]: **辭書形（辞書形）** - 動詞的基本形式，也稱為「原形」。是在字典中查詢時使用的形式。例：食べる、行く、見る。與「た形」不同，辭書形 + 時（〜時）表示習慣性或一般性的時間點。詳見 [辭書形](010_dictionary_form.md)

[^tara]: **〜たら** - 條件表達的文法形式，意為「如果...」「要是...」。雖然也使用た形，但「〜たら」表達條件或假設，而「〜た時」表達時間點。例：雨が降ったら行かない（如果下雨就不去）vs 雨が降った時行かなかった（下雨的時候沒去）。詳見 [〜たら](030_tara.md)
