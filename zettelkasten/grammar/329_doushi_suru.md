名詞與「する」結合形成サ變動詞的文法規則。

## 日文

動詞する（どうしする）

### 文法情報

| 項目 | 內容 |
|------|------|
| 文法類型 | 動詞變化 |
| JLPT | N3 |
| 使用場面 | 日常會話、正式場合 |
| 形成 | 動作性名詞 + する |

### 形成規則

**基本形式**
```
動作性名詞 + する → サ變動詞
```

**範例**
```
勉強 + する → 勉強する（べんきょうする）
運動 + する → 運動する（うんどうする）
結婚 + する → 結婚する（けっこんする）
```

## 日文解釋

「動詞する」とは、「（動作性の名詞を受けて）ある動作・行為を行う」という意味です。[^dict]

サ行変格活用動詞「する」は、日本語において最も重要な動詞の一つであり、動作性名詞と組み合わせることで、新しい動詞を作り出す機能を持っています。例えば、「勉強」「運動」「結婚」などの名詞に「する」を付けることで、「勉強する」「運動する」「結婚する」という動詞になります。

この文法は、外来語やカタカナ語を動詞化する際にも頻繁に使用されます。「ドライブする」「コピーする」「メールする」など、日常会話で非常に多く使われています。

[^dict]: 定義參考 スーパー大辞林（macOS 內建字典）

## 英文解釋

The verb "suru" (する) is one of the most fundamental verbs in Japanese, known as a "suru verb" or "irregular verb." When combined with action nouns (動作性名詞), it creates compound verbs that express various actions and activities.

This grammatical structure is extremely productive in Japanese, meaning it can be used to create new verbs from:
- Native Japanese nouns (勉強する, 運動する)
- Sino-Japanese compounds (結婚する, 卒業する)
- Loanwords from foreign languages (ドライブする, コピーする)

The conjugation follows the irregular サ行変格活用 pattern, which must be memorized as it differs from the regular u-verb and ru-verb patterns.

## 中文解釋

「動詞する」是日語中最重要的動詞之一，屬於サ行變格動詞（不規則動詞）。它的核心功能是與動作性名詞結合，形成複合動詞。

這個文法結構在日語中極為常用，可以將各種名詞轉化為動詞：
- 和語名詞：勉強する（學習）、運動する（運動）
- 漢語名詞：結婚する（結婚）、卒業する（畢業）
- 外來語：ドライブする（開車兜風）、コピーする（影印）

由於「する」是不規則動詞，其活用變化需要特別記憶，不同於一般的五段動詞或一段動詞。

## 核心用法

### 用法 1：動作性名詞 + する

將動作性名詞轉化為動詞，表示進行該動作。

**例句 1**
```
毎日三時間勉強しています。
I study for three hours every day.
我每天學習三個小時。
```

**例句 2**
```
来週友達と旅行するつもりです。
I plan to travel with my friend next week.
我打算下週和朋友去旅行。
```

### 用法 2：外來語 + する

將外來語名詞轉化為動詞。

**例句 3**
```
週末はよくドライブします。
I often go for drives on weekends.
週末我經常開車兜風。
```

**例句 4**
```
この書類をコピーしてください。
Please copy this document.
請影印這份文件。
```

### 用法 3：を + する

某些情況下，「を」與「する」搭配使用，強調動作的對象。

**例句 5**
```
彼女は医者をしています。
She works as a doctor.
她從事醫生工作。
```

## 文法規則

### 規則 1：名詞的選擇

並非所有名詞都可以與「する」結合。只有「動作性名詞」才能形成サ變動詞。

**可結合的名詞**：
- 表示動作的名詞：勉強、運動、仕事、研究
- 表示狀態變化的名詞：結婚、卒業、成功、失敗
- 外來語動作詞：ドライブ、コピー、メール、チェック

**不可結合的名詞**：
- 具體事物：本、机、車（不能說「本する」）
- 抽象概念（非動作性）：愛、夢、希望（一般不說「愛する」以外）

### 規則 2：「を」的省略

在「名詞＋する」結構中，助詞「を」通常省略：
- ✅ 勉強する（自然）
- △ 勉強をする（也正確，但較正式）

### 規則 3：可能形的特殊變化

サ變動詞的可能形不是「しられる」，而是使用「できる」：
- ✅ 勉強できる
- ❌ 勉強しられる

## 常見錯誤

### 錯誤 1：不當省略「する」

❌ 誤：私は毎日勉強です。
✅ 正：私は毎日勉強します。
說明：「勉強」單獨不能作為動詞使用，必須加上「する」。

### 錯誤 2：可能形使用錯誤

❌ 誤：日本語が勉強しられます。
✅ 正：日本語が勉強できます。
說明：サ變動詞的可能形使用「できる」，不使用「しられる」。

### 錯誤 3：與不適當的名詞結合

❌ 誤：本をする。
✅ 正：本を読む。
說明：「本」是具體事物，不是動作性名詞，不能與「する」結合。

## 學習要點

1. **サ變動詞是不規則變化**：活用形需要特別記憶，尤其是「し・せ・さ」的變化。
2. **可能形使用「できる」**：這是初學者常犯的錯誤，務必記住。
3. **動作性名詞的判斷**：只有表示動作或狀態變化的名詞才能與「する」結合。
4. **外來語動詞化**：現代日語中，外來語 + する 是非常常見的構詞方式。
5. **「を」的使用**：「を＋する」結構可以強調動作對象，但日常會話中常省略。
