## 日文
〜ために（〜ために）

**羅馬拼音**: ~tame ni

**詞性**：文法助詞・複合助詞

## 日文解釋

「〜ために」は、目的と理由の両方を表す重要な文法表現です。目的を示す場合は「〜するために」の形で「〜するために何かをする」という意味になり、理由を示す場合は「〜のために」の形で「〜が原因で何かが起こる」という意味になります。

### 接続方法

**1. 目的用法：動詞辞書形 + ために**
- 動詞辞書形[^dictionary-form] + ために
- 例：合格するために（為了合格）、買うために（為了購買）
- 意味：「〜という目的で」「〜することを目標として」

**2. 目的用法：名詞 + のために**
- 名詞 + のために
- 例：健康のために（為了健康）、家族のために（為了家人）
- 意味：「〜を目的として」「〜のために」

**3. 理由用法：動詞た形・い形容詞・な形容詞・名詞 + ために**
- 動詞た形[^ta-form] + ために：雨が降ったために（因為下雨了）
- い形容詞 + ために：忙しいために（因為忙碌）
- な形容詞語幹 + なために：静かなために（因為安靜）
- 名詞 + のために：病気のために（因為生病）
- 意味：「〜が原因で」「〜という理由で」

### 文法特徴

**目的用法的特徴**：
- 主文通常是意志動詞[^volitional-verb]（可以自己控制的動作）
- 表示有意圖、計畫性的行為
- 語感比較正式[^formal-context]，書面語常用
- 主文和從屬句的主語必須相同

**理由用法的特徴**：
- 表示客觀的原因、理由
- 語感正式，多用於書面語、新聞報導[^news-context]
- 強調因果關係的必然性
- 常與負面結果一起使用

### 使用場面

目的用法常用於表達學習目標、工作目的、生活規劃等有計畫性的行為。理由用法則多見於新聞報導、正式文書、說明文等，用來解釋事件發生的原因。在日常會話中，目的表達較常用「〜ように」，理由表達則常用「〜から」或「〜ので」等較口語的形式。

## 英文解釋

"~Tame ni" is a versatile grammar pattern that expresses both purpose ("in order to") and reason ("because of"). When expressing purpose, it follows the pattern "verb (dictionary form) + tame ni" meaning "for the purpose of doing something." When expressing reason, it can follow various forms to mean "due to" or "because of."

### Connection Patterns

**1. Purpose: Verb (dictionary form) + tame ni**
- Verb dictionary form + tame ni
- Example: goukaku suru tame ni (in order to pass), kau tame ni (in order to buy)
- Meaning: "for the purpose of ~", "with the goal of ~"

**2. Purpose: Noun + no tame ni**
- Noun + no tame ni
- Example: kenkou no tame ni (for health), kazoku no tame ni (for family)
- Meaning: "for the sake of ~", "for ~"

**3. Reason: Verb (ta-form) / Adjective / Noun + tame ni**
- Verb ta-form + tame ni: ame ga futta tame ni (because it rained)
- I-adjective + tame ni: isogashii tame ni (because busy)
- Na-adjective stem + na tame ni: shizuka na tame ni (because quiet)
- Noun + no tame ni: byouki no tame ni (because of illness)
- Meaning: "because of ~", "due to ~"

### Grammatical Features

**Purpose Usage Characteristics**:
- Main clause usually contains volitional verbs (controllable actions)
- Expresses intentional, planned behavior
- Relatively formal tone, common in written language
- Subject of main and subordinate clauses must be the same

**Reason Usage Characteristics**:
- Expresses objective cause or reason
- Formal tone, frequently used in written language and news reports
- Emphasizes inevitable causal relationship
- Often used with negative outcomes

### Common Usage

Purpose usage is common when expressing learning goals, work objectives, and life plans. Reason usage is frequently seen in news reports, formal documents, and expository writing to explain causes of events. In daily conversation, "~you ni" is more commonly used for purpose, while "~kara" or "~node" are preferred for reason.

## 核心用法

### 用法 1：表示目的（動詞）

表示「為了做某事而做另一件事」的目的關係。

**例句 1**
```
日本語を勉強するために、毎日アニメを見ます。
I watch anime every day in order to study Japanese.
為了學習日文，我每天看動漫。
```

**例句 2**
```
お金を貯めるために、アルバイトを始めました。
I started a part-time job in order to save money.
為了存錢，我開始打工了。
```

### 用法 2：表示目的（名詞）

表示「為了某事物或某人」的目的。

**例句 3**
```
健康のために、毎朝ジョギングをしています。
I jog every morning for my health.
為了健康，我每天早上慢跑。
```

**例句 4**
```
家族のために、一生懸命働いています。
I work hard for my family.
為了家人，我努力工作。
```

### 用法 3：表示理由・原因

表示某事發生的客觀原因或理由，常用於正式場合或書面語。

**例句 5**
```
台風のために、電車が止まってしまいました。
The trains stopped because of the typhoon.
因為颱風，電車停駛了。
```

**例句 6**
```
事故があったために、道路が渋滞しています。
The road is congested because there was an accident.
因為發生了事故，道路塞車了。
```

## 文法規則

### 規則 1：目的用法與〜ように的區別

| 文法 | 接續 | 主文動詞類型 | 語感 | 範例 |
|------|------|------------|------|------|
| 〜ために | 辭書形 + ために | 意志動詞（可控制） | 正式 | 合格するために勉強する |
| 〜ように | 辭書形 + ように | 無意志動詞（不可控制）或可能形 | 較自然 | 合格できるように勉強する |

**重要區別**：
- 「ために」用於可控制的行為：会うために行く（為了見面而去）
- 「ように」用於希望達成的狀態：会えるように祈る（祈禱能見到）

### 規則 2：理由用法與〜から・〜ので的區別

| 文法 | 語感 | 常見場合 | 因果關係 | 範例 |
|------|------|---------|---------|------|
| 〜ために | 正式、客觀 | 書面語、新聞 | 強調必然性 | 雨のために中止された |
| 〜ので | 禮貌、客觀 | 口語、書面語 | 自然因果 | 雨なので中止しました |
| 〜から | 口語、主觀 | 日常會話 | 個人理由 | 雨だから行かない |

**使用場景**：
- 新聞報導：「〜ために」最適合（客觀、正式）
- 正式解釋：「〜ので」適合（禮貌、客觀）
- 日常對話：「〜から」最自然（口語、直接）

### 規則 3：目的用法的主語限制

目的用法「〜ために」的主句和從屬句主語必須相同：

**正確**：
```
私は合格するために、毎日勉強します。
（我為了合格，每天學習。）
→ 主語都是「私」
```

**錯誤**：
```
❌ 私は合格するために、先生が教えてくれます。
（誤：我為了合格，老師教我。）
→ 主語不同：「私」和「先生」
```

**修正**：
```
✅ 私が合格できるように、先生が教えてくれます。
（為了讓我能合格，老師教我。）
→ 使用「〜ように」可以主語不同
```

## 常見錯誤

### 錯誤 1：目的用法使用た形

❌ 誤：日本へ行ったために、日本語を勉強します。
✅ 正：日本へ行くために、日本語を勉強します。
說明：表示目的時，動詞必須用辭書形，不能用た形。た形 + ために 表示理由（因為去了日本）。

### 錯誤 2：目的用法主語不一致

❌ 誤：私が合格するために、母が応援してくれます。
✅ 正：私が合格できるように、母が応援してくれます。
說明：目的用法「〜ために」要求主句和從屬句主語相同。主語不同時應使用「〜ように」。

### 錯誤 3：與から・ので混淆場合

❌ 不自然：雨だから、試合が中止になりました。（太口語，用於正式報告）
✅ 正式：雨のために、試合が中止になりました。
✅ 自然：雨なので、試合が中止になりました。
說明：在正式場合（如新聞報導、正式文書）使用「〜ために」或「〜ので」較適當，「〜から」過於口語。

## 學習要點

1. **兩種用法**：「〜ために」有目的和理由兩種用法，需根據接續形式和語境判斷。
2. **目的用法接續**：動詞辭書形 + ために（表示為了做某事），主句和從屬句主語必須相同。
3. **理由用法接續**：動詞た形・形容詞・名詞 + ために（表示因為某事），語感正式。
4. **與ように的區別**：目的表達時，「ために」用於意志動詞，「ように」用於無意志動詞或希望達成的狀態。
5. **正式語感**：無論目的或理由用法，「〜ために」都偏向正式、書面語，日常會話較少使用。

## 註解

[^dictionary-form]: **辞書形（辭書形）** - 動詞的基本形式，也稱為「原形」或「plain form」。是在字典中查詢時使用的形式。例：行く、食べる、する。在「〜ために」的目的用法中，動詞必須使用辭書形。

[^ta-form]: **た形** - 動詞的過去式形式，用於表達過去的動作或完成的動作。例：行く → 行った、食べる → 食べた。在「〜ために」的理由用法中，使用た形表示「因為做了某事」。

[^volitional-verb]: **意志動詞** - 表示可以由主語意志控制的動作的動詞。例：勉強する（學習）、行く（去）、買う（買）。與無意志動詞（如：雨が降る、病気になる）相對。在「〜ために」的目的用法中，主句通常使用意志動詞。

[^formal-context]: **正式語境** - 指書面語、商務場合、新聞報導、學術文章等需要使用較正式、禮貌語言的情境。「〜ために」屬於正式語感的文法，相較於口語的「〜から」或「〜ので」，更常用於正式場合。

[^news-context]: **新聞報導語境** - 新聞、報紙、正式通知等媒體語言。這類語境要求客觀、正式的表達方式。「〜ために」的理由用法特別常見於新聞報導中，用來說明事件發生的原因。
