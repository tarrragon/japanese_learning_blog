## 日文

目的語（もくてきご）

**羅馬拼音**：mokutekigo

**詞性**：文法術語

### 文法情報

| 項目 | 內容 |
|------|------|
| 文法類型 | 句子成分 |
| 使用場面 | 語言學分析 |
| 主要標記 | を（助詞） |
| 英文術語 | object, direct object |

### 基本定義

目的語とは、動詞が表す動作や作用が直接及ぶ対象を示す文の成分である。日本語では通常、格助詞「を」によって標示される。

**基本構造**
```
主語 + 目的語 + 動詞
（主格）（対格）（述語）
```

**例**
```
私は　りんごを　食べる
（主語）（目的語）（動詞）
```

## 日文解釋

目的語は、動詞が表す動作・作用の対象となる文の成分です。日本語文法では「対格」とも呼ばれ、主に格助詞「を」によって標示されます。

動詞には他動詞と自動詞があり、目的語を必要とするのは主に他動詞です。他動詞文では、動作が何に対して行われるのかを目的語が明示します。例えば「本を読む」では「本」が目的語であり、「読む」という動作の対象を示しています。

日本語は「主語-目的語-動詞（SOV）」の語順を基本とする言語です。これは英語の「主語-動詞-目的語（SVO）」とは異なる特徴です。また、日本語では助詞「を」が目的語を明確に標示するため、語順が比較的自由になる場合もあります。

目的語の概念を理解することは、他動詞と自動詞の使い分け、助詞「を」の正しい使用、受身文や使役文の構造理解に不可欠です。また、二重目的語構文（授受表現など）では、直接目的語と間接目的語の区別も重要になります。

## 英文解釋

The "mokutekigo" (目的語) is the grammatical term for the object in Japanese sentences, specifically referring to the direct object that receives the action of a verb. In linguistic terminology, it is also called the "accusative case" and is typically marked by the case particle "を" (wo/o).

In Japanese grammar, objects primarily appear with transitive verbs, which require a direct object to complete their meaning. The object indicates what or whom the action of the verb affects. For example, in "本を読む" (hon wo yomu, "read a book"), "本" (hon, "book") is the object that receives the action of reading.

Japanese follows a Subject-Object-Verb (SOV) word order, which differs from English's Subject-Verb-Object (SVO) pattern. However, because the particle "を" clearly marks the object, Japanese word order can be relatively flexible without causing ambiguity. Understanding the concept of the object is essential for distinguishing between transitive and intransitive verbs, using particles correctly, and comprehending passive and causative sentence structures.

## 中文解釋

「目的語」是日文文法術語中的賓語，指的是動詞所表示動作的直接對象。在語言學術語中也稱為「對格」，通常由格助詞「を」標示。

在日文文法中，賓語主要出現在他動詞句中。他動詞需要賓語來完成其意義，賓語說明動作施加於什麼對象。例如「本を読む」（讀書）中，「本」是賓語，接受「讀」這個動作。

日文採用「主語-賓語-動詞（SOV）」的基本語序，這與英文的「主語-動詞-賓語（SVO）」不同。但由於助詞「を」能清楚標示賓語，日文的語序相對靈活，不會造成歧義。理解賓語的概念對於區分他動詞和自動詞、正確使用助詞、理解被動句和使役句結構都很重要。

## 核心用法

### 用法 1：他動詞的目的語

他動詞需要目的語來完成句意，通常用「を」標示。

**例句 1**
```
私は毎朝コーヒーを飲みます。
I drink coffee every morning.
我每天早上喝咖啡。
```

**例句 2**
```
学生たちは日本語を勉強しています。
The students are studying Japanese.
學生們正在學習日文。
```

### 用法 2：移動動詞的經過點

移動動詞（行く、通る等）可用「を」標示經過的場所。

**例句 3**
```
毎日この道を通って学校へ行きます。
I go to school through this road every day.
我每天經過這條路去學校。
```

### 用法 3：離開動詞的起點

表示離開的動詞（出る、降りる等）用「を」標示出發點。

**例句 4**
```
朝7時に家を出ます。
I leave home at 7 a.m.
我早上七點離開家。
```

### 用法 4：二重目的語構文

部分動詞（教える、聞く等）可接兩個目的語。

**例句 5**
```
先生は学生に文法を教えます。
The teacher teaches grammar to students.
老師教學生文法。
```

## 文法規則

### 規則 1：「を」格標記

目的語通常用格助詞「を」標示，但在口語中有時會省略。

**格式**：
```
名詞 + を + 他動詞
```

**例**：
- 本を読む（讀書）
- ご飯を食べる（吃飯）
- 映画を見る（看電影）

### 規則 2：他動詞與自動詞的對應

許多動詞有他動詞和自動詞的對應形式，影響是否需要目的語。

| 他動詞（需要目的語） | 自動詞（不需要目的語） |
|-----------------|-------------------|
| 開ける（打開～）| 開く（自己開） |
| 閉める（關閉～）| 閉まる（自己關） |
| 始める（開始～）| 始まる（開始了） |
| 止める（停止～）| 止まる（停了） |

**例**：
- 窓を開ける（打開窗戶）← 他動詞 + 目的語
- 窓が開く（窗戶開了）← 自動詞，無目的語

### 規則 3：語序的靈活性

由於「を」明確標示目的語，日文語序相對靈活。

**標準語序（SOV）**：
```
私は　りんごを　食べる
```

**可能變化（OSV）**：
```
りんごを　私は　食べる（強調目的語）
```

### 規則 4：受身文中的目的語轉換

主動句的目的語在受身句中會成為主語。

**主動句**：
```
先生が　学生を　褒めた
（老師稱讚學生）
```

**受身句**：
```
学生が　先生に　褒められた
（學生被老師稱讚）
```

## 常見錯誤

### 錯誤 1：他動詞與自動詞混淆

❌ 誤：ドアが閉めた。
✅ 正：ドアが閉まった。（自動詞）
✅ 正：ドアを閉めた。（他動詞 + 目的語）

說明：「閉める」是他動詞，需要用「を」標示目的語；「閉まる」是自動詞，主語用「が」標示。

### 錯誤 2：格助詞「を」和「が」混用

❌ 誤：私は本が読みます。
✅ 正：私は本を読みます。

說明：「読む」是他動詞，其目的語應用「を」標示，而非「が」。

### 錯誤 3：自動詞誤用「を」

❌ 誤：バスを乗ります。
✅ 正：バスに乗ります。

說明：「乗る」是自動詞，應使用「に」標示目的地，而非「を」。但「降りる」（下車）卻使用「を」：バスを降りる。

### 錯誤 4：二重目的語的助詞錯誤

❌ 誤：先生は学生を文法を教えます。
✅ 正：先生は学生に文法を教えます。

說明：「教える」的授受對象用「に」標示（間接目的語），被教的內容用「を」標示（直接目的語）。

## 學習要點

1. **目的語的基本定義**：目的語是動詞動作的直接對象，通常用「を」標示，主要與他動詞搭配使用。

2. **他動詞與自動詞的區別**：他動詞需要目的語（用を），自動詞不需要目的語（主語用が）。掌握常見的他自動詞對應組很重要。

3. **語序特徵**：日文是 SOV 語言，但因助詞「を」能明確標示目的語，語序可靈活調整以達成不同的語義焦點。

4. **特殊用法**：「を」不僅標示直接目的語，還可標示移動動詞的經過點（道を通る）和離開動詞的起點（家を出る）。

5. **句型轉換**：理解目的語的角色對掌握被動句、使役句、授受表現等複雜句型至關重要。
