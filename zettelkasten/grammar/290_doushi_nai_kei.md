## 日文

動詞ない形（どうしないけい）

### 文法情報

| 項目 | 內容 |

### 形成規則

**一段動詞（上一段・下一段）**

語幹＋ない

```
食べる → 食べ + ない → 食べない
見る → 見 + ない → 見ない
起きる → 起き + ない → 起きない
```

**五段動詞**

語尾をあ段に変える＋ない

```
書く → 書か + ない → 書かない（く→か）
読む → 読ま + ない → 読まない（む→ま）
話す → 話さ + ない → 話さない（す→さ）
待つ → 待た + ない → 待たない（つ→た）
死ぬ → 死な + ない → 死なない（ぬ→な）
遊ぶ → 遊ば + ない → 遊ばない（ぶ→ば）
帰る → 帰ら + ない → 帰らない（る→ら）
買う → 買わ + ない → 買わない（う→わ）※特殊
```

**不規則動詞**

```
する → しない
来る（くる）→ 来ない（こない）
ある → ない（※最も特殊：あらない ではない）
```

## 日文解釋

動詞ない形は、動詞の否定を表す最も基本的な活用形である。日本語の動詞は「辞書形」から「ない形」へと変化させることで、その動作や状態を否定することができる。

この活用形は、単なる否定だけでなく、「〜なければならない」（義務）、「〜なくてもいい」（不必要）、「〜ないでください」（禁止の依頼）など、多くの文法表現の土台となる重要な形である。

動詞の種類によって変化のルールが異なるため、まず動詞が「一段動詞」「五段動詞」「不規則動詞」のどれに属するかを正確に判別することが、正しいない形を作る第一歩となる。

特に注意すべき点として、「ある」の否定形は「あらない」ではなく「ない」となる特殊な変化がある。また、五段動詞で語尾が「う」の場合、「あ」ではなく「わ」に変化する点も重要である。

## 英文解釋

The nai-form (negative form) is the fundamental conjugation pattern for negating Japanese verbs. By transforming verbs from their dictionary form to the nai-form, speakers can express that an action or state does not occur.

This conjugation serves not only for simple negation but also as the foundation for numerous grammar patterns including "nakereba naranai" (must), "nakutemo ii" (don't have to), and "naide kudasai" (please don't).

The conjugation rules vary depending on verb type. Ichidan (ru-verbs) simply drop the final "ru" and add "nai." Godan (u-verbs) change their final syllable to the corresponding "a-row" sound before adding "nai." The irregular verbs "suru" and "kuru" have unique negative forms.

A notable exception is the verb "aru" (to exist for inanimate objects), which becomes "nai" rather than the expected "aranai" - representing the most irregular case in Japanese verb negation.

## 中文解釋

動詞ない形是日語動詞表達否定的基本活用形式。通過將動詞從辭書形（原形）變化為ない形，可以表達「不做某事」或「某狀態不存在」的意思。

這個活用形不僅用於單純的否定，還是許多重要文法結構的基礎，包括「〜なければならない」（必須）、「〜なくてもいい」（不用）、「〜ないでください」（請不要）等表達方式。

變化規則因動詞類型而異：一段動詞（る動詞）去掉詞尾「る」加「ない」；五段動詞（う動詞）將詞尾變為あ段音後加「ない」；不規則動詞「する」和「来る」則有特殊變化。

學習者需特別注意：存在動詞「ある」的否定形是「ない」而非「あらない」，這是最特殊的例外。另外，以「う」結尾的五段動詞變化時用「わ」而非「あ」。

## 核心用法

### 用法 1：基本否定

表達動作或狀態的否定，是最基礎的用法。

**例句 1**
```
私は肉を食べない。
I don't eat meat.
我不吃肉。
```

**例句 2**
```
今日は雨が降らないでしょう。
It probably won't rain today.
今天大概不會下雨吧。
```

### 用法 2：習慣性否定

表達經常性地不做某事或某種習慣。

**例句 3**
```
兄は朝ごはんを全然食べない。
My older brother doesn't eat breakfast at all.
哥哥完全不吃早餐。
```

### 用法 3：意志・決心的否定

表達決定不做某事的意志。

**例句 4**
```
もう二度とあんな失敗はしない。
I will never make such a mistake again.
絕對不會再犯那種錯誤了。
```

**例句 5**
```
お酒はもう飲まないと決めた。
I've decided not to drink alcohol anymore.
我決定不再喝酒了。
```

## 文法規則

### 規則 1：一段動詞的變化

一段動詞的變化最為簡單，直接去掉詞尾「る」加上「ない」即可。

判斷一段動詞的方法：詞尾是「る」，且「る」前面的音節屬於「い段」或「え段」。

```
食べる（たべる）→ 食べない（e段）
見る（みる）→ 見ない（i段）
寝る（ねる）→ 寝ない（e段）
着る（きる）→ 着ない（i段）
```

### 規則 2：五段動詞的變化

五段動詞需要將詞尾的う段音變為對應的あ段音，再加上「ない」。

| 詞尾 | 變化 | 例詞 |
|------|------|------|
| く | か | 書く→書かない |
| ぐ | が | 泳ぐ→泳がない |
| す | さ | 話す→話さない |
| つ | た | 待つ→待たない |
| ぬ | な | 死ぬ→死なない |
| ぶ | ば | 遊ぶ→遊ばない |
| む | ま | 読む→読まない |
| る | ら | 帰る→帰らない |
| う | わ | 買う→買わない |

### 規則 3：特殊動詞

**不規則動詞**：
- する → しない
- 来る（くる）→ 来ない（こない）

**特殊否定**：
- ある → ない（不是「あらない」）

這是日語中最重要的特殊變化，必須熟記。

## 常見錯誤

### 錯誤 1：一段動詞誤用五段規則

❌ 誤：食べらない
✅ 正：食べない
說明：「食べる」是一段動詞，直接去「る」加「ない」，不需要變化詞幹。

### 錯誤 2：五段動詞誤用一段規則

❌ 誤：帰るない
✅ 正：帰らない
說明：「帰る」是五段動詞（る前是a段音），需要將「る」變成「ら」後加「ない」。

### 錯誤 3：「ある」的特殊否定

❌ 誤：あらない
✅ 正：ない
說明：「ある」是唯一否定形完全變形的動詞，其否定形就是「ない」本身。

### 錯誤 4：「う」結尾動詞的變化

❌ 誤：買あない
✅ 正：買わない
說明：以「う」結尾的五段動詞，否定形時變為「わ」而非「あ」。

## 學習要點

1. **先掌握動詞分類**：正確判斷動詞類型是學會ない形的前提。一段動詞詞尾「る」前是い段或え段音；五段動詞則不符合此規則。

2. **熟記あ段變化**：五段動詞的核心是將詞尾變為あ段音。可以用五十音圖輔助記憶：か、が、さ、た、な、ば、ま、ら、わ。

3. **特別記憶例外**：「ある→ない」和「う→わ」是兩個最重要的例外，需要特別記憶。

4. **從常用動詞開始**：先熟練「食べる、飲む、行く、来る、する、ある」等高頻動詞的ない形。

5. **連結延伸文法**：學會ない形後，可以進一步學習「〜ないで」「〜なければ」「〜なくても」等相關文法。
