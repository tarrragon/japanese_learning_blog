## 日文
従属節（じゅうぞくせつ）

**羅馬拼音**：juuzokusetsu

**品詞分類**：文法用語（統語論）

## 日文解釋
「従属節」とは、複文において主節に依存し、単独では完結した文として成立しない節のことです。従属節は主節を時間的、条件的、理由的、または様態的に修飾します。例えば、「雨が降ったら、傘を持っていきます」という文では、「雨が降ったら」が従属節であり、「傘を持っていきます」が主節です。従属節は接続助詞（から、ので、ば、たら、ても、など）や連体修飾節として現れることが多いです。

## 英文解釋
A subordinate clause (従属節, juuzokusetsu) is a clause within a complex sentence that depends on the main clause and cannot stand alone as a complete sentence. Subordinate clauses modify the main clause by providing temporal, conditional, causal, or manner information. For example, in "雨が降ったら、傘を持っていきます" (If it rains, I will bring an umbrella), "雨が降ったら" is the subordinate clause, while "傘を持っていきます" is the main clause. Subordinate clauses often appear through conjunctive particles (から, ので, ば, たら, ても, etc.) or as relative clauses.

## 中文解釋
從屬節（じゅうぞくせつ）是在複合句中依附於主節、無法單獨構成完整句子的子句。從屬節從時間、條件、原因或方式等方面修飾主節。例如在「雨が降ったら、傘を持っていきます」（如果下雨的話，我會帶傘）這個句子中，「雨が降ったら」是從屬節，而「傘を持っていきます」是主節。從屬節常透過接續助詞（から、ので、ば、たら、ても等）或作為連體修飾節出現。

## 例句

**例句1**
```
時間があれば、映画を見に行きたいです。
If I have time, I want to go see a movie.
如果有時間的話，我想去看電影。
```

**例句2**
```
彼が来るまで、ここで待ちましょう。
Let's wait here until he comes.
在他來之前，我們在這裡等吧。
```

**例句3**
```
日本語が上手になるように、毎日練習しています。
I practice every day so that I can become good at Japanese.
為了日語變好，我每天都在練習。
```

**例句4**
```
雨が降っているので、試合は中止になりました。
Because it is raining, the game was cancelled.
因為在下雨，比賽被取消了。
```

**例句5**
```
昨日買った本はとても面白かったです。
The book I bought yesterday was very interesting.
昨天買的書非常有趣。
```

## 補充說明

### 従属節の主要類型

1. **條件節**：ば・たら・なら・と（表達假設條件）
2. **原因理由節**：から・ので・ため（表達原因理由）
3. **時間節**：とき・まで・てから・まえに（表達時間關係）
4. **讓步節**：ても・のに・けれども（表達讓步轉折）
5. **連體修飾節**：修飾名詞的節（例：昨日買った本）

### 従属節 vs. 主節的區別[^1]

| 特徵 | 従属節 | 主節 |
|------|--------|------|
| 獨立性 | ❌ 無法單獨成句 | ✅ 可以單獨成句 |
| 位置 | 通常在前 | 通常在後 |
| 功能 | 修飾、補充 | 傳達核心意義 |
| 時態制約 | 受主節影響 | 自由決定 |

### 使用注意事項

1. **語序規則**：日語中從屬節通常放在主節之前
2. **主語省略**：當從屬節和主節主語相同時，從屬節主語常省略
3. **時態呼應**：某些從屬節的時態受主節限制
4. **敬語層級**：正式場合中從屬節內也需使用適當敬語

### 常見錯誤

❌ 従屬節獨立使用（如單獨說「雨が降ったら」）
⭕ 従屬節必須與主節配合使用

❌ 條件節後接過去式主節（*「時間があれば、行きました」）
⭕ 假設條件應配合非過去式（「時間があれば、行きます」）
