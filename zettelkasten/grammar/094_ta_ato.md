## 日文

〜た後で（〜たあとで）

時間順序を表す文法パターンで、「〜した後で」という意味です。動詞のた形[^ta-form]に「後で」を付けることで、ある動作が完了してから次の動作が行われることを示します。「〜後で」の基本形[^ato-de-basic]の中でも、動詞接続の具体的な使い方を表します。

### 文法情報

| 項目 | 內容 |
|------|------|
| 文法類型 | 時間表達・接續表現 |
| JLPT | N4 |
| 使用場面 | 日常會話・書面語 |
| 形成 | 動詞た形 + 後で |

### 形成規則

**基本形式**
```
動詞た形 + 後で
```

**接続例**
```
食べる → 食べた → 食べた後で（吃完之後）
行く → 行った → 行った後で（去了之後）
する → した → した後で（做完之後）
来る → 来た → 来た後で（來了之後）
```

**重要ポイント**
- 必ず動詞のた形を使用（辞書形[^dictionary-form]は使えない）
- 「で」は省略可能だが、初学者は付けることを推奨
- 名詞接続は「名詞 + の + 後で」となる

## 日文解釋

「〜た後で」は、動作の完了と次の動作への移行を表す時間接続表現です。前の動作がすでに完了していることを明示的に示すため、必ず動詞のた形を使用します。

日本語の時間表現[^time-expression]において、「前に」と対になる重要な文法です。「前に」が未実現の動作（辞書形）を使うのに対し、「〜た後で」は完了した動作（た形）を使うという明確な対比があります。

日常会話では「で」を省略して「〜た後」と言うこともありますが、文法として正確に表現する場合は「〜た後で」の形を使います。また、「〜てから」[^te-kara]と似た意味ですが、「〜た後で」の方が時間的な間隔があることを許容し、より一般的な時間順序を表します。

話し言葉でも書き言葉でも頻繁に使用される表現で、特に日常生活の習慣、予定、手順を説明する際に欠かせない文法です。

## 英文解釋

"~Ta ato de" is a temporal expression meaning "after doing..." or "after having done...". It uses the ta-form[^ta-form] of verbs followed by "ato de" to indicate that one action occurs after another action has been completed.

This pattern explicitly shows that the first action is already completed before the second action begins. The use of ta-form is mandatory, as it grammatically marks the completion aspect. This contrasts with "~mae ni"[^mae-ni] (before), which uses the dictionary form to indicate an action that hasn't yet occurred.

While "de" can be omitted in casual speech (becoming just "~ta ato"), keeping it makes the expression more formal and grammatically complete. The pattern is similar to "~te kara"[^te-kara], but "~ta ato de" allows for more temporal distance between the two actions, making it more general in expressing sequence.

This is one of the most frequently used time expressions in both spoken and written Japanese, essential for describing daily routines, schedules, and procedural sequences.

## 中文解釋

「〜た後で」是表示「做完...之後」的時間順序表達方式。使用動詞的た形[^ta-form]加上「後で」，來表示某個動作完成後才進行下一個動作。

這個文法明確地表示前一個動作已經完成，因此必須使用動詞的た形。這與「〜前に」[^mae-ni]（之前）形成對比：「前に」使用辭書形表示尚未發生的動作，而「〜た後で」使用た形表示已完成的動作。

在口語中可以省略「で」，變成「〜た後」，但為了文法的完整性，建議初學者使用「〜た後で」的形式。與「〜てから」[^te-kara]意思相近，但「〜た後で」允許兩個動作之間有較長的時間間隔，表達較為一般性的時間順序。

這是日語中非常常用的時間表達，無論是口語還是書面語都經常使用，特別是在描述日常習慣、計畫和步驟時不可或缺。

## 核心用法

### 用法 1：日常習慣（動作完成後的下一步）

表示日常生活中一個動作完成後進行另一個動作。

**例句 1**
```
朝ごはんを食べた後で、歯を磨きます。
After eating breakfast, I brush my teeth.
吃完早餐後刷牙。
```

**例句 2**
```
宿題をした後で、ゲームをします。
After finishing homework, I play games.
做完作業後玩遊戲。
```

### 用法 2：工作・學習順序

表示工作或學習中的步驟順序。

**例句 3**
```
会議が終わった後で、報告書を書きます。
After the meeting ends, I'll write the report.
開完會後寫報告。
```

**例句 4**
```
この本を読んだ後で、感想を聞かせてください。
After reading this book, please tell me your thoughts.
讀完這本書後，請告訴我你的感想。
```

### 用法 3：建議・計畫表達

表示建議某人在做完某事之後再進行下一步。

**例句 5**
```
よく考えた後で、返事をしてください。
Please give your answer after thinking it over carefully.
好好考慮之後再回覆。
```

## 文法規則

### 規則 1：必須使用た形

動詞接続時，必須使用た形，不能使用辞書形或其他形式。

**正確用法**
- ✅ 食べた後で（吃完之後）
- ✅ 行った後で（去了之後）
- ✅ 見た後で（看完之後）

**錯誤用法**
- ❌ 食べる後で（×）- 不能使用辞書形
- ❌ 食べて後で（×）- 不能使用て形
- ❌ 食べます後で（×）- 不能使用ます形

**說明**：「後」表示「之後」，邏輯上前一個動作必須已經完成，因此必須使用表示完成的た形。

### 規則 2：名詞接続需加「の」

名詞與「後で」接続時，中間必須加入「の」。

**正確用法**
- ✅ 授業の後で（課後）
- ✅ 仕事の後で（工作後）
- ✅ 会議の後で（會議後）

**錯誤用法**
- ❌ 授業後で（×）- 缺少「の」
- ❌ 仕事後で（×）- 缺少「の」

### 規則 3：「で」的省略與保留

「で」在口語中可以省略，但書面語和正式場合建議保留。

**完整形式**（推薦）
- 食べた後で、散歩します。
- 仕事の後で、飲みに行きます。

**省略形式**（口語可）
- 食べた後、散歩します。
- 仕事の後、飲みに行きます。

**建議**：
- 初學者：建議使用完整形式「〜た後で」
- 中高級：可根據場合選擇是否省略「で」

## 常見錯誤

### 錯誤 1：使用辞書形

❌ 誤：ご飯を食べる後で、出かけます。
✅ 正：ご飯を食べた後で、出かけます。

說明：「後で」表示動作已完成，必須使用た形，不能使用辞書形。辞書形用於「〜前に」（之前）的文法中。

### 錯誤 2：與原因表達「〜たから」混淆

❌ 誤：お腹が空いたから、ご飯を食べます。（これは正しいが、意味が異なる）
✅ 正：お腹が空いた後で、ご飯を食べます。（時間順序）
✅ 正：お腹が空いたから、ご飯を食べます。（原因）

說明：
- 「〜た後で」：表示時間順序（餓了之後吃飯）
- 「〜たから」：表示原因理由（因為餓了所以吃飯）

### 錯誤 3：名詞接続時忘記「の」

❌ 誤：会議後で、資料をまとめます。
✅ 正：会議の後で、資料をまとめます。

說明：名詞與「後で」接続時，必須在中間加入「の」作為連接。這是日語名詞修飾的基本規則。

## 學習要點

1. **た形是必須的**：「〜た後で」永遠使用た形，這是表示動作完成的標誌
2. **與「〜前に」對比記憶**：前に用辞書形、後で用た形，是重要的對比關係
3. **「で」可省略**：口語中常省略「で」，但初學者建議保留以建立正確文法意識
4. **時間間隔較寬鬆**：相比「〜てから」，「〜た後で」允許兩動作間有較長間隔
5. **名詞接続加「の」**：記住名詞接続時必須使用「名詞 + の + 後で」的格式

## 註解

[^ta-form]: **た形** - 動詞的過去形式，也稱為「過去形」或「完了形」。表示動作已經完成或發生過。例：食べる → 食べた、行く → 行った。在「〜た後で」中，使用た形來明確表示前一個動作已經完成。詳見 [た形](009_ta_form.md)

[^ato-de-basic]: **〜後で（基本形）** - 表示「在...之後」的時間表達基本文法，包含動詞接続（た形 + 後で）、名詞接続（名詞 + の + 後で）和時間表達（數字 + 時間單位 + 後）三種用法。詳見 [〜後で](014_ato_de.md)

[^dictionary-form]: **辞書形** - 動詞的基本形式，也稱為「原形」或「plain form」。是字典中查詢時使用的形式。例：食べる、行く、する。在「〜前に」文法中使用辞書形，與「〜た後で」使用た形形成對比。詳見 [辞書形](010_dictionary_form.md)

[^time-expression]: **時間表達** - 日語中表示時間關係的文法表現，包含「前に」（之前）、「後で」（之後）、「ながら」（同時）、「てから」（之後）等，是描述動作時間順序的重要文法系統。

[^te-kara]: **〜てから** - 表示「做完...之後」的時間順序文法，使用動詞て形接続。與「〜た後で」相似，但「〜てから」更強調動作的緊密連續性，適合表達「做完A馬上做B」的情況。詳見 [〜てから](015_te_kara.md)

[^mae-ni]: **〜前に** - 表示「在...之前」的時間文法，使用動詞辞書形接続。與「〜た後で」形成對比：「前に」用辞書形表示尚未發生的動作，「〜た後で」用た形表示已完成的動作。詳見 [〜前に](004_mae_ni.md)
