助詞「に」是日文中最常用且用法最多樣的格助詞之一，本卡片整理其主要用法體系。

<!--more-->

## 日文

に（格助詞）

### 文法情報

| 項目 | 內容 |
|------|------|
| 文法類型 | 格助詞 |
| JLPT | N5-N4 |
| 使用場面 | 所有場面 |
| 特徵 | 多義助詞、用法豐富 |

### 主要用法分類

```
1. 場所・存在（いる・ある）
2. 時間・時點（〜時に）
3. 目的地・方向（行く・来る）
4. 對象・相手（〜に言う）
5. 目的（〜に行く）
6. 變化結果（〜になる）
7. 被動動作主（〜に〜れる）
```

## 日文解釋

助詞「に」は日本語において最も多機能な格助詞の一つです。その用法は大きく七つの系統に分けることができます。

第一に、存在の場所を表します。「いる」「ある」という存在動詞と共に使い、人や物が存在する場所を示します。第二に、時間の一点を表します。「三時に」「月曜日に」のように、動作や出来事が起こる時点を示します。

第三に、移動の目的地や方向を表します。「東京に行く」のように、移動動詞と共に使います。第四に、動作の相手や対象を表します。「友達に話す」「先生に聞く」のように、動作が向かう相手を示します。

第五に、目的を表します。「買い物に行く」のように、移動の目的を示す場合に使います。第六に、変化の結果を表します。「医者になる」「赤くなる」のように、変化後の状態を示します。第七に、受身文での動作主を表します。「先生に褒められた」のように使います。

## 英文解釋

The particle "ni" (に) is one of the most versatile case particles in Japanese, with seven main categories of usage:

1. **Location of existence**: Marks where something/someone exists (with いる/ある)
2. **Point in time**: Indicates when something occurs (三時に = at 3 o'clock)
3. **Destination/Direction**: Marks where movement is directed (東京に行く = go to Tokyo)
4. **Target/Recipient**: Indicates the recipient of an action (友達に話す = speak to a friend)
5. **Purpose**: Shows the purpose of going somewhere (買い物に行く = go shopping)
6. **Result of change**: Marks what something becomes (医者になる = become a doctor)
7. **Agent in passive**: Indicates who performs the action in passive sentences

Understanding these different functions is essential for proper Japanese sentence construction.

## 中文解釋

助詞「に」是日文中用法最多樣的格助詞之一，主要有以下七種用法體系：

1. **存在場所**：表示人或物存在的位置（搭配いる/ある）
2. **時間點**：表示動作發生的時間（三時に＝三點時）
3. **目的地/方向**：表示移動的終點（東京に行く＝去東京）
4. **對象/接受者**：表示動作的對象（友達に話す＝跟朋友說）
5. **目的**：表示前往某處的目的（買い物に行く＝去購物）
6. **變化結果**：表示變化後的狀態（医者になる＝成為醫生）
7. **被動動作主**：在被動句中表示動作的執行者

掌握這些不同用法是正確使用日文的關鍵。

## 核心用法

### 用法 1：時間點

表示動作或事件發生的具體時間點。

**例句 1**
```
毎朝七時に起きます。
I wake up at 7 o'clock every morning.
我每天早上七點起床。
```

**例句 2**
```
来週の金曜日に会議があります。
There is a meeting next Friday.
下週五有會議。
```

### 用法 2：動作對象

表示動作的接受者或對象。

**例句 3**
```
母に手紙を書きました。
I wrote a letter to my mother.
我寫了一封信給媽媽。
```

### 用法 3：目的

與動詞連用形搭配，表示前往某處的目的。

**例句 4**
```
映画を見に行きませんか。
Won't you go see a movie?
要不要去看電影？
```

### 用法 4：變化結果

與「なる」搭配，表示變化後的結果狀態。

**例句 5**
```
将来、教師になりたいです。
I want to become a teacher in the future.
將來我想當老師。
```

## 文法規則

### 規則 1：時間詞與「に」的搭配

並非所有時間詞都需要「に」：

| 需要「に」 | 不需要「に」 |
|-----------|-------------|
| 三時に | 今日（✓） |
| 月曜日に | 明日（✓） |
| 一月に | 毎日（✓） |
| 2024年に | 来週（✓） |

**規則**：含有具體數字或特定名稱的時間詞通常需要「に」，相對時間詞通常不需要。

### 規則 2：「に」與「へ」的區別

兩者都可表示方向，但語感不同：
- **に**：強調到達點、目的地
- **へ**：強調移動的方向性

```
東京に行く（強調到達東京）
東京へ行く（強調往東京方向移動）
```

### 規則 3：目的用法的形成

動詞連用形（ます形去掉ます）+ に + 行く/来る

```
食べる → 食べ + に + 行く → 食べに行く
見る → 見 + に + 来る → 見に来る
```

## 常見錯誤

### 錯誤 1：相對時間詞誤用「に」

❌ 誤：明日に会いましょう。
✅ 正：明日会いましょう。
說明：「明日」是相對時間詞，不需要加「に」。

### 錯誤 2：混淆「に」和「で」

❌ 誤：図書館に勉強します。
✅ 正：図書館で勉強します。
說明：動作進行的場所用「で」，存在場所才用「に」。

### 錯誤 3：目的用法使用辭書形

❌ 誤：食べるに行きます。
✅ 正：食べに行きます。
說明：目的用法需使用動詞連用形（ます形去ます），不是辭書形。

## 學習要點

1. **多義性理解**：「に」有七種主要用法，需根據上下文判斷
2. **時間詞規則**：具體時間加「に」，相對時間不加
3. **與「で」區別**：存在/到達用「に」，動作場所用「で」
4. **與「へ」區別**：「に」強調到達點，「へ」強調方向
5. **目的用法形成**：動詞連用形 + に + 移動動詞
