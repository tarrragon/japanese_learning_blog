## 日文

〜ている

### 文法情報

| 項目 | 內容 |
|------|------|
| 文法類型 | 動詞變化句型 |
| JLPT | N5 |
| 使用場面 | 日常會話、書面、萬用 |
| 形成 | 動詞て形 + いる |

### 形成規則

**基本形式**
```
動詞て形 + いる
```

**範例**
```
食べる → 食べている（正在吃 / 吃著）
見る → 見ている（正在看 / 看著）
行く → 行っている（去了 / 正在去）
知る → 知っている（知道著）
住む → 住んでいる（住著）
```

**丁寧形**
```
食べている → 食べています
見ている → 見ています
行っている → 行っています
```

## 日文解釋

「〜ている」は日本語で最も重要な文法形式の一つで、動詞のて形に補助動詞「いる」を付けた表現です。この形には**二つの核心的な意味**があります。

第一の意味は**動作の進行**です。ちょうど英語の現在進行形（be + -ing）のように、「今まさに動作が行われている最中」を表します。例えば「今、ご飯を食べている」は「I am eating now」に相当し、食事という動作が現在進行中であることを示します。

第二の意味は**結果状態の持続**です。これは動作が完了した後、その結果として生じた状態が現在も続いていることを表します。例えば「結婚している」は「be married」に相当し、過去に結婚という出来事があり、その結果として既婚という状態が今も続いていることを示します。

どちらの意味になるかは、主に**動詞の性質**によって決まります。「食べる」「走る」「読む」などの継続的な動作を表す動詞は進行の意味になりやすく、「知る」「住む」「持つ」などの瞬間的な変化や状態を表す動詞は結果状態の意味になります。

日本語母語話者は文脈と動詞の種類から、どちらの意味かを自然に判断しています。学習者にとって、この二つの意味の区別を理解することは、自然な日本語表現を習得する上で極めて重要です。

## 英文解釋

"〜ている" (te iru) is one of the most fundamental grammar patterns in Japanese, formed by combining the te-form of a verb with the auxiliary verb "いる" (iru, to exist/be). This construction has **two core meanings** that learners must understand.

The first meaning is **progressive action**, similar to the English present progressive (be + -ing). It expresses that an action is currently in progress. For example, "今、ご飯を食べている" means "I am eating now," indicating that the action of eating is happening at this moment.

The second meaning is **resultant state**, which expresses that an action has been completed and the resulting state continues to the present. For example, "結婚している" means "to be married," indicating that the marriage event occurred in the past, and the state of being married continues now.

Which meaning applies depends primarily on **the nature of the verb**. Continuous action verbs like "eat," "run," and "read" typically express progressive action, while instantaneous or state-change verbs like "know," "live," and "have" typically express resultant state.

Native Japanese speakers naturally distinguish between these two meanings based on context and verb type. For learners, understanding this distinction is crucial for achieving natural Japanese expression and comprehension.

## 中文解釋

「〜ている」是日文中最基礎且重要的文法形式之一，由動詞的て形加上補助動詞「いる」構成。這個句型有**兩個核心意義**，學習者必須掌握。

第一個意義是**動作進行**，類似英文的現在進行式（be + -ing）。表示動作正在進行中。例如「今、ご飯を食べている」意思是「現在正在吃飯」，表示吃飯這個動作此刻正在發生。

第二個意義是**結果狀態持續**，表示動作已經完成，但其產生的結果狀態持續到現在。例如「結婚している」意思是「已婚」，表示過去發生了結婚這件事，而已婚這個狀態持續到現在。

使用哪個意義主要取決於**動詞的性質**。像「吃」「跑」「讀」等表示持續動作的動詞通常表達進行意義，而像「知道」「住」「擁有」等表示瞬間變化或狀態的動詞通常表達結果狀態。

日文母語者會根據上下文和動詞類型自然判斷是哪個意義。對學習者來說，理解這兩個意義的區別，是掌握自然日文表達的關鍵。

## 核心用法

### 用法 1：動作正在進行（進行式）

表示動作此刻正在發生，相當於「正在...」。

**例句 1**
```
今、宿題をしています。
I am doing my homework now.
我現在正在做作業。
```

**例句 2**
```
雨が降っている。
It's raining.
正在下雨。
```

### 用法 2：結果狀態持續（狀態式）

表示動作已完成，結果狀態持續存在。

**例句 3**
```
私は東京に住んでいます。
I live in Tokyo.
我住在東京。
```

**例句 4**
```
田中さんを知っていますか。
Do you know Tanaka-san?
你認識田中先生嗎？
```

### 用法 3：習慣性動作

表示經常反覆進行的動作。

**例句 5**
```
毎朝ジョギングをしています。
I jog every morning.
我每天早上都跑步。
```

## 文法規則

### 規則 1：動詞類型決定意義

**持續動作動詞 → 進行式**
```
食べる（吃）、読む（讀）、書く（寫）、走る（跑）
→ 表示「正在...」
```

**瞬間動詞 → 結果狀態**
```
知る（知道）、住む（住）、持つ（擁有）、結婚する（結婚）
→ 表示「...著」「已經...」
```

### 規則 2：特殊動詞「知る」

「知る」是瞬間動詞，**不能表示進行**，只能表示結果狀態。

```
✅ 知っている（知道 - 狀態）
❌ 知っている（正在知道 - 不自然）
✅ 知った（知道了 - 瞬間完成）
```

### 規則 3：移動動詞的特殊用法

移動動詞用「〜ている」表示「已經移動，現在在那裡」。

```
父は会社に行っている。
→ 父親去公司了（現在在公司）

友達が来ている。
→ 朋友來了（現在在這裡）
```

### 規則 4：否定形和過去形

**否定形**
```
食べていない（食べていません）- 沒在吃
知らない（知りません）- 不知道
```

**過去形**
```
食べていた（食べていました）- 以前在吃 / 曾經吃著
住んでいた（住んでいました）- 以前住著
```

## 常見錯誤

### 錯誤 1：「知る」誤用進行式

❌ 誤：今、知っている。（正在知道）
✅ 正：知っている。（知道）
說明：「知る」是瞬間動詞，不能表示「正在知道」，只能表示「知道」這個狀態。

### 錯誤 2：混淆「いる」和「ある」

❌ 誤：本を持ってある。
✅ 正：本を持っている。
說明：「持つ」搭配的是「いる」，不是「ある」。「〜てある」是另一個文法句型，只能接他動詞。

### 錯誤 3：移動動詞的誤解

❌ 誤理解：父は会社に行っている。= 父親正在去公司的路上。
✅ 正確理解：父は会社に行っている。= 父親去公司了（現在在公司）。
說明：移動動詞的「〜ている」表示移動已完成，人在目的地，而非正在移動中。

## 學習要點

1. **兩個核心意義**：進行式（正在...）和結果狀態（...著）
2. **動詞類型決定**：持續動詞 → 進行，瞬間動詞 → 狀態
3. **知る最特殊**：只能表示狀態「知道」，不能表示進行
4. **移動動詞特性**：表示「已經去了，在那裡」
5. **萬用性高**：日常會話和書面都常用，是N5最重要文法之一
