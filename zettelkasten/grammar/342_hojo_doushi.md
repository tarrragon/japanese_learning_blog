## 日文解釋

補助動詞とは、本動詞のて形に接続して、本来の動詞の意味に様々なニュアンスを加える働きをする動詞のことである。

補助動詞は単独では文を完結させることができず、必ず前に来る動詞のて形と組み合わせて使われる。この点で、独立した動詞として使う場合（本動詞用法）とは区別される。例えば「いる」は本動詞として「ここにいる」のように使えるが、補助動詞として「食べている」のように使う場合は、「食べる」という動作の進行や状態を表す機能語として働く。

補助動詞は日本語の時制・相（アスペクト）システムにおいて核心的な役割を果たす。英語では進行形（be + -ing）や完了形（have + 過去分詞）など異なる構造で表現する概念を、日本語では補助動詞の体系によって表現する。

## 英文解釋

Auxiliary verbs (hojo doushi) in Japanese are verbs that attach to the te-form of main verbs to add various grammatical meanings such as aspect, direction, benefaction, and completion.

Unlike main verbs that carry lexical meaning independently, auxiliary verbs function as grammatical markers when combined with te-form verbs. For example, "iru" as a main verb means "to exist/be," but as an auxiliary verb in "tabete iru," it indicates progressive or resultative aspect.

The auxiliary verb system is central to Japanese aspect marking. Concepts expressed through different structures in English (progressive with be + -ing, perfect with have + past participle) are conveyed through auxiliary verbs in Japanese. Understanding this system is essential for expressing temporal and aspectual nuances naturally.

## 中文解釋

補助動詞是接續在動詞て形後面，用來添加各種文法意義的動詞。這些動詞不獨立表達詞彙意義，而是作為文法標記，表示動作的進行、完成、方向、恩惠等概念。

例如「いる」作為本動詞時表示「存在」，但作為補助動詞時（如「食べている」），則表示動作的進行狀態或結果狀態。這種區別是理解日文時態和體態系統的關鍵。

補助動詞系統是日文文法的核心部分。英文用不同的結構表達的概念（進行式用 be + -ing，完成式用 have + 過去分詞），在日文中都透過補助動詞來表現。掌握這個系統對於自然地表達時間和狀態的細微差異非常重要。

## 核心用法

### 用法 1：進行・状態を表す（～ている）

最も基本的な補助動詞。動作の進行中、または結果状態を表す。

**例句 1**
```
今、本を読んでいます。
I am reading a book now.
我現在正在看書。
```

**例句 2**
```
窓が開いている。
The window is open.
窗戶開著。
```

### 用法 2：準備・放置を表す（～ておく）

事前準備や、ある状態のまま放置することを表す。

**例句 3**
```
明日の会議のために資料を準備しておきます。
I will prepare the materials in advance for tomorrow's meeting.
我會事先準備明天會議的資料。
```

### 用法 3：完了・遺憾を表す（～てしまう）

動作の完了、または予期せぬ結果への遺憾を表す。

**例句 4**
```
大切な書類をなくしてしまった。
I (unfortunately) lost the important documents.
我把重要文件弄丟了。
```

### 用法 4：恩恵の授受を表す（～てもらう/くれる/あげる）

行為の授受関係を表す。

**例句 5**
```
先輩に日本語を教えてもらいました。
I had my senior teach me Japanese. / My senior kindly taught me Japanese.
學長教了我日文。
```

## 文法規則

### 規則 1：て形への接続

補助動詞は必ず動詞のて形に接続する。形容詞やな形容詞には直接接続しない。

```
✅ 食べている（動詞て形 + いる）
❌ 美味しいいる（形容詞に直接接続は不可）
```

### 規則 2：本動詞用法との区別

同じ動詞でも、本動詞用法と補助動詞用法で意味が異なる。補助動詞として使う場合はひらがな表記が一般的。

```
本動詞：部屋にいる（部屋に存在する）
補助動詞：食べている（食べる動作の進行）

本動詞：荷物を置く（荷物を設置する）
補助動詞：買っておく（事前に買う）
```

### 規則 3：複数の補助動詞の組み合わせ

複数の補助動詞を組み合わせることも可能だが、順序に制約がある。

```
食べてしまっている（完了 + 状態）
準備しておいてください（準備 + 依頼）
```

## 常見錯誤

### 錯誤 1：表記の混同

❌ 誤：本を読んで居る
✅ 正：本を読んでいる
說明：補助動詞として使う場合はひらがな表記が標準。漢字表記は本動詞用法を示唆する。

### 錯誤 2：授受表現の主語混乱

❌ 誤：私は友達に本を借りてあげた
✅ 正：私は友達に本を借りてもらった / 友達は私に本を貸してくれた
說明：「てあげる」は話者が恩恵を与える場合、「てもらう/くれる」は話者が恩恵を受ける場合に使う。

### 錯誤 3：アスペクトの誤用

❌ 誤：昨日から雨が降っている（継続中なのに過去形の時間詞と）
✅ 正：昨日から雨が降っている / 昨日雨が降った
說明：「ている」は現在まで継続する状態を表す。過去の一時的な出来事には「た」形を使う。

## 學習要點

1. **て形の習得が前提**：補助動詞を使いこなすには、まずて形の作り方を完璧にマスターする必要がある。

2. **本動詞用法との区別**：同じ動詞でも補助動詞として使う場合は意味と表記が変わることを理解する。

3. **アスペクトの概念**：進行・完了・結果状態など、動作の時間的な側面を表現する概念を理解する。

4. **授受表現の視点**：「てもらう」「てくれる」「てあげる」は話者の視点と恩恵の方向によって使い分ける。

5. **口語での短縮形**：「ている→てる」「ておく→とく」「てしまう→ちゃう」など、会話での短縮形を覚える。
