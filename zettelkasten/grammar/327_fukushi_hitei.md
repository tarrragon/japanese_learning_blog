## 日文

副詞の否定用法

**羅馬拼音**: fukushi no hitei youhou

**概要**: 副詞と否定形の組み合わせで否定の程度や意味を表す文法パターン

## 日文解釋

副詞の否定用法とは、特定の副詞が否定形の述語と組み合わさって使用される文法パターンである。日本語では、副詞の種類によって否定の程度や意味合いが変わる。

**主要な否定副詞と使い方**:

| 副詞 | 意味 | 否定の程度 |
|------|------|----------|
| あまり | not very, not much | 部分否定 |
| 全然（ぜんぜん） | not at all | 完全否定 |
| ほとんど | hardly, almost not | ほぼ完全否定 |
| めったに | rarely, seldom | 頻度否定 |
| ちっとも | not at all | 完全否定（口語） |
| 少しも（すこしも） | not even a little | 完全否定 |
| さっぱり | not at all | 完全否定（口語） |

**文法構造**:
- 副詞 + 動詞（ない形）
- 副詞 + い形容詞（くない）
- 副詞 + な形容詞（ではない / じゃない）
- 副詞 + 名詞（ではない / じゃない）

## 英文解釋

Adverbial negation patterns in Japanese involve specific adverbs that pair with negative verb forms to express varying degrees of negation.

**Key negation adverbs**:

| Adverb | Meaning | Negation Level |
|--------|---------|----------------|
| amari | not very, not much | Partial negation |
| zenzen | not at all | Complete negation |
| hotondo | hardly, almost not | Near-complete negation |
| mettani | rarely, seldom | Frequency negation |
| chittomo | not at all | Complete negation (colloquial) |
| sukoshimo | not even a little | Complete negation |
| sappari | not at all | Complete negation (colloquial) |

**Grammar structure**:
- Adverb + Verb (nai-form)
- Adverb + i-adjective (kunai)
- Adverb + na-adjective (dewa nai / ja nai)
- Adverb + Noun (dewa nai / ja nai)

**主要否定副詞及用法**：

| 副詞 | 意思 | 否定程度 |
|------|------|---------|
| あまり | 不太、不怎麼 | 部分否定 |

- 副詞 + 動詞（ない形）
- 副詞 + い形容詞（くない）

- 副詞 + 名詞（ではない / じゃない）

## 例句

**例句1 - あまり〜ない（部分否定）**
```
この料理はあまり辛くない。
This dish isn't very spicy.
這道菜不太辣。
```

**例句2 - 全然〜ない（完全否定）**
```
昨日は全然眠れなかった。
I couldn't sleep at all yesterday.
昨天完全睡不著。
```

**例句3 - ほとんど〜ない（接近完全否定）**
```
冷蔵庫にはほとんど食べ物がない。
There's hardly any food in the refrigerator.
冰箱裡幾乎沒有食物。
```

**例句4 - めったに〜ない（頻率否定）**
```
彼女はめったに遅刻しない。
She rarely comes late.
她很少遲到。
```

**例句5 - ちっとも〜ない（口語完全否定）**
```
説明を聞いてもちっとも分からない。
I don't understand at all even after hearing the explanation.
聽了說明也一點都不懂。
```

**例句6 - 少しも〜ない（強調完全否定）**
```
少しも疲れていません。
I'm not tired at all.
我一點也不累。
```

**例句7 - さっぱり〜ない（口語完全否定）**
```
その話はさっぱり覚えていない。
I don't remember that story at all.
那件事我完全不記得。
```

**例句8 - 複合使用**
```
最近はあまり外出していないので、ほとんど日焼けしていない。
Since I haven't been going out much recently, I'm hardly sunburned.
最近不太外出，所以幾乎沒有曬黑。
```

## 對比與關聯

**否定程度的比較**:

```
完全否定（強）
  ↓ 全然〜ない / ちっとも〜ない / 少しも〜ない / さっぱり〜ない
  ↓ ほとんど〜ない
  ↓ あまり〜ない
部分否定（弱）
```

**「あまり」vs「全然」**:
- あまり：部分否定，表示「不太」「不怎麼」
- 全然：完全否定，表示「完全不」「一點也不」

**「ほとんど」的特殊性**:
- 表示「幾乎不」，意味著還有一點點例外
- 例：ほとんど食べなかった（幾乎沒吃，但可能吃了一點點）

**「めったに」的用法**:
- 專門用於表示頻率極低
- 例：めったに風邪を引かない（很少感冒）

## 文法筆記

**否定副詞的文體差異**:

| 副詞 | 文體 | 使用場合 |
|------|------|---------|
| 全然 | 口語・日常 | 日常會話 |
| 全く | 書面語・正式 | 報告、正式場合 |
| ちっとも | 口語 | 親密對話 |
| さっぱり | 口語 | 日常會話 |
| 少しも | 書面語 | 文章、演講 |

**注意事項**:
1. 這些副詞必須與否定形搭配使用
2. 錯誤用法：❌ あまり食べた / ❌ 全然分かる（傳統規則）
3. 現代日語中「全然」有時也用於肯定：全然大丈夫（完全沒問題）

**頻率與程度的區別**:
- めったに：表示頻率（很少發生）
- ほとんど：表示數量或程度（幾乎沒有）
