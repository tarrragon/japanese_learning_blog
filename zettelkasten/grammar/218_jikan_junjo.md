## 日文

時間順序表達

### 文法情報

| 項目 | 內容 |
|------|------|
| 文法類型 | 時間表達 |
| JLPT | N4 |
| 使用場面 | 敘事、說明 |
| 形成 | 接續詞 + 動詞て形/た形 |

### 形成規則

**基本形式**
```
まず → それから → 次に → 最後に
動詞て形 + から
動詞た + あとで
動詞前に
```

**範例**
```
起きる → 起きてから（起床之後）
食べる → 食べたあとで（吃完之後）
寝る → 寝る前に（睡覺之前）
```

## 日文解釋

時間順序を表す表現は、出来事が起こった順番や時間的な前後関係を明確に示すために使われます。日本語では、接続詞（まず、それから、次に、最後に）と動詞の活用形（て形、た形）を組み合わせて、複数の行動や出来事の時系列を整理して伝えることができます。

これらの表現は、料理の手順、旅行の計画、一日のスケジュール、物語の展開など、様々な場面で使用されます。特に「〜てから」は前の動作が完了してから次の動作に移ることを強調し、「〜前に」は後の動作より先に行うべきことを示します。「〜あとで」は動作完了後の時点を表し、より自然な会話表現として頻繁に使われます。

接続詞による順序表現（まず、それから、次に、最後に）は、手順を段階的に説明する際に非常に便利で、聞き手や読み手が内容を理解しやすくなります。ビジネスシーンや教育現場でも多用される重要な表現技法です。

## 英文解釋

Temporal sequence expressions are used to clearly indicate the order in which events occur and their chronological relationships. In Japanese, these expressions combine conjunctions (まず, それから, 次に, 最後に) with verb conjugations (て-form, た-form) to organize and communicate the timeline of multiple actions or events.

The pattern "〜てから" emphasizes that one action is completed before moving to the next, while "〜前に" indicates something that should be done before a subsequent action. "〜あとで" expresses a point in time after an action is completed and is frequently used in natural conversation.

Sequential conjunctions (まず, それから, 次に, 最後に) are particularly useful when explaining procedures step by step, making it easier for listeners or readers to follow the content. These expressions are essential in both business and educational contexts.

## 中文解釋

時間順序表達用於清楚地表示事件發生的順序和時間上的前後關係。日文中透過接續詞（まず、それから、次に、最後に）與動詞活用形（て形、た形）的組合，可以有條理地傳達多個動作或事件的時間序列。

「〜てから」強調前一個動作完成後才進行下一個動作，「〜前に」表示在後續動作之前應該做的事，「〜あとで」則表達動作完成後的時間點，是自然會話中常用的表達方式。

接續詞的順序表達（まず、それから、次に、最後に）在分步驟說明程序時非常實用，能讓聽者或讀者更容易理解內容。這些表達在商務場合和教育場景中也被廣泛使用。

## 核心用法

### 用法 1：接續詞順序（まず → それから → 次に → 最後に）

用於分步驟說明程序或過程。

**例句 1**
```
まず野菜を切ります。それから油で炒めます。次に調味料を加えます。最後に盛り付けて完成です。
First, cut the vegetables. Then stir-fry them in oil. Next, add the seasonings. Finally, plate and it's done.
首先切蔬菜。然後用油炒。接著加入調味料。最後裝盤就完成了。
```

### 用法 2：〜てから（動作完成後）

表示前一個動作完成後才進行下一個動作。

**例句 2**
```
朝起きてから、シャワーを浴びます。
After waking up in the morning, I take a shower.
早上起床後會洗澡。
```

**例句 3**
```
宿題を終えてから、遊びに行きましょう。
Let's go play after finishing homework.
做完作業後再去玩吧。
```

### 用法 3：〜前に（動作之前）

表示在某動作之前應該做的事。

**例句 4**
```
寝る前に歯を磨きます。
I brush my teeth before going to bed.
睡覺前刷牙。
```

### 用法 4：〜あとで（動作之後）

表示動作完成後的時間點，語氣較「〜てから」輕鬆。

**例句 5**
```
会議のあとで、ランチに行きませんか。
Would you like to go for lunch after the meeting?
開完會後要不要去吃午餐？
```

## 文法規則

### 規則 1：接續詞的使用順序

まず（首先）→ それから（然後）→ 次に（接著）→ 最後に（最後）的順序是固定的，不能隨意調換。

### 規則 2：〜てから的接續

動詞て形 + から，表示完成前一動作後才進行下一動作，強調順序性和完成性。

### 規則 3：〜前に的接續

動詞辞書形 + 前に，注意「前に」前面使用動詞原形（辞書形），而非て形或た形。

### 規則 4：〜あとで的接續

動詞た形 + あとで，可以省略「で」變成「〜あと」，意思相同但稍微正式一些。

## 常見錯誤

### 錯誤 1：〜前に使用た形

❌ 誤：寝た前に歯を磨きます。
✅ 正：寝る前に歯を磨きます。
說明：「前に」前面必須使用動詞辞書形，不能用た形。

### 錯誤 2：混淆〜てから和〜あとで

❌ 誤：食べるてから、薬を飲みます。
✅ 正：食べてから、薬を飲みます。
說明：「〜てから」使用て形，不需要在辞書形後加て。

### 錯誤 3：接續詞順序錯亂

❌ 誤：最後に材料を準備します。それから、まず切ります。
✅ 正：まず材料を準備します。それから切ります。
說明：接續詞應該按照實際動作順序使用，「まず」應該在最前面。

## 學習要點

1. **接續詞固定順序**：まず → それから → 次に → 最後に，順序不可顛倒
2. **動詞形態區別**：〜てから（て形）、〜前に（辞書形）、〜あとで（た形）
3. **〜てから強調完成**：表示前一動作必須完成才能進行下一動作
4. **〜前に用辞書形**：這是最容易犯錯的地方，要特別記住
5. **自然度差異**：〜あとで比〜てから語氣更輕鬆，日常會話中更常用
