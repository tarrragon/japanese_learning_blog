| 〜ない？ | ない形 + ？ | 親密・口語 | 行かない？ |

**動詞活用規則**

```
一段動詞：食べる → 食べましょう / 食べよう
五段動詞：行く → 行きましょう / 行こう
不規則動詞：
  する → しましょう / しよう
  来る → 来ましょう / 来よう
```

## 日文解釋

勧誘表現とは、話し手が聞き手に対して、一緒に何かをすることを提案したり、誘ったりするときに使う文法形式のことである。

日本語には複数の勧誘表現があり、場面や相手との関係によって使い分ける必要がある。「〜ましょう」は最も基本的な形で、自分も一緒にするという意志を含みながら相手を誘う。「〜ませんか」は相手の意向を尊重する丁寧な誘い方で、ビジネスシーンでも使える。「〜（よ）う」は友人や家族など親しい間柄で使うくだけた表現である。

勧誘表現を選ぶ際は、相手との上下関係、親疎関係、そして場面の公式度を考慮することが重要である。同じ「映画を見に行く」という誘いでも、上司には「映画を見に行きませんか」、友人には「映画見に行こうよ」と表現が変わる。

## 英文解釋

Invitation expressions (kanyuu hyougen) in Japanese are grammatical structures used to invite or suggest doing something together with the listener.

Japanese has several levels of invitation expressions, each carrying different nuances of politeness and formality. The form "〜ましょう" (mashou) is the standard polite invitation, implying "let's do something together." The form "〜ませんか" (masen ka) is more polite and considerate, literally meaning "won't you...?" and giving the listener more room to decline.

The casual form "〜（よ）う" (you) is used among close friends and family. Understanding when to use each form is essential for appropriate social interaction in Japanese, as using too casual a form with superiors or too formal a form with close friends can create awkwardness.

## 中文解釋

勸誘表現是日語中用於邀請對方一起做某事的文法結構。

日語有多種層次的勸誘表達方式，每種形式帶有不同程度的禮貌和正式感。「〜ましょう」是最基本的禮貌形式，表達「一起做吧」的意思。「〜ませんか」更為禮貌客氣，字面意思是「要不要...?」，給予對方更多拒絕的空間。

「〜（よ）う」是親密朋友和家人之間使用的口語形式。掌握何時使用哪種形式，對於在日語中進行得體的社交互動至關重要。對上司使用過於隨便的形式，或對親友使用過於正式的形式，都會造成不自然的印象。

## 核心用法

### 用法 1：〜ましょう（基本勸誘）

表達說話者提議與對方一起行動，同時包含自己也參與的意志。

**例句 1**
```
一緒に昼ごはんを食べましょう。
Let's have lunch together.
我們一起吃午餐吧。
```

**例句 2**
```
会議を始めましょう。
Let's start the meeting.
開始開會吧。
```

### 用法 2：〜ませんか（丁寧勸誘）

更加尊重對方意願的邀請方式，適用於初次見面或較正式的場合。

**例句 3**
```
今度の週末、美術館に行きませんか。
Would you like to go to the museum this weekend?
這個週末要不要去美術館呢？
```

**例句 4**
```
よろしければ、一杯いかがですか。
If you don't mind, would you like a drink?
方便的話，要不要喝一杯？
```

### 用法 3：〜（よ）う / 〜ない？（口語勸誘）

親密朋友之間使用的隨意形式。

**例句 5**
```
明日カラオケ行こうよ。
Let's go to karaoke tomorrow.
明天去唱卡拉OK吧。
```

## 常見錯誤

### 錯誤 1：對上司使用過於隨便的形式

❌ 誤：部長、今日一緒にランチ行こう。
✅ 正：部長、今日一緒にランチはいかがですか。
說明：對上司應使用較丁寧的「いかがですか」或「行きませんか」，而非「行こう」。

### 錯誤 2：混淆〜ましょう和〜ましょうか

❌ 誤：（服務生對客人）ご注文をしましょう。
✅ 正：ご注文をお伺いしましょうか。
說明：「〜ましょうか」用於詢問對方是否需要服務，「〜ましょう」則是共同行動的提議。

### 錯誤 3：在正式場合使用〜ない？形式

❌ 誤：（對客戶）この企画、検討しない？
✅ 正：この企画をご検討いただけませんでしょうか。
說明：商務場合應使用敬語形式的勸誘表達。

## 學習要點

1. **場面意識**：根據場合選擇適當的勸誘形式是日語交際的基本功。

2. **丁寧度階層**：ませんか ＞ ましょうか ＞ ましょう ＞ よう ＞ ない？

3. **聽話者本位**：「〜ませんか」比「〜ましょう」更尊重對方，給予拒絕的餘地。

4. **語調重要性**：同樣的句型，語調上揚或下降會改變意味。

5. **性別差異**：女性較常使用「〜ない？」「〜ましょうよ」等帶有柔和語氣的形式。
