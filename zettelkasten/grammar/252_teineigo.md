## 日文

丁寧語（ていねいご）

**羅馬拼音**: teineigo

**詞性**：敬語分類・語用系統

## 日文解釋

丁寧語は、聞き手に対して礼儀正しい話し方をするための敬語の一種です。「です」「ます」を使って丁寧に話すことで、相手への配慮を示します。尊敬語や謙譲語と異なり、動作主や動作の対象に関わらず使用できる最も基本的な敬語形式です。

### 丁寧語の特徴

1. **聞き手への配慮**：話の内容ではなく、話し方の丁寧さで敬意を表す
2. **普遍性**：自分・相手・第三者の誰の動作にも使用可能
3. **基本形**：日本語学習の最初に習う敬語表現
4. **社会的距離**：初対面や公的場面で使用

### 丁寧語の種類

**1. です・ます形**

動詞、形容詞、名詞に付けて丁寧さを表す最も基本的な形式。

- 動詞：[ます形](010_masu_form.md)を使う（食べる → 食べます）
- 形容詞：「です」を付ける（綺麗だ → 綺麗です）
- 名詞：「です」を付ける（学生だ → 学生です）

**2. ございます**

「ある」「いる」の最も丁寧な形。フォーマルな場面で使用。

- あります → ございます
- 「お早うございます」「ありがとうございます」など慣用表現

**3. 美化語**

名詞に「お」「ご」を付けて美しく丁寧に表現。

- 和語（日本固有語）：「お」を付ける → お茶、お水、お名前
- 漢語（中国由来語）：「ご」を付ける → ご飯、ご意見、ご住所

## 英文解釋

Teineigo (polite language) is a type of keigo (honorific language) used to speak politely to the listener. By using "desu" and "masu," it shows consideration for the listener. Unlike sonkeigo (respectful language) and kenjougo (humble language), teineigo can be used regardless of who performs the action, making it the most fundamental form of Japanese honorific language.

### Characteristics of Teineigo

1. **Listener-oriented**: Shows respect through polite speech style, not content
2. **Universality**: Can be used for anyone's actions - self, listener, or third party
3. **Basic form**: The first honorific expression taught in Japanese language learning
4. **Social distance**: Used in first meetings and formal situations

### Types of Teineigo

**1. Desu/Masu Forms**

The most basic form, attached to verbs, adjectives, and nouns.

- Verbs: Use [masu form](010_masu_form.md) (taberu → tabemasu)
- Adjectives: Add "desu" (kirei da → kirei desu)
- Nouns: Add "desu" (gakusei da → gakusei desu)

**2. Gozaimasu**

The most polite form of "aru" (to exist). Used in formal situations.

- arimasu → gozaimasu
- Fixed expressions: "ohayou gozaimasu," "arigatou gozaimasu"

**3. Bikago (Beautification Words)**

Adding "o" or "go" to nouns for refined, polite expression.

- Native Japanese words: Add "o" → ocha (tea), omizu (water)
- Sino-Japanese words: Add "go" → gohan (rice), goiken (opinion)

## 中文解釋

丁寧語是一種用於對聽者表示禮貌說話方式的敬語。透過使用「です」「ます」來禮貌地說話，表達對對方的尊重。與尊敬語和謙讓語不同，無論動作者是誰都可以使用，是最基本的敬語形式。

### 丁寧語的特徴

1. **以聽者為導向**：透過說話方式的禮貌來表示敬意，而非內容本身
2. **普遍性**：無論是自己、對方還是第三者的動作都可使用
3. **基本形式**：日語學習中最早學到的敬語表達
4. **社會距離**：用於初次見面或正式場合

### 丁寧語的種類

**1. です・ます形**

附加在動詞、形容詞、名詞後表示禮貌的最基本形式。

- 動詞：使用[ます形](010_masu_form.md)（食べる → 食べます）
- 形容詞：加「です」（綺麗だ → 綺麗です）
- 名詞：加「です」（学生だ → 学生です）

**2. ございます**

「ある」「いる」的最禮貌形式。用於正式場合。

- あります → ございます
- 慣用表達：「お早うございます」「ありがとうございます」

**3. 美化語**

在名詞前加「お」「ご」使表達更優美禮貌。

- 和語（日本固有詞）：加「お」→ お茶、お水、お名前
- 漢語（中文來源詞）：加「ご」→ ご飯、ご意見、ご住所

## 例句

**例句1：動詞的丁寧語（日常對話）**
```
明日、学校に行きます。
Tomorrow, I will go to school.
明天我要去學校。
```

**例句2：名詞的丁寧語（自我介紹）**
```
私は田中と申します。会社員です。
My name is Tanaka. I am an office worker.
我叫田中。我是上班族。
```

**例句3：ございます的使用（接待客人）**
```
いらっしゃいませ。こちらにメニューがございます。
Welcome. Here is the menu.
歡迎光臨。這裡有菜單。
```

**例句4：美化語的使用（日常禮貌表達）**
```
お茶をどうぞ。ご飯もございます。
Please have some tea. There is also rice.
請用茶。也有飯。
```

## 補充說明

### 丁寧語と尊敬語・謙譲語の違い

| 比較項目 | 丁寧語 | 尊敬語 | 謙讓語 |
|---------|--------|--------|--------|
| 敬意の対象 | 聞き手 | 動作の主体 | 動作の受け手 |
| 使用対象 | 誰の動作でも可 | 相手・目上の動作 | 自分の動作 |
| 表現方法 | です・ます | 特殊動詞・られる形 | 特殊動詞・お〜する |
| 習得時期 | 初級 | 中級〜上級 | 中級〜上級 |

### 場面による使い分け

**丁寧語が必須の場面**：
- 初対面の人との会話
- 職場での公式な場面
- 店員と客の対話
- 公的なスピーチや発表

**普通体（タメ語）を使う場面**：
- 家族との会話
- 親しい友人との会話
- 独り言や心の中の声

### 丁寧語の段階

日本語の丁寧さには段階があります：

1. **普通体**：食べる、行く、いい（最もカジュアル）
2. **丁寧体**：食べます、行きます、いいです（一般的な丁寧表現）
3. **より丁寧**：存じます、参ります、よろしゅうございます（フォーマル）

### よくある間違い

**間違い1：丁寧語の過剰使用**
- ❌ お水をお飲みになります（尊敬語が必要な場面で丁寧語のみ）
- ✅ お水をお飲みになります（上司など目上の人には尊敬語を）

**間違い2：形容詞の丁寧形**
- ❌ 寒いでした（い形容詞の過去丁寧形の誤り）
- ✅ 寒かったです

**間違い3：美化語の付け方**
- ❌ ご水（和語に「ご」を付ける誤り）
- ✅ お水

### 学習のポイント

1. **N5段階**：です・ます形を確実に使えるようにする
2. **N4段階**：ございます、美化語の使い分けを学ぶ
3. **N3以降**：場面に応じた丁寧さの調整ができるようになる

---

## 註解

丁寧語は敬語の中で最も基本的かつ重要な表現です。日常会話からビジネスまで幅広く使用されるため、まず丁寧語を確実にマスターしてから尊敬語・謙讓語の学習に進むことが推奨されます。
