## 日文

自他動詞対応（じたどうしたいおう）

**羅馬拼音**：jita doushi taiou

**文法類型**：動詞分類系統

### 基本定義

**自動詞**（じどうし）：表示主語自發動作或狀態變化的動詞，不需要受詞。

**他動詞**（たどうし）：表示主語對對象施加動作的動詞，需要受詞。

**対応**：許多日語動詞都有自動詞和他動詞成對出現的現象，表達同一事件的不同視角。

## 日文解釋

日本語には、同じ事柄を「自発的な変化」と「意図的な動作」の両方の視点から表現できる動詞の対応関係があります。これを「自他動詞の対」といいます。

例えば、「開く」（自動詞）と「開ける」（他動詞）は対応関係にあります。「ドアが開く」は自発的な変化を表し、「ドアを開ける」は誰かが意図的に行う動作を表します。

この対応関係を理解することは、日本語の文構造と助詞の使い分け（「が」と「を」）を正しく使うために非常に重要です。多くの学習者は、この概念を母語に持たないため、習得に時間がかかります。

自他動詞の対応には一定のパターンがありますが、完全に規則的ではありません。そのため、対応する動詞のペアを個別に学習する必要があります。重要なのは、文の視点（誰が動作の主体か）によって、自動詞と他動詞を使い分けることです。

## 英文解釋

Japanese has a systematic relationship between intransitive and transitive verbs that express the same event from different perspectives. This is called "jita doushi taiou" (intransitive-transitive verb pairs).

For example, "aku" (to open, intransitive) and "akeru" (to open, transitive) form a pair. "The door opens" uses the intransitive form, while "Someone opens the door" uses the transitive form. This distinction is crucial because it affects not only verb choice but also particle usage ("ga" vs "wo").

Unlike English, where many verbs can be both transitive and intransitive (e.g., "The door opened" / "I opened the door"), Japanese requires different verb forms. Understanding these pairs is essential for constructing grammatically correct sentences and expressing agency or spontaneous change accurately.

The pairing patterns are somewhat predictable but not entirely regular, requiring learners to memorize common pairs while recognizing formation patterns.

## 中文解釋

日語中存在一種系統性的動詞配對關係，稱為「自他動詞対応」。同一個事件可以用自動詞（表示自發變化）或他動詞（表示有意動作）來表達不同的視角。

例如：「開く」（自動詞，開啟）和「開ける」（他動詞，打開）是一組對應動詞。「ドアが開く」表示門自己開了，「ドアを開ける」表示某人打開門。這個區別不僅影響動詞選擇，還影響助詞的使用（「が」對「を」）。

與中文和英文不同，日語要求使用不同的動詞形式來表達及物和不及物的概念。理解這些配對關係對於正確構建句子、準確表達動作主體或自發變化至關重要。

配對模式有一定規律可循，但並非完全規則，學習者需要記憶常見配對，同時掌握構詞規律。

## 核心用法

### 用法 1：自動詞 - 表示自發變化（主語＋が）

自動詞描述事物自行發生的變化或狀態，不涉及外力，使用助詞「が」標示主語。

**例句 1**
```
窓が開いた。
The window opened.
窗戶打開了。
```

**例句 2**
```
電気が消えた。
The light went out.
電燈熄了。
```

### 用法 2：他動詞 - 表示有意動作（主語＋を）

他動詞描述某人有意對對象施加的動作，使用助詞「を」標示受詞。

**例句 3**
```
田中さんが窓を開けた。
Tanaka-san opened the window.
田中先生打開了窗戶。
```

**例句 4**
```
私が電気を消した。
I turned off the light.
我關了電燈。
```

### 用法 3：視角轉換 - 同一事件的不同表達

根據要強調的重點（自發變化或有意動作），選擇自動詞或他動詞。

**例句 5**
```
ドアが閉まっています。（狀態描述）
The door is closed.
門關著。

誰かがドアを閉めました。（動作者強調）
Someone closed the door.
有人關了門。
```

## 文法規則

### 規則 1：常見配對模式

雖然不完全規則，但許多自他動詞配對有可預測的形態變化模式：

| 模式 | 自動詞 | 他動詞 | 例子 |
|------|--------|--------|------|
| -u / -eru | 開く (aku) | 開ける (akeru) | 開啟 |
| -eru / -asu | 消える (kieru) | 消す (kesu) | 消失/消除 |
| -aru / -eru | 始まる (hajimaru) | 始める (hajimeru) | 開始 |
| -reru / -su | 壊れる (kowareru) | 壊す (kowasu) | 破壞 |

### 規則 2：助詞的對應使用

- **自動詞**：主語使用「が」
  - ドア**が**開く（門打開）

- **他動詞**：受詞使用「を」，動作者使用「が」或「は」
  - 田中さん**が**ドア**を**開ける（田中打開門）

### 規則 3：不規則配對

有些動詞的自他配對不遵循常見模式，需要個別記憶：

- 出る (deru, 自) / 出す (dasu, 他) - 出來/拿出
- 入る (hairu, 自) / 入れる (ireru, 他) - 進入/放入
- 落ちる (ochiru, 自) / 落とす (otosu, 他) - 掉落/使掉落

### 規則 4：只有自動詞或他動詞的動詞

並非所有動詞都有配對，有些只有自動詞或他動詞：

- **只有自動詞**：ある (存在)、いる (存在-生物)
- **只有他動詞**：食べる (吃)、読む (讀)

## 常見錯誤

### 錯誤 1：助詞使用混淆

❌ 誤：窓を開いた。
✅ 正：窓が開いた。
說明：自動詞「開く」必須使用助詞「が」，不能使用「を」。「を」只能用於他動詞。

### 錯誤 2：自他動詞選擇錯誤

❌ 誤：電気が消した。
✅ 正：電気が消えた。（自發）或 電気を消した。（有意）
說明：「消す」是他動詞，需要受詞「を」。如果主語是「電気」，應該使用自動詞「消える」。

### 錯誤 3：忽略動作者

❌ 誤：ドアを閉まった。
✅ 正：ドアが閉まった。（自發）或 私がドアを閉めた。（有意）
說明：他動詞「閉める」需要明確的動作者。如果只想描述門關上的狀態，應使用自動詞「閉まる」。

## 學習要點

1. **理解視角差異**：自動詞強調自發變化，他動詞強調有意動作和動作者。

2. **記憶常見配對**：優先學習日常高頻使用的自他動詞配對（開く/開ける、閉まる/閉める、消える/消す等）。

3. **掌握助詞規則**：自動詞用「が」，他動詞用「を」，這是最基本的區別標誌。

4. **識別配對模式**：雖然有例外，但認識常見的形態變化模式有助於學習新詞彙。

5. **實際應用練習**：通過造句練習，培養根據情境正確選擇自動詞或他動詞的語感。
