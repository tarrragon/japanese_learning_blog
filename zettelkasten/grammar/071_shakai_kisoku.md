# 社会規則（Social Rules）

## 日文

社会規則（しゃかいきそく）

**羅馬拼音**：Shakai Kisoku

## 日文解釋

社会規則とは、社会生活を営む上で守るべきルールや規範の総称である。法律のような明文化された規則から、暗黙の了解やマナーまで幅広い範囲を含む。日本語では、社会規則に関連する表現は義務・禁止・許可などの文法形式と密接に結びついており、「〜なければならない」「〜てはいけない」「〜べきだ」などの形式を用いて表現される。

社会規則は、集団の秩序を維持し、円滑な人間関係を築くために不可欠な要素である。特に日本社会においては、明確な規則だけでなく、場の空気を読むこと（空気を読む）や暗黙のルール（暗黙の了解）も重要視される。これらの規則を適切に理解し、従うことは、社会人として必須のスキルとされている。

## 英文解釋

Social rules (shakai kisoku) refer to the collective rules and norms that should be followed in social life. They encompass a wide range from codified rules like laws to implicit understandings and etiquette. In Japanese, expressions related to social rules are closely tied to grammatical forms expressing obligation, prohibition, and permission, using patterns such as "〜nakereba naranai" (must), "〜te wa ikenai" (must not), and "〜beki da" (should).

Social rules are essential elements for maintaining group order and building smooth interpersonal relationships. In Japanese society particularly, not only explicit rules but also reading the atmosphere (kuuki wo yomu) and implicit understandings (anmoku no ryoukai) are highly valued. Understanding and following these rules appropriately is considered an essential skill for members of society.

## 中文解釋

社會規則是指在社會生活中應當遵守的規則和規範的總稱。涵蓋範圍從法律等明文規定到心照不宣的默契和禮儀。在日文中，與社會規則相關的表達與表示義務、禁止、許可等的文法形式密切相關，使用「〜なければならない」（必須）、「〜てはいけない」（禁止）、「〜べきだ」（應該）等形式來表達。

社會規則是維持團體秩序、建立圓滑人際關係不可或缺的要素。特別是在日本社會中，不僅重視明確的規則，還非常看重察言觀色（空気を読む）和不言自明的規則（暗黙の了解）。適當地理解並遵守這些規則，被視為社會人士必備的技能。

## 核心用法

### 用法 1：明示的規則（明確規則）

使用「〜なければならない」表達必須遵守的規則。

**例句 1**
```
電車の中では携帯電話をマナーモードにしなければなりません。
On trains, mobile phones must be set to silent mode.
在電車上必須將手機設為靜音模式。
```

**例句 2**
```
ゴミは分別して捨てなければならない。
Garbage must be sorted before disposal.
垃圾必須分類丟棄。
```

### 用法 2：禁止事項（禁止規則）

使用「〜てはいけない」「〜てはならない」表達禁止行為。

**例句 3**
```
公共の場所で大声で話してはいけません。
You must not speak loudly in public places.
在公共場所禁止大聲說話。
```

**例句 4**
```
未成年者は酒を飲んではならない。
Minors must not drink alcohol.
未成年人禁止飲酒。
```

### 用法 3：規範的勸告（規範性建議）

使用「〜べきだ」表達社會期望或道德規範。

**例句 5**
```
社会人として、時間を守るべきです。
As a member of society, you should be punctual.
作為社會人士，應該遵守時間。
```

## 文法規則

### 規則 1：義務與禁止的對應

社會規則的表達可分為積極義務和消極禁止：

**積極義務（必須做）**：
- 〜なければならない
- 〜なくてはいけない
- 〜べきだ

**消極禁止（不可做）**：
- 〜てはいけない
- 〜てはならない
- 〜べきではない

### 規則 2：形式程度的選擇

根據場合正式程度選擇適當的表達：

**正式場合**：
```
〜なければなりません（丁寧語）
〜てはなりません（丁寧語）
```

**一般場合**：
```
〜なければならない（常體）
〜てはいけない（常體）
```

**口語**：
```
〜なきゃ（ダメ）
〜ちゃダメ
```

### 規則 3：規則來源的區分

| 規則類型 | 表達傾向 | 例子 |
|---------|---------|------|
| 法律規則 | 〜てはならない | 未成年者は飲酒してはならない |
| 社會常識 | 〜なければならない | 約束は守らなければならない |
| 道德規範 | 〜べきだ | 人には親切にするべきだ |
| 禮儀規範 | 〜ほうがいい | 敬語を使ったほうがいい |

## 常見錯誤

### 錯誤 1：規則強度選擇不當

❌ 誤：静かにしたほうがいいですよ。（在禁止喧嘩的場所）
✅ 正：静かにしなければなりません。

說明：在有明確規定的場所，應使用表示義務的「なければならない」而非建議的「ほうがいい」。

### 錯誤 2：正式度不匹配

❌ 誤：ここで煙草を吸っちゃダメ。（在正式告示中）
✅ 正：ここでの喫煙は禁止されています。

說明：正式告示應使用書面語，口語形式「ちゃダメ」不適當。

### 錯誤 3：過度強調

❌ 誤：電車で席を譲らなければならない。（對一般乘客）
✅ 正：電車で席を譲ったほうがいいですよ。

說明：讓座是美德而非強制規則，使用「ほうがいい」較為適當。

## 學習要點

1. **義務層級**：區分強制義務（法律）、社會義務（常識）、道德義務（規範）、建議（禮貌）
2. **場合適配**：根據正式程度選擇適當的語法形式
3. **文化理解**：認識日本社會中明示規則與暗示規則的差異
4. **雙向表達**：掌握義務與禁止的對應表達
5. **語氣把握**：避免過度或不足的義務表達
