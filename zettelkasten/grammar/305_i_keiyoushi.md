## 日文

い形容詞（いけいようし）/ い系形容詞

**羅馬拼音**: i-keiyoushi

**別名**: True adjectives、i-adjectives、descriptive adjectives

**類型**：品詞分類・形容詞

## 日文解釋

い形容詞（い系形容詞）は、日本語の形容詞の中で最も重要で基本的な類型で、すべてが「い」で終わる形容詞です。これらは特に動詞や名詞と異なり、時制や数などの活用を持つ独立した語です。

### 基本特徴

**語尾が「い」で終わる**
い形容詞の最大の特徴は、その名前の通り、すべての辞書形が「い」で終わることです。例えば「赤い」「美しい」「新しい」などです。

**活用能力を持つ**
い形容詞は、動詞と同じように活用形を持つ独立した品詞です。時制（現在形・過去形）や否定形など、複数の活用形を持ちます。

**述語機能**
い形容詞は直接述語として機能できます。例えば「このコンピュータは新しい」では、「新しい」が述語として機能しています。

### 識別方法

1. 辞書形が「〜い」で終わる
2. 後ろに「です」を付けてい形容詞の述語文を作成できる
3. 活用形の変化パターンに従う

### 活用パターン

い形容詞の活用は非常に規則的です：

| 活用形 | 規則 | 例：赤い |
|--------|------|---------|
| 辞書形 | 基本形 | 赤い |
| ます形相当 | い → い | 赤いです |
| て形 | い → くて | 赤くて |
| た形 | い → かった | 赤かった |
| ない形相当 | い → くない | 赤くない |
| 命令形 | い → くしろ | 赤くしろ |

### 特殊ない形容詞

**古い形容詞**：「いい」「よい」
- 「いい」は不規則活用で、部分的に「よい」の形式を使用する場合があります
- 例：よかった（「いい」の過去形）

**色彩形容詞**：
- 赤い（red）、青い（blue）、黄色い（yellow）、緑色い（greenish）など

## 英文解釋

The i-adjectives (i-keiyoushi) are the most fundamental and important type of adjectives in Japanese. They are called "i-adjectives" because all dictionary forms end in the character "i" (い).

### Key Characteristics

**Ending in "i"**
The defining feature of i-adjectives is that their dictionary forms all end in "い". Examples include "akai" (red), "utsukushii" (beautiful), and "atarashii" (new).

**Conjugation Ability**
Unlike certain other word types, i-adjectives are independent parts of speech with their own conjugation patterns. They can conjugate for tense (present/past) and negation, similar to verbs.

**Predicate Function**
I-adjectives can function directly as predicates in sentences. For example, in "Kono konpyūta wa atarashii" (This computer is new), "atarashii" functions as the main predicate without requiring a copula.

### Identification Method

1. Dictionary form ends in "〜い"
2. Can form a complete predicate sentence with "です" attached
3. Follows regular i-adjective conjugation patterns

### Conjugation Patterns

I-adjective conjugation is very regular:

| Form | Rule | Example: akai (red) |
|------|------|-----------|
| Dictionary | basic form | 赤い |
| Polite (です form) | い → いです | 赤いです |
| Te-form | い → くて | 赤くて |
| Past | い → かった | 赤かった |
| Negative | い → くない | 赤くない |
| Imperative | い → くしろ | 赤くしろ |

### Special I-Adjectives

**Irregular: いい / よい**
- "Ii" (good) conjugates irregularly, sometimes using "yoi" forms
- Example: Yokatta (was good - past tense)

**Color Adjectives**
- akai (red), aoi (blue), kiiroi (yellow), midoriiroi (greenish)

## 中文解釋

い形容詞（い系形容詞）是日語中最基本、最重要的形容詞類型。之所以稱為「い形容詞」，是因為所有的辭書形都以「い」（い）結尾。

### 基本特徵

**以い結尾**
い形容詞的決定性特徵是所有辭書形都以「い」結尾。例如「赤い」（紅色的）、「美しい」（美麗的）、「新しい」（新的）等。

**具有活用能力**
不同於某些詞語類型，い形容詞是獨立的詞類，具有自己的活用模式。它們可以根據時態（現在式/過去式）和否定形式進行變化，類似於動詞。

**述語功能**
い形容詞可以直接作為句子的述語。例如在「このコンピュータは新しい」（這台電腦是新的）中，「新しい」直接作為述語，無需使用系動詞。

### 識別方法

1. 辭書形以「〜い」結尾
2. 可以加「です」形成完整的述語句子
3. 遵循規則的い形容詞活用規則

### 活用規則

い形容詞的活用非常規則：

| 活用形 | 規則 | 例：赤い |
|--------|------|---------|
| 辭書形 | 基本形 | 赤い |
| です形 | い → いです | 赤いです |
| て形 | い → くて | 赤くて |
| た形 | い → かった | 赤かった |
| ない形 | い → くない | 赤くない |
| 命令形 | い → くしろ | 赤くしろ |

### 特殊い形容詞

**不規則：いい/よい**
- 「いい」（好的）以不規則方式活用，有時使用「よい」的形式
- 例：よかった（過去形 - 是好的）

**色彩形容詞**
- 赤い（紅色）、青い（藍色）、黄色い（黃色）、緑色い（帶綠色）

## 例句

**例句1：基本述語用法**
```
この花は美しいです。
This flower is beautiful.
這朵花很美麗。
```

**例句2：て形連接**
```
赤くて大きいりんごが好きです。
I like apples that are red and big.
我喜歡又紅又大的蘋果。
```

**例句3：過去形**
```
昨日のテストは難しかった。
Yesterday's test was difficult.
昨天的考試很難。
```

**例句4：否定形**
```
この映画は面白くない。
This movie is not interesting.
這部電影不有趣。
```

## 補充說明

### 對比：い形容詞 vs な形容詞

| 特徵 | い形容詞 | な形容詞 |
|------|---------|---------|
| 辭書形結尾 | 〜い | 〜(な) |
| 述語形式 | 直接接い | 需接な或です |
| て形 | い → くて | な → で |
| た形 | い → かった | な → だった |
| ない形 | い → くない | な → でない |
| 活用複雑性 | 相對規則 | 相對簡單 |
| 修飾名詞 | 直接連接 | 需加な |

### 不規則い形容詞

部分い形容詞有不規則活用：

| 形容詞 | 規則性 | 過去形 | 說明 |
|--------|--------|--------|------|
| いい | 不規則 | よかった | 使用「よ」系列 |
| ほしい | 半規則 | ほしかった | 活用基本規則 |
| 怖い | 規則 | 怖かった | 標準規則活用 |

### 形容詞的修飾功能

い形容詞可以直接修飾名詞，不需要添加額外的詞：
- 新しい本 (new book)
- 大きい家 (big house)
- 面白い話 (interesting story)
