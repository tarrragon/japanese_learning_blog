## 日文

〜てから

### 文法情報

| 項目 | 內容 |
|------|------|
| 文法類型 | 時間表達句型 |
| JLPT | N5 |
| 使用場面 | 日常會話、書面 |
| 形成 | 動詞て形 + から |

### 形成規則

**基本形式**
```
動詞て形 + から
```

**範例**
```
食べる → 食べてから（吃完之後）
見る → 見てから（看完之後）
行く → 行ってから（去了之後）
する → してから（做完之後）
来る → 来てから（來了之後）
```

## 日文解釋

「〜てから」は、動詞のて形に「から」を接続した時間表現で、**ある動作が完了してから次の動作を行う**ことを表します。この表現の最大の特徴は、**動作の順序性と連続性**を強調することにあります。

この文法には三つの重要な特徴があります。第一に、前の動作が**完全に終了してから**次の動作に移ることを明確に示します。第二に、二つの動作の間に**時間的な間隔があまりない**というニュアンスを含みます。第三に、前の動作が次の動作の**必要条件や前提**となる場合によく使われます。

例えば、「手を洗ってから、食事をします」という文は、手を洗う動作が完全に終わってから食事を始めるという順序を明確に示しています。この場合、手を洗うことは食事をするための前提条件として理解されます。

類似表現の「〜た後で」と比べると、「〜てから」の方が**動作の連続性が強く**、二つの動作が密接に関連していることを示します。また、日常会話では「〜てから」の方が**より自然で一般的**に使われる傾向があります。

さらに、「〜てから」には**「〜して以来」という意味の用法**もあります。「日本に来てから3年になります」のように、ある出来事からの経過時間を表すことができます。

N5レベルの基本文法として、日常生活の様々な場面で頻繁に使用される重要な表現です。

## 英文解釋

"〜てから" (te kara) is a time expression formed by adding "kara" to the te-form of a verb, expressing that **one action is completed before another action begins**. The key characteristic of this pattern is its emphasis on **sequential order and continuity** between actions.

This grammar pattern has three important features. First, it clearly indicates that the first action is **completely finished** before moving to the second action. Second, it carries a nuance that there is **little time gap** between the two actions. Third, it is commonly used when the first action serves as a **prerequisite or necessary condition** for the second action.

For example, "手を洗ってから、食事をします" (te o aratte kara, shokuji o shimasu) clearly shows the sequence: washing hands is completely finished before eating begins. In this case, washing hands is understood as a prerequisite for eating.

Compared to the similar expression "〜た後で" (ta ato de), "〜てから" has a **stronger sense of continuity** between actions and suggests that the two actions are closely related. In everyday conversation, "〜てから" tends to be **more natural and commonly used**.

Additionally, "〜てから" has a usage meaning **"since (doing something)"**. As in "日本に来てから3年になります" (Nihon ni kite kara san-nen ni narimasu), it can express the time elapsed since a certain event.

As a fundamental N5-level grammar pattern, it is an essential expression frequently used in various everyday situations.

## 中文解釋

「〜てから」是將動詞的て形加上「から」所形成的時間表達，表示**一個動作完成後才進行下一個動作**。這個表達的最大特點是強調**動作的順序性和連續性**。

這個文法有三個重要特徵。第一，明確表示前一個動作**完全結束後**才進入下一個動作。第二，含有兩個動作之間**時間間隔不長**的語感。第三，常用於前一個動作是下一個動作的**必要條件或前提**的情況。

例如，「手を洗ってから、食事をします」這句話明確表示洗手的動作完全結束後才開始用餐的順序。在這種情況下，洗手被理解為用餐的前提條件。

與類似表達「〜た後で」相比，「〜てから」的**動作連續性更強**，表示兩個動作密切相關。而且在日常會話中，「〜てから」往往**更自然、更常用**。

此外，「〜てから」還有**「自從...以來」的用法**。如「日本に来てから3年になります」，可以表示某個事件發生以來經過的時間。

作為 N5 級別的基本文法，這是日常生活各種場合中頻繁使用的重要表達。

## 核心用法

### 用法 1：表示動作的先後順序

前一個動作完成後，接著進行下一個動作。

**例句 1**
```
朝起きてから、顔を洗います。
I wash my face after waking up in the morning.
早上起床後洗臉。
```

**例句 2**
```
宿題をしてから、遊びに行きます。
I'll go out to play after finishing my homework.
做完作業後去玩。
```

### 用法 2：表示必要的前提條件

前一個動作是進行下一個動作的必要條件。

**例句 3**
```
よく考えてから、返事をしてください。
Please think carefully before giving an answer.
請好好考慮之後再回覆。
```

**例句 4**
```
医者に相談してから、薬を飲んだ方がいいです。
You should consult a doctor before taking medicine.
最好諮詢醫生之後再吃藥。
```

### 用法 3：表示「自從...以來」的時間經過

表示從某個動作發生以來經過的時間。

**例句 5**
```
日本に来てから、もう5年になります。
It's already been 5 years since I came to Japan.
來日本已經5年了。
```

## 文法規則

### 規則 1：必須使用動詞て形

「〜てから」只能接在動詞的て形後面，不能使用其他形式。

**正確形式**
```
✅ 食べてから（食べる的て形）
✅ 見てから（見る的て形）
✅ 行ってから（行く的て形）
```

**錯誤形式**
```
❌ 食べるから（辭書形 + から = 因為，意思完全不同）
❌ 食べたから（た形 + から = 因為...了，表示原因）
```

### 規則 2：強調動作的緊密連續性

「〜てから」暗示兩個動作之間沒有太長的時間間隔，且第二個動作通常是第一個動作的自然延續。

```
手を洗ってから、ご飯を食べます。
（洗手後立即吃飯，兩個動作緊密相連）
```

### 規則 3：與「から」的其他用法區分

要注意「から」可以有不同的意思，取決於前面的動詞形式：
- **て形 + から** = 「...之後」（時間順序）
- **普通形/た形 + から** = 「因為...」（原因理由）

```
食べてから行きます。（吃完之後去）← 時間順序
食べたから満腹です。（因為吃了所以飽了）← 原因理由
```

## 常見錯誤

### 錯誤 1：混淆時間用法和原因用法

❌ 誤：お腹が空いたから、ご飯を食べます。（這是原因，不是時間順序）
✅ 正：お腹が空いているから、ご飯を食べます。（因為餓了所以吃飯）
✅ 正：手を洗ってから、ご飯を食べます。（洗手後吃飯）
說明：「た形 + から」表示原因，「て形 + から」表示時間順序。要根據想表達的意思選擇正確形式。

### 錯誤 2：て形變化錯誤

❌ 誤：飲むてから、寝ます。
✅ 正：飲んでから、寝ます。
說明：「飲む」的て形是「飲んで」，不是「飲むて」。必須正確掌握て形的變化規則。

### 錯誤 3：用於時間間隔過長的情況

❌ 誤：大学を卒業してから、結婚しました。（不自然）
✅ 正：大学を卒業した後で、結婚しました。
✅ 正：大学を卒業してから5年後に、結婚しました。（加上具體時間就可以）
說明：「〜てから」通常用於動作緊密相連的情況。如果兩個動作之間有較長時間間隔，使用「〜た後で」更自然。

## 學習要點

1. **掌握て形變化**：因為「〜てから」使用て形，所以必須熟練掌握各類動詞的て形變化規則
2. **理解順序性**：重點在於表達「先做A，再做B」的明確順序關係
3. **區分から的用法**：要能區分「て形+から」（時間）和「た形+から」（原因）的不同
4. **注意動作連續性**：適用於兩個動作緊密相連的情況，不適合時間間隔過長的場合
5. **掌握雙重用法**：既可表示動作順序，也可表示「自從...以來」的時間經過
