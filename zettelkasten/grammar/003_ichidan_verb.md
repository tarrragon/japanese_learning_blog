## 日文
一段動詞（いちだんどうし）/ る動詞

**羅馬拼音**: ichidan doushi / ru-doushi

**別名**: Group II verbs（第二類動詞）、ru-verbs

**類型**：動詞分類・活用文法

## 日文解釋

一段動詞（る動詞）は、日本語動詞の三大分類の一つで、最も規則的で覚えやすい活用パターンを持つ動詞群です。

### 基本特徴

**語幹が不変**
一段動詞の最大の特徴は、語幹が決して変化しないことです。例えば「食べる」の場合、どんな活用形でも「食べ」という語幹は常に同じままです。変化するのは語尾の「る」の部分だけです。

**見分け方**
1. 辞書形が「〜る」で終わる
2. 「る」の直前が「え段」(e)または「い段」(i)の音
   - 例：たべる（taberu）、みる（miru）、ねる（neru）

**注意**: すべての「〜る」で終わる動詞が一段動詞とは限りません。「帰る」「走る」など、「る」の前が「あ・う・お段」の場合は五段動詞です。

### 活用パターン

一段動詞の活用は非常にシンプルです：

| 活用形 | 規則 | 例：食べる |
|--------|------|-----------|
| 辞書形 | 語幹 + る | 食べる |
| ます形 | 語幹 + ます | 食べます |
| て形 | 語幹 + て | 食べて |
| た形 | 語幹 + た | 食べた |
| ない形 | 語幹 + ない | 食べない |
| 可能形 | 語幹 + られる | 食べられる |
| 受身形 | 語幹 + られる | 食べられる |
| 使役形 | 語幹 + させる | 食べさせる |
| 命令形 | 語幹 + ろ/よ | 食べろ |
| 意志形 | 語幹 + よう | 食べよう |

すべての活用で語幹「食べ」は変わりません。

### 使用頻度

日本語には一段動詞が約200個あり、五段動詞（約4000個）より少ないですが、日常会話で非常によく使われる基本動詞が多く含まれています。

## 英文解釋

Ichidan verbs (also called ru-verbs or Group II verbs) are one of the three major verb categories in Japanese and have the most regular conjugation pattern, making them the easiest to learn.

### Key Characteristics

**Unchanging Stem**
The defining feature of ichidan verbs is that the stem never changes. For example, with "taberu" (to eat), the stem "tabe" remains constant in all conjugated forms. Only the "ru" ending changes.

**Identification Rules**
1. Dictionary form ends in "~ru"
2. The syllable immediately before "ru" is in the "e-row" (e) or "i-row" (i)
   - Examples: taberu (eat), miru (see), neru (sleep)

**Important**: Not all verbs ending in "~ru" are ichidan verbs. Verbs like "kaeru" (return) and "hashiru" (run), where the syllable before "ru" is in the "a," "u," or "o" rows, are godan verbs.

### Conjugation Pattern

Ichidan verb conjugation is very straightforward:

| Form | Rule | Example: taberu |
|------|------|----------------|
| Dictionary | stem + ru | taberu |
| Masu | stem + masu | tabemasu |
| Te | stem + te | tabete |
| Ta | stem + ta | tabeta |
| Nai | stem + nai | tabenai |
| Potential | stem + rareru | taberareru |
| Passive | stem + rareru | taberareru |
| Causative | stem + saseru | tabesaseru |
| Imperative | stem + ro/yo | tabero |
| Volitional | stem + you | tabeyou |

The stem "tabe" never changes across all forms.

### Usage Frequency

There are approximately 200 ichidan verbs in Japanese, far fewer than godan verbs (about 4,000), but many fundamental daily-use verbs are ichidan verbs.

## 中文解釋

一段動詞（又稱る動詞或第二類動詞）是日語動詞三大分類之一，擁有最規則的變化模式，是最容易學習的動詞類型。

### 基本特徵

**詞幹不變**
一段動詞最大的特點是詞幹永遠不變。例如「食べる」（吃），無論如何變化，詞幹「食べ」都保持不變，只有詞尾「る」會變化。

**識別方法**
1. 辭書形以「〜る」結尾
2. 「る」前面是「え段」(e)或「い段」(i)的音
   - 例如：たべる（taberu）、みる（miru）、ねる（neru）

**注意**：不是所有以「〜る」結尾的動詞都是一段動詞。「帰る」「走る」等「る」前面是「あ・う・お段」的動詞是五段動詞。

### 變化規則

一段動詞的變化非常簡單：

| 變化形 | 規則 | 例：食べる |
|--------|------|-----------|
| 辭書形 | 詞幹 + る | 食べる |
| ます形 | 詞幹 + ます | 食べます |
| て形 | 詞幹 + て | 食べて |
| た形 | 詞幹 + た | 食べた |
| ない形 | 詞幹 + ない | 食べない |
| 可能形 | 詞幹 + られる | 食べられる |
| 被動形 | 詞幹 + られる | 食べられる |
| 使役形 | 詞幹 + させる | 食べさせる |
| 命令形 | 詞幹 + ろ/よ | 食べろ |
| 意志形 | 詞幹 + よう | 食べよう |

所有變化中詞幹「食べ」都不改變。

### 使用頻率

日語中約有200個一段動詞，雖然比五段動詞（約4000個）少得多，但其中包含許多日常生活中極為常用的基本動詞。

## 例句

**例句1：基本用法**
```
毎朝7時に起きます。
I wake up at 7 o'clock every morning.
我每天早上7點起床。
```

**例句2：て形連接**
```
テレビを見て、ご飯を食べます。
I watch TV and then eat.
我看電視然後吃飯。
```

**例句3：ない形否定**
```
今日は肉を食べません。
I won't eat meat today.
我今天不吃肉。
```

**例句4：過去形**
```
昨日新しい服を着ました。
I wore new clothes yesterday.
我昨天穿了新衣服。
```

## 補充說明

### 對比：一段動詞 vs 五段動詞

| 特徵 | 一段動詞 | 五段動詞 |
|------|---------|---------|
| 數量 | 約200個 | 約4000個 |
| 詞幹 | 永不變化 | 會變化 |
| 辭書形結尾 | 〜る（前面是e/i段） | 〜う、〜く、〜ぐ、〜す等 |
| て形規則 | 詞幹 + て | 複雜（6種規則） |
| ない形規則 | 詞幹 + ない | 詞尾變あ段 + ない |
| 學習難度 | 簡單 | 中等 |
| 例子 | 食べる、見る、寝る | 書く、話す、読む |

### 常見混淆：看起來像一段動詞的五段動詞

有些動詞以「〜る」結尾，但實際上是五段動詞：

| 動詞 | 類型 | 理由 |
|------|------|------|
| 帰る (kaeru) | 五段 | 「る」前是「e」但實際是五段 |
| 走る (hashiru) | 五段 | 「る」前是「i」但實際是五段 |
| 切る (kiru) | 五段 | 「る」前是「i」但實際是五段 |
| 知る (shiru) | 五段 | 「る」前是「i」但實際是五段 |

**識別技巧**：當無法確定時，查看ます形：
- 一段動詞：食べる → 食べます（直接加ます）
- 五段動詞：帰る → 帰ります（る變り）

### 可能形特殊性

一段動詞的可能形「られる」在口語中常簡化為「れる」：
- 正式：食べられる
- 口語：食べれる（俗稱「ら抜き言葉」）

雖然「ら抜き言葉」在口語中很常見，但書面語和正式場合應使用完整形式。
