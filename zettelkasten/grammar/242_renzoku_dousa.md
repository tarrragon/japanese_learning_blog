## 日文
連続動作表現（れんぞくどうさひょうげん）

**羅馬拼音**：renzoku dousa hyougen

## 日文解釋
連続動作表現とは、複数の動作が時間的に続けて行われることを表す文法構造です。日本語では主に三つの方法があります：

1. **て形接続**：動作が順番に起こることを表す最も基本的な方法です。「AてB」の形で、Aが終わってからBが起こることを示します。

2. **～たり～たりする**：いくつかの動作を例として挙げる表現です。すべての動作を列挙するのではなく、代表的な動作を示す時に使います。

3. **～てから**：動作の順序を明確に強調する表現です。「Aてから B」で、Aが完全に終わってからBを始めることを強調します。

これらの表現は日常会話から書き言葉まで幅広く使われ、動作の流れを自然に伝えるために不可欠です。

## 英文解釋
Sequential action expressions are grammatical structures used to describe multiple actions occurring in succession. Japanese has three main ways to express this:

1. **Te-form connection**: The most basic method to show actions happening in order. In the "A-te B" pattern, it indicates that B happens after A is completed.

2. **~tari ~tari suru**: An expression for listing several actions as examples. Used when showing representative actions rather than listing all actions.

3. **~te kara**: An expression that clearly emphasizes the order of actions. "A-te kara B" emphasizes that B begins only after A is completely finished.

These expressions are widely used from daily conversation to written language and are essential for naturally conveying the flow of actions.

## 中文解釋
連續動作表達是用來描述多個動作依次發生的文法結構。日文主要有三種表達方式：

1. **て形連接**：表示動作依序發生的最基本方法。「Aて B」的形式表示 A 結束後 B 才發生。

2. **～たり～たりする**：列舉幾個動作作為例子的表達。不是列舉所有動作，而是用於表示代表性的動作。

3. **～てから**：明確強調動作順序的表達。「Aてから B」強調 A 完全結束後才開始 B。

這些表達從日常對話到書面語都廣泛使用，對於自然傳達動作的流程不可或缺。

## 文法結構

### 1. て形連接
**動詞て形 + 動詞**
- 順序性：按照提及的順序發生
- 時間關係：前一動作結束後進行下一動作

### 2. ～たり～たりする
**動詞た形（去だ）+ り + 動詞た形（去だ）+ り + する**
- 列舉性：舉出幾個動作作為代表
- 暗示性：暗示還有其他動作

### 3. ～てから
**動詞て形 + から + 動詞**
- 強調性：強調前一動作必須完成
- 明確性：明確表示時間順序

## 用法比較

| 表達 | 特點 | 使用場合 |
|------|------|---------|
| て形連接 | 自然連接，順序不強調 | 日常敘述 |
| ～たり～たりする | 列舉代表動作 | 描述例行活動 |
| ～てから | 強調順序 | 強調先後順序時 |

## 例句

**例句1：て形連接**
```
朝起きて、顔を洗って、朝ごはんを食べます。
I get up, wash my face, and eat breakfast.
早上起床、洗臉、吃早餐。
```

**例句2：て形連接（複雜動作）**
```
電車に乗って、会社に行って、仕事を始めます。
I take the train, go to the office, and start work.
搭電車、去公司、開始工作。
```

**例句3：～たり～たりする**
```
休みの日は本を読んだり、映画を見たりします。
On my days off, I read books, watch movies, and so on.
休假時看書、看電影等等。
```

**例句4：～てから**
```
宿題を終わってから、ゲームをします。
I play games after finishing my homework.
做完作業後玩遊戲。
```

**例句5：比較**
```
手を洗ってご飯を食べる。（自然連接）
手を洗ってからご飯を食べる。（強調順序）
I wash my hands and eat. / I eat after washing my hands.
洗手吃飯。/ 洗完手後吃飯。
```

## 注意事項

### て形連接 vs ～てから
- **て形**：自然的動作連接，不特別強調順序
- **てから**：明確強調必須先完成前一動作

### ～たり～たりする 的使用
- 通常需要至少兩個「～たり」
- 句尾必須有「する」或其他動詞
- 表示列舉的動作是代表性的，暗示還有其他動作
