## 日文

〜そうだ（様態）

**讀音**：〜そうだ

**文法類型**：様態を表す助動詞[^auxiliary-verb]

### 文法情報

| 項目 | 內容 |
|------|------|
| 文法類型 | 様態助動詞 |
| JLPT | N4 |
| 使用場面 | 日常會話[^daily-conversation]、書面語 |
| 形成 | 動詞・形容詞の語幹 + そうだ |

### 形成規則

**基本形式**

1. **動詞**：ます形語幹 + そうだ
   ```
   降る → 降りそうだ（看起來要下雨）
   ```

2. **い形容詞**：語幹[^i-adj-stem] + そうだ
   ```
   おいしい → おいしそうだ（看起來好吃）
   美しい → 美しそうだ（看起來美麗）
   ```

3. **な形容詞**：語幹 + そうだ
   ```
   元気（な） → 元気そうだ（看起來有精神）
   便利（な） → 便利そうだ（看起來方便）
   ```

**特殊變化**

- **いい（よい）** → よさそうだ（不規則變化）
- **ない** → なさそうだ

## 日文解釋

「〜そうだ」は、視覚的な情報[^visual-information]に基づいて、物事の状態や様子を推測する様態助動詞[^auxiliary-verb]です。話し手が実際に目で見て感じた印象を表現する際に使用されます。

この文法は主に二つの意味を持ちます。一つ目は、物事の外見から判断した性質や状態を表す用法（「おいしそうだ」「楽しそうだ」など）。二つ目は、動作や変化が今にも起こりそうな様子を表す用法（「降りそうだ」「倒れそうだ」など）です。

重要な特徴として、「〜そうだ」は話し手の主観的な印象[^subjective-impression]に基づく表現であり、確実な事実ではなく、見た目からの推測を表します。そのため、実際の状態とは異なる場合もあります。また、この表現は五感[^five-senses]の中でも特に視覚情報を重視する日本語の特徴をよく示しています。

否定形「〜そうにない」「〜そうもない」は、その事態が起こる可能性が非常に低いことを表します。また、過去形「〜そうだった」は、過去のある時点での様子や印象を回想する際に使用されます。

## 英文解釋

"〜そうだ" (sou da) is an auxiliary verb[^auxiliary-verb] that expresses appearance or likelihood based on visual observation[^visual-information]. It indicates the speaker's judgment about a situation or state based on what they can see.

This grammar pattern has two main uses. First, it describes the appearance or perceived quality of something ("looks delicious," "seems fun"). Second, it indicates that an action or change is about to happen ("looks like it will rain," "seems about to fall").

The key characteristic is that "〜そうだ" expresses subjective impression[^subjective-impression] rather than objective fact. The speaker makes an inference based on visual cues, which may or may not match reality. This reflects the Japanese language's emphasis on visual perception among the five senses[^five-senses].

The negative form "〜そうにない" or "〜そうもない" indicates that something is very unlikely to happen. The past form "〜そうだった" recalls an impression or appearance from a past moment.

## 中文解釋

「〜そうだ」是様態助動詞[^auxiliary-verb]，用於根據視覺觀察[^visual-information]推測事物的狀態或外觀。它表達說話者根據所見做出的判斷。

這個文法模式有兩個主要用法。第一，描述事物的外觀或感知特質（「看起來好吃」、「看起來很開心」）。第二，表示某個動作或變化即將發生（「看起來要下雨」、「看起來要倒」）。

關鍵特徵是「〜そうだ」表達主觀印象[^subjective-impression]而非客觀事實。說話者根據視覺線索做出推測，這可能與實際情況相符或不符。這反映了日語在五感[^five-senses]中特別重視視覺感知的特點。

否定形式「〜そうにない」或「〜そうもない」表示某事發生的可能性極低。過去式「〜そうだった」用於回憶過去某個時點的樣子或印象。

## 核心用法

### 用法 1：描述物事的外觀或性質

基於視覺觀察，描述事物看起來具有某種性質或特徵。

**例句 1**
```
このケーキはおいしそうですね。
This cake looks delicious.
這個蛋糕看起來很好吃。
```

**例句 2**
```
彼女は楽しそうに笑っています。
She is laughing happily (looks happy).
她看起來笑得很開心。
```

### 用法 2：表示即將發生的動作或變化

根據當前狀態推測某個動作或變化即將發生。

**例句 3**
```
空が暗くなってきた。雨が降りそうだ。
The sky is getting dark. It looks like it's going to rain.
天空變暗了。看起來要下雨了。
```

**例句 4**
```
この木は古くて、倒れそうです。
This tree is old and looks like it might fall down.
這棵樹很舊，看起來快要倒了。
```

### 用法 3：否定形式表示可能性極低

**例句 5**
```
彼は約束を守りそうにない。
He doesn't look like he'll keep his promise.
他看起來不會遵守約定。
```

## 文法規則

### 規則 1：接續方式的特殊性

「〜そうだ」接續時需要注意語幹的取法：
- い形容詞去掉「い」：美しい → 美し + そうだ
- な形容詞直接加：元気 + そうだ
- 動詞用ます形語幹：降ります → 降り + そうだ

### 規則 2：不規則變化

「いい（よい）」的變化是不規則的：
- いい → よさそうだ（×いそうだ）
- ない → なさそうだ

### 規則 3：與「〜そうだ（伝聞）」的區別

様態的「〜そうだ」和傳聞[^hearsay]的「〜そうだ」是不同的文法：
- 様態：動詞ます形語幹 + そうだ（看起來...）
- 傳聞：普通形 + そうだ（聽說...）

### 規則 4：時態變化

「〜そうだ」本身可以變化時態：
- 現在：〜そうです
- 過去：〜そうでした
- 否定：〜そうにない、〜そうもない
- 過去否定：〜そうになかった

## 常見錯誤

### 錯誤 1：い形容詞的「い」未去除

❌ 誤：このりんごはおいしいそうです。
✅ 正：このりんごはおいしそうです。
說明：い形容詞接續「〜そうだ」時，必須去掉「い」。「おいしいそうだ」會被誤解為傳聞「聽說好吃」。

### 錯誤 2：「いい」的不規則變化

❌ 誤：この本はいそうですね。
✅ 正：この本はよさそうですね。
說明：「いい」接續「〜そうだ」時，必須用「よさそう」，不能用「いそう」。

### 錯誤 3：與傳聞「〜そうだ」混淆

❌ 誤：雨が降るそうだ。（想表達「看起來要下雨」）
✅ 正：雨が降りそうだ。
說明：「降るそうだ」是傳聞「聽說要下雨」，「降りそうだ」才是様態「看起來要下雨」。接續方式不同。

### 錯誤 4：用於不適合的感官

❌ 誤：この部屋は静かそうですね。（只通過聽覺判斷）
✅ 正：この部屋は静かですね。（直接感知）
說明：「〜そうだ」主要用於視覺推測。如果是直接通過聽覺感知到安靜，應該用直接表達。

## 學習要點

1. **視覺為主的推測表達**：「〜そうだ」主要根據視覺觀察做出推測，強調「看起來」的感覺

2. **主觀性質**：表達的是說話者的主觀印象，可能與實際情況不符（看起來好吃但實際不好吃也可能）

3. **兩大用法**：一是描述事物的外觀性質，二是表示即將發生的動作或變化

4. **接續規則**：い形容詞去「い」、な形容詞直接加、動詞用ます形語幹，需要熟記

5. **與傳聞區別**：様態「〜そうだ」和傳聞「〜そうだ」接續方式不同，意思也完全不同，學習時要特別注意

## 註解

[^auxiliary-verb]: **助動詞** - 附加在動詞、形容詞等後面，增添文法意義的詞類。様態助動詞表示事物的外觀或狀態。詳見 [助動詞](../auxiliary/001_auxiliary_verb.md)

[^daily-conversation]: **日常會話** - 朋友、家人、同事之間的非正式對話情境。這種場合常用様態表達來描述事物的外觀或即將發生的事。詳見 [日常會話](../context/daily-conversation.md)

[^i-adj-stem]: **い形容詞語幹** - い形容詞去掉語尾「い」後的部分。例如「美しい」的語幹是「美し」。詳見 [い形容詞](../concept/i-adjective-stem.md)

[^visual-information]: **視覚情報** - 通過眼睛觀察獲得的資訊。「〜そうだ」主要基於視覺線索做出判斷，是日語重視視覺感知的體現。詳見 [視覚情報](../concept/visual-information.md)

[^subjective-impression]: **主観的印象** - 基於個人感覺和判斷的印象，不一定反映客觀事實。「〜そうだ」表達的是說話者的主觀感受。詳見 [主観性](../concept/subjectivity.md)

[^five-senses]: **五感** - 視覚、聴覚、嗅覚、味覚、触覚等五種感覺。日語在表達推測時，不同文法對應不同感官。詳見 [五感表現](../concept/five-senses.md)

[^hearsay]: **伝聞** - 表示從他人處聽到的資訊。傳聞的「〜そうだ」與様態的「〜そうだ」是不同的文法形式。詳見 [伝聞表現](../grammar/hearsay-sou-da.md)
