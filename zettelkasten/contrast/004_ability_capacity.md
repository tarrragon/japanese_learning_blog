## 概述

「能力」和「容量」在中文都可譯為「能力」或「容量」，但在日文中有明確的使用區別。「能力」指的是做某事的力量或才能，強調主動的行為能力；「容量」則指容器或系統能夠容納的量，強調被動的承載限度。理解這兩個詞的差異，有助於準確表達不同類型的「能力」概念。

## 核心區別表

| 特徵 | 能力 (nouryoku) | 容量 (youryou) |
|-----|----------------|----------------|
| **基本含義** | 做某事的能力、才能 | 容納的量、容積 |
| **性質** | 主動的、動態的 | 被動的、靜態的 |
| **焦點** | 行為、表現、技能 | 限度、上限、空間 |
| **適用對象** | 人的才能、技能 | 容器、記憶體、系統 |
| **可否提升** | 可以訓練提升 | 固定或需擴充硬體 |
| **英文對應** | ability, capability, competence | capacity, volume, storage |
| **使用場景** | 人力資源、教育、職場 | 技術規格、物理空間 |

## 能力（のうりょく）

### 日文解釋

「能力」は、ある事を成し遂げることができる力や才能を指します。人間の知的能力[^chitekinou]、身体能力[^shintainou]、社会的能力[^shakaitekinou]など、主体的に何かを行う力を表す言葉です。教育現場[^kyouiku]や人材評価[^jinzai]でよく使用され、訓練や経験によって向上させることができる動的な概念です。能力開発[^nouryokukaihatsu]という言葉があるように、成長可能性を含んでいます。

### 英文解釋

"Nouryoku" refers to the power or talent to accomplish something. It represents active forces such as human intellectual ability[^chitekinou], physical ability[^shintainou], and social competence[^shakaitekinou]. Commonly used in educational settings[^kyouiku] and human resource evaluation[^jinzai], it is a dynamic concept that can be improved through training and experience. The term "ability development"[^nouryokukaihatsu] illustrates that it encompasses growth potential.

### 中文解釋

「能力」指完成某事的力量或才能。代表人的知識能力[^chitekinou]、身體能力[^shintainou]、社交能力[^shakaitekinou]等主動執行某事的力量。常用於教育現場[^kyouiku]和人才評估[^jinzai]，是可透過訓練和經驗提升的動態概念。如「能力開發」[^nouryokukaihatsu]一詞所示，包含成長的可能性。

### 能力的核心用法

#### 1. 人的才能與技能

表示個人具備的各種能力：

**例句 1**
```
彼女は語学能力が高い。
She has high language ability.
她的語言能力很高。
```

**例句 2**
```
問題解決能力を身につける必要がある。
It's necessary to acquire problem-solving skills.
需要培養解決問題的能力。
```

**例句 3**
```
リーダーシップ能力が求められる。
Leadership ability is required.
需要領導能力。
```

#### 2. 職場與人力資源

在工作環境中評估能力：

**例句 4**
```
応募者の能力を評価する。
Evaluate the candidate's capabilities.
評估應徵者的能力。
```

**例句 5**
```
この仕事には専門的な能力が必要です。
This job requires specialized skills.
這份工作需要專業能力。
```

#### 3. 能力開發與成長

強調能力可以培養提升：

**例句 6**
```
従業員の能力開発に投資する。
Invest in employee skill development.
投資於員工的能力開發。
```

**例句 7**
```
コミュニケーション能力を向上させる。
Improve communication skills.
提升溝通能力。
```

## 容量（ようりょう）

### 日文解釋

「容量」は、容器[^youki]や空間、システムなどが収容できる量を表します。物理的な体積[^taiseki]だけでなく、コンピュータのメモリ容量[^memory]、電池の容量[^battery]など、抽象的な容れ物の限界値も指します。容量は基本的に固定された値であり、増やすためには物理的な拡張[^kakuchou]が必要です。技術仕様[^gijutsu]や製品スペック[^spec]を説明する際に使用される静的な概念です。

### 英文解釋

"Youryou" refers to the amount that a container[^youki], space, or system can hold. It indicates not only physical volume[^taiseki] but also the maximum limits of abstract containers such as computer memory capacity[^memory] and battery capacity[^battery]. Capacity is fundamentally a fixed value, and physical expansion[^kakuchou] is required to increase it. It is a static concept used when describing technical specifications[^gijutsu] and product specs[^spec].

### 中文解釋

「容量」表示容器[^youki]、空間或系統能容納的量。不僅指物理體積[^taiseki]，也指電腦記憶體容量[^memory]、電池容量[^battery]等抽象容器的上限值。容量基本上是固定值，要增加需要物理擴充[^kakuchou]。是說明技術規格[^gijutsu]和產品規格[^spec]時使用的靜態概念。

### 容量的核心用法

#### 1. 物理容器的容積

表示實體容器能裝的量：

**例句 8**
```
このペットボトルの容量は500mlです。
This plastic bottle has a capacity of 500ml.
這個寶特瓶的容量是 500ml。
```

**例句 9**
```
冷蔵庫の容量が足りない。
The refrigerator capacity is insufficient.
冰箱容量不夠。
```

#### 2. 技術規格

說明電子設備或系統的容量：

**例句 10**
```
ハードディスクの容量を増やす。
Increase the hard disk capacity.
增加硬碟容量。
```

**例句 11**
```
このスマホは128GBの容量があります。
This smartphone has 128GB of storage capacity.
這支智慧型手機有 128GB 的容量。
```

**例句 12**
```
バッテリー容量が大きいので長時間使える。
Because of the large battery capacity, it can be used for a long time.
因為電池容量大，可以長時間使用。
```

#### 3. 空間的承載限度

表示場所或系統的最大容納量：

**例句 13**
```
このホールの収容容量は1000人です。
This hall has a seating capacity of 1000 people.
這個會堂的收容容量是 1000 人。
```

**例句 14**
```
サーバーの処理容量を超えた。
The server's processing capacity was exceeded.
超過了伺服器的處理容量。
```

## 常見錯誤與提醒

### 錯誤 1：混淆記憶相關用法

❌ 錯誤：彼の記憶容量は素晴らしい。
✅ 正確：彼の記憶能力は素晴らしい。
說明：人的記憶力是「能力」，不是「容量」。「記憶容量」只用於電腦、設備。

### 錯誤 2：設備規格使用錯誤詞彙

❌ 錯誤：このパソコンは高い処理能力があります。
✅ 正確：このパソコンは高い処理能力があります。（這個其實可以）
△ 更準確：このパソコンの処理能力が高い。/ CPUの性能が高い。
說明：電腦的處理能力可用「能力」，但談規格時更常用「性能」。

### 錯誤 3：容量用於抽象能力

❌ 不自然：彼の思考容量が大きい。
✅ 自然：彼の思考能力が高い。
說明：思考是主動的能力，應使用「能力」而非「容量」。

## 學習要點

1. **「能力」= 主動、動態**：可透過訓練提升的力量
2. **「容量」= 被動、靜態**：固定的容納限度
3. **人用「能力」，物用「容量」**：基本判斷原則
4. **記憶的特殊情況**：人的記憶力用「能力」，設備用「容量」
5. **英文對應不同**：ability vs capacity 在日文中有明確區分

## 常見問題 (FAQ)

**Q1: 「処理能力」和「処理容量」有什麼差別？**
A: 兩者都可用，但重點不同：
- 「処理能力」= 強調處理的性能、速度、效率
- 「処理容量」= 強調能處理的資料量上限

**Q2: 「記憶力」應該說「記憶能力」還是「記憶容量」？**
A: 看對象：
- 人的記憶力 = 「記憶能力」✅（記憶力が良い）
- 電腦的記憶體 = 「記憶容量」✅（メモリ容量が大きい）

**Q3: 「大容量」可以用來形容人嗎？**
A: 不行，「大容量」只用於物品、設備：
- 「大容量バッテリー」✅
- 「大容量の人」❌

**Q4: 「許容量」和「容量」有什麼關係？**
A: 「許容量」（きょようりょう）是允許的限度，更強調承受範圍：
- 「ストレスの許容量」= 壓力的承受限度
- 「容量」= 物理或技術上的容納量

## 記憶技巧

**能力**：
- 「能」= 能夠做到 → 主動的力量
- 聯想：人的「才能」

- 「容」= 容納、包容 → 被動的空間
- 聯想：容器的「容積」
- 關鍵詞：規格、上限、擴充

## 註解

[^chitekinou]: **知的能力** - 思考、學習、理解等智力相關的能力。包括邏輯思考、記憶力、判斷力等。

[^shintainou]: **身體能力** - 體力、運動能力、反應速度等身體相關的能力。

[^shakaitekinou]: **社會能力** - 溝通、協調、領導等與他人互動相關的能力。

[^kyouiku]: **教育現場** - 學校、培訓機構等進行教育活動的場所。

[^jinzai]: **人材評價** - 評估員工或應徵者的才能、技能、潛力的過程。

[^nouryokukaihatsu]: **能力開發** - 透過訓練、教育提升個人能力的活動。

[^youki]: **容器** - 用來盛裝液體或物品的器具。

[^taiseki]: **體積** - 物體占據空間的大小。

[^memory]: **記憶體容量** - 電腦或設備能儲存資料的量。以 GB、TB 等單位表示。

[^battery]: **電池容量** - 電池能儲存的電量。以 mAh 等單位表示。

[^kakuchou]: **擴張** - 增加容量或規模。例如增設記憶體、更換大容量電池等。

[^gijutsu]: **技術規格** - 產品或系統的技術性能指標。

[^spec]: **產品規格** - 產品的詳細性能參數和特性。
