## 解釋

### 日文
何らかの音・単語を発する。話す、述べる、表現する。

### English
To say, to tell, to speak, to express.

### 中文
說、講、表達。

## 用法

### 1. 基本用法：發言、說話
```
何か言いましたか。
Did you say something?
你說了什麼嗎？
```

### 2. 引用用法：と言う
```
彼は「行きたくない」と言った。
He said, "I don't want to go."
他說「不想去」。
```

### 3. 稱呼、叫做：という
```
これは日本語で何と言いますか。
What do you call this in Japanese?
這個用日文怎麼說？
```

### 4. 定義、所謂：という
```
東京という町はとても大きいです。
The city called Tokyo is very large.
叫做東京的城市非常大。
```

### 5. 表達意見
```
私は賛成だと言いたい。
I want to say that I agree.
我想說我贊成。
```

## 備註

- 漢字可寫作「言う」「云う」「謂う」，最常用「言う」
- 「と言う」用於直接引用或間接引用
- 「という」常縮約為「って」（口語）
- ワ行動詞，ない形為「言わない」（注意音便）
