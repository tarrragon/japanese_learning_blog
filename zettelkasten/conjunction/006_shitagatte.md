## 日文

したがって

**羅馬拼音**: shitagatte

**詞性**: 接続詞（連接詞）

## 日文解釋

「したがって」は、前の文で述べた事実や前提から論理的[^logic]に導かれる結論を表す接続詞[^conjunction]です。「従う」という動詞から派生した形で、「前の内容に従って、次のことが言える」という意味を持ちます。

非常にフォーマルな表現[^formal]で、主に書き言葉[^written]、特に学術論文[^academic]、ビジネス文書[^business]、公式発表などで使われます。日常会話[^casual]ではほとんど使われず、代わりに「だから」や「それで」が用いられます。

論理的な推論[^reasoning]や因果関係[^causality]を明確に示す必要がある場面で特に重要です。「AはBである。したがってCだ」という形で、前提から結論を導く論理展開[^logical-development]に使用されます。

## 英文解釋

"Shitagatte" is a formal conjunction[^conjunction] used to express a conclusion that logically[^logic] follows from facts or premises stated in the previous sentence. It derives from the verb "shitagau" (従う, to follow), conveying the meaning "following from what was said before, we can conclude that..."

This is a highly formal expression[^formal] primarily used in written language[^written], especially in academic papers[^academic], business documents[^business], and official announcements. It is rarely used in everyday conversation[^casual], where "dakara" or "sorede" are preferred instead.

It is particularly important in situations requiring clear logical reasoning[^reasoning] and cause-effect relationships[^causality]. The pattern "A is B. Shitagatte, C" is used to develop logical arguments[^logical-development] from premises to conclusions.

## 中文解釋

「したがって」是正式的連接詞[^conjunction]，用於表示從前文陳述的事實或前提邏輯性[^logic]地推導出的結論。它源於動詞「従う」（跟隨），意思是「根據前面所說的內容，可以得出...」。

這是非常正式的表達[^formal]，主要用於書面語[^written]，特別是學術論文[^academic]、商業文件[^business]、正式公告等場合。在日常會話[^casual]中幾乎不使用，會用「だから」或「それで」代替。

在需要明確展示邏輯推論[^reasoning]和因果關係[^causality]的場合特別重要。通常以「A是B。因此C」的形式，用於從前提推導結論的邏輯展開[^logical-development]。

## 核心用法

### 用法 1：邏輯推論

表示從前提推導出必然的結論。

**例句 1**
```
この計画は予算を超過している。したがって、承認できない。
This plan exceeds the budget. Therefore, it cannot be approved.
這個計劃超出預算。因此，無法批准。
```

**例句 2**
```
実験結果は仮説を支持している。したがって、理論は正しいと言える。
The experimental results support the hypothesis. Therefore, we can say the theory is correct.
實驗結果支持假設。因此，可以說理論是正確的。
```

### 用法 2：因果關係說明

用於正式場合中說明客觀的因果關係。

**例句 3**
```
需要が減少している。したがって、生産量を調整する必要がある。
Demand is decreasing. Consequently, we need to adjust production volume.
需求正在減少。因此，需要調整生產量。
```

**例句 4**
```
データに誤りがあった。したがって、分析結果も信頼できない。
There were errors in the data. Therefore, the analysis results are also unreliable.
數據有錯誤。因此，分析結果也不可靠。
```

### 用法 3：政策或決定的說明

用於正式宣布根據某些情況做出的決定。

**例句 5**
```
安全性に問題が発見された。したがって、製品を回収することを決定した。
Safety issues have been discovered. Accordingly, we have decided to recall the product.
發現了安全問題。因此，決定回收產品。
```

## 文法規則

### 規則 1：句子連接

「したがって」必須連接兩個完整的句子，通常放在句首，前面使用句號。

**格式**
```
[前提句]。したがって、[結論句]。
[前提句]。したがって[結論句]。
```

**注意**: 「したがって」後面可以加逗號，也可以不加，兩種方式都正確。

### 規則 2：語域限制

只適用於正式場合，不適用於日常對話。

**適合場合**:
- 學術論文、研究報告
- 商業文件、提案書
- 正式演講、公告
- 法律文件

**不適合場合**:
- 朋友間對話（用「だから」）
- 日常郵件（用「それで」）
- 輕鬆的社交場合

### 規則 3：邏輯必然性

使用「したがって」時，結論必須是從前提邏輯推導出來的必然結果，不能用於主觀意見或偶然事件。

**✅ 正確使用**（邏輯必然）:
```
前提が誤っている。したがって、結論も誤りである。
前提錯誤。因此，結論也錯誤。
```

**❌ 錯誤使用**（主觀偶然）:
```
✗ 天気がいい。したがって、散歩に行こう。（應用「だから」）
天氣好。因此，去散步吧。
```

## 常見錯誤

### 錯誤 1：在日常會話中使用

❌ 誤: お腹が空いた。したがって、ご飯を食べよう。
✅ 正: お腹が空いた。だから、ご飯を食べよう。
說明: 日常對話應使用「だから」，「したがって」過於正式，聽起來不自然。

### 錯誤 2：連接非邏輯性內容

❌ 誤: 彼は学生だ。したがって、サッカーが好きだ。
✅ 正: 彼は学生だ。そして、サッカーが好きだ。
說明: 「是學生」和「喜歡足球」之間沒有邏輯必然關係，不能用「したがって」。應使用「そして」（而且）等並列連接詞。

### 錯誤 3：在句中間使用

❌ 誤: 雨が降った、したがって試合は中止です。
✅ 正: 雨が降った。したがって、試合は中止です。
說明: 「したがって」前必須是句號，不能用逗號。它連接的是句子與句子，不是子句。

## 學習要點

1. **正式度最高**: 「したがって」是因果連接詞中最正式的，主要用於書面語和正式場合。

2. **邏輯必然性**: 只用於表達邏輯上必然的因果關係，不用於主觀判斷或偶然事件。

3. **書面語特徵**: 在學術、商務、法律等正式文件中是標準用語，但在日常對話中幾乎不使用。

4. **語域選擇**: 根據場合選擇適當的因果連接詞：日常用「だから」，一般敘述用「それで」，正式場合用「したがって」。

5. **句子結構**: 必須連接完整句子，前面用句號分隔，後面可加可不加逗號。

## 註解

[^conjunction]: **接続詞（連接詞）** - 用來連接句子與句子、段落與段落的詞類，表達邏輯關係如因果、轉折、並列等。在日語中，連接詞獨立於句子結構之外，起到橋樑作用。

[^logic]: **論理的（邏輯性）** - 基於理性思維和推理規則的思考方式。在語言表達中，邏輯性指前後內容之間有必然的推導關係，而非偶然或主觀聯繫。

[^formal]: **フォーマルな表現（正式表達）** - 用於正式場合、商務環境、學術文章等需要保持專業和莊重的語境。特徵是用詞嚴謹、句式完整、避免口語化表達。

[^written]: **書き言葉（書面語）** - 主要用於書寫的語言形式，特徵是文法完整、結構嚴謹、避免省略。與口語相對，更注重正確性和正式性。

[^academic]: **学術論文（學術論文）** - 用於研究、教育等學術領域的正式文章。要求邏輯嚴密、用詞精確、論證充分，是「したがって」最典型的使用場合。

[^business]: **ビジネス文書（商業文件）** - 用於企業、組織間溝通的正式文件，如提案書、報告書、合約等。要求清晰、專業、正式，常使用「したがって」等正式連接詞。

[^casual]: **日常会話（日常會話）** - 朋友、家人、同輩之間的非正式對話。特徵是輕鬆自然、可省略部分成分、使用口語詞彙。「したがって」在此場合過於正式，不自然。

[^reasoning]: **推論（推論）** - 從已知前提通過邏輯思維得出結論的思考過程。「したがって」專門用於表達這種邏輯推論關係。

[^causality]: **因果関係（因果關係）** - 原因和結果之間的關聯。「したがって」用於表達客觀、必然的因果關係，強調邏輯上的必然性。

[^logical-development]: **論理展開（邏輯展開）** - 在論述中，按照邏輯順序從前提推導到結論的過程。學術寫作和正式文件常用這種結構，「したがって」是關鍵連接詞。
