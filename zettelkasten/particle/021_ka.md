# か (Ka)

## 中文解釋
日文疑問助詞，放在句尾表示疑問，相當於中文的「嗎」。也可用於表達不確定、懷疑或委婉的語氣。

## 英文解釋
A sentence-ending particle that indicates a question or uncertainty. It is equivalent to a question mark in English and transforms statements into questions.

## 日本語の解説
文末に付いて疑問を表す終助詞。話し手の疑問や不確かさを示し、相手に確認や判断を求める際に用いられる。

---

## 用法

### 1. 疑問文（Question Sentences）

**句型**：
- 普通體 + か
- 丁寧體（です/ます）+ か

**例句**：

```
これは本ですか。
Kore wa hon desu ka.
這是書嗎？
```

```
明日来ますか。
Ashita kimasu ka.
明天會來嗎？
```

```
あなたは学生か。
Anata wa gakusei ka.
你是學生嗎？（普通體）
```

### 2. 選擇疑問句（Alternative Questions）

**句型**：A か B か

**例句**：

```
コーヒーか紅茶か、どちらがいいですか。
Kōhī ka kōcha ka, dochira ga ii desu ka.
咖啡還是紅茶，哪個比較好？
```

```
行くか行かないか、決めてください。
Iku ka ikanai ka, kimete kudasai.
去還是不去，請決定。
```

### 3. 間接疑問句（Indirect Questions）

**句型**：疑問詞 + か

**例句**：

```
誰が来るか知っていますか。
Dare ga kuru ka shitte imasu ka.
你知道誰會來嗎？
```

```
いつ始まるか分かりません。
Itsu hajimaru ka wakarimasen.
不知道什麼時候開始。
```

### 4. 表達不確定（Expressing Uncertainty）

**例句**：

```
雨が降るかもしれない。
Ame ga furu ka mo shirenai.
可能會下雨。
```

```
彼は来るか来ないか分からない。
Kare wa kuru ka konai ka wakaranai.
不知道他會不會來。
```

### 5. 反問或懷疑（Rhetorical Questions）

**例句**：

```
そんなことできるか。
Sonna koto dekiru ka.
那種事怎麼可能做得到？（表示懷疑）
```

```
本当か。
Hontō ka.
真的嗎？（表示懷疑）
```

---

## 詳細說明

### 語氣差異

1. **丁寧體 + か**：禮貌的疑問
   - これは何ですか。（這是什麼？）

2. **普通體 + か**：較直接或非正式的疑問
   - これは何だか。（這是什麼？）

3. **省略か**：口語中常省略「か」，以語調上揚表示疑問
   - これは本？（這是書嗎？）

### 與其他疑問表現的比較

- **か vs の**：
  - か：標準疑問助詞
  - の：較柔和、女性化的疑問表現
  - 例：行くか vs 行くの

- **か vs かい**：
  - か：中性
  - かい：男性用語，較親切
  - 例：分かるか vs 分かるかい

### 書面與口語

- **書面語**：保留「か」
  - これは正しいか。

- **口語**：
  - 丁寧體常保留：これは正しいですか。
  - 普通體常省略：これ正しい？

---

## 常見搭配

- ～かもしれない（可能）
- ～かどうか（是否）
- ～か～か（或者）
- ～（の）ではないか（不是嗎）

---

## 學習要點

1. 丁寧體疑問句必須加「か」
2. 普通體疑問句可省略「か」，靠語調
3. 選擇疑問句用「AかBか」
4. 間接疑問句保留「か」
5. 與「の」比較，「か」較正式中性
