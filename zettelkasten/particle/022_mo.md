## 日文

も（mo）

## 日文解釋

「も」は副助詞の一つで、ある事柄に加えて別の事柄も同様であることを示します。基本的には「〜もまた」「〜も同様に」という意味で、複数の要素が同じ性質や状態を共有することを表現します。

主な機能として：
1. **添加**：Aに加えてBも同じであることを示す（「太郎も来た」）
2. **並列強調**：「〜も〜も」の形で複数要素を強調して列挙する（「雨も風も強い」）
3. **極端例示**：「〜さえも」「〜でも」の形で極端な例を挙げて全体を暗示する（「子供でもわかる」）
4. **譲歩・逆接**：「〜ても」の形で逆接条件を表す（「雨が降っても行く」）
5. **全称**：疑問詞と結合して全てを表す（「誰も」「何も」）

「は」との違いについて、「は」が対比や主題を提示するのに対し、「も」は既出の情報や前提に新しい要素を加える働きをします。

## 英文解釋

The particle "mo" is an adverbial particle (sometimes classified as a binding particle) that indicates addition, inclusion, or emphasis. It is one of the most fundamental particles in Japanese, essential from N5 level.

### Core Functions

1. **Addition/Inclusion**: Indicates that something shares the same property or state as something else ("also," "too")
2. **Parallel Emphasis**: Lists multiple items with emphasis using "〜mo〜mo" pattern ("both...and")
3. **Extreme Example**: Implies a whole category by citing an extreme case ("even")
4. **Concessive**: Forms concessive conditionals with "〜temo" ("even if")
5. **Universal Quantification**: Combines with question words to express totality ("everyone," "everything," "nowhere")

The key difference from "wa" is that while "wa" introduces topics or contrasts, "mo" adds new elements to an already established context.

## 中文解釋

「も」是一個副助詞（有時被歸類為係助詞），表示添加、包含或強調的意義。它是日語中最基礎的助詞之一，從N5級別就需要掌握。

### 核心功能

1. **添加**：表示某事物與另一事物具有相同的性質或狀態（「也」）
2. **並列強調**：使用「〜も〜も」的形式列舉多個項目並加以強調（「既...又...」）
3. **極端例示**：通過舉出極端的例子來暗示整個類別（「連...都...」「甚至」）
4. **讓步**：與「〜ても」形式結合表示讓步條件（「即使...也...」）
5. **全稱量化**：與疑問詞結合表示全部（「誰都」「什麼都」「哪裡都」）

與「は」的區別在於：「は」用於引入話題或表示對比，而「も」則是在已建立的語境中添加新的元素。

## 核心用法

### 用法 1：添加（〜も、また）

表示在已提及的事項基礎上，另一事項也具有相同特徵。

**例句 1**
```
田中さんは学生です。山田さんも学生です。
Tanaka is a student. Yamada is also a student.
田中是學生。山田也是學生。
```

**例句 2**
```
私はコーヒーが好きです。紅茶も好きです。
I like coffee. I also like tea.
我喜歡咖啡。我也喜歡紅茶。
```

### 用法 2：並列強調（〜も〜も）

用於強調列舉多個相同性質的事項。

**例句 3**
```
今日は雨も風も強くて、外出できなかった。
Today both the rain and wind were strong, so I couldn't go out.
今天雨大風也大，沒辦法出門。
```

**例句 4**
```
日本語も英語も話せます。
I can speak both Japanese and English.
日語和英語我都會說。
```

### 用法 3：極端例示（〜でも、〜さえも）

通過舉出極端或意外的例子來強調整體。

**例句 5**
```
こんな簡単な問題は子供でもわかる。
Even a child can understand such a simple problem.
這麼簡單的問題連小孩都懂。
```

### 用法 4：疑問詞 + も（全稱量化）

與疑問詞結合，在否定句中表示「沒有任何」，在肯定句中表示「全部」。

**例句 6**
```
今日は誰も来なかった。
Nobody came today.
今天沒有任何人來。
```

**例句 7**
```
彼女はどこへでも行く。
She goes everywhere.
她哪裡都去。
```

## 使用規則

### 與其他助詞的結合

| 組合 | 形式 | 例子 |
|------|------|------|
| が + も | がも → も | 私も行く（×私がも） |
| は + も | はも → も | 彼も知っている |
| を + も | をも | 彼をも招待した（文語的） |
| に + も | にも | 東京にも大阪にも行った |
| で + も | でも | 電車でもバスでも行ける |
| から + も | からも | 先生からも注意された |
| と + も | とも | 父とも母とも話した |

### 語順規則

1. 「が」「は」を置き換える場合、「も」が直接名詞に付く
2. 他の助詞とは結合して使用可能（「にも」「でも」「からも」等）
3. 「を」との結合「をも」は文語的で、現代語では「も」だけを使うことが多い

### 意味の違い

- **AはBだ。CもBだ。**：Aを話題として提示し、Cを追加情報として加える
- **AもBだ。CもBだ。**：AとC両方を同等に強調する

## 常見錯誤

### 錯誤 1：「も」と「は」の混同

❌ 誤：私は日本語が好きです。私は英語が好きです。
✅ 正：私は日本語が好きです。英語も好きです。
說明：二つ目の文では「も」を使って添加を表すのが自然です。

### 錯誤 2：疑問詞 + も の否定形忘れ

❌ 誤：誰もいます。（誰もがいるという意味なら）
✅ 正：誰もいません。/ 誰でもいます。
說明：「誰も」は否定文で「誰も〜ない」、肯定文では「誰でも」を使います。

### 錯誤 3：「も」の過剰使用

❌ 誤：私も昨日も学校も行きました。
✅ 正：私も昨日学校に行きました。
說明：「も」を複数の要素に付けると意味が不明確になります。

## 學習要點

1. **「も」は添加を表す基本助詞**：「〜もまた」「〜も同様に」という意味を持ちます
2. **「は」との使い分け**：「は」は対比・話題、「も」は添加・並列を表します
3. **疑問詞との結合**：否定文で「誰も〜ない」「何も〜ない」、肯定文で「誰でも」「何でも」
4. **助詞の結合順序**：「が」「は」は消えるが、「に」「で」「から」等とは結合する
5. **極端例示の用法**：「〜でも」「〜さえも」で強調効果を生み出す
